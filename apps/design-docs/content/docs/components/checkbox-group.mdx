## Overview

`CheckboxGroup` is a simple layout wrapper that groups multiple `Checkbox` items with:

- Accessible `role="group"` and `aria-label`
- Vertical or horizontal orientation
- Optional visible label

## Usage

```tsx
import { CheckboxGroup, Checkbox } from "@nebutra/ui/primitives";

// Vertical (default)
<CheckboxGroup label="Select features">
  <Checkbox>Analytics</Checkbox>
  <Checkbox>API Access</Checkbox>
  <Checkbox>SSO</Checkbox>
</CheckboxGroup>

// Horizontal
<CheckboxGroup orientation="horizontal" label="Options">
  <Checkbox>Option A</Checkbox>
  <Checkbox>Option B</Checkbox>
</CheckboxGroup>

// With disabled items
<CheckboxGroup label="Plan features">
  <Checkbox defaultChecked>100GB Storage</Checkbox>
  <Checkbox>Unlimited users</Checkbox>
  <Checkbox disabled>SSO (Enterprise only)</Checkbox>
</CheckboxGroup>
```

## Props

### CheckboxGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Accessible group label (also rendered visually) |
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | Layout direction |
| `children` | `ReactNode` | — | Checkbox items |
| `className` | `string` | — | Additional CSS classes |

### Checkbox

See [Checkbox](/docs/components/checkbox) for full props.
