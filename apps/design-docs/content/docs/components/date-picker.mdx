## Overview

`DatePicker` and `DateRangePicker` are popover-based date selection components powered by HeroUI. They combine a keyboard-editable date input with a calendar dropdown.

## Usage

```tsx
import { DatePicker, DateRangePicker } from "@nebutra/custom-ui/primitives";
```

## Examples

### Single Date

```tsx
<DatePicker label="Event date" />
```

### Date Range

```tsx
<DateRangePicker label="Trip dates" />
```

### Controlled

```tsx
import { parseDate } from "@nebutra/custom-ui/primitives";
import { useState } from "react";

const [date, setDate] = useState(parseDate("2024-04-04"));

<DatePicker label="Date" value={date} onChange={setDate} />
```

### With Min/Max

```tsx
import { today, getLocalTimeZone } from "@nebutra/custom-ui/primitives";

<DateRangePicker
  label="Period"
  minValue={today(getLocalTimeZone())}
  maxValue={today(getLocalTimeZone()).add({ months: 3 })}
/>
```

### Variants

```tsx
<DatePicker label="Default" />
<DatePicker label="Bordered" variant="bordered" />
<DatePicker label="Underlined" variant="underlined" />
<DatePicker label="Faded" variant="faded" />
```

### Two Visible Months

```tsx
<DateRangePicker label="Stay" visibleMonths={2} />
```

## Props

### DatePicker

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Input label |
| `value` | `DateValue` | — | Controlled value |
| `defaultValue` | `DateValue` | — | Uncontrolled default |
| `onChange` | `(value: DateValue) => void` | — | Change callback |
| `variant` | `"flat" \| "bordered" \| "faded" \| "underlined"` | `"flat"` | Visual style |
| `minValue` | `DateValue` | — | Earliest selectable date |
| `maxValue` | `DateValue` | — | Latest selectable date |
| `granularity` | `"day" \| "hour" \| "minute" \| "second"` | `"day"` | Segment detail level |
| `selectorButtonPlacement` | `"start" \| "end"` | `"end"` | Calendar icon placement |
| `isDisabled` | `boolean` | `false` | Disable interaction |
| `isRequired` | `boolean` | `false` | Mark as required |
| `description` | `string` | — | Helper text below input |
| `errorMessage` | `string` | — | Error text below input |

### DateRangePicker

Same props as DatePicker, plus:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `RangeValue<DateValue>` | — | Controlled range value |
| `onChange` | `(value: RangeValue<DateValue>) => void` | — | Range change callback |
| `visibleMonths` | `number` | `1` | Number of months shown |

## Accessibility

- Keyboard-editable date segments (Tab between year/month/day)
- Calendar popup with full arrow key navigation
- Screen reader announcements for date changes
- Focus management between input and popover
