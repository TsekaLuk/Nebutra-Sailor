---
title: FeatureCheckItem
description: Atomic feature item with a check icon (or custom icon), title, and optional description. Used in feature lists and comparison tables.
---

## Usage

```tsx
import { FeatureCheckItem } from "@nebutra/ui";

<FeatureCheckItem
  title="Unlimited team members"
  description="Invite your entire organization at no extra cost."
/>
```

## Examples

### Feature List

```tsx
const features = [
  { title: "Unlimited team members", description: "Scale without per-seat pricing." },
  { title: "99.99% uptime SLA", description: "Backed by financial guarantees." },
  { title: "SOC 2 Type II certified", description: "Enterprise security by default." },
  { title: "24/7 priority support", description: "Dedicated support channel." },
];

<div className="space-y-4">
  {features.map((feature) => (
    <FeatureCheckItem
      key={feature.title}
      title={feature.title}
      description={feature.description}
    />
  ))}
</div>
```

### Title Only (No Description)

```tsx
<div className="grid grid-cols-2 gap-2">
  <FeatureCheckItem title="Custom domains" />
  <FeatureCheckItem title="API access" />
  <FeatureCheckItem title="SSO / SAML" />
  <FeatureCheckItem title="Audit logs" />
  <FeatureCheckItem title="Data export" />
  <FeatureCheckItem title="Priority support" />
</div>
```

### Custom Icon

```tsx
import { Sparkles } from "lucide-react";

<FeatureCheckItem
  icon={Sparkles}
  iconClassName="text-yellow-500"
  title="AI-powered suggestions"
  description="Context-aware recommendations based on your team's workflow."
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Feature name |
| `description` | `string` | — | Optional supporting description |
| `icon` | `LucideIcon` | `Check` | Icon component |
| `iconClassName` | `string` | — | Override icon color/size |
| `className` | `string` | — | Additional container classes |
