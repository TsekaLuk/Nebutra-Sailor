---
title: "Glowing Effect"
description: "Mouse-tracking glowing border that follows the cursor around an element. Adds premium interactivity to cards, buttons, and panels."
---

## Usage

```tsx
import { GlowingEffect } from "@nebutra/custom-ui/primitives"
```

## Examples

### Glowing Card

```tsx
<div className="relative rounded-xl border p-6">
  <GlowingEffect
    spread={40}
    glow
    disabled={false}
    proximity={64}
    inactiveZone={0.7}
  />
  <div className="relative z-10">
    <h2 className="text-xl font-bold">Premium Card</h2>
    <p>Hover to see the glow track your cursor.</p>
  </div>
</div>
```

### White Variant (dark backgrounds)

```tsx
<div className="relative rounded-xl border border-white/20 bg-black p-6">
  <GlowingEffect
    variant="white"
    disabled={false}
    glow={false}
    spread={30}
  />
  <div className="relative z-10 text-white">Dark mode card</div>
</div>
```

### Always-on Glow (no mouse required)

```tsx
<div className="relative rounded-xl border p-6">
  <GlowingEffect
    glow={true}
    disabled={false}
    blur={4}
  />
  Content
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `disabled` | `boolean` | `true` | When `true`, renders a static border fallback |
| `glow` | `boolean` | `false` | Always show the glow (not just on proximity) |
| `variant` | `"default" \| "white"` | `"default"` | Color theme — default is multicolor, white is monochrome |
| `spread` | `number` | `20` | Spread angle of the glow in degrees |
| `blur` | `number` | `0` | Additional blur in pixels |
| `proximity` | `number` | `0` | Distance in px from element edge to activate |
| `inactiveZone` | `number` | `0.7` | Center zone ratio (0–1) where effect is inactive |
| `movementDuration` | `number` | `2` | Smoothing duration for glow angle transitions |
| `borderWidth` | `number` | `1` | Border width in pixels |
| `className` | `string` | — | Additional CSS classes |

## Notes

- Must be placed inside a `relative` parent.
- `disabled={false}` enables the mouse-tracking effect — it is disabled by default.
- The glow tracks the mouse position on the entire `document.body` with passive event listeners.
- Content inside the container should have `relative z-10` to render above the glow layer.
