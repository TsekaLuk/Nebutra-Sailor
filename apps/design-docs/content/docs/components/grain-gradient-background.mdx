---
title: "Grain Gradient Background"
description: "WebGL-powered animated background with organic grain texture and smooth gradient color transitions. Powered by @paper-design/shaders-react."
---

## Usage

```tsx
import { GrainGradientBackground } from "@nebutra/custom-ui/primitives"
```

## Examples

### Full Page Background

```tsx
<div className="relative min-h-screen">
  <GrainGradientBackground />
  <main className="relative z-10 p-8">Content</main>
</div>
```

### Custom Colors

```tsx
<div className="relative h-[400px]">
  <GrainGradientBackground
    colors={["#5100ff", "#00ff80", "#ffcc00"]}
    softness={0.8}
    noise={0.1}
    speed={0.5}
  />
</div>
```

### Hero Section

```tsx
<section className="relative overflow-hidden">
  <GrainGradientBackground
    colors={["#1a1a2e", "#16213e", "#0f3460"]}
    speed={0.3}
    softness={0.6}
  />
  <div className="relative z-10 py-24 text-center text-white">
    <h1 className="text-5xl font-bold">Hero Title</h1>
  </div>
</section>
```

## Props

`GrainGradientBackground` extends `React.HTMLAttributes<HTMLDivElement>` and accepts all `GrainGradientProps` from `@paper-design/shaders-react` (except `style`).

Key props from the underlying shader:

| Prop | Type | Description |
|------|------|-------------|
| `colors` | `string[]` | Array of CSS color strings for the gradient |
| `softness` | `number` | Gradient softness (0–1) |
| `noise` | `number` | Grain noise intensity (0–1) |
| `speed` | `number` | Animation speed |
| `className` | `string` | Additional CSS classes |

## Notes

- The component is absolutely positioned with `absolute inset-0 -z-10` by default.
- Requires WebGL support. Falls back gracefully in environments without WebGL.
- The deprecated `GrainCloudsBackground` alias is still exported for backwards compatibility.

## Deprecated Alias

```tsx
// Deprecated — use GrainGradientBackground
import { GrainCloudsBackground } from "@nebutra/custom-ui/primitives"
```
