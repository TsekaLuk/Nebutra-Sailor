---
title: "Light Rays"
description: "Atmospheric animated light rays shining from above a container. Creates a dramatic, cinematic lighting effect for hero sections and featured cards."
---

## Usage

```tsx
import { LightRays } from "@nebutra/custom-ui/primitives"
```

## Examples

### Basic

```tsx
<div className="relative h-[400px] w-full overflow-hidden rounded-xl border">
  <LightRays />
  <div className="relative z-10 p-8">Content</div>
</div>
```

### Golden Rays

```tsx
<div className="relative h-[400px] overflow-hidden rounded-xl bg-black">
  <LightRays
    count={10}
    color="rgba(255, 200, 100, 0.3)"
    blur={48}
    speed={18}
    length="80vh"
  />
</div>
```

### Subtle Blue Ambient

```tsx
<div className="relative overflow-hidden rounded-xl">
  <LightRays
    count={5}
    color="rgba(99, 102, 241, 0.15)"
    blur={60}
    speed={20}
  />
</div>
```

## Props

`LightRays` extends `React.HTMLAttributes<HTMLDivElement>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `count` | `number` | `7` | Number of animated rays |
| `color` | `string` | `"rgba(160,210,255,0.2)"` | Base color for ray gradients |
| `blur` | `number` | `36` | Blur filter radius in pixels |
| `speed` | `number` | `14` | Seconds per animation cycle (higher = slower) |
| `length` | `string` | `"70vh"` | CSS length for ray height |
| `ref` | `Ref<HTMLDivElement>` | — | Forward ref to the container |
| `className` | `string` | — | Additional CSS classes |

## Notes

- The component is `pointer-events-none` and absolutely fills its parent container.
- Each ray has randomized position, angle, width, swing, and timing for natural variety.
- Place content with `relative z-10` to render above the rays.
- Works best over dark or gradient backgrounds.
