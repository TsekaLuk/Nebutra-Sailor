## Overview

`SnippetStatus` is a compact code snippet inspired by Vercel's design system. It features status-colored backgrounds (success/warning/error), a `$` prompt prefix, multiline support, and animated copy-to-clipboard with icon swap animation.

Unlike the general `Snippet` component, `SnippetStatus` uses Vercel-style color tokens and includes built-in CSS keyframe animations for the copy icon transition.

## Installation

```tsx
import { SnippetStatus } from "@nebutra/ui/primitives";
```

## Usage

### Basic

```tsx
<SnippetStatus text="npm install package" />
```

### Status variants

```tsx
<SnippetStatus text="Deployment complete" type="success" />
<SnippetStatus text="Build timeout after 30s" type="warning" />
<SnippetStatus text="Error: module not found" type="error" />
```

### Dark mode

```tsx
<SnippetStatus text="npm run dev" dark />
```

### Multiline

```tsx
<SnippetStatus text={["npm install", "npm run dev"]} />
```

### Without prompt

```tsx
<SnippetStatus text="const x = 1" prompt={false} />
```

### Copy callback

```tsx
<SnippetStatus
  text="git clone https://github.com/org/repo"
  onCopy={() => console.log("Copied!")}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string \| string[]` | — | **Required.** Code text or array of lines |
| `width` | `string` | `"100%"` | CSS width of container |
| `onCopy` | `() => void` | — | Callback when text is copied |
| `prompt` | `boolean` | `true` | Show `$` prefix before each line |
| `dark` | `boolean` | `false` | Use dark/inverted theme |
| `type` | `"success" \| "warning" \| "error"` | — | Status color variant |
| `className` | `string` | — | Additional CSS classes |

## Color Variants

| Type | Background | Text |
|------|-----------|------|
| default | `bg-background-100` | `text-gray-1000` |
| inverted (dark) | `bg-gray-1000` | `text-gray-100` |
| success | `bg-blue-100` | `text-blue-900` |
| warning | `bg-amber-100` | `text-amber-900` |
| error | `bg-red-100` | `text-red-900` |

## Notes

- The copy animation uses inline `@keyframes` CSS injected via a `<style>` tag — no external animation libraries needed.
- The copy icon fades out (`animate-snippet-fade-out`) and the check icon fades in (`animate-snippet-fade-in`) for 2 seconds on click.
- Add `animate-snippet-fade-out` and `animate-snippet-fade-in` keyframe utilities to your global CSS if the animations aren't working.
