---
title: "Text Shimmer"
description: "A smooth shimmer/glint sweeps continuously through text. Perfect for AI loading states, premium labels, and call-to-action emphasis."
---

## Usage

```tsx
import { TextShimmer } from "@nebutra/custom-ui/primitives"
```

## Examples

### Loading State

```tsx
<TextShimmer duration={1}>Generating code...</TextShimmer>
```

### AI Typing Indicator

```tsx
<TextShimmer duration={1.5} className="text-sm text-muted-foreground">
  Claude is thinking...
</TextShimmer>
```

### Custom Colors via CSS Variables

```tsx
<TextShimmer
  duration={1.2}
  className="text-xl [--base-color:theme(colors.blue.600)] [--base-gradient-color:theme(colors.blue.200)]"
>
  Premium Feature
</TextShimmer>
```

### As Heading

```tsx
<TextShimmer as="h1" className="text-4xl font-bold" duration={3}>
  Welcome
</TextShimmer>
```

### Paused (static display)

```tsx
<TextShimmer paused>Static Text</TextShimmer>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text content (must be a string) |
| `as` | `React.ElementType` | `"p"` | HTML element to render as |
| `duration` | `number` | `2` | Shimmer sweep duration in seconds |
| `spread` | `number` | `2` | Shimmer beam spread (multiplied by text length) |
| `paused` | `boolean` | `false` | Pause the animation |
| `className` | `string` | — | Additional CSS classes |

## CSS Variables

Customize shimmer colors by overriding CSS variables:

| Variable | Default (light) | Default (dark) | Purpose |
|----------|-----------------|----------------|---------|
| `--base-color` | `#a1a1aa` | `#71717a` | Text base color |
| `--base-gradient-color` | `#000` | `#ffffff` | Shimmer highlight color |
