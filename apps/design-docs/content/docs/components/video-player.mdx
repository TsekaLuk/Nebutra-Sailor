## Overview

`VideoPlayer` is a fully-custom video player built on HTML5 `<video>` with Framer Motion animations:

- **Play/Pause** — click the video or the button
- **Progress bar** — animated spring slider; click to seek
- **Volume slider** — with mute toggle
- **Playback speed** — configurable speed options (default: 0.5×, 1×, 1.5×, 2×)
- **Auto-hiding controls** — overlays fade in on hover, out on mouse leave

## Usage

```tsx
import { VideoPlayer } from "@nebutra/custom-ui/primitives";

// Basic
<VideoPlayer src="/demo.mp4" />

// With poster and custom speeds
<VideoPlayer
  src="/tutorial.mp4"
  poster="/thumbnail.jpg"
  speeds={[0.5, 1, 1.25, 1.5, 2]}
  initialVolume={0.8}
/>

// Autoplay loop (muted required for browsers)
<VideoPlayer src="/background.mp4" autoPlay loop />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | **Required.** Video source URL |
| `poster` | `string` | — | Thumbnail image URL shown before play |
| `autoPlay` | `boolean` | `false` | Autoplay on mount (mutes automatically) |
| `loop` | `boolean` | `false` | Loop the video |
| `speeds` | `number[]` | `[0.5, 1, 1.5, 2]` | Available playback speed options |
| `initialVolume` | `number` | `1` | Starting volume (0–1) |
| `showControlsOnMount` | `boolean` | `false` | Show controls immediately on mount |
| `className` | `string` | — | Container CSS class |

## Notes

- Controls overlay is positioned at the bottom center of the video
- `autoPlay` automatically mutes the video (browser requirement)
- Volume icon adapts: `VolumeX` (muted), `Volume1` (< 50%), `Volume2` (≥ 50%)
- The player wraps in `max-w-4xl`; use `className` to constrain width
