---
title: "Video Text"
description: "Renders a looping video visible only through the silhouette of text letters using an SVG mask. Eye-catching hero effect."
---

## Usage

```tsx
import { VideoText } from "@nebutra/custom-ui/primitives"
```

## Examples

### Basic

```tsx
<div className="relative h-[300px] w-full overflow-hidden bg-black">
  <VideoText src="/videos/ocean.webm">OCEAN</VideoText>
</div>
```

### Custom Font Settings

```tsx
<div className="relative h-[400px] w-full overflow-hidden bg-black">
  <VideoText
    src="/videos/fire.webm"
    fontSize={15}
    fontWeight={900}
    fontFamily="Impact"
  >
    FIRE
  </VideoText>
</div>
```

### No Autoplay (manual control)

```tsx
<div className="relative h-[300px] w-full overflow-hidden">
  <VideoText src="/video.webm" autoPlay={false} loop={false} muted={false}>
    PLAY
  </VideoText>
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | Video source URL |
| `children` | `ReactNode` | — | Text content to use as the mask |
| `autoPlay` | `boolean` | `true` | Auto-play the video |
| `muted` | `boolean` | `true` | Mute the video |
| `loop` | `boolean` | `true` | Loop playback |
| `preload` | `"auto" \| "metadata" \| "none"` | `"auto"` | Video preload strategy |
| `fontSize` | `string \| number` | `20` | Font size in `vw` units (number) or CSS string |
| `fontWeight` | `string \| number` | `"bold"` | CSS font-weight |
| `fontFamily` | `string` | `"sans-serif"` | CSS font-family |
| `textAnchor` | `string` | `"middle"` | SVG text-anchor |
| `dominantBaseline` | `string` | `"middle"` | SVG dominant-baseline |
| `as` | `ElementType` | `"div"` | Container element type |
| `className` | `string` | — | Additional CSS classes on the container |

## Accessibility

- A `<span className="sr-only">` contains the text for screen readers.
- The video uses `playsInline` to prevent fullscreen on iOS.
- Provide a meaningful caption in `children` — it becomes the accessible text label.

## Notes

- Wrap in a container with `overflow-hidden` and explicit dimensions.
- The mask uses an inline `data:image/svg+xml` URL that updates on resize.
- Use high-contrast video (bright on dark or dark on light) for best legibility.
