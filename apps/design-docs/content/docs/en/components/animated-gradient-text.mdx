## Overview

`AnimatedGradientText` renders a `<span>` with a horizontally scrolling gradient background clipped to the text. The gradient cycles continuously from `colorFrom` through `colorTo` and back, creating a shimmering effect.

**Note:** The component uses a CSS class `animate-gradient` that must be defined in your global styles. The exported `ANIMATED_GRADIENT_CSS` constant contains the required CSS.

## Setup

Add the following CSS to your global stylesheet or Tailwind config:

```css
@keyframes gradient {
  to {
    background-position: var(--bg-size, 300%) 0;
  }
}

.animate-gradient {
  animation: gradient 8s linear infinite;
}
```

Or import the constant:

```ts
import { ANIMATED_GRADIENT_CSS } from "@nebutra/ui/primitives";
// Inject ANIMATED_GRADIENT_CSS into your global <style> tag
```

## Usage

```tsx
import { AnimatedGradientText } from "@nebutra/ui/primitives";

// Basic
<AnimatedGradientText>Animated Text</AnimatedGradientText>

// Custom colors
<AnimatedGradientText colorFrom="#ff0080" colorTo="#7928ca">
  Pink to Purple
</AnimatedGradientText>

// Faster animation
<AnimatedGradientText speed={2}>Fast Gradient</AnimatedGradientText>

// As heading
<h1 className="text-4xl font-bold">
  <AnimatedGradientText>Welcome</AnimatedGradientText>
</h1>
```

## Props

| Prop        | Type        | Default     | Description                                                                           |
| ----------- | ----------- | ----------- | ------------------------------------------------------------------------------------- |
| `speed`     | `number`    | `1`         | Speed multiplier. Higher = faster. Controls `--bg-size` CSS variable (`speed × 300%`) |
| `colorFrom` | `string`    | `"#ffaa40"` | Gradient start/end color (CSS color value)                                            |
| `colorTo`   | `string`    | `"#9c40ff"` | Gradient middle color (CSS color value)                                               |
| `className` | `string`    | —           | Additional CSS classes applied to the `<span>`                                        |
| `children`  | `ReactNode` | —           | Text content to display with gradient                                                 |

All other `<span>` props (`id`, `aria-*`, event handlers, etc.) are forwarded.

## Notes

- The gradient pattern is `colorFrom → colorTo → colorFrom`, producing a seamless loop
- `speed=0` effectively stops the animation (gradient still visible but static)
- Rendered as an `inline` element — wrap in a block container to control width
- Requires the `animate-gradient` CSS class to be globally available
