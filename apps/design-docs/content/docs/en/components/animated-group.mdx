## Usage

```tsx
import { AnimatedGroup } from "@nebutra/ui";

<AnimatedGroup preset="blur-slide" className="flex flex-col gap-4">
  <Card>Item 1</Card>
  <Card>Item 2</Card>
  <Card>Item 3</Card>
</AnimatedGroup>;
```

## Presets

| Preset       | Effect                                           |
| ------------ | ------------------------------------------------ |
| `fade`       | Opacity 0 → 1                                    |
| `slide`      | Opacity + slide up from y=20                     |
| `scale`      | Opacity + scale 0.8 → 1                          |
| `blur`       | Opacity + blur(4px) → blur(0)                    |
| `blur-slide` | Opacity + blur + slide up                        |
| `zoom`       | Opacity + scale 0.5 → 1 (spring)                 |
| `flip`       | Opacity + rotateX -90° → 0° (spring)             |
| `bounce`     | Opacity + drop from y=-50 (spring)               |
| `rotate`     | Opacity + rotate -180° → 0° (spring)             |
| `swing`      | Opacity + rotate -10° → 0° (spring, low damping) |

## Examples

### Blur Slide (Hero Sections)

```tsx
<AnimatedGroup preset="blur-slide" className="flex flex-col gap-6 text-center">
  <h1 className="text-5xl font-bold">Build faster</h1>
  <p className="text-xl text-muted-foreground">
    The platform for modern teams.
  </p>
  <div className="flex gap-4 justify-center">
    <Button>Get started</Button>
    <Button variant="outline">Learn more</Button>
  </div>
</AnimatedGroup>
```

### Slide Cards (Feature Grid)

```tsx
<AnimatedGroup preset="slide" className="grid grid-cols-3 gap-4">
  {features.map((f) => (
    <FeatureCard key={f.id}>...</FeatureCard>
  ))}
</AnimatedGroup>
```

### Custom Variants

```tsx
<AnimatedGroup
  variants={{
    container: {
      hidden: { opacity: 0 },
      visible: { opacity: 1, transition: { staggerChildren: 0.2 } },
    },
    item: {
      hidden: { opacity: 0, x: -30 },
      visible: { opacity: 1, x: 0 },
    },
  }}
  className="flex flex-row gap-4"
>
  <Card>Left slide item 1</Card>
  <Card>Left slide item 2</Card>
</AnimatedGroup>
```

### No Preset (Default Fade)

```tsx
<AnimatedGroup className="space-y-2">
  <p>Paragraph 1</p>
  <p>Paragraph 2</p>
  <p>Paragraph 3</p>
</AnimatedGroup>
```

## Props

| Prop        | Type                                        | Default | Description                                                |
| ----------- | ------------------------------------------- | ------- | ---------------------------------------------------------- |
| `children`  | `ReactNode`                                 | —       | Items to animate (each child is wrapped in a `motion.div`) |
| `preset`    | `PresetType`                                | —       | Built-in animation preset                                  |
| `variants`  | `{ container?: Variants; item?: Variants }` | —       | Custom Framer Motion variants                              |
| `className` | `string`                                    | —       | Classes on the outer `motion.div` container                |

## Notes

- Children are animated with `staggerChildren: 0.1` by default — each child starts 100ms after the previous.
- Custom `variants` override the preset. If both `preset` and `variants.item` are specified, `variants.item` takes precedence.
- The container plays `initial="hidden"` → `animate="visible"` immediately on mount. For scroll-triggered animation, use Framer Motion's `useInView` with a ref on the parent.
