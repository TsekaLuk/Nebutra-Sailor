---
title: AnimatedList
description: Sequential list animation that reveals children one by one with a spring animation. Items accumulate from top, perfect for notification feeds on landing pages.
---

## Usage

```tsx
import { AnimatedList } from "@nebutra/ui";

<AnimatedList delay={1000}>
  <NotificationCard key="1" message="New user signed up" />
  <NotificationCard key="2" message="Payment received $49" />
  <NotificationCard key="3" message="Deploy completed" />
</AnimatedList>;
```

<Callout>
  Each child **must** have a stable `key` prop — `AnimatedList` uses it
  internally for `AnimatePresence` keying.
</Callout>

## Examples

### Notification Feed

```tsx
const notifications = [
  { id: "1", title: "New signup", body: "john@example.com joined", icon: "👤" },
  { id: "2", title: "Payment", body: "$49/mo plan activated", icon: "💳" },
  {
    id: "3",
    title: "Deploy",
    body: "v2.1.0 deployed to production",
    icon: "🚀",
  },
  { id: "4", title: "Alert", body: "CPU usage above 80%", icon: "⚠️" },
];

<AnimatedList delay={800}>
  {notifications.map((n) => (
    <div
      key={n.id}
      className="flex items-center gap-3 rounded-xl border bg-background px-4 py-3 shadow-sm w-72"
    >
      <span className="text-2xl">{n.icon}</span>
      <div>
        <p className="text-sm font-medium">{n.title}</p>
        <p className="text-xs text-muted-foreground">{n.body}</p>
      </div>
    </div>
  ))}
</AnimatedList>;
```

### Fast Reveal (500ms)

```tsx
<AnimatedList delay={500}>
  <Card key="a">First</Card>
  <Card key="b">Second</Card>
  <Card key="c">Third</Card>
</AnimatedList>
```

### Slow Reveal (2s)

```tsx
<AnimatedList delay={2000}>
  <Step key="1">Step 1: Initialize</Step>
  <Step key="2">Step 2: Configure</Step>
  <Step key="3">Step 3: Deploy</Step>
</AnimatedList>
```

## Props

### AnimatedList

| Prop        | Type        | Default | Description                               |
| ----------- | ----------- | ------- | ----------------------------------------- |
| `children`  | `ReactNode` | —       | List items (each must have a `key` prop)  |
| `delay`     | `number`    | `1000`  | Milliseconds between each item appearing  |
| `className` | `string`    | —       | Additional classes for the flex container |

### AnimatedListItem

The internal item animation wrapper — can also be used standalone:

```tsx
import { AnimatedListItem } from "@nebutra/ui";

<AnimatePresence>
  <AnimatedListItem key={item.id}>
    <MyCard {...item} />
  </AnimatedListItem>
</AnimatePresence>;
```

Item animation: `scale: 0, opacity: 0` → `scale: 1, opacity: 1` with `spring(stiffness: 350, damping: 40)`.

## Notes

- Items accumulate **in reverse order** (newest on top) — this matches notification feed UX where the latest item is most prominent.
- After all items are shown, the animation stops. It does **not** loop unless items are added dynamically.
- For landing page demos, the list often loops — implement looping by resetting children based on an interval in the parent component.
