## Usage

```tsx
import { AnimatedShinyText } from "@nebutra/ui/primitives";
```

## Examples

### Basic

```tsx
<AnimatedShinyText>✨ Introducing our new feature</AnimatedShinyText>
```

### In an Announcement Badge

```tsx
<div className="rounded-full border px-4 py-1.5 text-sm">
  <AnimatedShinyText>🎉 Now in public beta</AnimatedShinyText>
</div>
```

### Custom Shimmer Width

```tsx
<AnimatedShinyText shimmerWidth={200} className="text-xl font-bold">
  Premium Feature
</AnimatedShinyText>
```

### Faster Animation

```tsx
<AnimatedShinyText duration={4} className="text-lg">
  Ultra Fast Shimmer
</AnimatedShinyText>
```

## Props

`AnimatedShinyText` extends `React.ComponentPropsWithoutRef<"span">`.

| Prop           | Type        | Default | Description                              |
| -------------- | ----------- | ------- | ---------------------------------------- |
| `shimmerWidth` | `number`    | `100`   | Width of the shimmer highlight in pixels |
| `duration`     | `number`    | `8`     | Full animation cycle duration in seconds |
| `className`    | `string`    | —       | Additional CSS classes                   |
| `children`     | `ReactNode` | —       | Text content                             |

## Accessibility

- Renders as a `<span>` — semantically neutral.
- The shimmer is purely visual; screen readers read the text content unchanged.
- Avoid using this on large paragraphs; keep it to short phrases or labels.
