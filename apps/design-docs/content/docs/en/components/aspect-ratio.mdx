## Usage

```tsx
import { AspectRatio } from "@nebutra/ui/primitives";
```

## Props

| Prop        | Type        | Default | Description                                      |
| ----------- | ----------- | ------- | ------------------------------------------------ |
| `ratio`     | `number`    | `1`     | Width-to-height ratio (e.g., `16/9`, `4/3`, `1`) |
| `className` | `string`    | —       | Additional CSS classes                           |
| `children`  | `ReactNode` | —       | Content to constrain within the ratio            |

## Examples

### Common Ratios

```tsx
{
  /* 16:9 — Widescreen video */
}
<AspectRatio ratio={16 / 9}>
  <video src="/intro.mp4" className="h-full w-full object-cover" />
</AspectRatio>;

{
  /* 4:3 — Traditional photo */
}
<AspectRatio ratio={4 / 3}>
  <img src="/photo.jpg" className="h-full w-full object-cover" />
</AspectRatio>;

{
  /* 1:1 — Square */
}
<AspectRatio ratio={1}>
  <img src="/avatar.jpg" className="h-full w-full rounded-full object-cover" />
</AspectRatio>;
```

### With Iframe Embed

```tsx
<AspectRatio ratio={16 / 9}>
  <iframe
    src="https://www.youtube.com/embed/dQw4w9WgXcQ"
    className="h-full w-full"
    allowFullScreen
    title="YouTube video"
  />
</AspectRatio>
```

## Accessibility

- No inherent ARIA role — apply appropriate roles to the child element.
- For images, always include `alt` text on the `<img>` element.
- For video or audio content, provide captions and transcripts.
