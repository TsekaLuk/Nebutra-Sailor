## Usage

```tsx
import { AuroraText } from "@nebutra/ui/primitives";
```

## Examples

### Basic

```tsx
<AuroraText>Aurora</AuroraText>
```

### As a Hero Heading

```tsx
<h1 className="text-6xl font-bold">
  Build <AuroraText>beautiful</AuroraText> apps
</h1>
```

### Custom Colors

```tsx
<AuroraText colors={["#00ff87", "#60efff", "#0061ff"]}>
  Ocean Gradient
</AuroraText>
```

### Faster Speed

```tsx
<AuroraText speed={3} className="text-5xl font-black">
  Fast Aurora
</AuroraText>
```

## Tailwind Config Requirement

The `animate-aurora` animation must be registered in your Tailwind config:

```js
// tailwind.config.ts
export default {
  theme: {
    extend: {
      animation: {
        aurora: "aurora 8s ease-in-out infinite alternate",
      },
      keyframes: {
        aurora: {
          "0%": {
            backgroundPosition: "0% 50%",
            transform: "rotate(-5deg) scale(0.9)",
          },
          "50%": {
            backgroundPosition: "100% 50%",
            transform: "rotate(-3deg) scale(0.95)",
          },
          "100%": {
            backgroundPosition: "0% 50%",
            transform: "rotate(-5deg) scale(0.9)",
          },
        },
      },
    },
  },
};
```

## Props

| Prop        | Type        | Default                                     | Description                                  |
| ----------- | ----------- | ------------------------------------------- | -------------------------------------------- |
| `children`  | `ReactNode` | —                                           | Text content                                 |
| `colors`    | `string[]`  | `["#FF0080","#7928CA","#0070F3","#38bdf8"]` | Gradient color stops                         |
| `speed`     | `number`    | `1`                                         | Animation speed multiplier (higher = faster) |
| `className` | `string`    | —                                           | Additional CSS classes                       |

## Accessibility

- Includes a visually-hidden `<span className="sr-only">` with the full text for screen readers.
- The visible gradient span uses `aria-hidden="true"`.
