---
title: "Avatar"
description: "User profile image with automatic fallback to initials. Supports size variants, stacked groups, git platform badges, status icons, and generated avatars."
---

## Import

```tsx
import {
  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarGroup,
  GitHubAvatar,
  GitLabAvatar,
  BitbucketAvatar,
  AvatarWithIcon,
  DiceBearAvatar,
} from "@nebutra/ui/primitives";
```

## Fallback

When the image URL fails or is omitted, `AvatarFallback` renders initials on a muted background.

<ComponentPreview name="avatar-fallback-demo">

  ```tsx

<Tip>
  Always pass the same `size` to both `Avatar` and `AvatarFallback` — this
  ensures the font size matches the container.
</Tip>

  ```

| ------- | ------------------- | ------- | ------------------------------------- |
| `items` | `AvatarGroupItem[]` | —       | Array of `{ src?, alt, fallback }`    |
| `max`   | `number`            | `4`     | Max avatars before collapsing to `+N` |

## Generated Avatars (DiceBear)

Procedurally generated avatars via [DiceBear](https://www.dicebear.com/) — deterministic for the same `seed`. No npm package required.

<ComponentPreview name="dicebear-avatar-demo">
  <DiceBearAvatarDemo />

  ```tsx
<DiceBearAvatarDemo />
  ```
</ComponentPreview>

| Prop          | Type                               | Default            | Description                                 |
| ------------- | ---------------------------------- | ------------------ | ------------------------------------------- |
| `seed`        | `string`                           | —                  | Deterministic seed (username, email, UUID…) |
| `avatarStyle` | `DiceBearStyle`                    | `"bottts-neutral"` | Avatar collection                           |
| `options`     | `Record<string, string \| number>` | `{}`               | Extra DiceBear query params                 |
| `size`        | `AvatarSize`                       | `"md"`             | Size preset                                 |

<Tip>
  Use `bottts-neutral` for system/bot entities, `avataaars` or `lorelei` for
  human placeholders, `fun-emoji` for anonymous/guest sessions.
</Tip>

## Smart Avatar Group

`AvatarSmartGroup` is a dynamic, animated stack of avatars powered by `framer-motion`. It elegantly reveals hidden avatars upon hovering or expanding the group.

<ComponentPreview name="avatar-smart-group-demo">
  <AvatarSmartGroupDemo />

  ```tsx
<AvatarSmartGroupDemo />
  ```
</ComponentPreview>

| Prop                | Type                                            | Default                | Description                                                                  |
| ------------------- | ----------------------------------------------- | ---------------------- | ---------------------------------------------------------------------------- |
| `avatars`           | `{ src: string; fallback: string; alt: string; }[]` | —                  | Array of avatar objects containing image sources and fallbacks               |
| `maxVisible`        | `number`                                        | `3`                    | Maximum number of avatars directly visible when the component is stacked     |
| `mode`              | `"expand" \| "stack"`                           | `"stack"`              | Determines the layout behavior: either fully expanded or stacked with overlap|
| `size`              | `"sm" \| "md" \| "lg"`                          | `"md"`                 | Size preset for the individual avatars                                       |
| `onAvatarClick`     | `(avatar) => void`                              | `undefined`            | Optional event handler, triggered when an individual avatar is clicked       |

<Tip>
  Hover over the stacked component or change the \`mode\` prop to \`expand\` to see the smooth, animated unveiling of the hidden profiles.
</Tip>

| Prop  | Type     | Description                       |
| ----- | -------- | --------------------------------- |

| Prop      | Type         | Default | Description                                         |
| --------- | ------------ | ------- | --------------------------------------------------- |
| `size`    | `AvatarSize` | `"md"`  | Match parent Avatar size for correct font           |
| `delayMs` | `number`     | `600`   | Delay before showing — prevents flash on fast loads |

## Accessibility

- Always provide a meaningful `alt` on `AvatarImage`.
- When the avatar triggers an action (e.g., opens a user menu), wrap in `<button aria-label="...">`.
- `AvatarFallback` background (`bg-muted`) meets 4.5:1 contrast against `text-muted-foreground`.
- `DiceBearAvatar` SVGs are decorative — the `AvatarFallback` provides a text alternative.
