## Overview

`Awards` is a single polymorphic component with 6 distinct visual variants for displaying achievements, certifications, and recognitions. Choose the style that best fits your UI context.

## Installation

```tsx
import { Awards } from "@nebutra/ui/primitives";
```

## Variants

### `badge` (default)

Horizontal card with a decorative badge SVG icon and text content.

```tsx
<Awards
  variant="badge"
  title="Top Contributor"
  subtitle="Open Source Award"
  recipient="Jane Smith"
  date="March 2026"
/>
```

### `award`

Vertical layout with a laurel wreath SVG and a colored level ribbon.

```tsx
<Awards
  variant="award"
  title="Excellence Award"
  subtitle="For outstanding performance"
  recipient="Jane Smith"
  date="March 2026"
  level="gold"
/>
```

### `certificate`

Formal certificate design with a bookmark icon and recipient line.

```tsx
<Awards
  variant="certificate"
  title="Achievement"
  subtitle="Successfully completed the program"
  recipient="Jane Smith"
  date="March 2026"
/>
```

### `stamp`

Circular stamp with serrated edge and curved text paths.

```tsx
<Awards
  variant="stamp"
  title="CERTIFIED"
  subtitle="PROFESSIONAL"
  recipient="Jane Smith"
  date="2026"
/>
```

### `sticker`

Bold typographic sticker with layered stroke shadows for a 3D effect.

```tsx
<Awards variant="sticker" title="PRO" />
```

### `id-card`

Vertical ID card with a live status indicator and logo area.

```tsx
<Awards
  variant="id-card"
  title="Jane Smith"
  subtitle="Senior Engineer"
  description="Engineering"
  date="ENG-2026"
/>
```

## Props

| Prop          | Type                                                                       | Default   | Description                                   |
| ------------- | -------------------------------------------------------------------------- | --------- | --------------------------------------------- |
| `variant`     | `"stamp" \| "award" \| "certificate" \| "badge" \| "sticker" \| "id-card"` | `"badge"` | Visual style variant                          |
| `title`       | `string`                                                                   | —         | **Required.** Main title text                 |
| `subtitle`    | `string`                                                                   | —         | Secondary text (award name, role, etc.)       |
| `description` | `string`                                                                   | —         | Additional description or department          |
| `date`        | `string`                                                                   | —         | Date or ID string                             |
| `recipient`   | `string`                                                                   | —         | Name of the award recipient                   |
| `level`       | `"bronze" \| "silver" \| "gold" \| "platinum"`                             | `"gold"`  | Medal level (used in `award` variant)         |
| `showIcon`    | `boolean`                                                                  | `true`    | Whether to show the star icon (stamp variant) |
| `customIcon`  | `ReactNode`                                                                | —         | Custom icon to replace the default            |
| `className`   | `string`                                                                   | —         | Additional CSS classes                        |

## Level Colors

| Level      | Gradient                |
| ---------- | ----------------------- |
| `bronze`   | amber-600 → amber-800   |
| `silver`   | gray-400 → gray-600     |
| `gold`     | yellow-400 → yellow-600 |
| `platinum` | slate-300 → slate-500   |

## Notes

- The `sticker` variant uses `WebkitTextStroke` with layered spans for the 3D border effect and adapts to light/dark mode.
- The `stamp` variant generates a serrated edge via a dynamically computed SVG path with 40 teeth.
- The `id-card` variant includes an animated ping indicator for the "live" status dot.
