## Usage

```tsx
import { BentoGrid, BentoCard } from "@nebutra/ui";
import { Sparkles } from "lucide-react";

<BentoGrid>
  <BentoCard
    name="AI Integration"
    description="Seamlessly integrate AI into your workflow"
    Icon={Sparkles}
    href="/features/ai"
    cta="Learn more"
  />
</BentoGrid>;
```

## Examples

### Basic Grid

```tsx
import { Sparkles, Zap, Lock } from "lucide-react";

<BentoGrid>
  <BentoCard
    name="AI-powered"
    description="Built-in intelligence that learns from your workflow."
    Icon={Sparkles}
    className="lg:col-span-2"
    href="/features/ai"
    cta="Explore AI features"
  />
  <BentoCard
    name="Lightning fast"
    description="Sub-second response times, globally distributed."
    Icon={Zap}
    href="/features/performance"
    cta="See benchmarks"
  />
  <BentoCard
    name="Enterprise security"
    description="SOC 2 Type II certified. GDPR compliant."
    Icon={Lock}
    href="/features/security"
    cta="View compliance"
  />
</BentoGrid>;
```

### With Background Slot

The `background` prop accepts any ReactNode — use gradients, images, or animation components:

```tsx
<BentoCard
  name="Wave Visualization"
  description="Real-time audio visualization for your data streams."
  Icon={Radio}
  background={
    <div className="absolute inset-0 opacity-60">
      <WaveAnimation position="absolute" />
    </div>
  }
  href="/features/streams"
/>
```

### Custom Row/Column Span

Control grid layout with `className`:

```tsx
<BentoGrid className="lg:grid-cols-3">
  <BentoCard
    name="Wide"
    description="..."
    Icon={Maximize}
    className="lg:col-span-2 lg:row-span-2"
  />
  <BentoCard
    name="Tall"
    description="..."
    Icon={ArrowUp}
    className="lg:row-span-2"
  />
  <BentoCard name="Normal" description="..." Icon={Square} />
  <BentoCard name="Normal" description="..." Icon={Circle} />
</BentoGrid>
```

## Props

### BentoGrid

| Prop        | Type        | Default | Description                |
| ----------- | ----------- | ------- | -------------------------- |
| `children`  | `ReactNode` | —       | BentoCard items            |
| `className` | `string`    | —       | Override grid columns/rows |

Default grid: `grid-cols-1 md:grid-cols-2 lg:grid-cols-3`, `auto-rows-[22rem]`, `gap-4`.

### BentoCard

| Prop          | Type                | Default        | Description                                            |
| ------------- | ------------------- | -------------- | ------------------------------------------------------ |
| `name`        | `string`            | —              | Card title                                             |
| `description` | `string`            | —              | Card description text                                  |
| `Icon`        | `React.ElementType` | —              | Icon component (displayed above title)                 |
| `background`  | `ReactNode`         | —              | Background slot — absolutely positioned behind content |
| `href`        | `string`            | —              | CTA link URL                                           |
| `cta`         | `string`            | `"Learn more"` | CTA button text                                        |
| `className`   | `string`            | —              | Grid span classes (`lg:col-span-2`, etc.)              |

## Notes

- On **desktop** (lg), the CTA button is hidden and reveals on hover by sliding up.
- On **mobile**, the CTA is always visible below the description.
- The background slot renders in a `div` container at the top of the card — use `absolute inset-0` positioning inside the background for full coverage.
- Icon animates to 75% scale on hover (`group-hover:scale-75`), and content slides up (`group-hover:-translate-y-10`).
