## Usage

```tsx
import { BorderTrail } from "@nebutra/ui/primitives";
```

## Examples

### Basic

```tsx
<div className="relative rounded-md border p-6">
  <BorderTrail />
  <p>Content here</p>
</div>
```

### Glowing Trail on Card

```tsx
<div className="relative rounded-xl border p-8">
  <BorderTrail
    className="bg-gradient-to-r from-blue-500 via-purple-500 to-pink-500"
    size={120}
  />
  <h2 className="text-xl font-bold">Premium Card</h2>
  <p>With a glowing border trail</p>
</div>
```

### Slower Animation with Delay

```tsx
<div className="relative rounded-lg border p-4">
  <BorderTrail
    size={80}
    transition={{ duration: 8, ease: "linear", repeat: Infinity }}
    delay={2}
  />
  Content
</div>
```

## Props

| Prop                  | Type            | Default                                             | Description                                        |
| --------------------- | --------------- | --------------------------------------------------- | -------------------------------------------------- |
| `size`                | `number`        | `60`                                                | Size of the trail element in pixels                |
| `transition`          | `Transition`    | `{ repeat: Infinity, duration: 5, ease: "linear" }` | Framer Motion transition config                    |
| `delay`               | `number`        | —                                                   | Delay before animation starts in seconds           |
| `onAnimationComplete` | `() => void`    | —                                                   | Callback when animation completes a cycle          |
| `style`               | `CSSProperties` | —                                                   | Additional inline styles for the trail element     |
| `className`           | `string`        | —                                                   | CSS classes on the trail (use for gradient colors) |

## Setup Requirements

- Parent must have `position: relative` (or use `relative` class).
- Parent must have a defined `border-radius` for the trail to follow correctly.
- The component uses CSS `offset-path: rect(...)` — modern browsers only.

## Notes

- The trail element uses `mix-blend-mode: normal` by default. Add `mix-blend-mode: screen` via className for a glow effect on dark backgrounds.
- Use `className` to customize the trail color — e.g., `bg-gradient-to-r from-blue-500 to-purple-500`.
