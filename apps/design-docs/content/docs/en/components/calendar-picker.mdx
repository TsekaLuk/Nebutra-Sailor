## Overview

`CalendarPicker` is a Geist-inspired date range picker that combines a `RangeCalendar` popover with optional preset shortcuts. It supports multiple layout modes: default (side-by-side), compact, stacked, and horizontal.

## Usage

```tsx
import { CalendarPicker } from "@nebutra/ui/primitives";
```

## Examples

### Basic

```tsx
const [range, setRange] = useState<RangeValue<DateValue>>();

<CalendarPicker value={range} onChange={setRange} />
```

### With Presets

```tsx
import { startOfDay, subDays, subWeeks, subMonths, endOfDay } from "date-fns";

const presets = {
  "last-3-days": {
    text: "Last 3 Days",
    start: startOfDay(subDays(new Date(), 3)),
    end: endOfDay(new Date()),
  },
  "last-7-days": {
    text: "Last 7 Days",
    start: startOfDay(subWeeks(new Date(), 1)),
    end: endOfDay(new Date()),
  },
  "last-month": {
    text: "Last Month",
    start: startOfDay(subMonths(new Date(), 1)),
    end: endOfDay(new Date()),
  },
};

<CalendarPicker
  value={range}
  onChange={setRange}
  presets={presets}
  allowClear
/>
```

### Compact Layout

Presets appear as a horizontal row below the calendar.

```tsx
<CalendarPicker presets={presets} compact />
```

### Stacked Layout

Presets appear vertically below the calendar.

```tsx
<CalendarPicker presets={presets} stacked />
```

### Horizontal Layout

Two calendar months displayed side by side.

```tsx
<CalendarPicker horizontalLayout />
```

### Small Size

```tsx
<CalendarPicker size="small" presets={presets} />
```

### Default Preset Selected

```tsx
<CalendarPicker presets={presets} presetIndex={2} stacked />
```

### With Min/Max Dates

```tsx
const now = new Date();
const minDate = new Date(now.getFullYear(), now.getMonth() - 2, now.getDate());
const maxDate = new Date(now.getFullYear(), now.getMonth() + 2, now.getDate());

<CalendarPicker minValue={minDate} maxValue={maxDate} allowClear />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|

| `compact` | `boolean` | `false` | Compact layout ã presets as horizontal row |

| `maxValue` | `Date` | ã | Latest selectable date |
| `className` | `string` | ã | Additional trigger class |

### CalendarPreset

```tsx
interface CalendarPreset {
  text: string;   // Display label
  start: Date;    // Range start
  end: Date;      // Range end
}
```

## Layout Modes

| Mode | Description |
|------|-------------|
| **default** | Calendar left, presets right (side-by-side) |
| **compact** | Presets as horizontal buttons below calendar |
| **stacked** | Calendar top, presets below (vertical) |
| **horizontal** | Two calendar months, no presets sidebar |

## Accessibility

- Trigger button shows current selection or "Select period"
- Calendar popup uses full keyboard navigation (arrow keys, Page Up/Down)
- Clear button has `aria-label="Clear date selection"`
- Preset buttons are keyboard-focusable with focus ring
- Popover closes on Escape
