## Overview

`Calendar` and `RangeCalendar` are inline calendar grid components powered by HeroUI and `@internationalized/date`. They support 13 international calendar systems, date constraints, and full keyboard/ARIA accessibility.

## Usage

```tsx
import { Calendar, RangeCalendar, today, getLocalTimeZone } from "@nebutra/ui/primitives";
```

## Props

### Calendar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `DateValue` | — | Controlled selected date |
| `defaultValue` | `DateValue` | — | Uncontrolled default |
| `onChange` | `(value: DateValue) => void` | — | Selection callback |
| `minValue` | `DateValue` | — | Earliest selectable date |
| `maxValue` | `DateValue` | — | Latest selectable date |
| `isDateUnavailable` | `(date: DateValue) => boolean` | — | Predicate for unavailable dates |
| `visibleMonths` | `number` | `1` | Number of months displayed |
| `showMonthAndYearPickers` | `boolean` | `false` | Enable month/year dropdown selectors |
| `isDisabled` | `boolean` | `false` | Disable all interaction |
| `isReadOnly` | `boolean` | `false` | Read-only mode |

### RangeCalendar

Same props as Calendar, except:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `RangeValue<DateValue>` | — | Selected date range |
| `onChange` | `(value: RangeValue<DateValue>) => void` | — | Range change callback |

## Date Utilities

Re-exported from `@internationalized/date` for convenience:

```tsx
import {
  CalendarDate, CalendarDateTime, ZonedDateTime,
  today, getLocalTimeZone, now,
  parseDate, parseDateTime, parseZonedDateTime,
  isWeekend, isToday, isSameDay, isSameMonth, isSameYear,
  startOfWeek, startOfMonth, startOfYear,
  endOfWeek, endOfMonth, endOfYear,
} from "@nebutra/ui/primitives";
```

## Accessibility

- Full keyboard navigation: arrow keys between days, Page Up/Down for months
- ARIA labels and live regions for screen readers
- Focus management and selection announcements
- Disabled/unavailable dates have `aria-disabled="true"`
