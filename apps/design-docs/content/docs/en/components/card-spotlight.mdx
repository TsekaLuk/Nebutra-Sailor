---
title: CardSpotlight
description: Card with a radial spotlight that follows the mouse cursor, combined with a WebGL canvas reveal effect on hover.
---

## Usage

```tsx
import { CardSpotlight } from "@nebutra/ui";

<CardSpotlight className="h-96 w-96">
  <h2 className="relative z-20 text-xl font-bold text-white">
    Your Feature
  </h2>
  <p className="relative z-20 text-neutral-300">
    Feature description here.
  </p>
</CardSpotlight>
```

<Callout>
  Requires `three` and `@react-three/fiber` to be installed for the canvas reveal animation on hover.
</Callout>

## Examples

### Basic Card

```tsx
<CardSpotlight className="h-80 w-80">
  <h2 className="relative z-20 mt-2 text-lg font-bold text-white">
    Enterprise Security
  </h2>
  <p className="relative z-20 mt-4 text-sm text-neutral-300">
    SOC 2 Type II certified. Zero-trust architecture. GDPR compliant.
  </p>
</CardSpotlight>
```

### Custom Spotlight Color

```tsx
<CardSpotlight
  color="#1a0533"
  radius={300}
  className="h-80 w-80"
>
  <h2 className="relative z-20 text-lg font-bold text-white">
    Purple spotlight
  </h2>
</CardSpotlight>
```

### Checklist Content

```tsx
<CardSpotlight className="w-96 p-8">
  <h2 className="relative z-20 text-lg font-bold text-white mb-4">
    What's included
  </h2>
  {["Unlimited projects", "Priority support", "Custom domain", "API access"].map((item) => (
    <div key={item} className="relative z-20 flex items-center gap-2 mb-2">
      <CheckCircle2 className="h-4 w-4 text-green-400" />
      <span className="text-sm text-neutral-300">{item}</span>
    </div>
  ))}
</CardSpotlight>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Card content |
| `radius` | `number` | `350` | Spotlight radius in pixels |
| `color` | `string` | `"#262626"` | Spotlight background color |
| `className` | `string` | — | Additional Tailwind classes |

## Notes

- Content must use `relative z-20` to appear above the spotlight overlay.
- The canvas reveal effect (blue/purple dots) activates on mouse enter. It uses Three.js — install `three` and `@react-three/fiber` if not already present.
- Default background is `bg-black` — customize via `className`.
- `radius` controls how far the spotlight gradient spreads from the cursor.
