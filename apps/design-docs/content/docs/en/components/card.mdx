### All Variants

```tsx
<Card variant="default" padding="md">...</Card>
<Card variant="elevated" padding="md">...</Card>   {/* box shadow */}
<Card variant="bordered" padding="md">...</Card>   {/* strong border, no shadow */}
<Card variant="ghost" padding="md">...</Card>      {/* no border or shadow */}
<Card variant="gradient" padding="md">...</Card>   {/* brand gradient border */}
```

### Padding Sizes

```tsx
<Card variant="bordered" padding="sm">...</Card>  {/* 16px */}
<Card variant="bordered" padding="md">...</Card>  {/* 24px (default) */}
<Card variant="bordered" padding="lg">...</Card>  {/* 32px */}
<Card variant="bordered" padding="none">...</Card> {/* no padding */}
```

## Props

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "elevated" \| "bordered" \| "ghost" \| "gradient"` | `"default"` | Visual style |
| `padding` | `"none" \| "sm" \| "md" \| "lg"` | `"md"` | Padding preset |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Card content via subcomponents |

### Subcomponents

| Component | Element | Description |
|-----------|---------|-------------|
| `Card.Header` | `div` | Top section — usually contains `Card.Icon` + `Card.Title` |
| `Card.Body` | `div` | Main content area — `Card.Description` and other content |
| `Card.Footer` | `div` | Bottom section — typically action buttons |
| `Card.Icon` | `div` | Icon wrapper with `size` prop (`sm`, `md`, `lg`) |
| `Card.Title` | `h3` | Card heading — maintains document heading hierarchy |
| `Card.Description` | `p` | Descriptive text in `text-muted-foreground` |

## Design Tokens

Padding: `cardTokens.padding` — sm=16px, md=24px, lg=32px.
Shadow scale: `cardTokens.shadow` — maps to `primitiveShadow.sm/md/lg`.

## Accessibility

- `Card.Title` renders as `<h3>` — maintain proper heading hierarchy in your page.
- For interactive cards (e.g., clickable feature cards), wrap in `<a>` or `<button>`.
- Use `role="article"` when cards represent discrete, standalone content items.
