## Overview

`Checkbox` is a Geist Design System implementation via [21st.dev](https://21st.dev), providing a fully-accessible toggle control with checked, unchecked, and indeterminate states. Supports dark mode via `--ds-gray-*` CSS variables.

## Usage

```tsx
import { Checkbox } from "@nebutra/ui/primitives";
```

## Examples

### Default

```tsx
const [checked, setChecked] = useState(false);

<Checkbox checked={checked} onChange={(v) => setChecked(v)}>
  Option 1
</Checkbox>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | `false` | Checked state |
| `onChange` | `(checked: boolean) => void` | — | Callback when toggled |
| `disabled` | `boolean` | `false` | Disable the checkbox |
| `indeterminate` | `boolean` | `false` | Show indeterminate (dash) state |
| `children` | `ReactNode` | — | Label text |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Hidden native `<input type="checkbox">` with `aria-label`
- Keyboard: parent `<div>` is clickable (toggle on click)
- Disabled state prevents interaction and grays out the control
