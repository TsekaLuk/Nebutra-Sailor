## Accessibility

- Built on Radix UI Collapsible.
- `CollapsibleTrigger` has `aria-expanded` to communicate open/closed state.
- `CollapsibleContent` has `aria-hidden="true"` when collapsed.
- Use `CollapsibleTrigger` with descriptive text or `aria-label` for screen reader clarity.
