## Overview

`ColorBadge` is a versatile badge component with solid and subtle color variants. It supports leading icons and can render as a Next.js link. Includes gradient variants for trial/promo contexts.

## Installation

```tsx
import { ColorBadge } from "@nebutra/ui/primitives";
```

## Usage

### Basic

```tsx
<ColorBadge variant="blue">New</ColorBadge>
<ColorBadge variant="green-subtle" size="lg">Success</ColorBadge>
```

### With icon

```tsx
import { Shield } from "lucide-react";

<ColorBadge variant="purple" icon={<Shield />}>
  Protected
</ColorBadge>
```

### As a link

```tsx
<ColorBadge variant="blue" asLink href="/docs">
  View Docs
</ColorBadge>
```

### Status badges

```tsx
<ColorBadge variant="green">Active</ColorBadge>
<ColorBadge variant="amber">Pending</ColorBadge>
<ColorBadge variant="red">Error</ColorBadge>
```

### Gradient badges

```tsx
<ColorBadge variant="trial">Trial</ColorBadge>
<ColorBadge variant="turbo">Turbo</ColorBadge>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `ColorBadgeVariant` | `"gray"` | Color style variant |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Size of the badge |
| `capitalize` | `boolean` | `true` | Auto-capitalize text |
| `icon` | `ReactNode` | — | Optional icon shown before text |
| `asLink` | `boolean` | `false` | Render as a Next.js Link |
| `href` | `string` | — | URL for link variant |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Badge text content |

## Variants

| Variant | Style |
|---------|-------|
| `gray` | Solid gray |
| `gray-subtle` | Light gray |
| `blue` | Solid blue |
| `blue-subtle` | Light blue |
| `purple` | Solid purple |
| `purple-subtle` | Light purple |
| `amber` | Solid amber |
| `amber-subtle` | Light amber |
| `red` | Solid red |
| `red-subtle` | Light red |
| `pink` | Solid pink |
| `pink-subtle` | Light pink |
| `green` | Solid green |
| `green-subtle` | Light green |
| `teal` | Solid teal |
| `teal-subtle` | Light teal |
| `inverted` | Dark/light adaptive |
| `trial` | Blue → pink gradient |
| `turbo` | Rose → blue gradient |
| `pill` | Outline pill style |
