## Overview

`CommandMenu` is a full-screen Dialog-based command palette built on `cmdk`. It opens as a centered overlay with a search input and grouped action items. Pressing `Escape` closes it.

## Installation

```tsx
import { CommandMenu } from "@nebutra/ui/primitives";
```

## Usage

### Default

```tsx
import { useState } from "react"
import { Button, CommandMenu } from "@nebutra/ui/primitives"

export function CommandMenuDemo() {
  const [open, setOpen] = useState(false)

  function handleAction(): void {
    setOpen(false)
  }

  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Command Menu</Button>
      <CommandMenu.Root open={open} setOpen={setOpen}>
        <CommandMenu.Input placeholder="What do you need?" />
        <CommandMenu.List>
          <CommandMenu.Group heading="Suggestions">
            <CommandMenu.Item callback={handleAction}>
              Figma Import
            </CommandMenu.Item>
          </CommandMenu.Group>
          <CommandMenu.Group heading="Commands">
            <CommandMenu.Item callback={handleAction}>
              Import Extension
            </CommandMenu.Item>
            <CommandMenu.Item callback={handleAction}>
              Manage Extensions
            </CommandMenu.Item>
          </CommandMenu.Group>
          <CommandMenu.Group heading="Collaboration">
            <CommandMenu.Item callback={handleAction}>
              Flags Explorer
            </CommandMenu.Item>
          </CommandMenu.Group>
        </CommandMenu.List>
      </CommandMenu.Root>
    </>
  )
}
```

### With keyboard shortcut (⌘K)

Wire up a `useEffect` at your app root or layout to toggle on `Cmd+K`:

```tsx
useEffect(() => {
  const handler = (e: KeyboardEvent) => {
    if (e.key === "k" && (e.metaKey || e.ctrlKey)) {
      e.preventDefault()
      setOpen((prev) => !prev)
    }
  }
  document.addEventListener("keydown", handler)
  return () => document.removeEventListener("keydown", handler)
}, [])
```

### With separators

```tsx
<CommandMenu.Root open={open} setOpen={setOpen}>
  <CommandMenu.Input placeholder="Search commands..." />
  <CommandMenu.List>
    <CommandMenu.Group heading="Files">
      <CommandMenu.Item callback={handleAction}>New File</CommandMenu.Item>
    </CommandMenu.Group>
    <CommandMenu.Separator />
    <CommandMenu.Group heading="Edit">
      <CommandMenu.Item callback={handleAction}>Copy</CommandMenu.Item>
    </CommandMenu.Group>
  </CommandMenu.List>
</CommandMenu.Root>
```

### With shortcuts

```tsx
<CommandMenu.Item callback={handleAction}>
  New File
  <CommandMenu.Shortcut>⌘N</CommandMenu.Shortcut>
</CommandMenu.Item>
```

### With disabled items

```tsx
<CommandMenu.Item disabled callback={handleAction}>
  Unavailable Action
</CommandMenu.Item>
```

## Props

### CommandMenu.Root

| Prop | Type | Description |
|------|------|-------------|
| `open` | `boolean` | Whether the command menu is visible |
| `setOpen` | `(open: boolean) => void` | Setter to control open state |
| `children` | `ReactNode` | `CommandMenu.Input` + `CommandMenu.List` |

### CommandMenu.Item

| Prop | Type | Description |
|------|------|-------------|
| `callback` | `() => void` | Invoked on click or Enter key |
| `disabled` | `boolean` | Disable this item |
| `className` | `string` | Additional CSS classes |
| `children` | `ReactNode` | Item label (+ optional `CommandMenu.Shortcut`) |

### Sub-Components

| Component | Description |
|-----------|-------------|
| `CommandMenu.Input` | Search input with search icon |
| `CommandMenu.List` | Scrollable results container |
| `CommandMenu.Group` | Group with optional `heading` label |
| `CommandMenu.Item` | Selectable action item with `callback` |
| `CommandMenu.Empty` | Shown when no results match the search |
| `CommandMenu.Shortcut` | Keyboard shortcut hint aligned to the right |
| `CommandMenu.Separator` | Visual divider between groups |

## Accessibility

- Dialog overlay traps focus and restores on close.
- `Escape` closes the overlay.
- `ArrowUp` / `ArrowDown` navigate between items.
- `Enter` triggers the focused item's `callback`.
- Search filters items in real-time via cmdk.
- No close button — command palettes rely on `Escape` and item selection as exit paths.
