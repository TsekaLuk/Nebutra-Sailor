## Usage

```tsx
import {
  Command,
  CommandEmpty,
  CommandGroup,
  CommandInput,
  CommandItem,
  CommandList,
  CommandSeparator,
  CommandShortcut,
  CommandDialog,
} from "@nebutra/ui/primitives"
```

## Props

### Command (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `onSelect` | `(value: string) => void` | — | Callback when an item is selected |
| `shouldFilter` | `boolean` | `true` | Whether to filter items based on search input |
| `filter` | `(value: string, search: string) => number` | — | Custom filter function |
| `children` | `ReactNode` | — | Command content |

### CommandInput

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | — | Placeholder text |
| `value` | `string` | — | Controlled search value |
| `onValueChange` | `(value: string) => void` | — | Callback on input change |
| `className` | `string` | — | Additional CSS classes |

### CommandItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Value for filtering and selection |
| `disabled` | `boolean` | `false` | Prevents selection |
| `onSelect` | `(value: string) => void` | — | Callback when item is selected |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Item content |

## Accessibility

- Built on cmdk — keyboard-first design.
- `CommandInput` has `role="combobox"` with `aria-expanded` and `aria-haspopup="listbox"`.
- `CommandList` has `role="listbox"`.
- Items have `role="option"` with `aria-selected`.
- Keyboard: type to search, `ArrowUp`/`ArrowDown` to navigate, `Enter` to select.
