## Overview

`ContextMenu` is a right-click context menu built on Radix ContextMenu. It supports prefix icons, suffix hints (shortcuts/badges), disabled items, link mode, and labeled groups.

### With suffix hints (shortcuts)

```tsx
<ContextMenu.Item
  prefix={<Edit2 className="h-4 w-4" />}
  suffix={<span className="text-xs text-muted-foreground">⌘E</span>}
  onSelect={() => {}}
>
  Edit
</ContextMenu.Item>
```

### With labels and sections

```tsx
<ContextMenu.Content>
  <ContextMenu.Label>Actions</ContextMenu.Label>
  <ContextMenu.Item onSelect={() => {}}>Edit</ContextMenu.Item>
  <ContextMenu.Item onSelect={() => {}}>Copy</ContextMenu.Item>
  <ContextMenu.Separator />
  <ContextMenu.Label>Danger</ContextMenu.Label>
  <ContextMenu.Item
    onSelect={() => {}}
    className="text-destructive focus:text-destructive"
  >
    Delete
  </ContextMenu.Item>
</ContextMenu.Content>
```

### Link mode

Items with `href` render as `<a>` tags:

```tsx
<ContextMenu.Item value="docs" href="/docs" prefix={<Link className="h-4 w-4" />}>
  Documentation
</ContextMenu.Item>
```

### Disabled items

```tsx
<ContextMenu.Item disabled onSelect={() => {}}>
  Paste
</ContextMenu.Item>
```

## Props

### ContextMenu.Root

Accepts all Radix `ContextMenu.Root` props.

### ContextMenu.Trigger

Accepts all Radix `ContextMenu.Trigger` props. Use `asChild` to make any element the trigger.

### ContextMenu.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |

Accepts all Radix `ContextMenu.Content` props.

### ContextMenu.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Item label |
| `onSelect` | `() => void` | — | Called on click or keyboard selection (maps to Radix `onSelect`). Not fired when `href` is set. |
| `disabled` | `boolean` | `false` | Disables interaction |
| `href` | `string` | — | Renders item as `<a>` link; navigation is the selection action |
| `prefix` | `ReactNode` | — | Element before the label (icon) |
| `suffix` | `ReactNode` | — | Element after the label (shortcut/badge) |
| `className` | `string` | — | Additional CSS classes |

### ContextMenu.Label

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Label text |
| `className` | `string` | — | Additional CSS classes |

### ContextMenu.Separator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Built on Radix ContextMenu — fully keyboard navigable.
- Arrow keys move between items; `Enter` or `Space` selects.
- `Escape` closes the menu.
- Disabled items are skipped by keyboard navigation.
- Screen readers announce the menu via `role="menu"` and items via `role="menuitem"`.
