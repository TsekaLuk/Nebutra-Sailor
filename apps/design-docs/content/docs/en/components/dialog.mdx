## Props

### Dialog (Root)

Extends `@radix-ui/react-dialog` `Root` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Uncontrolled default open state |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |
| `modal` | `boolean` | `true` | Whether to use modal behavior |

### DialogTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Merge props onto the immediate child element |
| `children` | `ReactNode` | — | Trigger element |

### DialogContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Dialog content |

Renders with a built-in close button (`X`) in the top-right corner.

### DialogHeader / DialogFooter

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Header or footer content |

### DialogTitle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Dialog title text |

### DialogDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Supplementary description text |

### DialogClose

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Merge props onto the immediate child element |
| `children` | `ReactNode` | — | Close trigger element |

## Examples

### Controlled Dialog

```tsx
const [open, setOpen] = useState(false)

<Dialog open={open} onOpenChange={setOpen}>
  <DialogTrigger asChild>
    <Button>Open</Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Controlled Dialog</DialogTitle>
    </DialogHeader>
    <p>This dialog is controlled externally.</p>
  </DialogContent>
</Dialog>
```

### Confirmation Dialog

```tsx
<Dialog>
  <DialogTrigger asChild>
    <Button variant="destructive">Delete Account</Button>
  </DialogTrigger>

    <DialogHeader>
      <DialogTitle>Are you absolutely sure?</DialogTitle>
      <DialogDescription>

    </DialogHeader>

      <DialogClose asChild>
        <Button variant="outline">Cancel</Button>
      </DialogClose>

  </DialogContent>
</Dialog>
```

## Accessibility

- Built on Radix UI Dialog — follows the WAI-ARIA [Dialog pattern](https://www.w3.org/WAI/ARIA/apg/patterns/dialog-modal/).
- `DialogContent` has `role="dialog"` and `aria-modal="true"`.
- `DialogTitle` is linked via `aria-labelledby` for screen reader announcement.
- `DialogDescription` is linked via `aria-describedby`.
- Focus is trapped inside the dialog while open.
- Pressing `Escape` closes the dialog.
- Focus returns to the trigger element on close.
