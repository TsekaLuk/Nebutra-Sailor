## Overview

`DisplayCards` renders multiple `DisplayCard` elements stacked on top of each other using CSS Grid's `[grid-area:stack]` trick. Cards have a `-skew-y-[8deg]` tilt and animate outward on hover, revealing the content beneath. Ideal for decorative hero sections and feature showcases.

## Installation

```tsx
import { DisplayCards } from "@nebutra/ui/primitives";
```

## Usage

### Default stacked cards

```tsx
<DisplayCards />
```

Renders three default cards with different offsets — the front card is fully visible, the middle card is shifted right and down with a grayscale overlay, and the back card is shifted furthest with the most offset.

### Custom cards

```tsx
<DisplayCards
  cards={[
    {
      title: "Analytics",
      description: "Track everything in real time",
      date: "Today",
    },
    {
      title: "Automation",
      description: "Let your workflows run themselves",
      date: "2 hours ago",
    },
    {
      title: "Insights",
      description: "Surface what matters most",
      date: "Just now",
    },
  ]}
/>
```

### Single card with custom icon

```tsx
import { Zap } from "lucide-react";

<DisplayCards
  cards={[
    {
      icon: <Zap className="size-4 text-yellow-300" />,
      title: "Lightning Fast",
      description: "Sub-50ms response times globally",
      date: "Live",
      iconClassName: "bg-yellow-800",
      titleClassName: "text-yellow-400",
    },
  ]}
/>
```

## Props

### DisplayCards

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `cards` | `DisplayCardProps[]` | default stack | Array of card configs |
| `className` | `string` | — | Additional CSS classes for the grid container |

### DisplayCardProps

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `"Featured"` | Card title |
| `description` | `string` | `"Discover amazing content"` | Card description text |
| `date` | `string` | `"Just now"` | Timestamp shown at bottom |
| `icon` | `ReactNode` | `<Sparkles />` | Icon element in the header |
| `iconClassName` | `string` | — | Background class for icon container |
| `titleClassName` | `string` | `"text-blue-500"` | Color class for title |
| `className` | `string` | — | Additional CSS classes for this card |

## Notes

- Cards use `select-none` to prevent text selection during interaction.
- The stacking effect relies on all cards being in the same `[grid-area:stack]` grid cell.
- The default cards include a `before:` pseudo-element overlay that fades on hover, giving a desaturation-to-color reveal effect.
- Combine with a background gradient or image for a decorative hero section.
