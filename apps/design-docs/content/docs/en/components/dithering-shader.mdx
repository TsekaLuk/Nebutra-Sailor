## Overview

`DitheringShader` renders an animated dithering effect directly on a `<canvas>` element using WebGL2. It requires no external libraries â€” all GLSL shaders are bundled in the component.

Features:
- 7 shape patterns: simplex noise, warp, dots, wave, ripple, swirl, sphere
- 4 dithering types: random noise, 2Ă—2, 4Ă—4, 8Ă—8 Bayer ordered matrices
- Configurable foreground/background colors (hex only)
- Variable pixel size and animation speed
- `speed=0` renders a single static frame

Use `DitheringBackground` instead if you need a full-screen theme-aware background with grain/vignette/glow overlays.

## Usage

```tsx
import { DitheringShader } from "@nebutra/ui/primitives";

// Default (simplex noise, 8Ă—8 Bayer, white on black)
<DitheringShader width={800} height={800} />

// Swirl in a card
<DitheringShader
  shape="swirl"
  type="4x4"
  colorBack="#220011"
  colorFront="#00ffff"
  pxSize={4}
  speed={0.9}
/>

// Static dark scheme
<DitheringShader
  shape="sphere"
  type="8x8"
  colorBack="#0f0f0f"
  colorFront="#e5e7eb"
  speed={0.3}
/>

// Paused (single frame)
<DitheringShader shape="wave" speed={0} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `number` | `800` | Canvas width in pixels |
| `height` | `number` | `800` | Canvas height in pixels |
| `colorBack` | `string` | `"#000000"` | Background color (hex format only) |
| `colorFront` | `string` | `"#ffffff"` | Foreground/dither color (hex format only) |
| `shape` | `DitheringShaderShape` | `"simplex"` | Shape pattern (see table below) |
| `type` | `DitheringShaderType` | `"8x8"` | Dithering matrix type (see table below) |
| `pxSize` | `number` | `4` | Pixel size for dithering cells. Higher = blockier |
| `speed` | `number` | `1` | Animation speed multiplier. `0` = paused (renders one static frame) |
| `className` | `string` | â€” | Additional CSS classes for the wrapper `<div>` |
| `style` | `CSSProperties` | â€” | Additional inline styles for the wrapper `<div>` |

## Shape Patterns

| Value | Description |
|-------|-------------|
| `"simplex"` | Smooth simplex noise clouds |
| `"warp"` | Domain-warped flowing lines |
| `"dots"` | Animated sine-wave dot columns |
| `"wave"` | Horizontal cosine wave |
| `"ripple"` | Concentric radial ripple from center |
| `"swirl"` | Logarithmic spiral swirl |
| `"sphere"` | 3D sphere with moving point light |

## Dithering Types

| Value | Description |
|-------|-------------|
| `"random"` | Pseudo-random noise dithering |
| `"2x2"` | 2Ă—2 ordered Bayer matrix |
| `"4x4"` | 4Ă—4 ordered Bayer matrix |
| `"8x8"` | 8Ă—8 ordered Bayer matrix (most detail) |

## Notes

- Colors must be hex strings (`#rrggbb`). CSS variables and `hsl()` are not supported
- The component wraps the canvas in a `<div>` sized to `width Ă— height` in pixels â€” it does **not** auto-scale to its container
- Requires WebGL2 support in the browser. Falls back gracefully (blank canvas) in unsupported environments
- All effect parameters trigger a full WebGL re-initialization on change (via `useEffect` dependency array)
- No external library required â€” GLSL shaders are compiled at runtime in the browser
