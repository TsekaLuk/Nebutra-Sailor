## Usage

```tsx
import { DotPattern } from "@nebutra/ui/primitives"
```

## Examples

### Static Background

```tsx
<div className="relative h-[400px] w-full overflow-hidden">
  <DotPattern className="text-muted-foreground/30" />
  <div className="relative z-10 p-8">Content</div>
</div>
```

### Glowing Dots

```tsx
<div className="relative h-[400px] w-full overflow-hidden">
  <DotPattern glow className="text-primary/50" />
</div>
```

### Gradient Mask (fade toward corner)

```tsx
<div className="relative h-[400px] w-full overflow-hidden">
  <DotPattern
    className="[mask-image:linear-gradient(to_bottom_right,white,transparent,transparent)]"
  />
</div>
```

### Custom Spacing and Size

```tsx
<DotPattern
  width={24}
  height={24}
  cr={1.5}
  className="text-blue-400/40"
/>
```

## Props

`DotPattern` extends `React.SVGProps<SVGSVGElement>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `number` | `16` | Horizontal spacing between dots |
| `height` | `number` | `16` | Vertical spacing between dots |
| `x` | `number` | `0` | X-offset of the pattern |
| `y` | `number` | `0` | Y-offset of the pattern |
| `cx` | `number` | `1` | X-offset of individual dots |
| `cy` | `number` | `1` | Y-offset of individual dots |
| `cr` | `number` | `1` | Radius of each dot |
| `glow` | `boolean` | `false` | Enable animated glow pulsing |
| `className` | `string` | — | CSS classes (use `text-` for dot color) |

## Notes

- The component is absolutely positioned, filling its container (`absolute inset-0 h-full w-full`).
- Dot color is controlled via Tailwind `text-*` utility (uses `currentColor`).
- With `glow`, each dot animates independently with randomized timing.
- Use `mask-image` utilities for directional fades.
