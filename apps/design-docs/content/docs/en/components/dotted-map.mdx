## Overview

`DottedMap` renders a world map as an SVG filled with dot points. Use it to show global presence, office locations, or user distribution. Markers are plotted by latitude/longitude.

## Usage

```tsx
import { DottedMap } from "@nebutra/ui/primitives";

// Basic (empty world map)
<div className="h-[400px] w-full">
  <DottedMap />
</div>

// With location markers
<div className="h-[400px] w-full">
  <DottedMap
    markers={[
      { lat: 40.7128, lng: -74.006 },   // New York
      { lat: 51.5074, lng: -0.1278 },   // London
      { lat: 35.6762, lng: 139.6503 },  // Tokyo
    ]}
    markerColor="#FF6900"
  />
</div>

// On dark background
<div className="h-[400px] w-full bg-zinc-950 rounded-xl p-4">
  <DottedMap
    markers={[{ lat: 37.7749, lng: -122.4194 }]}
    markerColor="#00e5ff"
    className="text-zinc-700"
  />
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `number` | `150` | Internal SVG width units |
| `height` | `number` | `75` | Internal SVG height units |
| `mapSamples` | `number` | `5000` | Number of dot sample points (higher = more detail) |
| `markers` | `DottedMapMarker[]` | `[]` | Location markers to plot |
| `markerColor` | `string` | `"#FF6900"` | Color of marker dots |
| `dotRadius` | `number` | `0.2` | Radius of map dots |
| `stagger` | `boolean` | `true` | Enable alternating row offset for better density |
| `className` | `string` | — | SVG className (use `text-*` for dot color via `currentColor`) |

### `DottedMapMarker`

| Prop | Type | Description |
|------|------|-------------|
| `lat` | `number` | Latitude (–90 to 90) |
| `lng` | `number` | Longitude (–180 to 180) |
| `size` | `number` | Marker dot size (optional, defaults to `dotRadius`) |

## Notes

- Dot color uses `currentColor` — set via `text-*` Tailwind class on the SVG
- Increasing `mapSamples` improves land coverage detail but may impact render performance
- Markers are positioned using the same SVG coordinate system as the dots
- Requires `svg-dotted-map`: `pnpm add svg-dotted-map`
