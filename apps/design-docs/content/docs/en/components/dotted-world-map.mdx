## Overview

`DottedWorldMap` renders a simplified world map using a pre-generated diagonal dot grid from the `dotted-map` library. Unlike `DottedMap`, it uses a static pre-computed point set (`height: 55, grid: "diagonal"`) â€” no runtime generation â€” making it very fast.

Best used as a decorative background element on landing pages or dashboards.

## Usage

```tsx
import { DottedWorldMap } from "@nebutra/ui/primitives";

// Default
<DottedWorldMap className="w-full" />

// Dark theme
<div className="bg-zinc-950 p-4">
  <DottedWorldMap
    dotColor="#3f3f46"
    backgroundColor="transparent"
  />
</div>

// Primary color dots
<DottedWorldMap
  dotColor="hsl(var(--primary))"
  backgroundColor="transparent"
  dotRadius={0.2}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `dotRadius` | `number` | `0.15` | Radius of each dot |
| `dotColor` | `string` | `"currentColor"` | CSS color for dots |
| `backgroundColor` | `string` | `"var(--color-background)"` | Background color of the SVG |
| `className` | `string` | â€” | SVG CSS class |

## Notes

- Map points are pre-computed at module initialization â€” no render-time computation
- Uses `dotted-map` with `{ height: 55, grid: "diagonal" }` â€” not configurable at runtime; use `DottedMap` for runtime-generated maps with markers
- Inherits the width of its container via `w-full`; height follows the SVG viewBox aspect ratio (`120 Ă— 60`)
- Requires `dotted-map`: `pnpm add dotted-map`
