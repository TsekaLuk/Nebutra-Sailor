## Overview

`Entity` is a horizontal row layout with three zones: `left` (avatar, icon, etc.), a content area (`Entity.Content`), and `right` (actions, metadata). `Entity.List` wraps multiple rows in a bordered `<ul>`.

## Installation

```tsx
import { Entity } from "@nebutra/ui/primitives";
```

## Usage

### Default

```tsx
import { Entity } from "@nebutra/ui/primitives";
import { Avatar } from "@nebutra/ui/primitives";

<Entity
  left={<Avatar size={32} username="evilrabbit" />}
  right={<p className="text-sm text-muted-foreground">Connected 1h ago</p>}
>
  <Entity.Content
    title="Evil Rabbit"
    description="@Glenn Hitchcock (@gin)"
    fill
  />
</Entity>
```

### Entity.List

Use `Entity.List` as the container and pass `as="li"` on each `Entity` row.

```tsx
<Entity.List>
  <Entity
    as="li"
    right={<Button size="small" variant="outline">Decline</Button>}
  >
    <Entity.Content
      title="GitHub Desktop on MacBook Pro"
      description="Last used just now"
      fill
    />
  </Entity>
  <Entity
    as="li"
    right={<Button size="small" variant="outline">Decline</Button>}
  >
    <Entity.Content
      title="VS Code on Windows 11"
      description="Last used 10min ago"
      fill
    />
  </Entity>
</Entity.List>
```

### With Skeleton (loading state)

```tsx
<Entity>
  <div className="flex flex-1 flex-col gap-2">
    <Skeleton className="h-4 w-full" />
    <div className="flex gap-2">
      <Skeleton className="h-4 w-16" />
      <Skeleton className="h-4 w-14" />
    </div>
  </div>
</Entity>
```

## Props

### `Entity`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `React.ElementType` | `"div"` | Override the root element (e.g. `"li"` inside `Entity.List`) |
| `left` | `ReactNode` | — | Left slot — avatar, icon, or other leading content |
| `right` | `ReactNode` | — | Right slot — actions, metadata, or trailing content |
| `children` | `ReactNode` | — | Typically one or more `Entity.Content` blocks |
| `className` | `string` | — | Additional CSS classes |

### `Entity.Content`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | **Required.** Primary label |
| `description` | `ReactNode` | — | Supporting text shown below the title |
| `fill` | `boolean` | `false` | When `true`, the block expands to fill available horizontal space |
| `className` | `string` | — | Additional CSS classes |

### `Entity.List`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | `Entity` rows with `as="li"` |
| `className` | `string` | — | Additional CSS classes |
