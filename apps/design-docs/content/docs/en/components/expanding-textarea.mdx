## Usage

```tsx
import { ExpandingTextarea } from "@nebutra/ui/primitives"
```

## Props

Extends `React.TextareaHTMLAttributes<HTMLTextAreaElement>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | ‚Äî | Placeholder text when empty |
| `value` | `string` | ‚Äî | Controlled value |
| `defaultValue` | `string` | ‚Äî | Uncontrolled default value |
| `onChange` | `React.ChangeEventHandler<HTMLTextAreaElement>` | ‚Äî | Change event handler |
| `minRows` | `number` | `1` | Minimum number of visible rows |
| `maxRows` | `number` | ‚Äî | Maximum rows before scroll begins |
| `disabled` | `boolean` | `false` | Disables the textarea |
| `className` | `string` | ‚Äî | Additional CSS classes |

## Examples

### Controlled Expanding Textarea

```tsx
const [value, setValue] = useState("")

<ExpandingTextarea
  value={value}
  onChange={(e) => setValue(e.target.value)}
  placeholder="Enter a description..."
  className="w-full min-h-[80px]"
/>
```

### With Min/Max Rows

```tsx
<ExpandingTextarea
  minRows={3}
  maxRows={10}
  placeholder="Write your message..."
/>
```

### In a Form

```tsx
<div className="space-y-2">
  <Label htmlFor="bio">Bio</Label>
  <ExpandingTextarea
    id="bio"
    name="bio"
    placeholder="Tell us about yourself..."
    className="w-full"
  />
</div>
```

## Accessibility

- Renders a native `<textarea>` element ‚Äî inherits full keyboard and screen reader support.
- Always pair with a `<Label>` linked via `htmlFor`/`id`.
- Height expansion happens automatically ‚Äî no user action required beyond typing.
- Visible focus ring via `focus-visible:ring-2` for keyboard navigation.
