## Overview

`GithubInlineDiff` renders a unified diff view identical to GitHub's PR review interface. Features:

- Hunk headers (`@@ -x,y +x,y @@`)
- Context lines (gray background)
- Addition lines (green background, `+` prefix)
- Deletion lines (red background, `-` prefix)
- Line-level comment threads (click `+` button to open)
- Thread resolve toggle
- Current user avatar with initials

## Usage

```tsx
import { GithubInlineDiff, type DiffLine } from "@nebutra/ui/primitives";

const diff: DiffLine[] = [
  { kind: "hunk", content: "@@ -1,5 +1,6 @@" },
  { kind: "context", old: 1, new: 1, content: "import React from 'react';" },
  { kind: "del", old: 2, new: null, content: "const title = 'Old Title';" },
  { kind: "add", old: null, new: 2, content: "const title = 'New Title';" },
  { kind: "add", old: null, new: 3, content: "const version = '2.0.0';" },
  { kind: "context", old: 3, new: 4, content: "" },
];

<GithubInlineDiff
  diff={diff}
  fileName="src/config.ts"
  fileStatus="Modified"
  currentUser={{ name: "Alice Chen", initials: "AC" }}
  onAddComment={(lineIndex, comment) => console.log("Comment added", comment)}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `diff` | `readonly DiffLine[]` | — | **Required.** Array of diff lines |
| `fileName` | `string` | — | **Required.** File name shown in header |
| `fileStatus` | `string` | — | Badge text (e.g., "Modified", "Added") |
| `initialComments` | `Record<number, DiffComment[]>` | — | Pre-existing comments per line index |
| `onAddComment` | `(lineIndex: number, comment: DiffComment) => void` | — | Called when a new comment is submitted |
| `onResolveChange` | `(lineIndex: number, resolved: boolean) => void` | — | Called when thread is resolved/unresolved |
| `currentUser` | `{ name: string; initials: string }` | — | Current user for new comments |
| `className` | `string` | — | Container CSS class |

## Types

### `DiffLine`

```ts
type DiffLine =
  | { kind: "hunk"; content: string }
  | { kind: "context"; old: number | null; new: number | null; content: string }
  | { kind: "add"; old: null; new: number; content: string }
  | { kind: "del"; old: number; new: null; content: string };
```

### `DiffComment`

```ts
interface DiffComment {
  id: string;
  author: string;
  initials: string;
  body: string;
  createdAt: string;
}
```

## Notes

- Hover a line to see the `+` comment button (appears on the left of `context`, `add`, and `del` lines)
- Comment threads are stored in component state; use `onAddComment` to persist to your backend
- The `initialComments` prop keys are **line indices** in the diff array (0-indexed), not line numbers
