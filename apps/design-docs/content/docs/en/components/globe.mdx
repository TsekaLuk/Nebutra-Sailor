## Overview

`Globe` renders a WebGL-powered 3D globe using the [COBE](https://github.com/shuding/cobe) library. It supports:

- Auto-rotation (configurable speed)
- Drag to rotate (pointer events)
- Custom markers with lat/lng positions
- Configurable colors (base, marker, glow)
- Responsive sizing via CSS

The component fills its absolutely-positioned container, so wrap it in a `relative` div with a fixed size.

## Usage

```tsx
import { Globe } from "@nebutra/ui/primitives";

// Basic (uses default markers and rotation)
<div className="relative h-[400px] w-[400px]">
  <Globe />
</div>

// Custom config
<div className="relative h-[400px] w-[400px]">
  <Globe
    config={{
      markers: [
        { location: [40.7128, -74.006], size: 0.1 },  // NYC
        { location: [51.5074, -0.1278], size: 0.08 }, // London
      ],
      markerColor: [0.1, 0.8, 1],
      dark: 1,
    }}
    rotationSpeed={0.003}
  />
</div>
```

## Props

### `GlobeProps`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `config` | `GlobeConfig` | See below | Globe configuration object |
| `rotationSpeed` | `number` | `0.005` | Auto-rotation speed (set to `0` to disable) |
| `className` | `string` | — | Container className |

### `GlobeConfig`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `phi` | `number` | `0` | Initial horizontal rotation |
| `theta` | `number` | `0.3` | Initial vertical tilt |
| `dark` | `number` | `0` | Dark mode (0 = light, 1 = dark) |
| `diffuse` | `number` | `0.4` | Diffuse lighting strength |
| `mapSamples` | `number` | `16000` | Map detail level |
| `mapBrightness` | `number` | `1.2` | Land brightness |
| `baseColor` | `[r, g, b]` | `[1, 1, 1]` | Base color (0–1 per channel) |
| `markerColor` | `[r, g, b]` | orange | Marker dot color |
| `glowColor` | `[r, g, b]` | `[1, 1, 1]` | Glow halo color |
| `markers` | `GlobeMarker[]` | Major cities | Array of location markers |

### `GlobeMarker`

| Prop | Type | Description |
|------|------|-------------|
| `location` | `[lat, lng]` | Geographic coordinates |
| `size` | `number` | Dot size (0.03–0.1 recommended) |

## Notes

- Requires the `cobe` package: `pnpm add cobe`
- The container must be `position: relative` with a fixed aspect ratio
- The canvas fades in with a 500ms opacity transition
- Drag interaction works on pointer and touch devices
