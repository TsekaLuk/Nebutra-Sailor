
## Examples

### Full Page Background

```tsx
<div className="relative min-h-screen">
  <GrainGradientBackground />
  <main className="relative z-10 p-8">Content</main>
</div>
```

## Props

`GrainGradientBackground` extends `React.HTMLAttributes<HTMLDivElement>` and accepts all `GrainGradientProps` from `@paper-design/shaders-react` (except `style`).

Key props from the underlying shader:

| Prop | Type | Description |
|------|------|-------------|
| `colors` | `string[]` | Array of CSS color strings for the gradient |
| `softness` | `number` | Gradient softness (0–1) |
| `noise` | `number` | Grain noise intensity (0–1) |
| `speed` | `number` | Animation speed |
| `className` | `string` | Additional CSS classes |

## Notes

- The component is absolutely positioned with `absolute inset-0 -z-10` by default.
- Requires WebGL support. Falls back gracefully in environments without WebGL.
- The deprecated `GrainCloudsBackground` alias is still exported for backwards compatibility.
