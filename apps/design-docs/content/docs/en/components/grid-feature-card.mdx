## Usage

```tsx
import { GridFeatureCard } from "@nebutra/ui";
import { Zap } from "lucide-react";

<GridFeatureCard
  icon={Zap}
  title="Performance"
  description="Sub-second response times across 35 global edge locations."
/>
```

### Custom Grid Size

The `gridSize` prop controls the density of the background SVG pattern:

```tsx
<GridFeatureCard
  icon={Code}
  title="Developer First"
  description="Comprehensive APIs and SDKs for every platform."
  gridSize={10}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | — | Feature icon (24×24, thin stroke) |
| `title` | `string` | — | Feature title |
| `description` | `string` | — | Feature description |
| `gridSize` | `number` | `20` | Size of grid squares in the SVG pattern |
| `className` | `string` | — | Additional Tailwind classes |

## Notes

- The SVG grid pattern is **randomly generated** on each mount using `React.useMemo` — squares are placed at random positions in the upper-right area.
- The pattern is decorative and hidden from screen readers via `aria-hidden="true"`.
- Use `divide-x divide-y` on the grid container to create clean card borders without doubling up.
- The pattern uses `fill-foreground/5 stroke-foreground/25` — automatically adapts to light/dark mode.
