## Usage

```tsx
import { GridPatternCard, GridPatternCardBody } from "@nebutra/ui";

<GridPatternCard>
  <GridPatternCardBody>
    <h3 className="text-xl font-semibold">Feature Title</h3>
    <p className="text-muted-foreground mt-2">Feature description goes here.</p>
  </GridPatternCardBody>
</GridPatternCard>
```

## Examples

### Basic Card

```tsx
<GridPatternCard className="max-w-sm">
  <GridPatternCardBody>
    <h3 className="text-xl font-semibold">AI Integration</h3>
    <p className="mt-2 text-muted-foreground">
      Seamlessly connect your workflow with the latest AI models.
    </p>
  </GridPatternCardBody>
</GridPatternCard>
```

### Without Entrance Animation

```tsx
<GridPatternCard disableAnimation className="max-w-sm">
  <GridPatternCardBody>
    <h3>Static display</h3>
    <p>No fade-in — instant render.</p>
  </GridPatternCardBody>
</GridPatternCard>
```

### Custom Gradient Overlay

Override the gradient that sits between the grid pattern and content:

```tsx
<GridPatternCard
  gradientClassName="from-blue-950/90 via-blue-950/50 to-blue-950/10"
  className="max-w-sm"
>
  <GridPatternCardBody>
    <h3 className="text-white">Custom Gradient</h3>
  </GridPatternCardBody>
</GridPatternCard>
```

### Grid of Cards

```tsx
<div className="grid grid-cols-1 md:grid-cols-3 gap-4">
  {features.map((feature) => (
    <GridPatternCard key={feature.id}>
      <GridPatternCardBody>
        <feature.Icon className="h-8 w-8 mb-4" />
        <h3>{feature.title}</h3>
        <p className="text-muted-foreground">{feature.description}</p>
      </GridPatternCardBody>
    </GridPatternCard>
  ))}
</div>
```

## Props

### GridPatternCard

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Card content |
| `className` | `string` | — | Additional classes for the card container |
| `patternClassName` | `string` | — | Additional classes for the grid pattern layer |
| `gradientClassName` | `string` | — | Override gradient overlay classes |
| `disableAnimation` | `boolean` | `false` | Disable the fade-in entrance animation |

### GridPatternCardBody

Re-export of a `div` with `text-left p-4 md:p-6` — accepts standard `div` props and `className`.

## Notes

- The grid pattern is inlined as SVG data URLs — no external assets required.
- Dark mode: the grid color switches from `hsla(215, 16%, 47%)` (slate) to `white` automatically via a CSS variable set on `.dark` class.
- Entrance animation: `opacity: 0 → 1, y: -20 → 0` over 0.8 seconds with `easeOut`. Use `disableAnimation` for server-rendered contexts or when the card is already visible on mount.
