## Overview

`HeroPopover` wraps [HeroUI's Popover](https://www.heroui.com/docs/components/popover) component with typed props. Use `HeroPopoverTrigger` to define the element that opens the popover, and `HeroPopoverContent` for the floating content area.

## Installation

```tsx
import {
  HeroPopover,
  HeroPopoverTrigger,
  HeroPopoverContent,
} from "@nebutra/ui/primitives";
```

## Usage

### Basic

```tsx
<HeroPopover>
  <HeroPopoverTrigger>
    <button>Open Popover</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">
      <p>This is the popover content.</p>
    </div>
  </HeroPopoverContent>
</HeroPopover>
```

### With arrow and placement

```tsx
<HeroPopover showArrow placement="right">
  <HeroPopoverTrigger>
    <button>Show right</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">Content with arrow</div>
  </HeroPopoverContent>
</HeroPopover>
```

### With backdrop

```tsx
<HeroPopover backdrop="blur">
  <HeroPopoverTrigger>
    <button>Blurred backdrop</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">Blurred background behind popover</div>
  </HeroPopoverContent>
</HeroPopover>
```

### Controlled

```tsx
const [isOpen, setIsOpen] = useState(false);

<HeroPopover isOpen={isOpen} onOpenChange={setIsOpen}>
  <HeroPopoverTrigger>
    <button onClick={() => setIsOpen(true)}>Open controlled</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">
      <p>Controlled popover</p>
      <button onClick={() => setIsOpen(false)}>Close</button>
    </div>
  </HeroPopoverContent>
</HeroPopover>
```

### With accessible title

```tsx
<HeroPopover>
  <HeroPopoverTrigger>
    <button>Info</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    {(titleProps) => (
      <div className="p-4">
        <h3 {...titleProps} className="font-semibold">
          Accessible Title
        </h3>
        <p className="mt-1 text-sm">Content here.</p>
      </div>
    )}
  </HeroPopoverContent>
</HeroPopover>
```

## Notes

- Requires `@heroui/popover` and HeroUI provider setup in your app.
- `HeroPopoverContent` can accept a render function `(titleProps: object) => ReactNode` for accessible title elements.
