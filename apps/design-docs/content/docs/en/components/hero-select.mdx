## Overview

`HeroSelect`, `HeroSelectItem`, and `HeroSelectSection` are re-exports of [HeroUI's Select](https://www.heroui.com/docs/components/select) components, namespaced under the `HeroSelect` prefix for design system consistency.

## Installation

```tsx
import {
  HeroSelect,
  HeroSelectItem,
  HeroSelectSection,
} from "@nebutra/ui/primitives";
```

## Usage

### Basic

```tsx
<HeroSelect label="Favorite Animal" placeholder="Select an animal">
  <HeroSelectItem key="cat">Cat</HeroSelectItem>
  <HeroSelectItem key="dog">Dog</HeroSelectItem>
  <HeroSelectItem key="elephant">Elephant</HeroSelectItem>
  <HeroSelectItem key="penguin">Penguin</HeroSelectItem>
</HeroSelect>
```

### Multiple selection

```tsx
<HeroSelect
  label="Animals"
  selectionMode="multiple"
  placeholder="Select animals"
>
  <HeroSelectItem key="cat">Cat</HeroSelectItem>
  <HeroSelectItem key="dog">Dog</HeroSelectItem>
  <HeroSelectItem key="elephant">Elephant</HeroSelectItem>
</HeroSelect>
```

### With sections

```tsx
<HeroSelect label="Location" placeholder="Select location">
  <HeroSelectSection title="North America">
    <HeroSelectItem key="us">United States</HeroSelectItem>
    <HeroSelectItem key="ca">Canada</HeroSelectItem>
    <HeroSelectItem key="mx">Mexico</HeroSelectItem>
  </HeroSelectSection>
  <HeroSelectSection title="Europe">
    <HeroSelectItem key="uk">United Kingdom</HeroSelectItem>
    <HeroSelectItem key="fr">France</HeroSelectItem>
    <HeroSelectItem key="de">Germany</HeroSelectItem>
  </HeroSelectSection>
</HeroSelect>
```

### Controlled value

```tsx
const [value, setValue] = useState<string>("");

<HeroSelect
  label="Framework"
  selectedKeys={[value]}
  onSelectionChange={(keys) => setValue(Array.from(keys)[0] as string)}
>
  <HeroSelectItem key="react">React</HeroSelectItem>
  <HeroSelectItem key="vue">Vue</HeroSelectItem>
  <HeroSelectItem key="svelte">Svelte</HeroSelectItem>
</HeroSelect>
```

### Virtualized (large lists)

```tsx
<HeroSelect
  isVirtualized
  items={largeItemList}
  maxListboxHeight={400}
  label="Country"
>
  {(item) => <HeroSelectItem key={item.key}>{item.label}</HeroSelectItem>}
</HeroSelect>
```

## Notes

- These are direct re-exports from `@heroui/select` — refer to the [HeroUI Select docs](https://www.heroui.com/docs/components/select) for the full API reference.
- All HeroUI props are passed through unchanged.
- Requires `@heroui/select` and HeroUI provider setup in your app.
