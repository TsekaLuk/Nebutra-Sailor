## Overview

`Highlighter` wraps text in a `<span>` and applies a hand-drawn marker stroke animation using the `rough-notation` library. Supports 7 annotation types, custom colors, stroke widths, and viewport-triggered animations.

## Installation

```tsx
import { Highlighter } from "@nebutra/ui/primitives";
```

`rough-notation` is a peer dependency — install it if not already present:

```bash
pnpm add rough-notation
```

## Usage

### Basic highlight

```tsx
<p>
  This is <Highlighter>very important</Highlighter> text.
</p>
```

### Underline with custom color

```tsx
<Highlighter action="underline" color="#FF9800">
  Underlined text
</Highlighter>
```

### Circle annotation

```tsx
<Highlighter action="circle" color="#ef4444" strokeWidth={2}>
  Circled!
</Highlighter>
```

### Viewport-triggered

The annotation only plays when the element scrolls into view:

```tsx
<Highlighter triggerOnView color="#a5f3fc">
  Triggered on scroll
</Highlighter>
```

### Mixed annotations in a paragraph

```tsx
<p className="text-xl leading-relaxed">
  The{" "}
  <Highlighter action="underline" color="#FF9800">
    Magic UI Highlighter
  </Highlighter>{" "}
  makes important{" "}
  <Highlighter action="highlight" color="#87CEFA">
    text stand out
  </Highlighter>{" "}
  effortlessly.
</p>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Content to annotate |
| `action` | `HighlighterAction` | `"highlight"` | Type of annotation effect |
| `color` | `string` | `"#ffd1dc"` | Annotation stroke/fill color (any CSS color) |
| `strokeWidth` | `number` | `1.5` | Width of the annotation stroke in pixels |
| `animationDuration` | `number` | `600` | Duration of draw animation in milliseconds |
| `iterations` | `number` | `2` | Draw iterations — higher values add sketchy character |
| `padding` | `number` | `2` | Padding between element and annotation in pixels |
| `multiline` | `boolean` | `true` | Whether annotation wraps across multiple lines |
| `triggerOnView` | `boolean` | `false` | Only animate when element enters the viewport |

## Annotation Types

| Value | Description |
|-------|-------------|
| `highlight` | Filled highlight behind text (like a marker) |
| `underline` | Hand-drawn line under text |
| `box` | Rectangular box around text |
| `circle` | Oval/circle around text |
| `strike-through` | Line through the middle of text |
| `crossed-off` | Two diagonal lines crossing the text |
| `bracket` | Square bracket(s) beside text |

## Notes

- The component re-renders the annotation when any prop changes (color, action, etc.).
- A `ResizeObserver` redraws the annotation if the element or document body resizes.
- Annotations are properly cleaned up when the component unmounts.
- Use `iterations={3}` or higher for a more organic, hand-drawn appearance.
