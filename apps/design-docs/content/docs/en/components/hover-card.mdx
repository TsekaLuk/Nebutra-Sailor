## Accessibility

- Built on Radix UI HoverCard.
- HoverCard content is not shown to screen readers who navigate by tab — it is purely visual.
- For keyboard users, consider `Tooltip` or `Popover` as an alternative for critical information.
- The card is focusable and explorable once open via mouse hover.
