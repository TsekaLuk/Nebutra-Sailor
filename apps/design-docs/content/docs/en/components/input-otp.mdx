## Usage

```tsx
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSlot,
  InputOTPSeparator,
} from "@nebutra/ui/primitives"
```

## Props

### InputOTP (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxLength` | `number` | — | **Required.** Total number of OTP characters |
| `value` | `string` | — | Controlled OTP value |
| `onChange` | `(value: string) => void` | — | Callback on value change |
| `disabled` | `boolean` | `false` | Disables all slots |
| `pattern` | `string` | — | Regex pattern for allowed characters (e.g., numeric only) |
| `className` | `string` | — | Additional CSS classes |

### InputOTPSlot

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | — | **Required.** Zero-based slot position |
| `className` | `string` | — | Additional CSS classes |

## Examples

### Numeric Only (4-digit PIN)

```tsx
import { REGEXP_ONLY_DIGITS } from "input-otp"

<ComponentPreview className="min-h-[200px] flex justify-center items-center">
  <InputOTP maxLength={4} pattern={REGEXP_ONLY_DIGITS}>
    <InputOTPGroup>
      <InputOTPSlot index={0} />
      <InputOTPSlot index={1} />
      <InputOTPSlot index={2} />
      <InputOTPSlot index={3} />
    </InputOTPGroup>
  </InputOTP>
</ComponentPreview>
```

### Controlled OTP

```tsx
const [otp, setOtp] = useState("")

<InputOTP maxLength={6} value={otp} onChange={setOtp}>
  <InputOTPGroup>
    {Array.from({ length: 6 }).map((_, i) => (
      <InputOTPSlot key={i} index={i} />
    ))}
  </InputOTPGroup>
</InputOTP>

{otp.length === 6 && <p>Complete OTP: {otp}</p>}
```

## Accessibility

- Renders a hidden native `<input>` element — screen readers can interact with it normally.
- Each visual slot visually tracks the cursor position.
- Auto-advance: focus moves to next slot automatically after entry.
- Supports paste of full OTP string.
- Descriptive `aria-label="One-time password"` on the input.
