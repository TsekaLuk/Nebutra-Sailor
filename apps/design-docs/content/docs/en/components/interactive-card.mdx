## Usage

```tsx
import { InteractiveCard } from "@nebutra/ui";

<InteractiveCard
  step="STEP 1"
  title="Create Project"
  description="Start by creating a new project in your workspace."
/>
```

## Examples

### Default with Animated Icon

The default icon shows a folder with a PPT file floating upward and a download button that pulses on hover:

```tsx
<InteractiveCard
  step="STEP 1"
  title="Export your data"
  description="Select the format and export your content in one click."
/>
```

### Three-Step Process

```tsx
<div className="grid grid-cols-3 gap-6">
  <InteractiveCard
    step="STEP 1"
    title="Connect"
    description="Link your data sources with our one-click integrations."
  />
  <InteractiveCard
    step="STEP 2"
    title="Configure"
    description="Set up your pipeline with drag-and-drop simplicity."
  />
  <InteractiveCard
    step="STEP 3"
    title="Deploy"
    description="Push to production with a single command."
  />
</div>
```

### Custom Icon

Replace the default animated icon with your own:

```tsx
import { Sparkles } from "lucide-react";

<InteractiveCard
  step="STEP 1"
  title="AI Analysis"
  description="Let AI analyze your data and surface insights automatically."
  icon={
    <div className="mb-8 flex h-40 w-full items-center justify-center">
      <Sparkles className="h-20 w-20 text-primary" />
    </div>
  }
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `step` | `string` | — | Step label shown above the title (e.g., `"STEP 1"`) |
| `title` | `string` | — | Card heading |
| `description` | `string` | — | Card description text |
| `icon` | `ReactNode` | `AnimatedExportIcon` | Custom icon replacing the default animated SVG |
| `className` | `string` | — | Additional Tailwind classes |

## Notes

- The default `AnimatedExportIcon` is also exported and can be used standalone.
- Hover animation works via Framer Motion's `whileHover="hover"` / `initial="initial"` group — custom icons can respond to this by using `useAnimation` or `variants` from the `whileHover` context.
- The card renders with `text-center` alignment — override via `className` if left alignment is needed.
- Accessible: `role="group"` and `aria-label` are set automatically from `title` and `description`.
