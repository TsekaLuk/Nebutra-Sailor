## Usage

```tsx
import { Label } from "@nebutra/ui/primitives"
```

## Props

Extends `@radix-ui/react-label` `Root` props (extends `React.LabelHTMLAttributes<HTMLLabelElement>`).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `htmlFor` | `string` | — | Links label to its control via the control's `id` |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Label text |

## Accessibility

- Built on Radix UI Label — renders as a native `<label>` element.
- Clicking the label transfers focus to the associated control via `htmlFor`.
- When the associated control is disabled, the label automatically reflects the disabled cursor.
- Do not use `<span>` or `<div>` as labels — always use this `Label` component.
