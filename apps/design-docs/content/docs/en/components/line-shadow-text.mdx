## Usage

```tsx
import { LineShadowText } from "@nebutra/ui/primitives"
```

## CSS Setup

Add the `animate-line-shadow` keyframe to your global CSS or Tailwind config:

```css
@keyframes line-shadow {
  0%   { background-position: 0 0; }
  100% { background-position: 100% -100%; }
}
.animate-line-shadow::after {
  animation: line-shadow 15s linear infinite;
}
```

The component also exports `LINE_SHADOW_CSS` as a string for programmatic injection:

```tsx
import { LINE_SHADOW_CSS } from "@nebutra/ui/primitives"
```

## Props

`LineShadowText` extends both `React.HTMLAttributes<HTMLElement>` and Framer Motion `MotionProps`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text content (must be a string) |
| `shadowColor` | `string` | `"black"` | Color of the diagonal line shadow |
| `as` | `React.ElementType` | `"span"` | HTML element to render as |
| `className` | `string` | — | Additional CSS classes |
| `...motionProps` | `MotionProps` | — | All Framer Motion props (initial, animate, etc.) |

## Notes

- `children` must be a plain string — component will throw if passed JSX nodes.
- Shadow renders via CSS `::after` pseudo-element using `data-text` attribute.
