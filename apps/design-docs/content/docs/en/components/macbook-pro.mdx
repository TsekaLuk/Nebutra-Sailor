## Props

The `MacbookPro` component accepts the following props:

| Prop | Type | Default | Description |
|---|---|---|---|
| `width` | `number` | `650` | The SVG width parameter. |
| `height` | `number` | `400` | The SVG height parameter. |
| `src` | `string` | `undefined` | Optional URL to an image that will be rendered inside the screen mask. |
| `...props` | `SVGProps<SVGSVGElement>` | `{}` | All native React SVG element props are supported. |

## Usage

```tsx
import { MacbookPro } from "@nebutra/ui/primitives"

export function MockupSection() {
  return (
    <MacbookPro 
      src="/images/dashboard-screenshot.png"
      className="w-full max-w-4xl" 
    />
  )
}
```
