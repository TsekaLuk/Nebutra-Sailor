## Usage

```tsx
import { MagicCard } from "@nebutra/ui";

<MagicCard className="rounded-xl p-6">
  <h3 className="text-xl font-bold">Pro Plan</h3>
  <p className="mt-2 text-muted-foreground">$49/month</p>
</MagicCard>
```


## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Card content |
| `gradientSize` | `number` | `200` | Radius of the spotlight gradient in pixels |
| `gradientColor` | `string` | `"#262626"` | Color of the inner spotlight fill |
| `gradientOpacity` | `number` | `0.8` | Opacity of the inner spotlight |
| `gradientFrom` | `string` | `"#9E7AFF"` | Start color of the border gradient |
| `gradientTo` | `string` | `"#FE8BBB"` | End color of the border gradient |
| `className` | `string` | — | Applied to the outer wrapper |

## Notes

- `MagicCard` wraps children in a `relative rounded-[inherit]` container — the border radius should be set via `className`.
- The gradient border is only visible when the cursor is over the card (tracked via `onPointerMove`).
- The spotlight resets to off-screen when the cursor leaves the card, the window loses focus, or the tab becomes hidden.
- For the border effect to look right, ensure the parent element doesn't clip overflow.
