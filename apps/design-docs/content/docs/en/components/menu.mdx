## Overview

`Menu` is a semantic dropdown component built on top of `DropdownMenu` (Radix). It exposes a high-level API with `prefix`, `suffix`, `locked`, `disabled`, and `href` slots on each item — covering all standard UI Menu variants without exposing Radix internals at the usage site.

Use `DropdownMenu` directly when you need raw building blocks like checkbox items, radio groups, or sub-menus. Use `Menu` for standard navigation and action lists.

## Permissions & States

### Locked Items

Locked items are permission-gated — visible but non-interactive, automatically rendering a lock icon. Semantically distinct from `disabled`.

## Link Items

Automatically renders the item as an `<a>` tag when the `href` prop is provided. Keyboard and mouse navigation is preserved via Radix `asChild`.

## Custom Trigger

You can completely replace the `Menu.Trigger` appearance by overriding classes or using other elements via `asChild` (if you use the raw `DropdownMenu` primitives). With `<Menu.Trigger>`, you can pass custom children.

## Props

The API mirrors Radix DropdownMenu exactly, but the `.Item` accepts custom rendering props.

### `Menu.Trigger`

Extends `<button>` HTML attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `chevron` | `boolean` | `false` | Adds a rotating ChevronDown icon |
| `className` | `string` | — | Override trigger styles |

### `Menu.Content`

Extends Radix `DropdownMenuContent` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"start"` | Horizontal alignment relative to trigger |
| `sideOffset` | `number` | `4` | Gap between trigger and content in px |

### `Menu.Item`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `prefix` | `ReactNode` | — | Icon/element on the left |
| `suffix` | `ReactNode` | — | Icon/element on the right (overridden by lock icon when `locked`) |
| `disabled` | `boolean` | `false` | Grays out item, removes pointer events |
| `locked` | `boolean` | `false` | Permission-gated state — shows lock icon, prevents interaction |
| `href` | `string` | — | Renders item as `<a>` navigating to this URL |
| `onSelect` | `(e: Event) => void` | — | Called on item selection |
| `className` | `string` | — | Additional CSS classes |
