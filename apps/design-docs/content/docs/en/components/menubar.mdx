## Usage

```tsx

  Menubar,
  MenubarMenu,
  MenubarTrigger,
  MenubarContent,
  MenubarItem,
  MenubarLabel,
  MenubarSeparator,
  MenubarShortcut,
  MenubarCheckboxItem,
  MenubarRadioGroup,
  MenubarRadioItem,
  MenubarSub,
  MenubarSubContent,
  MenubarSubTrigger,
} from "@nebutra/ui/primitives"
```

## Props

### Menubar (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | `MenubarMenu` elements |

### MenubarContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"start"` | Content alignment |
| `alignOffset` | `number` | `-4` | Offset from trigger |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Menu items |

### MenubarItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `inset` | `boolean` | `false` | Adds left padding for alignment |
| `disabled` | `boolean` | `false` | Prevents selection |
| `onSelect` | `() => void` | — | Callback when item is selected |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Item content |

## Accessibility

- Built on Radix UI Menubar — follows the WAI-ARIA [Menu Button pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menu-button/).
- `Menubar` has `role="menubar"`.
- Each `MenubarTrigger` has `role="menuitem"` with `aria-haspopup="menu"`.
- Keyboard: `ArrowRight`/`ArrowLeft` to navigate between menus, `ArrowDown` to open, `Escape` to close.
