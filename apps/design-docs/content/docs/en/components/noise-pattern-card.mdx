## Overview

`NoisePatternCard` wraps content in a dark card with a repeating grain/noise texture. The card fades in with a subtle upward motion on mount. Use `NoisePatternCardBody` for consistent internal padding.

The noise pattern requires a `bg-noise-pattern` Tailwind utility to be configured in your project's CSS.

## Installation

```tsx
import {
  NoisePatternCard,
  NoisePatternCardBody,
} from "@nebutra/ui/primitives";
```

## Tailwind Config

Add the noise pattern background to your CSS or Tailwind config:

```css
/* globals.css */
@layer utilities {
  .bg-noise-pattern {
    background-image: url("data:image/svg+xml,..."); /* SVG noise filter */
  }
}
```

## Usage

### Basic card

```tsx
<NoisePatternCard className="max-w-sm">
  <NoisePatternCardBody>
    <h3 className="text-lg font-semibold text-white">Title</h3>
    <p className="mt-2 text-sm text-zinc-400">
      A sophisticated dark card with organic grain texture.
    </p>
  </NoisePatternCardBody>
</NoisePatternCard>
```

### Custom pattern and overlay

```tsx
<NoisePatternCard
  patternClassName="opacity-50"
  overlayClassName="bg-zinc-900/50"
  className="rounded-xl"
>
  <NoisePatternCardBody>
    <p className="text-zinc-200">Custom opacity noise texture</p>
  </NoisePatternCardBody>
</NoisePatternCard>
```

## Props

### NoisePatternCard

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Card content |
| `className` | `string` | — | CSS classes for the outer card container |
| `patternClassName` | `string` | — | CSS classes for the noise pattern layer |
| `overlayClassName` | `string` | — | CSS classes for the overlay div wrapping children |

### NoisePatternCardBody

Extends `React.HTMLAttributes<HTMLDivElement>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes (default: `p-4 text-left md:p-6`) |

## Notes

- The card background is `bg-zinc-950` in both light and dark modes — designed for dark UI contexts.
- Entrance animation uses Framer Motion with `opacity: 0 → 1` and `y: -20 → 0` over 0.8s.
- The noise pattern tile is 500×500px via `bg-[length:500px_500px]`.
- The overlay adds `bg-zinc-950/30` transparency over the pattern.
