## Usage

<ComponentPreview name="popover-demo" className="min-h-[250px] flex justify-center items-start pt-10">
    <Popover>
    <PopoverTrigger asChild>
      <Button variant="ghost" size="icon">

      </Button>
    </PopoverTrigger>
    <PopoverContent side="right" align="start" className="w-64">
      <div className="space-y-3">
        <div className="flex items-center justify-between">
          <Label htmlFor="notifications">Notifications</Label>
          <Switch id="notifications" />

        <div className="flex items-center justify-between">

## Accessibility

- Built on Radix UI Popover — `PopoverContent` has `role="dialog"`.
- Trigger has `aria-expanded` and `aria-haspopup="dialog"`.
- Focus is moved into the popover when opened; returns to trigger on close.
- Pressing `Escape` closes the popover.
- Clicking outside the popover closes it by default.
