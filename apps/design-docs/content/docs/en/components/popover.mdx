## Props

### Popover (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Uncontrolled default open state |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |
| `modal` | `boolean` | `false` | Whether to use modal behavior |

### PopoverTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Merge props onto the immediate child element |
| `children` | `ReactNode` | — | Trigger element |

### PopoverContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Preferred side to open |
| `sideOffset` | `number` | `4` | Distance from trigger |
| `align` | `"start" \| "center" \| "end"` | `"center"` | Alignment along trigger |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Popover body content |

## Accessibility

- Built on Radix UI Popover — `PopoverContent` has `role="dialog"`.
- Trigger has `aria-expanded` and `aria-haspopup="dialog"`.
- Focus is moved into the popover when opened; returns to trigger on close.
- Pressing `Escape` closes the popover.
- Clicking outside the popover closes it by default.
