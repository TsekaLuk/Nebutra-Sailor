## Overview

`PricingCard` is a namespace export with composable sub-components for building pricing cards. Features a glass effect header, plan name, optional badge, price with strikethrough support, and a feature list with section separators.

## Installation

```tsx
import { PricingCard } from "@nebutra/ui/primitives";
```

## Usage

### Basic pricing card

```tsx
import { Check } from "lucide-react";
import { PricingCard } from "@nebutra/ui/primitives";

<PricingCard.Card>
  <PricingCard.Header>
    <PricingCard.Plan>
      <PricingCard.PlanName>Pro</PricingCard.PlanName>
      <PricingCard.Badge>Popular</PricingCard.Badge>
    </PricingCard.Plan>
    <PricingCard.Price>
      <PricingCard.MainPrice>$29</PricingCard.MainPrice>
      <PricingCard.Period>/month</PricingCard.Period>
    </PricingCard.Price>
    <PricingCard.Description>Per workspace, billed annually</PricingCard.Description>
  </PricingCard.Header>
  <PricingCard.Body>
    <PricingCard.List>
      <PricingCard.ListItem>
        <Check className="mt-0.5 h-4 w-4 flex-shrink-0 text-primary" />
        Unlimited projects
      </PricingCard.ListItem>
      <PricingCard.ListItem>
        <Check className="mt-0.5 h-4 w-4 flex-shrink-0 text-primary" />
        Priority support
      </PricingCard.ListItem>
    </PricingCard.List>
  </PricingCard.Body>
</PricingCard.Card>
```

### With original price (strikethrough)

```tsx
<PricingCard.Price>
  <PricingCard.OriginalPrice>$49</PricingCard.OriginalPrice>
  <PricingCard.MainPrice>$29</PricingCard.MainPrice>
  <PricingCard.Period>/month</PricingCard.Period>
</PricingCard.Price>
```

### With separator (locked features)

```tsx
<PricingCard.Body>
  <PricingCard.List>
    <PricingCard.ListItem>
      <Check className="h-4 w-4 text-primary" />
      5 projects
    </PricingCard.ListItem>
  </PricingCard.List>
  <PricingCard.Separator>Upgrade to unlock</PricingCard.Separator>
  <PricingCard.List>
    <PricingCard.ListItem className="opacity-50">
      <Lock className="h-4 w-4" />
      Unlimited projects
    </PricingCard.ListItem>
  </PricingCard.List>
</PricingCard.Body>
```

### With icon in plan name

```tsx
import { Zap } from "lucide-react";

<PricingCard.PlanName>
  <Zap />
  Pro
</PricingCard.PlanName>
```

## Components

| Sub-component | HTML element | Description |
|--------------|-------------|-------------|
| `PricingCard.Card` | `<div>` | Outer card container |
| `PricingCard.Header` | `<div>` | Glass-effect header section |
| `PricingCard.Plan` | `<div>` | Row with plan name and badge |
| `PricingCard.PlanName` | `<div>` | Plan name (supports icon + label) |
| `PricingCard.Badge` | `<span>` | Rounded pill badge |
| `PricingCard.Description` | `<p>` | Subtitle/description text |
| `PricingCard.Price` | `<div>` | Price row container |
| `PricingCard.MainPrice` | `<span>` | Large bold price number |
| `PricingCard.Period` | `<span>` | Period text (e.g., `/month`) |
| `PricingCard.OriginalPrice` | `<span>` | Struck-through original price |
| `PricingCard.Body` | `<div>` | Feature list section |
| `PricingCard.List` | `<ul>` | Feature list container |
| `PricingCard.ListItem` | `<li>` | Single feature item (icon + text) |
| `PricingCard.Separator` | `<div>` | Divider with center label text |

## Props

All sub-components extend their underlying HTML element's props.

### PricingCard.Header specific

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `glassEffect` | `boolean` | `true` | Apply gradient glass overlay |

### PricingCard.Separator specific

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | `"Upgrade to access"` | Separator label text |

## Notes

- Individual components are also exported (`PricingCardRoot`, `PricingCardHeader`, etc.) for flexibility without the namespace pattern.
- The `PlanName` component automatically sizes icons to `size-4` unless the icon has an explicit size class.
- The header glass effect is a `linear-gradient` overlay — it works best on slightly dark/muted backgrounds.
