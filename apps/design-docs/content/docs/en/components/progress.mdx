## Usage

```tsx
import { Progress } from "@nebutra/ui/primitives"
```

## Props

Extends `@radix-ui/react-progress` `Root` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number \| null` | — | Progress value from `0` to `max`. `null` for indeterminate |
| `max` | `number` | `100` | Maximum value |
| `getValueLabel` | `(value: number, max: number) => string` | — | Custom label for screen readers |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Built on Radix UI Progress — has `role="progressbar"`.
- Sets `aria-valuemin="0"`, `aria-valuemax` (default 100), and `aria-valuenow` based on `value`.
- `aria-label` or `aria-labelledby` should be added for context.
- Indeterminate state (`value={null}`) omits `aria-valuenow`.
