## Usage

```tsx
import { ProgressiveBlur } from "@nebutra/ui/primitives"
```

## Examples

### Bottom Scroll Fade

```tsx
<div className="relative h-[400px] overflow-auto">
  <div className="space-y-4 p-4">
    {items.map(item => <Card key={item.id} />)}
  </div>
  <ProgressiveBlur position="bottom" height="30%" />
</div>
```

### Top and Bottom

```tsx
<div className="relative h-[300px] overflow-auto">
  <ProgressiveBlur position="both" />
  <Content />
</div>
```

### Subtle Fade (fewer blur levels)

```tsx
<ProgressiveBlur
  blurLevels={[0.5, 1, 2, 4, 8]}
  height="20%"
  position="bottom"
/>
```

### With a "See More" CTA

```tsx
<div className="relative h-[300px] overflow-hidden">
  <ProgressiveBlur position="bottom" height="40%">
    <button className="absolute bottom-4 left-1/2 -translate-x-1/2">
      Show more
    </button>
  </ProgressiveBlur>
  <Content />
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `"top" \| "bottom" \| "both"` | `"bottom"` | Where to apply the blur effect |
| `height` | `string` | `"30%"` | Height of the blur zone (CSS length) |
| `blurLevels` | `number[]` | `[0.5, 1, 2, 4, 8, 16, 32, 64]` | Blur values in px for each layer |
| `children` | `ReactNode` | — | Optional content inside the blur zone (e.g., a CTA button) |
| `className` | `string` | — | Additional CSS classes |

## Notes

- The component is `pointer-events-none` by default. Children inside it are interactive.
- Uses `backdrop-filter` with gradient masks for the progressive blur effect.
- Safari compatibility included via `-webkit-backdrop-filter`.
- When `position="both"`, the `height` prop is ignored and the component fills the container.
