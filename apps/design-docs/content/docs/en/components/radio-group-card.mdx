## Usage

```tsx
import { RadioGroupCard, RadioGroupCardItem } from "@nebutra/ui/primitives"
```

## Props

### RadioGroupCard

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled selected value |
| `defaultValue` | `string` | — | Uncontrolled default selected value |
| `onValueChange` | `(value: string) => void` | — | Callback when selection changes |
| `disabled` | `boolean` | `false` | Disables all items |
| `className` | `string` | — | Additional CSS classes (use `grid` utilities for layout) |
| `children` | `ReactNode` | — | `RadioGroupCardItem` elements |

### RadioGroupCardItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | **Required.** Value for this option |
| `id` | `string` | — | For accessibility linking |
| `disabled` | `boolean` | `false` | Disables this specific item |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Card content (icon + label + description) |

## Accessibility

- Selected item has a visible checked indicator and border highlight.
- Each card item has `role="radio"` with `aria-checked`.
- Keyboard: `ArrowLeft`/`ArrowRight` or `ArrowUp`/`ArrowDown` to navigate between cards.
- `Space` to select the focused card.
