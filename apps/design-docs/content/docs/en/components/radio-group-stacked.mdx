## Usage

```tsx
import { RadioGroupStacked, RadioGroupStackedItem } from "@nebutra/ui/primitives"
```

## Props

### RadioGroupStacked

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled selected value |
| `defaultValue` | `string` | — | Uncontrolled default selected value |
| `onValueChange` | `(value: string) => void` | — | Callback when selection changes |
| `disabled` | `boolean` | `false` | Disables all items |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | `RadioGroupStackedItem` elements |

### RadioGroupStackedItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | **Required.** Value for this option |
| `id` | `string` | — | For accessibility linking |
| `disabled` | `boolean` | `false` | Disables this item |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Item content |

## Examples

### Notification Preference

```tsx
<RadioGroupStacked defaultValue="all">
  <RadioGroupStackedItem value="all" id="notif-all">
    <div>
      <div className="font-medium">All notifications</div>
      <div className="text-sm text-muted-foreground">Receive all activity.</div>
    </div>
  </RadioGroupStackedItem>
  <RadioGroupStackedItem value="mentions" id="notif-mentions">
    <div>
      <div className="font-medium">Mentions only</div>
      <div className="text-sm text-muted-foreground">Only when mentioned directly.</div>
    </div>
  </RadioGroupStackedItem>
  <RadioGroupStackedItem value="none" id="notif-none">
    <div>
      <div className="font-medium">No notifications</div>
      <div className="text-sm text-muted-foreground">Completely silent.</div>
    </div>
  </RadioGroupStackedItem>
</RadioGroupStacked>
```

## Accessibility

- Has `role="radiogroup"` on the root container.
- Each item has `role="radio"` with `aria-checked`.
- Selected item is visually distinguished with border highlight.
- Keyboard navigation: `ArrowUp`/`ArrowDown` to move, `Space` to select.
