## Usage

```tsx
import { RadioGroupStacked, RadioGroupStackedItem } from "@nebutra/ui/primitives"
```

## Props

### RadioGroupStacked

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled selected value |
| `defaultValue` | `string` | — | Uncontrolled default selected value |
| `onValueChange` | `(value: string) => void` | — | Callback when selection changes |
| `disabled` | `boolean` | `false` | Disables all items |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | `RadioGroupStackedItem` elements |

### RadioGroupStackedItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | **Required.** Value for this option |
| `id` | `string` | — | For accessibility linking |
| `disabled` | `boolean` | `false` | Disables this item |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Item content |

## Accessibility

- Has `role="radiogroup"` on the root container.
- Each item has `role="radio"` with `aria-checked`.
- Selected item is visually distinguished with border highlight.
- Keyboard navigation: `ArrowUp`/`ArrowDown` to move, `Space` to select.
