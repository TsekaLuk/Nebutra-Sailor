## Usage

```tsx
import { RadioGroup, RadioGroupItem } from "@nebutra/ui/primitives"
```

## Props

### RadioGroup (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | ŌĆö | Controlled selected value |
| `defaultValue` | `string` | ŌĆö | Uncontrolled default selected value |
| `onValueChange` | `(value: string) => void` | ŌĆö | Callback when selection changes |
| `disabled` | `boolean` | `false` | Disables all radio items |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | Layout direction |
| `className` | `string` | ŌĆö | Additional CSS classes |

### RadioGroupItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | ŌĆö | **Required.** Value for this option |
| `disabled` | `boolean` | `false` | Disables this specific item |
| `id` | `string` | ŌĆö | For linking with a `<Label>` |
| `className` | `string` | ŌĆö | Additional CSS classes |

## Accessibility

- Built on Radix UI RadioGroup ŌĆö has `role="radiogroup"`.
- Each item has `role="radio"` with `aria-checked`.
- Always pair items with `<Label>` via matching `id`/`htmlFor`.
- Keyboard: `ArrowUp`/`ArrowDown` (or `ArrowLeft`/`ArrowRight`) to navigate, `Space` to select.
