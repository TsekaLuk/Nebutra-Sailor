## Usage

```tsx
import { ReactionChip, ReactionBadge, MessageWithReactions } from "@nebutra/ui";
import { useState } from "react";

// Pre-composed message with reactions
<MessageWithReactions text="Great work on this feature! 🚀" />

// Standalone chip
const [selected, setSelected] = useState<string>();
<ReactionChip onSelect={setSelected} selected={selected} />
```

## Examples

### MessageWithReactions (Pre-composed)

The simplest way — a message card that shows the reaction picker on hover:

```tsx
<MessageWithReactions text="This new design is looking great!" />
```

With custom emojis:

```tsx
<MessageWithReactions
  text="Check out this new component"
  reactionOptions={["🔥", "💯", "🙌", "✨", "👀"]}
/>
```

With custom children:

```tsx
<MessageWithReactions reactionOptions={["👍", "❤️", "😂"]}>
  <div className="flex items-center gap-2">
    <Avatar src="/user.jpg" />
    <p>Custom message content with rich formatting</p>
  </div>
</MessageWithReactions>
```

### Standalone ReactionChip

```tsx
const [selected, setSelected] = useState<string | undefined>();

<div className="relative inline-block">
  <div className="rounded-lg border p-3 text-sm">
    Hover to react
  </div>
  <div className="absolute -top-3 right-0">
    <ReactionChip
      emojis={["👍", "❤️", "😂", "🎉", "🚀"]}
      onSelect={setSelected}
      selected={selected}
    />
  </div>
</div>
```

## Props

### ReactionChip

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onSelect` | `(emoji: string) => void` | — | Callback when emoji is clicked |
| `emojis` | `string[]` | `["👍", "❤️", "😂", "🎉"]` | Available emoji options |
| `selected` | `string` | — | Currently selected emoji (highlighted) |
| `className` | `string` | — | Additional classes |

### ReactionBadge

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `emoji` | `string` | — | Emoji to display |
| `count` | `number` | — | Reaction count |
| `bump` | `boolean` | `false` | Triggers brief scale animation |
| `className` | `string` | — | Additional classes |

### MessageWithReactions

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | — | Message text content |
| `reactionOptions` | `string[]` | `["👍", "❤️", "😂", "🎉"]` | Available emoji options |
| `children` | `ReactNode` | — | Custom content replacing `text` |
| `className` | `string` | — | Additional classes |

## Notes

- `ReactionChip` uses `onMouseDown` with `preventDefault` to prevent focus loss from the triggering element before the emoji is selected.
- The bump animation in `MessageWithReactions` plays for 180ms when a reaction is added.
- Selecting the same emoji twice deselects it (toggles off).
- Accessibility: `role="group"`, `aria-label`, `aria-pressed` are set on picker buttons.
