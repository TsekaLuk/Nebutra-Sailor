## Usage

```tsx
import { ScrollArea, ScrollBar } from "@nebutra/ui/primitives"
```

## Props

### ScrollArea

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `"auto" \| "always" \| "scroll" \| "hover"` | `"hover"` | When to show the scrollbar |
| `scrollHideDelay` | `number` | `600` | Delay in ms before hiding scroll bar |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Scrollable content |

### ScrollBar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | Scrollbar orientation |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Scroll area content is accessible to keyboard and screen reader users — native scrolling semantics are preserved.
- The custom scrollbar is `aria-hidden="true"` — it is purely visual.
- Focus remains on content elements within the scroll area, not the scrollbar itself.
