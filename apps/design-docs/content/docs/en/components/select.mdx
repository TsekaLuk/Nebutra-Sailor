## Usage

```tsx
import {
  Select,
  SelectTrigger,
  SelectValue,
  SelectContent,
  SelectItem,
  SelectGroup,
  SelectLabel,
  SelectSeparator,
  SelectScrollUpButton,
  SelectScrollDownButton,
} from "@nebutra/ui/primitives"
```

## Props

### Select (Root)

Extends `@radix-ui/react-select` `Root` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled selected value |
| `defaultValue` | `string` | — | Uncontrolled default value |
| `onValueChange` | `(value: string) => void` | — | Callback when selection changes |
| `open` | `boolean` | — | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |
| `disabled` | `boolean` | `false` | Disables the select |

### SelectTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Trigger content (typically `SelectValue`) |

### SelectValue

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | — | Text shown when no value selected |

### SelectContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `position` | `"item-aligned" \| "popper"` | `"popper"` | Positioning strategy |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Preferred side to open |
| `align` | `"start" \| "center" \| "end"` | `"start"` | Alignment along the trigger |

### SelectItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | **Required.** The value for this option |
| `disabled` | `boolean` | `false` | Prevents this item from being selected |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Item label text |

### SelectGroup / SelectLabel / SelectSeparator

| Component | Description |
|-----------|-------------|
| `SelectGroup` | Groups related items together |
| `SelectLabel` | Non-selectable label for a group |
| `SelectSeparator` | Visual divider between groups |

## Accessibility

- Built on Radix UI Select — follows the WAI-ARIA [Listbox pattern](https://www.w3.org/WAI/ARIA/apg/patterns/listbox/).
- `SelectTrigger` has `role="combobox"` with `aria-expanded` and `aria-haspopup="listbox"`.
- `SelectContent` has `role="listbox"`.
- Each `SelectItem` has `role="option"` and `aria-selected`.
- Full keyboard navigation: `ArrowUp`/`ArrowDown` to navigate, `Enter`/`Space` to select, `Escape` to close.
