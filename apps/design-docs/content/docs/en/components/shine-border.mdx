---

## Usage

```tsx
import { ShineBorder } from "@nebutra/ui/primitives"
```

## Examples

### Overlay Mode (no children)

Add a shine effect to any existing element:

```tsx
<div className="relative rounded-xl p-6">
  <ShineBorder shineColor={["#A07CFE", "#FE8FB5", "#FFBE7B"]} />
  <p>Your content here</p>
</div>
```

### Container Mode (with children)

Standalone card with shine border:

```tsx
<ShineBorder
  shineColor="#A07CFE"
  borderRadius={16}
  borderWidth={2}
>
  <h2>Shine Card</h2>
  <p>Content inside</p>
</ShineBorder>
```

### Multi-Color Gradient

```tsx
<ShineBorder shineColor={["#FF6B6B", "#FFE66D", "#4ECDC4"]}>
  <span>Rainbow border</span>
</ShineBorder>
```

### Faster Animation

```tsx
<ShineBorder shineColor="#6366f1" duration={6}>
  Content
</ShineBorder>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `shineColor` | `string \| string[]` | — | Color(s) for the shine gradient |
| `color` | `string \| string[]` | `"#000000"` | Deprecated alias for `shineColor` |
| `borderWidth` | `number` | `1` | Border width in pixels |
| `duration` | `number` | `14` | Animation cycle duration in seconds |
| `borderRadius` | `number` | `8` | Border radius (container mode only) |
| `children` | `ReactNode` | — | Content (switches to container mode) |
| `className` | `string` | — | Additional CSS classes |

## Modes

| Mode | When | Positioning |
|------|------|-------------|
| **Overlay** | No `children` | `absolute inset-0`, inherits parent's `border-radius` |
| **Container** | With `children` | Standalone `div`, wraps content |

## Notes

- Requires the `animate-shine` keyframe in your Tailwind config or global CSS.
- `motion-safe:` prefix is applied — animation is disabled when `prefers-reduced-motion` is set.
