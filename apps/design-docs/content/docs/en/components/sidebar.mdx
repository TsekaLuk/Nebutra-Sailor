## Demo

```tsx

  Sidebar,
  SidebarContent,
  SidebarHeader,
  SidebarFooter,
  SidebarGroup,
  SidebarGroupLabel,
  SidebarGroupContent,

  SidebarMenuButton,

  SidebarTrigger,

import { LayoutDashboard, Settings, Users, HelpCircle } from "lucide-react"

export function SidebarDemo() {
  return (
    <SidebarProvider>
      <Sidebar>
        <SidebarHeader>
          <div className="px-4 py-2 font-semibold">Nebutra</div>
        </SidebarHeader>
        <SidebarContent>
          <SidebarGroup>
            <SidebarGroupLabel>Application</SidebarGroupLabel>
            <SidebarGroupContent>
              <SidebarMenu>
                <SidebarMenuItem>
                  <SidebarMenuButton asChild>
                    <a href="/dashboard">
                      <LayoutDashboard />
                      Dashboard
                    </a>
                  </SidebarMenuButton>
                </SidebarMenuItem>
                <SidebarMenuItem>
                  <SidebarMenuButton asChild>
                    <a href="/users">
                      <Users />
                      Users
                    </a>
                  </SidebarMenuButton>
                </SidebarMenuItem>
              </SidebarMenu>
            </SidebarGroupContent>
          </SidebarGroup>
        </SidebarContent>
        <SidebarFooter>
          <SidebarMenu>
            <SidebarMenuItem>
              <SidebarMenuButton asChild>
                <a href="/settings">
                  <Settings />
                  Settings
                </a>
              </SidebarMenuButton>
            </SidebarMenuItem>
          </SidebarMenu>
        </SidebarFooter>
      </Sidebar>
      <main>
        <SidebarTrigger />
        {/* Page content */}
      </main>
    </SidebarProvider>
  )
}
```

## Props

### SidebarProvider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultOpen` | `boolean` | `true` | Uncontrolled default open state |
| `open` | `boolean` | — | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |

### Sidebar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"left" \| "right"` | `"left"` | Which side the sidebar appears on |
| `variant` | `"sidebar" \| "floating" \| "inset"` | `"sidebar"` | Visual variant |
| `collapsible` | `"offcanvas" \| "icon" \| "none"` | `"offcanvas"` | Collapse behavior |
| `className` | `string` | — | Additional CSS classes |

### SidebarMenuButton

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isActive` | `boolean` | `false` | Highlights the button as the active route |
| `asChild` | `boolean` | `false` | Merge props onto the child element |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Button size |
| `tooltip` | `string \| TooltipContent` | — | Tooltip shown in collapsed icon mode |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- `Sidebar` renders as a `<aside>` element with `aria-label`.
- `SidebarTrigger` has `aria-expanded` to communicate sidebar state.
- Active menu buttons have `aria-current="page"`.
- Keyboard shortcut: `⌘B` (or `Ctrl+B`) toggles the sidebar via `useSidebar()` hook.
- In collapsed icon mode, `tooltip` provides screen reader text for icon-only buttons.
