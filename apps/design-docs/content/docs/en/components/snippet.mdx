## Overview

`Snippet` renders code with a `$` prefix and a copy button. Supports single-line and multiline code, 4 visual variants, 6 color schemes, 3 sizes, and configurable border radius.

## Installation

```tsx
import { Snippet } from "@nebutra/ui/primitives";
```

## Usage

### Basic

```tsx
<Snippet>npm install @nebutra/ui</Snippet>
```

### Without symbol

```tsx
<Snippet hideSymbol>const x = 1;</Snippet>
```

### Custom symbol

```tsx
<Snippet symbol=">">{`git clone https://github.com/org/repo`}</Snippet>
```

### Multiline

```tsx
<Snippet>
  {`npm install package
yarn add package
pnpm add package`}
</Snippet>
```

### Colored variants

```tsx
<Snippet color="success">Deployment successful!</Snippet>
<Snippet color="danger">Build failed: missing dependency</Snippet>
<Snippet color="warning">Deprecated API in use</Snippet>
```

### Visual variants

```tsx
<Snippet variant="flat">Flat background</Snippet>
<Snippet variant="bordered">Bordered</Snippet>
<Snippet variant="shadow">With shadow</Snippet>
<Snippet variant="solid">Inverted solid</Snippet>
```

### Without copy button

```tsx
<Snippet hideCopyButton>Read-only code</Snippet>
```

### Copy callback

```tsx
<Snippet onCopy={(text) => console.log("Copied:", text)}>
  npm run dev
</Snippet>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Code content |
| `codeString` | `string` | — | Override copy text (if different from children) |
| `variant` | `"flat" \| "bordered" \| "shadow" \| "solid"` | `"flat"` | Visual style |
| `color` | `"default" \| "primary" \| "secondary" \| "success" \| "warning" \| "danger"` | `"default"` | Color scheme |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Size |
| `radius` | `"none" \| "sm" \| "md" \| "lg" \| "full"` | `"lg"` | Border radius |
| `symbol` | `string \| ReactNode` | `"$"` | Prefix symbol |
| `hideSymbol` | `boolean` | `false` | Hide the symbol prefix |
| `hideCopyButton` | `boolean` | `false` | Hide the copy button |
| `disableCopy` | `boolean` | `false` | Disable copy functionality |
| `copyIcon` | `ReactNode` | `<Copy />` | Custom copy icon |
| `checkIcon` | `ReactNode` | `<Check />` | Custom copied state icon |
| `timeout` | `number` | `2000` | Time (ms) before copied state resets |
| `onCopy` | `(value: string) => void` | — | Callback when code is copied |
| `className` | `string` | — | Additional CSS classes |
