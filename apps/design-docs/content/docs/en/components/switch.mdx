## Props

### Switch (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | `"default"` | Form field name group for the internal radio inputs |
| `size` | `"small" \| "medium" \| "large"` | `"medium"` | Defines the size of the controls |
| `className` | `string` | — | Additional CSS classes |
| `style` | `React.CSSProperties` | — | Additional inline styles |

### Switch.Control

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Text label for the control option |
| `value` | `string` | — | **Required.** The value of this specific option |
| `defaultChecked` | `boolean` | `false` | Whether this option should be selected by default |
| `disabled` | `boolean` | `false` | Disables this specific control |
| `icon` | `ReactNode` | — | Optional icon to display instead of a text label |
