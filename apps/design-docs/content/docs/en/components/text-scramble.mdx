## Usage

```tsx
import { TextScramble } from "@nebutra/ui/primitives"
```

## Examples

### Auto-play on Mount

```tsx
<TextScramble className="font-mono text-sm uppercase">
  Hello World
</TextScramble>
```

### Hover Trigger

```tsx
function HoverScramble() {
  const [trigger, setTrigger] = useState(false);

  return (
    <TextScramble
      as="span"
      trigger={trigger}
      onScrambleComplete={() => setTrigger(false)}
      onHoverStart={() => setTrigger(true)}
      className="font-mono cursor-pointer"
    >
      Hover me
    </TextScramble>
  );
}
```

### Custom Character Set

```tsx
<TextScramble characterSet="░▒▓█▀▄▌▐" duration={1.5}>
  Cyber Mode
</TextScramble>
```

### Slow Motion

```tsx
<TextScramble duration={2} speed={0.08} className="text-2xl">
  Slow Reveal
</TextScramble>
```

## Props

`TextScramble` extends Framer Motion `MotionProps`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Final text content to resolve to |
| `duration` | `number` | `0.8` | Total scramble duration in seconds |
| `speed` | `number` | `0.04` | Character update interval in seconds |
| `characterSet` | `string` | alphanumeric | Characters used during scramble phase |
| `as` | `React.ElementType` | `"p"` | HTML element to render as |
| `trigger` | `boolean` | `true` | Set to `false` to pause; flip to `true` to replay |
| `onScrambleComplete` | `() => void` | — | Callback when animation finishes |
| `className` | `string` | — | Additional CSS classes |
| `...motionProps` | `MotionProps` | — | All Framer Motion props |

## Notes

- Spaces in the source text are preserved and never scrambled.
- Setting `trigger={false}` then `true` replays the animation from the start.
