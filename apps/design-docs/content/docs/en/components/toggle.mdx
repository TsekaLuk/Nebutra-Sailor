## Usage

```tsx
import { Toggle } from "@nebutra/ui";
```

## Props

| Prop             | Type                          | Default     | Description                                      |
| ---------------- | ----------------------------- | ----------- | ------------------------------------------------ |
| `pressed`        | `boolean`                     | —           | Controlled pressed state.                        |
| `defaultPressed` | `boolean`                     | `false`     | Initial pressed state (uncontrolled).            |
| `onPressedChange`| `(pressed: boolean) => void`  | —           | Callback when pressed state changes.             |
| `variant`        | `"default" \| "outline"`      | `"default"` | Visual variant.                                  |
| `size`           | `"default" \| "sm" \| "lg"`   | `"default"` | Size variant.                                    |
| `disabled`       | `boolean`                     | `false`     | Disables the toggle.                             |
| `asChild`        | `boolean`                     | `false`     | Render as child element.                         |

## Accessibility

Toggle renders a `<button>` with `aria-pressed`. Provide an `aria-label` when the toggle contains only an icon. Keyboard: `Space` or `Enter` activates.
