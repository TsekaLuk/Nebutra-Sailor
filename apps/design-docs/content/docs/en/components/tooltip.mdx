## Accessibility

- Built on Radix UI Tooltip — follows the WAI-ARIA [Tooltip pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tooltip/).
- Tooltip content is linked to the trigger via `aria-describedby`.
- Opens on focus as well as hover — keyboard users can access tooltips.
- Pressing `Escape` closes the tooltip.
- Do not place essential, non-redundant information only in tooltips — they are not accessible on touch devices.
