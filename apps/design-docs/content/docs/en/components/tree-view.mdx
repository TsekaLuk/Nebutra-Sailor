## Usage

```tsx
import { TreeView, TreeViewItem } from "@nebutra/ui";
```

## Props

### TreeView

| Prop        | Type        | Default | Description                                      |
| ----------- | ----------- | ------- | ------------------------------------------------ |
| `className` | `string`    | —       | Additional CSS classes.                          |
| `children`  | `ReactNode` | —       | `TreeViewItem` children.                         |

### TreeViewItem

| Prop               | Type        | Default | Description                                      |
| ------------------ | ----------- | ------- | ------------------------------------------------ |
| `label`            | `string`    | **required** | Display label.                             |
| `icon`             | `ReactNode` | —       | Icon displayed before the label.                 |
| `defaultExpanded`  | `boolean`   | `false` | Whether this item starts expanded.               |
| `expanded`         | `boolean`   | —       | Controlled expanded state.                       |
| `onExpandedChange` | `(expanded: boolean) => void` | — | Callback for expansion change.     |
| `selected`         | `boolean`   | `false` | Whether this item is selected.                   |
| `onSelect`         | `() => void`| —       | Callback when item is selected.                  |
| `disabled`         | `boolean`   | `false` | Disables selection and expansion.                |
| `children`         | `ReactNode` | —       | Nested `TreeViewItem` children.                  |

## Examples

### Controlled Expansion

```tsx
const [expanded, setExpanded] = useState<Record<string, boolean>>({});

function ControlledTree() {
  return (
    <TreeView>
      <TreeViewItem
        label="root"
        expanded={expanded["root"] ?? false}
        onExpandedChange={(open) =>
          setExpanded((prev) => ({ ...prev, root: open }))
        }
      >
        <TreeViewItem label="child-1" />
        <TreeViewItem label="child-2" />
      </TreeViewItem>
    </TreeView>
  );
}
```

### Selectable Items

```tsx
const [selected, setSelected] = useState<string | null>(null);

<TreeView>
  {items.map((item) => (
    <TreeViewItem
      key={item.id}
      label={item.name}
      selected={selected === item.id}
      onSelect={() => setSelected(item.id)}
    />
  ))}
</TreeView>
```

## Accessibility

Implements WAI-ARIA `tree` and `treeitem` roles. Keyboard navigation: `ArrowDown`/`ArrowUp` move between items, `ArrowRight` expands, `ArrowLeft` collapses or moves to parent, `Enter`/`Space` selects.
