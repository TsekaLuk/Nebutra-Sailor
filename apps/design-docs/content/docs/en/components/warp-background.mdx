## Props

`WarpBackground` extends `React.HTMLAttributes<HTMLDivElement>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Content rendered inside the container |
| `perspective` | `number` | `100` | CSS perspective depth in px (lower = more dramatic) |
| `beamsPerSide` | `number` | `3` | Number of beams per grid side |
| `beamSize` | `number` | `5` | Beam size as % of container |
| `beamDelayMax` | `number` | `3` | Maximum start delay in seconds |
| `beamDelayMin` | `number` | `0` | Minimum start delay in seconds |
| `beamDuration` | `number` | `3` | Duration of each beam animation in seconds |
| `gridColor` | `string` | `"var(--border)"` | CSS color for grid lines |
| `className` | `string` | — | Additional CSS classes (default: `p-20`) |

## Notes

- The grid background uses CSS container queries (`cqmax`, `cqh`, `cqi`) — requires modern browsers.
- Each beam has a randomly generated color (HSL) and aspect ratio for visual variety.
- The 3D effect is created with CSS `perspective` and `rotateX` transforms on four grid faces.
