---
title: "MetricCard"
description: "KPI display card showing a metric value with trend indicator, label, and optional sparkline."
---

## Demo

```tsx
<MetricCard
  label="Events ingested"
  value="2.4M"
  trend={{ direction: "up", value: "+12%", label: "vs last week" }}
  icon={<Zap className="w-5 h-5" />}
/>
```

## Usage

```tsx
import { MetricCard } from "@nebutra/ui";

export function DashboardMetrics() {
  const { metrics } = useMetrics();

  return (
    <div className="grid grid-cols-3 gap-4">
      <MetricCard
        label="Events ingested"
        value={formatNumber(metrics.eventsIngested)}
        trend={{ direction: "up", value: "+12%", label: "vs last week" }}
        icon={<Zap className="w-5 h-5" />}
      />
      <MetricCard
        label="Error rate"
        value={`${metrics.errorRate}%`}
        trend={{ direction: "down", value: "-0.3%", label: "vs last week" }}
        icon={<AlertCircle className="w-5 h-5" />}
        trendPositive="down"
      />
      <MetricCard
        label="Avg latency"
        value={`${metrics.p50Latency}ms`}
        trend={{ direction: "up", value: "+4ms", label: "vs last week" }}
        icon={<Clock className="w-5 h-5" />}
        trendPositive="down"
      />
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | — | **Required.** Short descriptor above the value |
| `value` | `string \| number` | — | **Required.** The primary metric value to display |
| `trend` | `MetricTrend` | — | Optional trend indicator shown below the value |
| `trendPositive` | `"up" \| "down"` | `"up"` | Which direction is considered positive (affects color) |
| `icon` | `React.ReactNode` | — | Icon in the top-right corner of the card |
| `description` | `string` | — | Muted secondary text beneath the trend |
| `isLoading` | `boolean` | `false` | Renders a skeleton placeholder |
| `onClick` | `() => void` | — | Makes the card interactive; renders with hover styles |
| `className` | `string` | — | Additional classes on the card root |

### `MetricTrend`

```ts
interface MetricTrend {
  direction: "up" | "down" | "neutral";
  value: string;    // e.g. "+12%" or "-0.3%"
  label?: string;   // e.g. "vs last week"
}
```

## Examples

### Loading state

```tsx
<MetricCard
  label="Events ingested"
  value=""
  isLoading={true}
/>
```

### Neutral trend

```tsx
<MetricCard
  label="Active pipelines"
  value={14}
  trend={{ direction: "neutral", value: "0", label: "no change" }}
/>
```

### Clickable card

```tsx
<MetricCard
  label="Failed jobs"
  value={7}
  trend={{ direction: "up", value: "+3", label: "today" }}
  trendPositive="down"
  onClick={() => router.push("/jobs?status=failed")}
/>
```

## Accessibility

- Renders as a `<div>` with `role="group"` and `aria-label="{label}: {value}"`.
- Clickable variant renders as a `<button>` with full keyboard support.
- Trend direction icons are `aria-hidden="true"`; trend value is included in the text node for screen readers.
- `isLoading` state has `aria-busy="true"` on the card root.
- Color is not the sole indicator of trend direction — arrow icons and text labels are always present.
