## Visual Philosophy: Hexagonal Intelligence — 六角智能

The Nebutra visual identity is rooted in the **hexagon** — the most efficient tessellation found in nature (honeycombs, molecular structures, crystalline lattices). The hexagonal N logo represents structured intelligence emerging from interconnected nodes.

Five principles govern every visual decision:

1. **Dark-first** — all surfaces default to dark mode; light is the adaptation
2. **Gradient as signature** — the Blue-to-Cyan gradient is a first-class brand asset, not decoration
3. **Motion with meaning** — every animation maps to a data metaphor (emerge, flow, pulse)
4. **Texture for depth** — grain, noise, and glassmorphism add analog warmth to digital surfaces
5. **Systematic precision** — every value is tokenized and shipped as code

## Logo

The Nebutra logo uses the letter **N** as its foundational form — constructed through positive and negative geometric space to create an implied hexagonal structure. The Blue-to-Cyan gradient fill gives it a forward-looking, technology-driven character.

### Logo Components

| Component | Package | How it loads | When to use |
|-----------|---------|-------------|-------------|
| `Logo` / `Logomark` / `Wordmark` | `@nebutra/brand` | `<img src="/brand/logo-*.svg">` | Web app with public folder (Next.js) |
| `LogomarkSVG` / `WordmarkEnSVG` / `LogoEnSVG` | `@nebutra/brand` | Inline SVG (`fill="currentColor"`) | No public folder; color via Tailwind `text-*`; email, PDF, Storybook |

```tsx
import { Logo, Logomark, Wordmark } from "@nebutra/brand"

// Hexagonal N icon
<Logomark variant="inverse" size={32} />

// Full logo with English wordmark
<Logo variant="en" size={150} inverted />
```

```tsx
import { LogomarkSVG, WordmarkEnSVG, LogoEnSVG } from "@nebutra/brand"

// Inline SVG — color controlled via CSS
<LogomarkSVG className="text-white w-8 h-8" />
<LogoEnSVG className="text-white" width={160} />
```

### Clearance and Minimum Size

- **Minimum clearance**: all sides >= 1/4 of logo height
- **Digital minimum height**: >= 35 px
- **Print minimum height**: >= 6 mm

<Warning>
  Do not distort, rotate, recolor, add drop shadows, or place on low-contrast backgrounds.
  Always use the provided brand gradient — never substitute with a solid color.
  See [Brand Guidelines](/design-system/foundations/brand-guidelines) for the complete list of prohibited uses.
</Warning>
