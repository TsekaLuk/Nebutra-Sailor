---
title: "InnerSideMenu"
description: "In-page secondary navigation for settings, docs, or multi-section layouts."
---

## Demo

```tsx
<InnerSideMenu
  items={[
    { id: "general", label: "General", href: "#general" },
    { id: "security", label: "Security", href: "#security" },
    { id: "members", label: "Members", href: "#members", badge: 3 },
    { id: "billing", label: "Billing", href: "#billing" },
    { id: "danger", label: "Danger zone", href: "#danger", variant: "danger" },
  ]}
  activeId="security"
/>
```

## Usage

```tsx
import { InnerSideMenu } from "@nebutra/custom-ui";

const SETTINGS_NAV = [
  { id: "profile", label: "Profile", href: "/settings/profile" },
  { id: "notifications", label: "Notifications", href: "/settings/notifications" },
  { id: "api-keys", label: "API keys", href: "/settings/api-keys" },
];

export function SettingsLayout({ children }: { children: React.ReactNode }) {
  const pathname = usePathname();
  const activeId = SETTINGS_NAV.find((item) => pathname === item.href)?.id;

  return (
    <div className="flex gap-8">
      <aside className="w-48 shrink-0">
        <InnerSideMenu items={SETTINGS_NAV} activeId={activeId} />
      </aside>
      <main className="flex-1">{children}</main>
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `items` | `SideMenuItem[]` | — | **Required.** Ordered list of navigation items |
| `activeId` | `string` | — | `id` of the currently active item |
| `onItemClick` | `(id: string) => void` | — | Optional click handler (in addition to native link navigation) |
| `className` | `string` | — | Additional classes on the `<nav>` root |

### `SideMenuItem`

```ts
interface SideMenuItem {
  id: string;
  label: string;
  href?: string;
  badge?: number;
  icon?: React.ReactNode;
  variant?: "default" | "danger";
  disabled?: boolean;
}
```

## Examples

### With icons

```tsx
import { Settings, Shield, Users, CreditCard } from "lucide-react";

<InnerSideMenu
  items={[
    { id: "general", label: "General", href: "#general", icon: <Settings className="w-4 h-4" /> },
    { id: "security", label: "Security", href: "#security", icon: <Shield className="w-4 h-4" /> },
    { id: "members", label: "Members", href: "#members", icon: <Users className="w-4 h-4" />, badge: 12 },
    { id: "billing", label: "Billing", href: "#billing", icon: <CreditCard className="w-4 h-4" /> },
  ]}
  activeId="general"
/>
```

### Scroll-spy mode

Pass `href` values as anchor links (`#section-id`) and omit `onItemClick`. Pair with `TableOfContents`
for automatic active state tracking on scroll.

```tsx
<InnerSideMenu
  items={scrollItems}
  activeId={activeSection}
/>
```

## Accessibility

- Renders as `<nav aria-label="Page sections">`.
- Active item has `aria-current="page"` (for page-level routes) or `aria-current="true"` (for anchors).
- Disabled items have `aria-disabled="true"` and are excluded from tab order.
- Badge counts are wrapped in `<span aria-label="{count} pending">` for screen reader context.
- Keyboard: `Tab` moves between items; `Enter` / `Space` activates.
