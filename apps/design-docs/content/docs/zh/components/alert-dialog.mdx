## 属性 (Props)

### AlertDialog (根组件)

| 属性 | 类型 | 默认值 | 描述 |
| -------------- | ------------------------- | ------- | -------------------------------- |
| `open` | `boolean` | — | 受控的打开状态 |
| `defaultOpen` | `boolean` | `false` | 非受控的默认打开状态 |
| `onOpenChange` | `(open: boolean) => void` | — | 打开状态改变时的回调函数 |

### AlertDialogContent

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ----------- | ------- | ---------------------- |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 对话框内容 |

### AlertDialogAction

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ------------ | ------- | ------------------------------ |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 确认操作按钮标签 |
| `onClick` | `() => void` | — | 确认操作的处理函数 |

### AlertDialogCancel

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ----------- | ------- | ---------------------- |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 取消按钮标签 |

## 可访问性 (Accessibility)

- 基于 Radix UI AlertDialog 构建 — 遵循 WAI-ARIA [警告和消息对话框模式 (Alert and Message Dialogs pattern)](https://www.w3.org/WAI/ARIA/apg/patterns/alertdialog/)。
- 渲染时带有 `role="alertdialog"` 和 `aria-modal="true"`。
- 打开时焦点将被锁定在对话框内。
- 与 `Dialog` 不同，`AlertDialog` 在点击遮罩层 (overlay) 时不会关闭 — 它需要用户执行显式操作。
