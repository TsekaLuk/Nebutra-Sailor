## 使用方法 (Usage)

```tsx
import {
  Alert,
  AlertIcon,
  AlertTitle,
  AlertDescription,
  AlertContent,
  AlertToolbar,
} from "@nebutra/ui/primitives";
```

## 属性 (Props)

### Alert

| 属性         | 类型                                                                                      | 默认值       | 描述                                     |
| ------------ | ----------------------------------------------------------------------------------------- | ------------- | ----------------------------------------------- |
| `variant`    | `"secondary" \| "primary" \| "destructive" \| "success" \| "info" \| "mono" \| "warning"` | `"secondary"` | 颜色和语义意图                       |
| `appearance` | `"solid" \| "outline" \| "light" \| "stroke"`                                             | `"solid"`     | 视觉样式处理                          |
| `size`       | `"sm" \| "md" \| "lg"`                                                                    | `"md"`        | 警告提示的尺寸                               |
| `icon`       | `"primary" \| "destructive" \| "success" \| "info" \| "warning"`                          | —             | 覆盖图标颜色 (需与 `variant="mono"` 一起使用) |
| `close`      | `boolean`                                                                                 | `false`       | 是否显示关闭按钮              |
| `onClose`    | `() => void`                                                                              | —             | 点击关闭按钮时的回调函数         |
| `className`  | `string`                                                                                  | —             | 额外的 CSS 类名                          |
| `children`   | `ReactNode`                                                                               | —             | 警告提示内容                                   |

### AlertIcon

| 属性        | 类型        | 默认值 | 描述                           |
| ----------- | ----------- | ------- | ------------------------------------- |
| `children`  | `ReactNode` | —       | 图标元素 (例如 lucide-react 图标) |
| `className` | `string`    | —       | 额外的 CSS 类名                |

### AlertTitle

| 属性        | 类型        | 默认值 | 描述            |
| ----------- | ----------- | ------- | ---------------------- |
| `children`  | `ReactNode` | —       | 标题文本内容     |
| `className` | `string`    | —       | 额外的 CSS 类名 |

### AlertDescription

| 属性        | 类型        | 默认值 | 描述              |
| ----------- | ----------- | ------- | ------------------------ |
| `children`  | `ReactNode` | —       | 描述文本内容 |
| `className` | `string`    | —       | 额外的 CSS 类名   |

### AlertContent

| 属性        | 类型        | 默认值 | 描述                                       |
| ----------- | ----------- | ------- | ------------------------------------------------- |
| `children`  | `ReactNode` | —       | 包裹 `AlertTitle` 和 `AlertDescription` 的容器 |
| `className` | `string`    | —       | 额外的 CSS 类名                            |

### AlertToolbar

| 属性        | 类型        | 默认值 | 描述             |
| ----------- | ----------- | ------- | ----------------------- |
| `children`  | `ReactNode` | —       | 操作按钮或链接 |
| `className` | `string`    | —       | 额外的 CSS 类名  |

## 示例 (Examples)

### 实色变体 (Solid Variants)

```tsx
<Alert variant="primary" appearance="solid">
  <AlertTitle>主警告提示 — 实色外观</AlertTitle>
</Alert>

<Alert variant="destructive" appearance="solid">
  <AlertTitle>破坏性警告提示 — 实色外观</AlertTitle>
</Alert>
```

### 可关闭的警告提示 (Dismissible Alert)

```tsx
const [visible, setVisible] = useState(true);

{
  visible && (
    <Alert
      variant="info"
      appearance="light"
      close
      onClose={() => setVisible(false)}
    >
      <AlertIcon>
        <Info />
      </AlertIcon>
      <AlertTitle>此警告提示可被关闭</AlertTitle>
    </Alert>
  );
}
```

### 带有工具栏 (With Toolbar)

```tsx
<Alert variant="warning" appearance="light">
  <AlertIcon>
    <TriangleAlert />
  </AlertIcon>
  <AlertContent>
    <AlertTitle>需要采取措施</AlertTitle>
    <AlertDescription>请查看您的账单信息。</AlertDescription>
  </AlertContent>
  <AlertToolbar>
    <Button size="sm" variant="outline">
      查看 (Review)
    </Button>
  </AlertToolbar>
</Alert>
```

### 尺寸变体 (Size Variants)

```tsx
<Alert variant="success" size="sm">
  <AlertTitle>小尺寸警告提示 (Small alert)</AlertTitle>
</Alert>

<Alert variant="success" size="md">
  <AlertTitle>中尺寸警告提示（默认）(Medium alert)</AlertTitle>
</Alert>

<Alert variant="success" size="lg">
  <AlertTitle>大尺寸警告提示 (Large alert)</AlertTitle>
</Alert>
```

## 可访问性 (Accessibility)

- 根元素使用 `role="alert"` 进行渲染 — 屏幕阅读器会立即朗读该内容。
- 关闭按钮包含 `aria-label="Dismiss"`，以便于屏幕阅读器进行清晰理解。
- 图标容器使用 `data-slot` 属性来进行针对性样式设计，而不会影响底层语义。
- 支持关闭按钮的键盘交互操作 (`Enter` 键, `Space` 空格键)。
