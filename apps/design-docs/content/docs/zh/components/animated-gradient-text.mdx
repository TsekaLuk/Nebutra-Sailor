## 概览 (Overview)

`AnimatedGradientText` 渲染一个 `<span>`，其背景为水平滚动的渐变效果，并裁剪应用于文本。渐变效果在 `colorFrom` 和 `colorTo` 之间持续循环，营造出闪烁的视觉效果。

**注意：** 该组件使用了 `animate-gradient` CSS 类，必须在您的全局样式中进行定义。导出的 `ANIMATED_GRADIENT_CSS` 常量包含了所需的 CSS 代码。

## 配置 (Setup)

将以下 CSS 添加到您的全局样式表或 Tailwind 配置中：

```css
@keyframes gradient {
  to {
    background-position: var(--bg-size, 300%) 0;
  }
}

.animate-gradient {
  animation: gradient 8s linear infinite;
}
```

或者导入常量：

```ts
import { ANIMATED_GRADIENT_CSS } from "@nebutra/ui/primitives";
// 将 ANIMATED_GRADIENT_CSS 注入到您的全局 <style> 标签中
```

## 使用方法 (Usage)

```tsx
import { AnimatedGradientText } from "@nebutra/ui/primitives";

// 基础用法
<AnimatedGradientText>动画文本 (Animated Text)</AnimatedGradientText>

// 自定义颜色
<AnimatedGradientText colorFrom="#ff0080" colorTo="#7928ca">
  粉色到紫色 (Pink to Purple)
</AnimatedGradientText>

// 更快的动画速度
<AnimatedGradientText speed={2}>快速渐变 (Fast Gradient)</AnimatedGradientText>

// 作为标题使用
<h1 className="text-4xl font-bold">
  <AnimatedGradientText>欢迎 (Welcome)</AnimatedGradientText>
</h1>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ----------- | ----------- | ------------------------------------------------------------------------------------- |
| `speed`     | `number`    | `1`         | 速度倍率。数值越高速度越快。控制 `--bg-size` CSS 变量 (`speed × 300%`) |
| `colorFrom` | `string`    | `"#ffaa40"` | 渐变起始/结束颜色 (CSS 颜色值) |
| `colorTo`   | `string`    | `"#9c40ff"` | 渐变中间颜色 (CSS 颜色值) |
| `className` | `string`    | —           | 应用于 `<span>` 的额外 CSS 类名 |
| `children`  | `ReactNode` | —           | 要显示渐变效果的文本内容 |

所有其他的 `<span>` 属性（如 `id`、`aria-*`、事件处理程序等）都会透传。

## 注意事项 (Notes)

- 渐变模式为 `colorFrom → colorTo → colorFrom`，从而产生无缝循环效果。
- `speed=0` 会有效地停止动画（渐变依然可见但保持静态）。
- 作为行内 (`inline`) 元素渲染 —— 请将其包裹在块级容器中以控制宽度。
- 需要 `animate-gradient` CSS 类在全局可用。
