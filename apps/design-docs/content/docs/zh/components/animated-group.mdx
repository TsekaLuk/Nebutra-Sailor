description: 交错动画容器，将每个子元素包裹在一个 motion 容器中，内置 10 种动画预设或支持完全自定义变体。
---

## 使用方法 (Usage)

```tsx
import { AnimatedGroup } from "@nebutra/ui";

<AnimatedGroup preset="blur-slide" className="flex flex-col gap-4">
  <Card>条目 1 (Item 1)</Card>
  <Card>条目 2 (Item 2)</Card>
  <Card>条目 3 (Item 3)</Card>
</AnimatedGroup>;
```

## 预设 (Presets)

| 预设 | 效果 |
| ------------ | ------------------------------------------------ |
| `fade`       | 透明度 0 → 1 |
| `slide`      | 透明度 + 从 y=20 向上滑动 |
| `scale`      | 透明度 + 缩放 0.8 → 1 |
| `blur`       | 透明度 + 模糊(4px) → 模糊(0) |
| `blur-slide` | 透明度 + 模糊 + 向上滑动 |
| `zoom`       | 透明度 + 缩放 0.5 → 1 (弹簧动画) |
| `flip`       | 透明度 + X轴旋转 -90° → 0° (弹簧动画) |
| `bounce`     | 透明度 + 从 y=-50 掉落 (弹簧动画) |
| `rotate`     | 透明度 + 旋转 -180° → 0° (弹簧动画) |
| `swing`      | 透明度 + 旋转 -10° → 0° (弹簧动画，低阻尼) |

## 示例 (Examples)

### 模糊滑动 - 用于 Hero 板块 (Blur Slide - Hero Sections)

```tsx
<AnimatedGroup preset="blur-slide" className="flex flex-col gap-6 text-center">
  <h1 className="text-5xl font-bold">加速构建 (Build faster)</h1>
  <p className="text-xl text-muted-foreground">
    为现代团队打造的平台。 (The platform for modern teams.)
  </p>
  <div className="flex gap-4 justify-center">
    <Button>开始体验 (Get started)</Button>
    <Button variant="outline">了解更多 (Learn more)</Button>
  </div>
</AnimatedGroup>
```

### 滑动卡片 - 用于功能网格 (Slide Cards - Feature Grid)

```tsx
<AnimatedGroup preset="slide" className="grid grid-cols-3 gap-4">
  {features.map((f) => (
    <FeatureCard key={f.id}>...</FeatureCard>
  ))}
</AnimatedGroup>
```

### 自定义变体 (Custom Variants)

```tsx
<AnimatedGroup
  variants={{
    container: {
      hidden: { opacity: 0 },
      visible: { opacity: 1, transition: { staggerChildren: 0.2 } },
    },
    item: {
      hidden: { opacity: 0, x: -30 },
      visible: { opacity: 1, x: 0 },
    },
  }}
  className="flex flex-row gap-4"
>
  <Card>左滑条目 1 (Left slide item 1)</Card>
  <Card>左滑条目 2 (Left slide item 2)</Card>
</AnimatedGroup>
```

### 无预设 - 默认淡入 (No Preset - Default Fade)

```tsx
<AnimatedGroup className="space-y-2">
  <p>段落 1 (Paragraph 1)</p>
  <p>段落 2 (Paragraph 2)</p>
  <p>段落 3 (Paragraph 3)</p>
</AnimatedGroup>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ------------------------------------------- | ------- | ---------------------------------------------------------- |
| `children`  | `ReactNode`                                 | —       | 要执行动画的条目（每个子元素都会被包裹在 `motion.div` 中） |
| `preset`    | `PresetType`                                | —       | 内置动画预设 |
| `variants`  | `{ container?: Variants; item?: Variants }` | —       | 自定义 Framer Motion 变体 |
| `className` | `string`                                    | —       | 外部 `motion.div` 容器的类名 |

## 注意事项 (Notes)

- 子元素默认以 `staggerChildren: 0.1` 的频率交错显示 —— 即每个子元素比前一个晚 100ms 开始动画。
- 自定义 `variants` 会覆盖预设。如果同时指定了 `preset` 和 `variants.item`，则 `variants.item` 优先级更高。
- 容器在挂载时立即执行 `initial="hidden"` → `animate="visible"`。若需滚动触发动画，请在父元素上结合 Framer Motion 的 `useInView` 使用 ref。
