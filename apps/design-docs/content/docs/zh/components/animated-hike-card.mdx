## 使用方法 (Usage)

```tsx
import { AnimatedHikeCard } from "@nebutra/ui";
import { Clock, Mountain } from "lucide-react";

<AnimatedHikeCard
  title="山地健行 (Mountain Trek)"
  images={["/img1.jpg", "/img2.jpg", "/img3.jpg"]}
  stats={[
    { icon: <Clock className="h-4 w-4" />, label: "约 6 小时" },
    { icon: <Mountain className="h-4 w-4" />, label: "8 公里" },
  ]}
  description="一条穿过高山草甸和岩石峰顶的美丽步道。"
  href="/hikes/mountain-trek"
/>;
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| ------------- | ---------- | ------- | ------------------------------------------------------ |
| `title`       | `string`   | —       | 卡片标题（大号粗体） |
| `images`      | `string[]` | —       | 图像 URL 数组 —— 建议使用 3 张以获得最佳展开效果 |
| `stats`       | `Stat[]`   | —       | `{ icon: ReactNode, label: string }` 标签数组 |
| `description` | `string`   | —       | 状态下方的描述文本 |
| `href`        | `string`   | —       | 链接地址（渲染为 `<a>`） |
| `className`   | `string`   | —       | 额外的 Tailwind 类名 |

## 注意事项 (Notes)

- 卡片渲染为 `<a>` 元素 —— `href` 是保证正确链接语义的必填项。
- **图像展开效果**：3 张图像通过绝对定位叠加。在悬停时，图像通过 CSS `group-hover` 变换向左右展开。
- **可访问性**：卡片的外部锚点具有 `aria-label="了解更多关于 {title} 的信息"`。
- `max-w-sm`（移动端）和 `max-w-md`（桌面端）限制了卡片宽度 —— 可通过 `className` 覆盖以适应不同尺寸。
