---
title: 动画列表 (AnimatedList)
description: 顺序列表动画，通过弹簧动画逐一展示子元素。条目从顶部累积，非常适合用于落地页上的通知流。
---

## 使用方法 (Usage)

```tsx
import { AnimatedList } from "@nebutra/ui";

<AnimatedList delay={1000}>
  <NotificationCard key="1" message="新用户已注册 (New user signed up)" />
  <NotificationCard key="2" message="收到付款 $49 (Payment received $49)" />
  <NotificationCard key="3" message="部署已完成 (Deploy completed)" />
</AnimatedList>;
```

<Callout>
  每个子元素**必须**具有稳定的 `key` 属性 —— `AnimatedList` 在内部使用它进行 `AnimatePresence` 键值管理。
</Callout>

## 示例 (Examples)

### 通知流 (Notification Feed)

```tsx
const notifications = [
  { id: "1", title: "新注册", body: "john@example.com 已加入", icon: "👤" },
  { id: "2", title: "付款", body: "$49/月 方案已激活", icon: "💳" },
  {
    id: "3",
    title: "部署",
    body: "v2.1.0 已部署至生产环境",
    icon: "🚀",
  },
  { id: "4", title: "告警", body: "CPU 使用率超过 80%", icon: "⚠️" },
];

<AnimatedList delay={800}>
  {notifications.map((n) => (
    <div
      key={n.id}
      className="flex items-center gap-3 rounded-xl border bg-background px-4 py-3 shadow-sm w-72"
    >
      <span className="text-2xl">{n.icon}</span>
      <div>
        <p className="text-sm font-medium">{n.title}</p>
        <p className="text-xs text-muted-foreground">{n.body}</p>
      </div>
    </div>
  ))}
</AnimatedList>;
```

### 快速展示 (500ms)

```tsx
<AnimatedList delay={500}>
  <Card key="a">第一 (First)</Card>
  <Card key="b">第二 (Second)</Card>
  <Card key="c">第三 (Third)</Card>
</AnimatedList>
```

### 慢速展示 (2s)

```tsx
<AnimatedList delay={2000}>
  <Step key="1">步骤 1：初始化 (Initialize)</Step>
  <Step key="2">步骤 2：配置 (Configure)</Step>
  <Step key="3">步骤 3：部署 (Deploy)</Step>
</AnimatedList>
```

## 属性 (Props)

### AnimatedList (动画列表)

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ----------- | ------- | ----------------------------------------- |
| `children`  | `ReactNode` | —       | 列表条目（每个都必须带有 `key` 属性） |
| `delay`     | `number`    | `1000`  | 每个条目出现之间的延迟时间（毫秒） |
| `className` | `string`    | —       | flex 容器的额外类名 |

### AnimatedListItem (列表动画项)

内部的条目动画包装器 —— 也可以单独使用：

```tsx
import { AnimatedListItem } from "@nebutra/ui";

<AnimatePresence>
  <AnimatedListItem key={item.id}>
    <MyCard {...item} />
  </AnimatedListItem>
</AnimatePresence>;
```

条目动画：`scale: 0, opacity: 0` → `scale: 1, opacity: 1`，使用 `spring(stiffness: 350, damping: 40)` 弹簧效果。

## 注意事项 (Notes)

- 条目以**反序累积**（最新的在最上方） —— 这符合通知流的用户体验逻辑，即最新的条目最为显眼。
- 在所有条目显示完毕后，动画会自动停止。除非动态添加条目，否则它**不会**循环播放。
- 对于落地页演示，列表通常需要循环播放 —— 您可以通过在父组件中根据时间间隔重置子元素来实现循环逻辑。
