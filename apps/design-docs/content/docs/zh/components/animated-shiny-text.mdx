## 使用方法 (Usage)

```tsx
import { AnimatedShinyText } from "@nebutra/ui/primitives";
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<AnimatedShinyText>✨ 介绍我们的新功能 (Introducing our new feature)</AnimatedShinyText>
```

### 在公告徽章中 (In an Announcement Badge)

```tsx
<div className="rounded-full border px-4 py-1.5 text-sm">
  <AnimatedShinyText>🎉 现已开启公开测试 (Now in public beta)</AnimatedShinyText>
</div>
```

### 自定义闪烁宽度 (Custom Shimmer Width)

```tsx
<AnimatedShinyText shimmerWidth={200} className="text-xl font-bold">
  高级功能 (Premium Feature)
</AnimatedShinyText>
```

### 更快的动画速度 (Faster Animation)

```tsx
<AnimatedShinyText duration={4} className="text-lg">
  超快闪烁 (Ultra Fast Shimmer)
</AnimatedShinyText>
```

## 属性 (Props)

`AnimatedShinyText` 继承自 `React.ComponentPropsWithoutRef<"span">`。

| 属性 | 类型 | 默认值 | 描述 |
| -------------- | ----------- | ------- | ---------------------------------------- |
| `shimmerWidth` | `number`    | `100`   | 闪烁高亮的宽度（像素） |
| `duration`     | `number`    | `8`     | 完整动画循环持续时间（秒） |
| `className`    | `string`    | —       | 额外的 CSS 类名 |
| `children`     | `ReactNode` | —       | 文本内容 |

## 可访问性 (Accessibility)

- 渲染为 `<span>` —— 在语义上保持中立。
- 闪烁效果纯粹是视觉上的；屏幕阅读器读取的文本内容保持不变。
- 避免在大段落中使用此效果；请将其限制在简短的短语或标签中。
