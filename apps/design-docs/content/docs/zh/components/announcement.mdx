title: 公告 (Announcement)
description: 胶囊形状的公告徽章复合组件，具有可选的标签 (tag) 和标题 (title) 区域。基于 Badge 组件构建。
---

## 使用方法 (Usage)

```tsx
import { Announcement, AnnouncementTag, AnnouncementTitle } from "@nebutra/ui";

<Announcement>
  <AnnouncementTag>新功能 (New)</AnnouncementTag>
  <AnnouncementTitle>
    介绍我们的最新功能
    <ArrowUpRightIcon size={14} />
  </AnnouncementTitle>
</Announcement>;
```

## 示例 (Examples)

### 基础公告 (Basic Announcement)

带有类别标签和标题的标准模式：

```tsx
<Announcement>
  <AnnouncementTag>测试版 (Beta)</AnnouncementTag>
  <AnnouncementTitle>多智能体工作流现已可用 (Multi-agent workflows are now available)</AnnouncementTitle>
</Announcement>
```

### 带有箭头图标 (With Arrow Icon)

```tsx
import { ArrowUpRight } from "lucide-react";

<Announcement>
  <AnnouncementTag>新功能 (New)</AnnouncementTag>
  <AnnouncementTitle>
    阅读发布说明 (Read the release notes)
    <ArrowUpRight size={14} />
  </AnnouncementTitle>
</Announcement>;
```

### 作为链接使用 (As a Link)

使用锚点标签 (`<a>`) 包裹，使整个徽章可点击：

```tsx
<a href="/blog/release-v2">
  <Announcement className="cursor-pointer">
    <AnnouncementTag>v2.0</AnnouncementTag>
    <AnnouncementTitle>
      查看新功能 (See what's new)
      <ArrowUpRight size={14} />
    </AnnouncementTitle>
  </Announcement>
</a>
```

### 主题化变体 (Themed Variants)

将 `themed` 属性与带有颜色的 `className` 结合使用，用于状态样式的公告：

```tsx
{
  /* 成功 (Success) */
}
<Announcement themed className="bg-green-100 text-green-700">
  <AnnouncementTag>成功 (Success)</AnnouncementTag>
  <AnnouncementTitle>部署已完成 (Deployment completed)</AnnouncementTitle>
</Announcement>;

{
  /* 错误 (Error) */
}
<Announcement themed className="bg-rose-100 text-rose-700">
  <AnnouncementTag>错误 (Error)</AnnouncementTag>
  <AnnouncementTitle>构建失败 —— 请检查日志 (Build failed — check logs)</AnnouncementTitle>
</Announcement>;

{
  /* 警告 (Warning) */
}
<Announcement themed className="bg-amber-100 text-amber-700">
  <AnnouncementTag>警告 (Warning)</AnnouncementTag>
  <AnnouncementTitle>使用了废弃的 API (Deprecated API in use)</AnnouncementTitle>
</Announcement>;
```

### 仅显示标题 - 无标签 (Title Only - No Tag)

```tsx
<Announcement>
  <AnnouncementTitle>
    对开源项目免费 —— 立即开始使用 (Free for open-source projects — get started today)
  </AnnouncementTitle>
</Announcement>
```

## 属性 (Props)

### Announcement (根组件)

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | -------------- | ----------- | ----------------------------------------------------------------------- |
| `themed`    | `boolean`      | `false`     | 应用主题化样式 —— 用于带有颜色的 `className` 状态徽章 |
| `variant`   | `BadgeVariant` | `"outline"` | 继承自 Badge 的变体样式 |
| `className` | `string`       | —           | 额外的 Tailwind 类名 |

### AnnouncementTag (公告标签)

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ----------------- | ------- | --------------------------------------- |
| `className` | `string`          | —       | 额外的 Tailwind 类名 |
| `children`  | `React.ReactNode` | —       | 标签文本（例如 "New"、"Beta"、"v2.0"） |

### AnnouncementTitle (公告标题)

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ----------------- | ------- | ---------------------------- |
| `className` | `string`          | —       | 额外的 Tailwind 类名 |
| `children`  | `React.ReactNode` | —       | 标题文本和可选图标 |

## 注意事项 (Notes)

- `AnnouncementTag` 和 `AnnouncementTitle` **必须**在 `Announcement` 内部使用 —— 它们通过 React Context 读取 `themed` 标志。在外部使用会抛出错误。
- `themed` 属性会将 `AnnouncementTag` 的背景更改为 `bg-background/60`（而不是 `bg-foreground/5`），这在有色背景上效果更好。
- 内置了悬停状态 (`hover:shadow-md`) —— 非常适合可点击的公告横幅。
