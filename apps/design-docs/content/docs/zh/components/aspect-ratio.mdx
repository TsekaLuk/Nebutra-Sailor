## 使用方法 (Usage)

```tsx
import { AspectRatio } from "@nebutra/ui/primitives";
```

## 属性 (Props)

| 属性        | 类型        | 默认值 | 描述                                      |
| ----------- | ----------- | ------- | ------------------------------------------------ |
| `ratio`     | `number`    | `1`     | 宽高比 (例如： `16/9`, `4/3`, `1`) |
| `className` | `string`    | —       | 额外的 CSS 类名                           |
| `children`  | `ReactNode` | —       | 需要在该比例下约束的内容            |

## 示例 (Examples)

### 常用比例 (Common Ratios)

```tsx
{
  /* 16:9 — 宽屏视频 */
}
<AspectRatio ratio={16 / 9}>
  <video src="/intro.mp4" className="h-full w-full object-cover" />
</AspectRatio>;

{
  /* 4:3 — 传统照片 */
}
<AspectRatio ratio={4 / 3}>
  <img src="/photo.jpg" className="h-full w-full object-cover" />
</AspectRatio>;

{
  /* 1:1 — 正方形 */
}
<AspectRatio ratio={1}>
  <img src="/avatar.jpg" className="h-full w-full rounded-full object-cover" />
</AspectRatio>;
```

### 配合 Iframe 嵌入 (With Iframe Embed)

```tsx
<AspectRatio ratio={16 / 9}>
  <iframe
    src="https://www.youtube.com/embed/dQw4w9WgXcQ"
    className="h-full w-full"
    allowFullScreen
    title="YouTube video"
  />
</AspectRatio>
```

## 可访问性 (Accessibility)

- 无固有 ARIA 角色 — 请为子元素应用合适的角色。
- 对于图片，请始终在 `<img>` 元素上包含 `alt` 文本。
- 对于视频或音频内容，请提供字幕和文本转录。
