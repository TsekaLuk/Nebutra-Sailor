## 概览 (Overview)

`AssistedPasswordConfirmation` 在用户输入确认密码时提供实时的视觉反馈：

- 与原始密码匹配的字符会以 **绿色** 突出显示
- 不匹配的字符会以 **红色** 突出显示
- 如果用户尝试输入超过密码长度的内容，输入框会 **抖动** (Framer Motion)
- 可选的提示信息会显示被遮盖的原始密码字符

## 使用方法 (Usage)

```tsx
import { AssistedPasswordConfirmation } from "@nebutra/ui/primitives";

const [password, setPassword] = useState("");
const [isMatched, setIsMatched] = useState(false);

// 在表单中
<div>
  <input
    type="password"
    value={password}
    onChange={(e) => setPassword(e.target.value)}
    placeholder="密码 (Password)"
  />
  <AssistedPasswordConfirmation
    password={password}
    onMatch={() => setIsMatched(true)}
    onConfirmChange={(value) => {
      if (value !== password) setIsMatched(false);
    }}
  />
</div>;
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
| ----------------- | ------------------------- | -------------------- | ---------------------------------------------------- |
| `password`        | `string`                  | —                    | **必填。** 要匹配的原始密码 |
| `placeholder`     | `string`                  | `"Confirm Password"` | 输入框占位符文本 |
| `onConfirmChange` | `(value: string) => void` | — | 确认输入内容变化时的回调 |
| `onMatch`         | `() => void`              | — | 密码完全匹配时的回调 |
| `showHint`        | `boolean`                 | `true`               | 是否显示掩码形式的密码字符指示器 |
| `className`       | `string`                  | — | 容器 CSS 类名 |

## 注意事项 (Notes)

- 该组件在内部跟踪确认文本 —— 它是 **非受控 (uncontrolled)** 组件。
- 一旦确认文本长度达到 `password` 的长度，输入将被锁定；进一步的按键将触发抖动动画。
- 结合使用 `onConfirmChange` 和 `onMatch` 来集成到表单状态管理中。
- 抖动动画使用了 Framer Motion 的 `motion.div` 配合 `animate` 变体实现。
