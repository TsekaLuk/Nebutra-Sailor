## 使用方法 (Usage)

```tsx
import { AuroraText } from "@nebutra/ui/primitives";
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<AuroraText>极光 (Aurora)</AuroraText>
```

### 作为 Hero 标题 (As a Hero Heading)

```tsx
<h1 className="text-6xl font-bold">
  构建 <AuroraText>精美 (beautiful)</AuroraText> 应用
</h1>
```

### 自定义颜色 (Custom Colors)

```tsx
<AuroraText colors={["#00ff87", "#60efff", "#0061ff"]}>
  海洋渐变 (Ocean Gradient)
</AuroraText>
```

### 更快速度 (Faster Speed)

```tsx
<AuroraText speed={3} className="text-5xl font-black">
  快速极光 (Fast Aurora)
</AuroraText>
```

## Tailwind 配置要求 (Tailwind Config Requirement)

`animate-aurora` 动画必须在您的 Tailwind 配置中进行注册：

```js
// tailwind.config.ts
export default {
  theme: {
    extend: {
      animation: {
        aurora: "aurora 8s ease-in-out infinite alternate",
      },
      keyframes: {
        aurora: {
          "0%": {
            backgroundPosition: "0% 50%",
            transform: "rotate(-5deg) scale(0.9)",
          },
          "50%": {
            backgroundPosition: "100% 50%",
            transform: "rotate(-3deg) scale(0.95)",
          },
          "100%": {
            backgroundPosition: "0% 50%",
            transform: "rotate(-5deg) scale(0.9)",
          },
        },
      },
    },
  },
};
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | ----------- | ------------------------------------------- | -------------------------------------------- |
| `children`  | `ReactNode` | — | 文本内容 |
| `colors`    | `string[]`  | `["#FF0080","#7928CA","#0070F3","#38bdf8"]` | 渐变颜色停止点 |
| `speed`     | `number`    | `1` | 动画速度倍数（数值越高速度越快） |
| `className` | `string`    | — | 额外的 CSS 类名 |

## 可访问性 (Accessibility)

- 包含一个使用 `.sr-only` 类名的可视化隐藏 `<span>` 标签，为屏幕阅读器提供完整文本。
- 可见的渐变 `<span>` 标签使用了 `aria-hidden="true"`。
