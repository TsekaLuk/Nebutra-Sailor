## 概览 (Overview)

`AvatarCircles` 渲染一行重叠的圆形头像图像，采用 `-space-x-4` 的负间距。在最后一个头像后会显示一个可选的超出计数器（例如 `+99`），该计数器可以链接至 `overflowUrl`。

常见用途：

- 团队成员预览
- GitHub 贡献者列表
- 社会认同展示（“加入 1,200+ 用户”）
- 评论区参与者展示

## 使用方法 (Usage)

```tsx
import { AvatarCircles } from "@nebutra/ui/primitives";

const avatars = [
  { imageUrl: "https://github.com/octocat.png", profileUrl: "https://github.com/octocat" },
  { imageUrl: "https://github.com/user2.png", profileUrl: "https://github.com/user2" },
  { imageUrl: "https://github.com/user3.png", profileUrl: "https://github.com/user3" },
];

// 基础用法
<AvatarCircles avatarUrls={avatars} />

// 带有超出数量
<AvatarCircles avatarUrls={avatars} numPeople={99} />

// 带有自定义尺寸
<AvatarCircles avatarUrls={avatars} numPeople={50} size={48} />

// 社会认同模式
<div className="flex items-center gap-4">
  <AvatarCircles avatarUrls={avatars} numPeople={1200} />
  <span className="text-sm text-muted-foreground">加入 1,200+ 名开发者</span>
</div>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| ------------- | -------------------- | ------- | -------------------------------------------- |
| `avatarUrls`  | `AvatarCircleItem[]` | — | **必填。** 头像配置数组 |
| `numPeople`   | `number`             | — | 在超出徽标中显示的数字 |
| `size`        | `number`             | `40` | 头像直径（像素） |
| `overflowUrl` | `string`             | `""` | 超出数量徽标的链接地址 |
| `className`   | `string`             | — | 容器的 CSS 类名 |

### `AvatarCircleItem`

| 属性 | 类型 | 描述 |
| ------------ | -------- | ----------------------------------------- |
| `imageUrl`   | `string` | 头像图像 URL |
| `profileUrl` | `string` | 点击头像时的链接地址 |
| `alt`        | `string` | 替代文本（默认为 "Avatar &#123;index&#125;"） |

## 注意事项 (Notes)

- 头像在新标签页中打开 (`target="_blank" rel="noopener noreferrer"`)。
- 悬停效果：使用 `scale-110` 和 `z-10` 使头像放大并置于最前。
- 超出徽标具备暗黑模式感知能力（亮色模式下为黑色，暗黑模式下为白色）。
- RTL 支持：在从右向左的布局中使用 `rtl:space-x-reverse`。
