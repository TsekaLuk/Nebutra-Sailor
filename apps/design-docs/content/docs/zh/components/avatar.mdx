## 回退机制 (Fallback)

当图片地址失效或未提供时，`AvatarFallback` 会在柔和背景上渲染首字母。

<ComponentPreview>
  <Avatar size="md">
    <AvatarFallback size="md">NN</AvatarFallback>
  </Avatar>

  ```tsx
<Avatar size="md">
    <AvatarFallback size="md">NN</AvatarFallback>
  </Avatar>
  ```
</ComponentPreview>

```tsx
<Avatar size="md">
  <AvatarFallback size="md">AC</AvatarFallback>
</Avatar>
```

<Tip>
  请务必为 `Avatar` 和 `AvatarFallback` 传入相同的 `size` — 这能确保字体大小与容器匹配。
</Tip>

## 头像组 (Group)

`AvatarGroup` 可以将头像堆叠显示，并带有自动生成的 `+N` 溢出指示器。下方的第二行示例包含 5 个成员，设置 `max={4}`，显示为 `+2`。

<ComponentPreview>
  <AvatarGroup
    items={[
      { alt: "user1", fallback: "U1" },
      { alt: "user2", fallback: "U2" },
      { alt: "user3", fallback: "U3" },
      { alt: "user4", fallback: "U4" },
      { alt: "user5", fallback: "U5" },
    ]}
    max={4}
    size="sm"
  />

  ```tsx
<AvatarGroup
    items={[
      { alt: "user1", fallback: "U1" },
      { alt: "user2", fallback: "U2" },
      { alt: "user3", fallback: "U3" },
      { alt: "user4", fallback: "U4" },
      { alt: "user5", fallback: "U5" },
    ]}
    max={4}
    size="sm"
  />
  ```
</ComponentPreview>

```tsx
<AvatarGroup
  items={[
    {
      src: "https://avatar.vercel.sh/evilrabbit",
      alt: "evilrabbit",
      fallback: "ER",
    },
    { src: "https://avatar.vercel.sh/leerob", alt: "leerob", fallback: "LR" },
    { src: "https://avatar.vercel.sh/rauchg", alt: "rauchg", fallback: "RG" },
    { src: "https://avatar.vercel.sh/shuding", alt: "shuding", fallback: "SH" },
    { src: "https://avatar.vercel.sh/almonk", alt: "almonk", fallback: "AL" },
  ]}
  max={4}
  size="sm"
/>
```

| 属性 | 类型 | 默认值 | 描述 |
| ------- | ------------------- | ------- | ------------------------------------- |
| `items` | `AvatarGroupItem[]` | — | `{ src?, alt, fallback }` 的数组 |
| `max` | `number` | `4` | 折叠前允许显示的最大头像数 |
| `size` | `AvatarSize` | `"sm"` | 应用于所有头像的尺寸 |

| 属性 | 类型 | 默认值 | 描述 |

  <DiceBearAvatar seed="my-bot-agent" avatarStyle="bottts" size="md" />

  ```tsx
    size="md"
  />
  <DiceBearAvatar seed="guest-session-123" avatarStyle="fun-emoji" size="md" />

  ```
</ComponentPreview>

```tsx
<DiceBearAvatar seed="rauchg" avatarStyle="bottts-neutral" size="md" />

<DiceBearAvatar
  seed="user@example.com"
  avatarStyle="pixel-art"
  options={{ radius: 50 }}
  size="md"
/>

<DiceBearAvatar seed="guest-session-123" avatarStyle="fun-emoji" size="md" />

<DiceBearAvatar seed="my-bot-agent" avatarStyle="bottts" size="md" />
```

| 属性 | 类型 | 默认值 | 描述 |
| ------------- | ---------------------------------- | ------------------ | ------------------------------------------- |
| `seed` | `string` | — | 确定性种子 (用户名、邮箱、UUID 等) |
| `avatarStyle` | `DiceBearStyle` | `"bottts-neutral"` | 头像集合样式 |
| `options` | `Record<string, string \| number>` | `{}` | 额外的 DiceBear 查询参数 |
| `size` | `AvatarSize` | `"md"` | 尺寸预设 |

<Tip>
  对系统/机器人实体使用 `bottts-neutral`；对人类占位符使用 `avataaars` 或 `lorelei`；对匿名/访客会话使用 `fun-emoji`。
</Tip>

## 属性参考 (Props Reference)

### Avatar

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | -------------------------------------- | ------- | ---------------------- |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | 尺寸预设 |
| `className` | `string` | — | 额外的 CSS 类名 |

### AvatarImage

| 属性 | 类型 | 描述 |
| ----- | -------- | --------------------------------- |
| `src` | `string` | 图片 URL |
| `alt` | `string` | **必填。** 屏幕阅读器文本 |

### AvatarFallback

| 属性 | 类型 | 默认值 | 描述 |
| --------- | ------------ | ------- | --------------------------------------------------- |
| `size` | `AvatarSize` | `"md"` | 需匹配父级 Avatar 的尺寸以确保字体正确 |
| `delayMs` | `number` | `600` | 显示前的延迟时间 — 防止在快速加载时闪烁 |

## 可访问性 (Accessibility)

- 始终为 `AvatarImage` 提供有意义的 `alt` 属性。
- 当头像触发操作时（例如打开用户菜单），请将其包裹在 `<button aria-label="...">` 中。
- `AvatarFallback` 的背景色 (`bg-muted`) 与 `text-muted-foreground` 的对比度符合 4.5:1 标准。
- `DiceBearAvatar` 的 SVG 是装饰性的 — `AvatarFallback` 提供了对应的文字替代方案。
