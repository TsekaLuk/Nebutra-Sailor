## 降级回退 (Fallback)

当图片 URL 加载失败或未提供时，`AvatarFallback` 会在柔和的背景上呈现用户的首字母。

<ComponentPreview name="avatar-fallback-demo">
  <AvatarFallbackDemo />

  ```tsx
<AvatarFallbackDemo />
  ```
</ComponentPreview>

<Tip>
  始终为 `Avatar` 和 `AvatarFallback` 传递相同的 `size` — 这能确保字体大小与容器正确匹配。
</Tip>

| `items` | `AvatarGroupItem[]` | —       | `{ src?, alt, fallback }` 的数组对象  |

## 生成头像 (DiceBear Generated)

通过 [DiceBear](https://www.dicebear.com/) 过程化生成的头像 — 对于相同的 `seed`，生成结果是确定性的。无需安装额外的 npm 包。

<ComponentPreview name="dice-bear-avatar-demo">
  <DiceBearAvatarDemo />

  ```tsx
<DiceBearAvatarDemo />
  ```
</ComponentPreview>

| 属性          | 类型                               | 默认值             | 描述                                    |
| ------------- | ---------------------------------- | ------------------ | --------------------------------------- |
| `seed`        | `string`                           | —                  | 确定性种子 (用户名，电子邮件，UUID 等) |
| `avatarStyle` | `DiceBearStyle`                    | `"bottts-neutral"` | 头像主题样式系列                        |
| `options`     | `Record<string, string \| number>` | `{}`               | 额外的 DiceBear URL 查询参数            |
| `size`        | `AvatarSize`                       | `"md"`             | 尺寸预设                                |

<Tip>
  对系统/机器人实体使用 `bottts-neutral`；对人类占位符使用 `avataaars` 或 `lorelei`；对匿名/访客会话使用 `fun-emoji`。
</Tip>

## 智能悬停头像组 (Smart Avatar Group)

`AvatarSmartGroup` 是一个动态的动画头像堆叠组件，由 `framer-motion` 驱动。它可以在悬停或展开组时优雅地显示隐藏的头像。

<ComponentPreview name="avatar-smart-group-demo">
  <AvatarSmartGroupDemo />

  ```tsx
<AvatarSmartGroupDemo />
  ```
</ComponentPreview>

| 属性                | 类型                                            | 默认值                 | 描述                                                                          |
| ------------------- | ----------------------------------------------- | ---------------------- | ----------------------------------------------------------------------------- |
| `avatars`           | `{ src: string; fallback: string; alt: string; }[]` | —                  | 包含图像来源和回退信息的头像对象数组                                            |
| `maxVisible`        | `number`                                        | `3`                    | 组件处于堆叠状态时可直接看到的最大头像数量                                    |
| `mode`              | `"expand" \| "stack"`                           | `"stack"`              | 决定布局行为：完全展开或堆叠重叠                                              |
| `size`              | `"sm" \| "md" \| "lg"`                          | `"md"`                 | 单个头像的尺寸预设                                                            |
| `onAvatarClick`     | `(avatar) => void`                              | `undefined`            | 可选事件处理程序，点击单个头像时触发                                          |

<Tip>
  将鼠标悬停在堆叠组件上，或者将 \`mode\` 属性更改为 \`expand\`，可以看到隐藏配置文件的平滑展开动画。
</Tip>

## 属性参考 (Props Reference)

### Avatar

| 属性 | 类型 | 默认值 | 描述 |
| ----------- | -------------------------------------- | ------- | ---------------------- |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | 尺寸预设 |
| `className` | `string` | — | 额外的 CSS 类名 |

### AvatarImage

| 属性 | 类型 | 描述 |
| ----- | -------- | --------------------------------- |
| `src` | `string` | 图片 URL |
| `alt` | `string` | **必填。** 屏幕阅读器文本 |

### AvatarFallback

| 属性 | 类型 | 默认值 | 描述 |
| --------- | ------------ | ------- | --------------------------------------------------- |
| `size` | `AvatarSize` | `"md"` | 需匹配父级 Avatar 的尺寸以确保字体正确 |
| `delayMs` | `number` | `600` | 显示前的延迟时间 — 防止在快速加载时闪烁 |

## 可访问性 (Accessibility)

- 始终为 `AvatarImage` 提供有意义的 `alt` 属性。
- 当头像触发操作时（例如打开用户菜单），请将其包裹在 `<button aria-label="...">` 中。
- `AvatarFallback` 的背景色 (`bg-muted`) 与 `text-muted-foreground` 的对比度符合 4.5:1 标准。
- `DiceBearAvatar` 的 SVG 是装饰性的 — `AvatarFallback` 提供了对应的文字替代方案。
