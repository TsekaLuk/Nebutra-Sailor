## 概览 (Overview)

`Awards` 是一个单一的多态组件，具有 6 种不同的视觉变体，用于展示成就、认证和认可。请根据您的 UI 环境选择最合适的样式。

## 安装 (Installation)

```tsx
import { Awards } from "@nebutra/ui/primitives";
```

## 变体 (Variants)

### `badge` (默认)

带有装饰性徽章 SVG 图标和文本内容的水平卡片。

```tsx
<Awards
  variant="badge"
  title="顶级贡献者 (Top Contributor)"
  subtitle="开源奖项 (Open Source Award)"
  recipient="Jane Smith"
  date="2026年3月"
/>
```

### `award`

带有月桂花环 SVG 和带有颜色的等级丝带的垂直布局。

```tsx
<Awards
  variant="award"
  title="卓越奖 (Excellence Award)"
  subtitle="表彰杰出表现"
  recipient="Jane Smith"
  date="2026年3月"
  level="gold"
/>
```

### `certificate`

带有书签图标和获奖者姓名行的正式证书设计。

```tsx
<Awards
  variant="certificate"
  title="达成荣誉 (Achievement)"
  subtitle="成功完成项目"
  recipient="Jane Smith"
  date="2026年3月"
/>
```

### `stamp`

具有锯齿边和弧形文本路径的圆形印章。

```tsx
<Awards
  variant="stamp"
  title="已认证 (CERTIFIED)"
  subtitle="专业级 (PROFESSIONAL)"
  recipient="Jane Smith"
  date="2026"
/>
```

### `sticker`

具有分层描边阴影以实现 3D 效果的大胆排版贴纸。

```tsx
<Awards variant="sticker" title="专业 (PRO)" />
```

### `id-card`

带有实时状态指示器和 Logo 区域的垂直 ID 卡。

```tsx
<Awards
  variant="id-card"
  title="Jane Smith"
  subtitle="资深工程师"
  description="工程部"
  date="ENG-2026"
/>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| ------------- | -------------------------------------------------------------------------- | --------- | --------------------------------------------- |
| `variant`     | `"stamp" \| "award" \| "certificate" \| "badge" \| "sticker" \| "id-card"` | `"badge"` | 视觉样式变体 |
| `title`       | `string`                                                                   | — | **必填。** 主标题文本 |
| `subtitle`    | `string`                                                                   | — | 次级文本（奖项名称、角色等） |
| `description` | `string`                                                                   | — | 额外描述或部门信息 |
| `date`        | `string`                                                                   | — | 日期或 ID 字符串 |
| `recipient`   | `string`                                                                   | — | 获奖者姓名 |
| `level`       | `"bronze" \| "silver" \| "gold" \| "platinum"`                             | `"gold"`  | 等级（用于 `award` 变体） |
| `showIcon`    | `boolean`                                                                  | `true` | 是否显示星形图标（用于 `stamp` 变体） |
| `customIcon`  | `ReactNode`                                                                | — | 用于替换默认图标的自定义图标 |
| `className`   | `string`                                                                   | — | 额外的 CSS 类名 |

## 等级颜色 (Level Colors)

| 等级 | 渐变 |
| ---------- | ----------------------- |
| `bronze`   | amber-600 → amber-800 |
| `silver`   | gray-400 → gray-600 |
| `gold`     | yellow-400 → yellow-600 |
| `platinum` | slate-300 → slate-500 |

## 注意事项 (Notes)

- `sticker` 变体使用 `WebkitTextStroke` 和分层 `<span>` 来实现 3D 边框效果，并适配亮/暗模式。
- `stamp` 变体通过动态计算的具有 40 个齿的 SVG 路径生成锯齿边缘。
- `id-card` 变体包含一个用于“实时”状态点的动画 Ping 指示器。
