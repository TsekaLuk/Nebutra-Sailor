## 尺寸 (Sizes)

通过 `size` 属性控制三种尺寸：

| `size` | 字体大小 | 内边距         | 使用场景                        |
| ------ | ---- | --------------- | ------------------------------- |
| `sm`   | 10px | `px-1.5 py-px`  | 紧凑的 UI, 表格单元格           |
| `md`   | 12px | `px-2.5 py-0.5` | 默认 — 大多数环境               |
| `lg`   | 14px | `px-3 py-1`     | 突出的标签, 头图区域            |

## 属性 (Props)

| 属性        | 类型                   | 默认值     | 描述                                       |
| ----------- | ---------------------- | ----------- | ------------------------------------------------- |
| `variant`   | 见下方              | `"default"` | 颜色 / 语义变体                          |
| `size`      | `"sm" \| "md" \| "lg"` | `"md"`      | 尺寸预设                                       |
| `icon`      | `ReactNode`            | —           | 渲染在标签前的图标，自动调整大小至 12×12px |
| `dot`       | `boolean`              | `false`     | 在标签前渲染一个 6px 的状态圆点             |
| `asChild`   | `boolean`              | `false`     | 通过 Radix Slot 渲染为子元素           |
| `className` | `string`               | —           | 额外的 CSS 类名                            |
| `children`  | `ReactNode`            | —           | 徽标标签文本                                       |

### 变体值 (Variant values)

**语义 (Semantic):** `default` `secondary` `destructive` `outline` `success` `warning` `info` `error`

**实色 (Color solid):** `gray` `blue` `purple` `amber` `red` `pink` `green` `teal` `inverted`

**柔和 (Color subtle):** `gray-subtle` `blue-subtle` `purple-subtle` `amber-subtle` `red-subtle` `pink-subtle` `green-subtle` `teal-subtle`

## 可访问性 (Accessibility)

- 默认情况下渲染为 `<div>` — 不可交互。
- 针对可点击的徽标，可以将 `asChild` 结合原生 `<a>` 或 Next.js 的 `<Link>` 一起使用。
- 针对动态状态更新，请在父容器添加 `aria-live="polite"` 属性。
- 当用作通知计数时，请在触发元素上配对使用 `aria-label`。
- `dot` 和 `icon` 插槽默认带有 `aria-hidden="true"` — 因为标签文本已提供了可访问的无障碍名称。
