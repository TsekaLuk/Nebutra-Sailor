```tsx
import { Book } from "@nebutra/ui";
```

---

## 默认用法 (Default)

具有书脊透视效果、琥珀色封面和标题的书籍。

```tsx
<Book title="前端云的用户体验 (The user experience of the Frontend Cloud)" />
```

---

## 变体 (Variants)

`"default"` 渲染具有可见书脊阴影的 3D 透视书籍。
`"simple"` 渲染一个扁平、无阴影的封面 —— 适用于网格或列表布局。

```tsx
<Book
  title="前端云的用户体验 (The user experience of the Frontend Cloud)"
  variant="simple"
  width={196}
/>

<Book
  title="前端云的用户体验 (The user experience of the Frontend Cloud)"
  variant="default"
  width={196}
/>
```

---

## 自定义颜色 (Custom Color)

可以向 `color` 传递任何 CSS 颜色字符串。设置 `textColor` 以获取对比色。

```tsx
<Book
  color="#9D2127"
  title="Nebutra 如何优化您的部署流水线 (How Nebutra improves your deployment pipeline)"
/>

<Book
  color="#7DC1C1"
  textColor="white"
  title="Nebutra 的设计工程 (Design Engineering at Nebutra)"
  variant="simple"
/>

<Book
  color="#0033FE"
  textColor="white"
  title="AI 驱动的基础设施指南 (AI-Powered Infrastructure Guide)"
/>
```

## 自定义插图 (Custom Illustration)

`illustration` 属性接受任何 ReactNode，并填充封面约上方的 55% 区域。

```tsx
// 彩色圆圈（社交证明 / 团队插图）
<Book
  title="前端云的用户体验 (The user experience of the Frontend Cloud)"
  illustration={
    <div className="flex items-end justify-center gap-3 h-full pb-4">
      <div className="h-14 w-14 rounded-full bg-blue-500" />
      <div className="h-10 w-10 rounded-full bg-red-500" />
      <div className="h-12 w-12 rounded-full bg-green-500" />
    </div>
  }
/>

// 全屏图像
<Book
  title="Nebutra 架构深度解析 (Nebutra Architecture Deep Dive)"
  color="#0f172a"
  textColor="white"
  illustration={
    <img
      src="/covers/architecture.png"
      className="h-full w-full object-cover"
      alt=""
    />
  }
/>
```

## 纹理效果 (Textured)

为任何颜色添加细粒/纸张纹理。在高饱和度颜色上效果最佳。

```tsx
<Book color="#7DC1C1" textured title="云计算模式 (Cloud Computing Patterns)" />
<Book color="#9D2127" textured title="安全工程 (Security Engineering)" />
<Book color="#FED954" textured title="前端性能 (Frontend Performance)" />

{/* 纹理 + 简单变体 (Textured + simple variant) */}
<Book
  color="#7DC1C1"
  textColor="white"
  textured
  variant="simple"
  title="设计工程 (Design Engineering)"
/>
```

## 响应式宽度 (Responsive Width)

向 `width` 传递 `{ sm, md }` 对象，以便使用 CSS `clamp()` 进行响应式缩放。

```tsx
<Book title="AI 驱动的搜索 (AI-Powered Search)" width={{ sm: 130, md: 196 }} />
```

---

## 宽度选项 (Width Options)

```tsx
<Book title="宽版书籍" width={300} />
<Book title="默认书籍" width={196} />
<Book title="紧凑书籍" width={140} />
```

---

## 带有 Logo 标记 (With Logo Mark)

`logo` 属性在封面的右下角渲染品牌标记。

```tsx
import { LogomarkSVG } from "@nebutra/brand";

<Book
  title="Nebutra 平台指南 (Nebutra Platform Guide)"
  color="#0033FE"
  textColor="white"
  logo={<LogomarkSVG width={16} height={16} />}
/>;
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
| -------------- | ------------------------ | ----------- | -------------------------------------- |
| `title`        | `string`                 | —           | **必填。** 封面标题文本 |
| `variant`      | `"default" \| "simple"`  | `"default"` | 3D 书脊 vs 扁平封面 |
| `color`        | `string`                 | `"#f59e0b"` | 封面背景颜色 (CSS 颜色) |
| `textColor`    | `string`                 | `"#000000"` | 标题和 Logo 的文本颜色 |
| `illustration` | `ReactNode`              | —           | 在封面上方 55% 区域渲染的 JSX |
| `width`        | `number \| { sm?, md? }` | `160`       | 宽度 (px)，或响应式断点 |
| `textured`     | `boolean`                | `false`     | 细粒/纸张纹理叠加层 |
| `logo`         | `ReactNode`              | —           | 右下角的品牌标记 |
| `className`    | `string`                 | — | 额外的 CSS 类名 |

## 设计说明 (Design Notes)

- **纵横比** 固定为 `1 : 1.45`（纵向）。高度根据 `width` 自动计算。
- **标题字体大小** 与 `width` 成比例缩放 —— 无需手动调整大小。
- **书脊效果**（仅限默认变体）使用 CSS `perspective(700px) rotateY(-10deg)` 并配合从左到右的渐变叠加，以模拟印刷书籍的书脊效果。
- **纹理** 使用 SVG `feTurbulence` 分形噪声滤镜，在 `200 × 200px` 的平铺块上渲染，并通过 `mix-blend-mode: overlay` 混合。

---

## 可访问性 (Accessibility)

- 组件渲染为一个非交互式的 `<div>`。
- 所有装饰性叠加层（书脊渐变、光泽、纹理）都带有 `aria-hidden`。
- 请务必提供有意义的 `title` 文本 —— 这是唯一的无障碍标签。
- 如果书籍是可点击的（例如链接到 PDF），请用 `<a>` 包裹并添加 `aria-label`。
