## 使用方法 (Usage)

```tsx
import { BorderTrail } from "@nebutra/ui/primitives";
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<div className="relative rounded-md border p-6">
  <BorderTrail />
  <p>此处放置内容 (Content here)</p>
</div>
```

### 卡片上的发光轨迹 (Glowing Trail on Card)

```tsx
<div className="relative rounded-xl border p-8">
  <BorderTrail
    className="bg-gradient-to-r from-blue-500 via-purple-500 to-pink-500"
    size={120}
  />
  <h2 className="text-xl font-bold">高级卡片 (Premium Card)</h2>
  <p>带有发光的边框轨迹 (With a glowing border trail)</p>
</div>
```

### 带有延迟的慢速动画 (Slower Animation with Delay)

```tsx
<div className="relative rounded-lg border p-4">
  <BorderTrail
    size={80}
    transition={{ duration: 8, ease: "linear", repeat: Infinity }}
    delay={2}
  />
  内容 (Content)
</div>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| --------------------- | --------------- | --------------------------------------------------- | -------------------------------------------------- |
| `size`                | `number`        | `60` | 轨迹元素的尺寸（像素） |
| `transition`          | `Transition`    | `{ repeat: Infinity, duration: 5, ease: "linear" }` | Framer Motion 转换配置 |
| `delay`               | `number`        | — | 动画开始前的延迟（秒） |
| `onAnimationComplete` | `() => void`    | — | 动画完成一个周期时的回调函数 |
| `style`               | `CSSProperties` | — | 轨迹元素的额外行内样式 |
| `className`           | `string`        | — | 轨迹的 CSS 类名（用于设置渐变颜色） |

## 配置要求 (Setup Requirements)

- 父元素必须具有 `position: relative`（或使用 `relative` 类名）。
- 父元素必须定义 `border-radius`，以便轨迹能够正确遵循边框路径。
- 该组件使用了 CSS `offset-path: rect(...)` —— 仅支持现代浏览器。

## 注意事项 (Notes)

- 轨迹元素默认使用 `mix-blend-mode: normal`。通过 `className` 添加 `mix-blend-mode: screen` 可以在深色背景上实现发光效果。
- 使用 `className` 自定义轨迹颜色 —— 例如 `bg-gradient-to-r from-blue-500 to-purple-500`。
