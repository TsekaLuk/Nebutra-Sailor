## 使用方法 (Usage)

```tsx
import { BubbleText } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<BubbleText className="text-5xl text-indigo-300">
  悬停在我身上！ (Hover over me!)
</BubbleText>
```

### 作为 Hero 标题 (As a Hero Heading)

```tsx
<BubbleText as="h1" className="text-6xl font-bold text-blue-400">
  交互式 Hero (Interactive Hero)
</BubbleText>
```

### 更宽的涟漪距离 (Wider Ripple Distance)

```tsx
<BubbleText rippleDistance={4} className="text-4xl text-purple-400">
  宽涟漪效果 (Wide Ripple Effect)
</BubbleText>
```

### 窄涟漪效果 (Narrow Ripple)

```tsx
<BubbleText rippleDistance={1} className="text-5xl text-emerald-400">
  紧凑气泡 (Tight Bubble)
</BubbleText>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `string` | — | 文本内容（必须是字符串） |
| `as` | `"h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6" \| "p" \| "span"` | `"h2"` | 要渲染的 HTML 元素 |
| `rippleDistance` | `number` | `2` | 悬停涟漪在每个方向上受影响的字符数 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 设计建议 (Design Notes)

涟漪效果应用了字重过渡：
- **悬停字符**: `font-black` + `brightness-150`
- **±1 距离**: `font-medium` + `brightness-125`
- **±2 距离**: `font-light`
- **超出范围**: 无变化

## 可访问性 (Accessibility)

- 将每个字符渲染为 `<span>` —— 语义中性。
- 仅支持鼠标交互；无键盘事件处理。请确保文本在无悬停交互的情况下仍清晰易读。
