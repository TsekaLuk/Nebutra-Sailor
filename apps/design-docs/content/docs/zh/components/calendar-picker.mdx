## 概览 (Overview)

`CalendarPicker` 是一款受 Geist 启发的日期范围选择器，它结合了 `RangeCalendar` 弹出层和可选的预设快捷方式。它支持多种布局模式：默认（并排）、紧凑、堆叠和水平。

## 使用方法 (Usage)

```tsx
import { CalendarPicker } from "@nebutra/ui/primitives";
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
const [range, setRange] = useState<RangeValue<DateValue>>();

<CalendarPicker value={range} onChange={setRange} />
```

### 带有预设 (With Presets)

```tsx
import { startOfDay, subDays, subWeeks, subMonths, endOfDay } from "date-fns";

const presets = {
  "last-3-days": {
    text: "最近 3 天 (Last 3 Days)",
    start: startOfDay(subDays(new Date(), 3)),
    end: endOfDay(new Date()),
  },
  "last-7-days": {
    text: "最近 7 天 (Last 7 Days)",
    start: startOfDay(subWeeks(new Date(), 1)),
    end: endOfDay(new Date()),
  },
  "last-month": {
    text: "最近一个月 (Last Month)",
    start: startOfDay(subMonths(new Date(), 1)),
    end: endOfDay(new Date()),
  },
};

<CalendarPicker
  value={range}
  onChange={setRange}
  presets={presets}
  allowClear
/>
```

### 紧凑布局 (Compact Layout)

预设以水平排布的方式显示在日历下方。

```tsx
<CalendarPicker presets={presets} compact />
```

### 堆叠布局 (Stacked Layout)

预设以垂直排布的方式显示在日历下方。

```tsx
<CalendarPicker presets={presets} stacked />
```

### 水平布局 (Horizontal Layout)

水平显示两个月份的日历。

```tsx
<CalendarPicker horizontalLayout />
```

### 小尺寸 (Small Size)

```tsx
<CalendarPicker size="small" presets={presets} />
```

### 默认选择预设 (Default Preset Selected)

```tsx
<CalendarPicker presets={presets} presetIndex={2} stacked />
```

### 带有最小/最大日期限制 (With Min/Max Dates)

```tsx
const now = new Date();
const minDate = new Date(now.getFullYear(), now.getMonth() - 2, now.getDate());
const maxDate = new Date(now.getFullYear(), now.getMonth() + 2, now.getDate());

<CalendarPicker minValue={minDate} maxValue={maxDate} allowClear />
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `RangeValue<DateValue>` | — | 受控的选择日期范围 |
| `onChange` | `(value: RangeValue<DateValue> \| undefined) => void` | — | 改变时的回调 |
| `presets` | `Record<string, CalendarPreset>` | — | 日期范围预设快捷方式 |
| `presetIndex` | `number` | — | 初始选择的预设（从 0 开始） |
| `size` | `"small" \| "default"` | `"default"` | 尺寸变体 |
| `allowClear` | `boolean` | `false` | 显示清除按钮 |
| `compact` | `boolean` | `false` | 紧凑布局 — 预设为水平行 |
| `stacked` | `boolean` | `false` | 堆叠布局 — 预设位于日历下方 |
| `horizontalLayout` | `boolean` | `false` | 显示两个月的日历 |
| `minValue` | `Date` | — | 最早可选日期 |
| `maxValue` | `Date` | — | 最晚可选日期 |
| `className` | `string` | — | 触发器的额外类名 |

### CalendarPreset (日历预设接口)

```tsx
interface CalendarPreset {
  text: string;   // 显示标签
  start: Date;    // 范围开始
  end: Date;      // 范围结束
}
```

## 布局模式 (Layout Modes)

| 模式 | 描述 |
|------|-------------|
| **default** | 日历在左，预设在右（并排） |
| **compact** | 预设作为水平按钮位于日历下方 |
| **stacked** | 日历在上，预设在下（垂直） |
| **horizontal** | 显示两个月日历，且没有预设侧面栏 |

## 可访问性 (Accessibility)

- 触发按钮显示当前选择或“选择时段 (Select period)”。
- 日历弹出窗口支持全键盘导航（方向键、Page Up/Down）。
- 清除按钮具有 `aria-label="清除日期选择 (Clear date selection)"` 属性。
- 预设按钮是可聚焦的并带有焦点环。
- 按下 `Escape` 键可关闭弹出层。
