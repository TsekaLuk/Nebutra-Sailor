## 概览 (Overview)

`Calendar` 和 `RangeCalendar` 是基于 HeroUI 和 `@internationalized/date` 构建的内联日历网格组件。它们支持 13 种国际日历系统、日期限制以及完整的键盘/ARIA 可访问性。

## 使用方法 (Usage)

```tsx
import { Calendar, RangeCalendar, today, getLocalTimeZone } from "@nebutra/ui/primitives";
```

## 属性 (Props)

### Calendar

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `DateValue` | — | 受控的选择日期 |
| `defaultValue` | `DateValue` | — | 非受控的默认值 |
| `onChange` | `(value: DateValue) => void` | — | 选择改变时的回调 |
| `minValue` | `DateValue` | — | 最早可选日期 |
| `maxValue` | `DateValue` | — | 最晚可选日期 |
| `isDateUnavailable` | `(date: DateValue) => boolean` | — | 判断日期是否不可选的断言函数 |
| `visibleMonths` | `number` | `1` | 显示的月份数量 |
| `showMonthAndYearPickers` | `boolean` | `false` | 启用月份/年份下拉选择器 |
| `isDisabled` | `boolean` | `false` | 禁用所有交互 |
| `isReadOnly` | `boolean` | `false` | 只读模式 |

### RangeCalendar

属性与 Calendar 相同，除以下各项外：

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `RangeValue<DateValue>` | — | 受控的选择日期范围 |
| `onChange` | `(value: RangeValue<DateValue>) => void` | — | 范围改变时的回调 |

## 可访问性 (Accessibility)

- 完整的键盘导航：使用方向键在日期之间移动，Page Up/Down 切换月份。
- 屏幕阅读器的 ARIA 标签和活动区域。
- 焦点管理和选择播报。
- 禁用/不可用的日期具有 `aria-disabled="true"` 属性。
