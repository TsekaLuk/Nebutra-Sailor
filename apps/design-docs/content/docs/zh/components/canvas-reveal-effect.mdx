## 依赖 (Dependencies)

需要安装 `three` 和 `@react-three/fiber`：

```bash
pnpm add three @react-three/fiber
```

## 使用方法 (Usage)

```tsx
import { CanvasRevealEffect } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<div className="relative h-[400px] w-full overflow-hidden rounded-xl">
  <CanvasRevealEffect />
</div>
```

### 青色点 (Cyan Dots)

```tsx
<CanvasRevealEffect
  animationSpeed={0.5}
  colors={[[0, 255, 255]]}
  dotSize={3}
/>
```

### 多色效果 (Multi-color)

```tsx
<CanvasRevealEffect
  animationSpeed={0.4}
  colors={[
    [59, 130, 246],   // 蓝色
    [139, 92, 246],   // 紫色
  ]}
  dotSize={4}
  showGradient
/>
```

### 悬停显示卡片 (On Hover Card)

```tsx
function RevealCard() {
  const [hovered, setHovered] = useState(false);
  return (
    <div
      className="relative h-[300px] w-[400px] rounded-xl border p-8"
      onMouseEnter={() => setHovered(true)}
      onMouseLeave={() => setHovered(false)}
    >
      {hovered && <CanvasRevealEffect animationSpeed={0.3} />}
      <p>悬停以显示 (Hover to reveal)</p>
    </div>
  );
}
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `animationSpeed` | `number` | `0.4` | 动画速度 (0.1 = 慢, 1.0 = 快) |
| `opacities` | `number[]` | `[0.3,0.3,0.3,0.5,0.5,0.5,0.8,0.8,0.8,1]` | 点的透明度分布 |
| `colors` | `number[][]` | `[[0,255,255]]` | 点的 RGB 颜色数组 |
| `containerClassName` | `string` | — | 容器的 CSS 类名 |
| `dotSize` | `number` | — | 以像素为单位的点尺寸 |
| `showGradient` | `boolean` | — | 覆盖径向渐变淡出效果 |

## 注意事项 (Notes)

- 由于 `@react-three/fiber` 与 React 19 的 JSX 不兼容，类型检查已被禁用 (`@ts-nocheck`)。该组件在运行时可正常工作。
- 请将组件包裹在具有 `overflow-hidden` 的容器中，以确保边缘整洁。
