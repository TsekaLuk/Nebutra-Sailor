---
title: 卡片聚光灯 (CardSpotlight)
description: 带有随鼠标光标移动的径向聚光灯的卡片，悬停时结合了 WebGL 画布显示效果。
---

## 使用方法 (Usage)

```tsx
import { CardSpotlight } from "@nebutra/ui";

<CardSpotlight className="h-96 w-96">
  <h2 className="relative z-20 text-xl font-bold text-white">
    您的功能 (Your Feature)
  </h2>
  <p className="relative z-20 text-neutral-300">
    此处放置功能描述。
  </p>
</CardSpotlight>
```

<Callout>
  悬停时的画布显示动画需要安装 `three` 和 `@react-three/fiber`。
</Callout>

## 示例 (Examples)

### 基础卡片 (Basic Card)

```tsx
<CardSpotlight className="h-80 w-80">
  <h2 className="relative z-20 mt-2 text-lg font-bold text-white">
    企业级安全 (Enterprise Security)
  </h2>
  <p className="relative z-20 mt-4 text-sm text-neutral-300">
    通过 SOC 2 Type II 认证。零信任架构。符合 GDPR 标准。
  </p>
</CardSpotlight>
```

### 自定义聚光灯颜色 (Custom Spotlight Color)

```tsx
<CardSpotlight
  color="#1a0533"
  radius={300}
  className="h-80 w-80"
>
  <h2 className="relative z-20 text-lg font-bold text-white">
    紫色聚光灯 (Purple spotlight)
  </h2>
</CardSpotlight>
```

### 清单内容 (Checklist Content)

```tsx
<CardSpotlight className="w-96 p-8">
  <h2 className="relative z-20 text-lg font-bold text-white mb-4">
    包含内容 (What's included)
  </h2>
  {["无限项目", "优先支持", "自定义域名", "API 访问"].map((item) => (
    <div key={item} className="relative z-20 flex items-center gap-2 mb-2">
      <CheckCircle2 className="h-4 w-4 text-green-400" />
      <span className="text-sm text-neutral-300">{item}</span>
    </div>
  ))}
</CardSpotlight>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 卡片内容 |
| `radius` | `number` | `350` | 以像素为单位的聚光灯半径 |
| `color` | `string` | `"#262626"` | 聚光灯背景颜色 |
| `className` | `string` | — | 额外的 Tailwind 类名 |

## 注意事项 (Notes)

- 内容必须使用 `relative z-20` 以显示在聚光灯遮罩层之上。
- 画布显示效果（蓝色/紫色点）在鼠标进入时激活。它使用 Three.js —— 如果尚未安装，请安装 `three` 和 `@react-three/fiber`。
- 默认背景为 `bg-black` —— 可通过 `className` 自定义。
- `radius` 控制聚光灯渐变从光标位置扩散的距离。
