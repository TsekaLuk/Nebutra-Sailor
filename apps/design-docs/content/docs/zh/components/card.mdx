### 所有变体 (All Variants)

```tsx
<Card variant="default" padding="md">...</Card>
<Card variant="elevated" padding="md">...</Card>   {/* 带有阴影 (box shadow) */}
<Card variant="bordered" padding="md">...</Card>   {/* 强边框，无阴影 */}
<Card variant="ghost" padding="md">...</Card>      {/* 无边框或阴影 */}
<Card variant="gradient" padding="md">...</Card>   {/* 品牌渐变边框 */}
```

### 内边距尺寸 (Padding Sizes)

```tsx
<Card variant="bordered" padding="sm">...</Card>  {/* 16px */}
<Card variant="bordered" padding="md">...</Card>  {/* 24px (默认) */}
<Card variant="bordered" padding="lg">...</Card>  {/* 32px */}
<Card variant="bordered" padding="none">...</Card> {/* 无内边距 */}
```

## 属性 (Props)

### Card

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `variant` | `"default" \| "elevated" \| "bordered" \| "ghost" \| "gradient"` | `"default"` | 视觉风格 |
| `padding` | `"none" \| "sm" \| "md" \| "lg"` | `"md"` | 内边距预设 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 通过子组件传入卡片内容 |

### 子组件 (Subcomponents)

| 组件 | 元素 | 描述 |
|-----------|---------|-------------|
| `Card.Header` | `div` | 顶部区域 — 通常包含 `Card.Icon` + `Card.Title` |
| `Card.Body` | `div` | 主体内容区域 — 包含 `Card.Description` 及其他内容 |
| `Card.Footer` | `div` | 底部区域 — 通常包含操作按钮 |
| `Card.Icon` | `div` | 图标包装组件，带有 `size` 属性 (`sm`, `md`, `lg`) |
| `Card.Title` | `h3` | 卡片标题 — 请维护文档的标题层级 |
| `Card.Description` | `p` | 使用 `text-muted-foreground` 样式的描述性文本 |

## 设计令牌 (Design Tokens)

内边距：`cardTokens.padding` — sm=16px, md=24px, lg=32px。
阴影等级：`cardTokens.shadow` — 映射到 `primitiveShadow.sm/md/lg`。

## 可访问性 (Accessibility)

- `Card.Title` 渲染为 `<h3>` — 在页面中请保持正确的标题层级。
- 对于交互式卡片（例如可点击的功能卡片），请使用 `<a>` 或 `<button>` 标签进行包裹。
- 当卡片代表离散的、独立的内容条目时，请使用 `role="article"`。
