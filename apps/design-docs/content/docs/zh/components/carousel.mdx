## 使用方法 (Usage)

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
} from "@nebutra/ui/primitives"
```

## 属性 (Props)

### Carousel (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 滚动方向 |
| `opts` | `EmblaOptionsType` | — | Embla Carousel 选项 |
| `plugins` | `EmblaPluginType[]` | — | Embla Carousel 插件 |
| `setApi` | `(api: CarouselApi) => void` | — | 获取 Embla carousel API 实例 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 轮播内容 |

### CarouselItem (轮播项)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名。使用 `basis-*` 来控制条目宽度 |
| `children` | `ReactNode` | — | 幻灯片内容 |

## 可访问性 (Accessibility)

- 导航按钮 (`CarouselPrevious`, `CarouselNext`) 具有描述性的 `aria-label` 属性。
- 轮播区域在上下文中使用时应具有 `aria-label` 或 `aria-labelledby`。
- 为自动播放的轮播提供键盘可访问的替代方案。
- 确保导航控件具有足够的色彩对比度。
