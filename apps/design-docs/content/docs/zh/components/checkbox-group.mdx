## 概述 (Overview)

`CheckboxGroup` 是一个简单的布局包装器，它将多个 `Checkbox` 项分组，并提供以下功能：

- 具有可访问性的 `role="group"` 和 `aria-label`
- 支持垂直或水平方向布局
- 可选的选择性可见标签

## 使用方法 (Usage)

```tsx
import { CheckboxGroup, Checkbox } from "@nebutra/ui/primitives";

// 垂直布局 (默认)
<CheckboxGroup label="选择功能 (Select features)">
  <Checkbox>分析 (Analytics)</Checkbox>
  <Checkbox>API 访问 (API Access)</Checkbox>
  <Checkbox>SSO</Checkbox>
</CheckboxGroup>

// 水平布局
<CheckboxGroup orientation="horizontal" label="选项 (Options)">
  <Checkbox>选项 A (Option A)</Checkbox>
  <Checkbox>选项 B (Option B)</Checkbox>
</CheckboxGroup>

// 带有禁用项
<CheckboxGroup label="方案功能 (Plan features)">
  <Checkbox defaultChecked>100GB 存储 (100GB Storage)</Checkbox>
  <Checkbox>无限用户 (Unlimited users)</Checkbox>
  <Checkbox disabled>SSO (仅限企业版 (Enterprise only))</Checkbox>
</CheckboxGroup>
```

## 属性 (Props)

### CheckboxGroup

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `label` | `string` | — | 可访问的分组标签（也会在视觉上渲染） |
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | 布局方向 |
| `children` | `ReactNode` | — | 复选框项 |
| `className` | `string` | — | 额外的 CSS 类名 |

### Checkbox

完整属性请参阅 [复选框 (Checkbox)](/docs/components/checkbox)。
