## 概览 (Overview)

`ChoiceboxGroup` 是一个提供卡片样式选择项的复合组件。每个条目都有标题、描述以及在选中时显示的可选自定义内容。支持单选（radio）和多选（checkbox）模式。

## 使用方法 (Usage)

```tsx
import { ChoiceboxGroup } from "@nebutra/ui/primitives";
```

## 示例 (Examples)

### 单选模式 (Radio Mode - Single Select)

```tsx
const [value, setValue] = useState("trial");

<ChoiceboxGroup
  direction="row"
  label="选择方案 (Select a plan)"
  type="radio"
  value={value}
  onChange={(v) => setValue(v as string)}
>
  <ChoiceboxGroup.Item
    title="专业试用版 (Pro Trial)"
    description="免费试用两周"
    value="trial"
  />
  <ChoiceboxGroup.Item
    title="专业版 (Pro)"
    description="现在就开始"
    value="pro"
  />
</ChoiceboxGroup>
```

### 多选模式 (Checkbox Mode - Multi Select)

```tsx
const [value, setValue] = useState<string[]>([]);

<ChoiceboxGroup
  direction="row"
  label="选择方案 (Select plans)"
  type="checkbox"
  value={value}
  onChange={(v) => setValue(v as string[])}
>
  <ChoiceboxGroup.Item title="专业试用版 (Pro Trial)" description="免费试用两周" value="trial" />
  <ChoiceboxGroup.Item title="专业版 (Pro)" description="现在就开始" value="pro" />
</ChoiceboxGroup>
```

### 禁用状态 (Disabled)

```tsx
{/* 组级禁用 */}
<ChoiceboxGroup disabled type="radio" label="..." ...>
  <ChoiceboxGroup.Item title="..." value="..." />
</ChoiceboxGroup>

{/* 条目级禁用 */}
<ChoiceboxGroup type="checkbox" label="..." ...>
  <ChoiceboxGroup.Item disabled title="专业试用版 (Pro Trial)" value="trial" />
  <ChoiceboxGroup.Item title="专业版 (Pro)" value="pro" />
</ChoiceboxGroup>
```

### 自定义内容 (Custom Content)

当条目被选中时，自定义内容会渲染在标题/描述下方。

```tsx
<ChoiceboxGroup.Item title="专业试用版 (Pro Trial)" description="免费试用两周" value="trial">
  <div className="flex justify-center p-2">
    <Badge variant="blue">剩余 14 天 (14 days remaining)</Badge>
  </div>
</ChoiceboxGroup.Item>
```

### 纵向布局 (Column Layout)

```tsx
<ChoiceboxGroup direction="column" type="radio" label="选择等级 (Choose a tier)" ...>
  <ChoiceboxGroup.Item title="基础版 (Starter)" description="适用于个人项目" value="starter" />
  <ChoiceboxGroup.Item title="团队版 (Team)" description="适用于小型团队" value="team" />
</ChoiceboxGroup>
```

## 属性 (Props)

### ChoiceboxGroup

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `type` | `"radio" \| "checkbox"` | — | **必填。** 选择模式 |
| `direction` | `"row" \| "column"` | `"row"` | 布局方向 |
| `label` | `string` | — | **必填。** 可访问的组标签 |
| `showLabel` | `boolean` | `false` | 是否视觉上显示标签 |
| `value` | `string \| string[]` | — | 受控的值 |
| `defaultValue` | `string \| string[]` | — | 非受控的默认值 |
| `onChange` | `(value: string \| string[]) => void` | — | 选择改变时的回调 |
| `disabled` | `boolean` | `false` | 禁用所有条目 |
| `className` | `string` | — | 额外的 CSS 类名 |

### ChoiceboxGroup.Item

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `title` | `string` | — | **必填。** 主要标签 |
| `description` | `string` | — | 次要文本 |
| `value` | `string` | — | **必填。** 条目的值 |
| `disabled` | `boolean` | `false` | 禁用此条目 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 选中时显示的自定义内容 |

## 可访问性 (Accessibility)

- 单选模式：组上带有 `role="radiogroup"`，条目上带有 `role="radio"`。
- 多选模式：组上带有 `role="group"`，条目上带有 `role="checkbox"`。
- `aria-checked` 反映选择状态。
- 禁用的条目带有 `aria-disabled` 属性。
- 键盘操作：方向键在单选条目间导航（ roving tabindex ）；Tab + 空格/回车键操作多选条目。
- 焦点环：2px 品牌色及 2px 偏移。
