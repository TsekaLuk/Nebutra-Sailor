## 属性 (Props)

### Collapsible (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `open` | `boolean` | — | 受控的打开状态 |
| `defaultOpen` | `boolean` | `false` | 非受控的默认打开状态 |
| `onOpenChange` | `(open: boolean) => void` | — | 打开状态改变时的回调函数 |
| `disabled` | `boolean` | `false` | 防止切换状态 |
| `className` | `string` | — | 额外的 CSS 类名 |

### CollapsibleTrigger (触发器)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | 将属性合并到直接子元素上 |
| `children` | `ReactNode` | — | 触发元素 |

### CollapsibleContent (内容区)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 折叠主体内容 |

## 可访问性 (Accessibility)

- 基于 Radix UI Collapsible 构建。
- `CollapsibleTrigger` 具有 `aria-expanded` 属性，用于通报打开/关闭状态。
- `CollapsibleContent` 在折叠时具有 `aria-hidden="true"` 属性。
- 请为 `CollapsibleTrigger` 使用描述性文本或 `aria-label`，以便屏幕阅读器清晰识别。
