## 可访问性 (Accessibility)

- 基于 Radix UI Collapsible 构建。
- `CollapsibleTrigger` 具有 `aria-expanded` 属性，用于通报打开/关闭状态。
- `CollapsibleContent` 在折叠时具有 `aria-hidden="true"` 属性。
- 请为 `CollapsibleTrigger` 使用描述性文本或 `aria-label`，以便屏幕阅读器清晰识别。
