## 概览 (Overview)

`ColorBadge` 是一个多功能的徽标组件，具有纯色和淡色两种颜色变体。它支持前置图标，并可以渲染为 Next.js 链接。包含用于试用/促销场景的渐变变体。

## 安装 (Installation)

```tsx
import { ColorBadge } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础用法 (Basic)

```tsx
<ColorBadge variant="blue">新建 (New)</ColorBadge>
<ColorBadge variant="green-subtle" size="lg">成功 (Success)</ColorBadge>
```

### 带有图标 (With icon)

```tsx
import { Shield } from "lucide-react";

<ColorBadge variant="purple" icon={<Shield />}>
  受保护 (Protected)
</ColorBadge>
```

### 作为链接 (As a link)

```tsx
<ColorBadge variant="blue" asLink href="/docs">
  查看文档 (View Docs)
</ColorBadge>
```

### 状态徽标 (Status badges)

```tsx
<ColorBadge variant="green">活跃 (Active)</ColorBadge>
<ColorBadge variant="amber">待处理 (Pending)</ColorBadge>
<ColorBadge variant="red">错误 (Error)</ColorBadge>
```

### 渐变徽标 (Gradient badges)

```tsx
<ColorBadge variant="trial">试用 (Trial)</ColorBadge>
<ColorBadge variant="turbo">加速 (Turbo)</ColorBadge>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `variant` | `ColorBadgeVariant` | `"gray"` | 颜色样式变体 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 徽标尺寸 |
| `capitalize` | `boolean` | `true` | 是否自动将文本首字母大写 |
| `icon` | `ReactNode` | — | 在文本前显示的可选图标 |
| `asLink` | `boolean` | `false` | 是否渲染为 Next.js 链接 |
| `href` | `string` | — | 链接变体的 URL |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 徽标文本内容 |

## 变体 (Variants)

| 变体 (Variant) | 样式 (Style) |
|---------|-------|
| `gray` | 纯灰色 |
| `gray-subtle` | 浅灰色 |
| `blue` | 纯蓝色 |
| `blue-subtle` | 浅蓝色 |
| `purple` | 纯紫色 |
| `purple-subtle` | 浅紫色 |
| `amber` | 纯琥珀色 |
| `amber-subtle` | 浅琥珀色 |
| `red` | 纯红色 |
| `red-subtle` | 浅红色 |
| `pink` | 纯粉色 |
| `pink-subtle` | 浅粉色 |
| `green` | 纯绿色 |
| `green-subtle` | 浅绿色 |
| `teal` | 纯青色 |
| `teal-subtle` | 浅青色 |
| `inverted` | 深/浅自适应 |
| `trial` | 蓝色 → 粉色渐变 |
| `turbo` | 玫瑰色 → 蓝色渐变 |
| `pill` | 药丸边框样式 |
