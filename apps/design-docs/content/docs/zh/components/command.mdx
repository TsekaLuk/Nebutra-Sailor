## 属性 (Props)

### Command (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `onSelect` | `(value: string) => void` | — | 当一项被选中时的回调函数 |
| `shouldFilter` | `boolean` | `true` | 是否根据搜索输入过滤项目 |
| `filter` | `(value: string, search: string) => number` | — | 自定义的过滤函数 |
| `children` | `ReactNode` | — | 命令菜单内容 |

### CommandInput

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `placeholder` | `string` | — | 占位符文本 |
| `value` | `string` | — | 受控的搜索值 |
| `onValueChange` | `(value: string) => void` | — | 输入内容改变时的回调函数 |
| `className` | `string` | — | 额外的 CSS 类名 |

### CommandItem

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | 用于过滤和选中的值 |
| `disabled` | `boolean` | `false` | 防止被选中 |
| `onSelect` | `(value: string) => void` | — | 项被选中时的回调函数 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 项内容 |

## 可访问性 (Accessibility)

- 基于 cmdk 构建 — 键盘优先的设计。
- `CommandInput` 具有 `role="combobox"` 、 `aria-expanded` 和 `aria-haspopup="listbox"` 属性。
- `CommandList` 具有 `role="listbox"` 属性。
- 子项具有 `role="option"` 和 `aria-selected` 属性。
- 键盘导航：键入进行搜索，上箭头/下箭头 (`ArrowUp`/`ArrowDown`) 进行导航，`Enter` 键进行选择。
