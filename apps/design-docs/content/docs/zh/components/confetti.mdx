## 概览 (Overview)

`Confetti` 将 `canvas-confetti` 库封装在 React 组件中，支持：

- **挂载时自动播放**（默认）—— 组件渲染时立即发射
- **通过 ref 手动控制** —— 命令式调用 `ref.current.fire(options)`
- **五彩纸屑按钮 (ConfettiButton)** —— 便捷的 `<Button>`，点击时从自身位置发射纸屑
- 支持 Web Worker 进行离线渲染
- 画布自动调整尺寸

典型使用场景：购买成功、解锁成就、表单提交、里程碑庆祝。

## 使用方法 (Usage)

```tsx
import { Confetti, ConfettiButton, type ConfettiRef } from "@nebutra/ui/primitives";
import { useRef } from "react";

// 挂载时自动播放 (覆盖整个视口)
<Confetti className="fixed inset-0 pointer-events-none" />

// 通过 ref 手动控制
const confettiRef = useRef<ConfettiRef>(null);

<Confetti ref={confettiRef} manualstart className="fixed inset-0 pointer-events-none" />
<button onClick={() => confettiRef.current?.fire({ particleCount: 100, spread: 80 })}>
  庆祝！ (Celebrate!)
</button>

// 便捷按钮 (从按钮位置发射)
<ConfettiButton variant="default">
  🎉 庆祝一下 (Celebrate)
</ConfettiButton>

// 自定义颜色
confettiRef.current?.fire({
  particleCount: 150,
  spread: 100,
  colors: ["#0033FE", "#00e5ff", "#ffffff"],
  origin: { y: 0.8 },
});
```

## Confetti 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `options` | `ConfettiOptions` | — | 默认五彩纸屑动画选项（会被 `fire()` 调用时的选项覆盖） |
| `globalOptions` | `ConfettiGlobalOptions` | `{ resize: true, useWorker: true }` | Canvas-confetti 全局选项 |
| `manualstart` | `boolean` | `false` | 为 `true` 时，禁止挂载时自动播放 |
| `children` | `ReactNode` | — | 子组件可以通过 `ConfettiContext` 访问 `fire` 方法 |
| `className` | `string` | — | `<canvas>` 元素的 CSS 类名 |

所有其他 `<canvas>` 属性都会被转发。

## ConfettiButton 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `options` | `ConfettiOptions & ConfettiGlobalOptions` | — | 五彩纸屑选项。`origin` 会自动设置为按钮的中心位置 |

所有 `Button` 属性（`variant`、`size`、`disabled`、`onClick` 等）都会被转发。

## ConfettiRef

```ts
type ConfettiRef = {
  fire: (options?: ConfettiOptions) => Promise<void>;
} | null;
```

配合 `useRef<ConfettiRef>(null)` 使用，并将其传递给 `<Confetti ref={confettiRef} manualstart />`。

## 注意事项 (Notes)

- 画布默认应用了 `pointer-events-none`，以避免阻塞点击操作。
- `ConfettiButton` 通过按钮的 `getBoundingClientRect()` 计算发射起点 `origin` —— 无需额外配置。
- 当省略 `manualstart`（或设为 `false`）时，五彩纸屑在挂载时会自动发射一次。
- 传递给 `fire()` 的 `options` 会与组件级的 `options` 属性合并（并覆盖后者）。
- 需要安装 `canvas-confetti`：`pnpm add canvas-confetti`
