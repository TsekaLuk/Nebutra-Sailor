## 概览 (Overview)

`ContextCard` 是一款基于 Radix Tooltip 构建的悬停激活卡片。与普通提示框不同，它支持任意 `ReactNode` 内容 —— 文本、富标记、图像或链接。它会根据请求侧自动定位并带有动画。

## 安装 (Installation)

```tsx
import { ContextCard } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 默认情况（所有侧面）

```tsx
import { ContextCard } from "@nebutra/ui/primitives"

<div className="flex flex-row items-center justify-around gap-8">
  <ContextCard.Trigger
    content="The Evil Rabbit Jumped over the Fence"
    side="top"
  >
    <span>顶部 (Top)</span>
  </ContextCard.Trigger>

  <ContextCard.Trigger
    content="The Evil Rabbit Jumped over the Fence"
    side="bottom"
  >
    <span>底部 (Bottom)</span>
  </ContextCard.Trigger>

  <ContextCard.Trigger
    content="The Evil Rabbit Jumped over the Fence"
    side="left"
  >
    <span>左侧 (Left)</span>
  </ContextCard.Trigger>

  <ContextCard.Trigger
    content="The Evil Rabbit Jumped over the Fence"
    side="right"
  >
    <span>右侧 (Right)</span>
  </ContextCard.Trigger>
</div>
```

### 富内容 (Rich content)

```tsx
<ContextCard.Trigger
  side="top"
  content={
    <div className="flex flex-col gap-1">
      <p className="font-medium">Evil Rabbit</p>
      <p className="text-muted-foreground text-xs">Jumped over the Fence</p>
    </div>
  }
>
  <span>悬停查看详情 (Hover for details)</span>
</ContextCard.Trigger>
```

### 应用在按钮上 (On a button)

```tsx
<ContextCard.Trigger content="点击提交表单 (Click to submit your form)" side="bottom">
  <Button>提交 (Submit)</Button>
</ContextCard.Trigger>
```

## 属性 (Props)

### ContextCard.Trigger

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `content` | `ReactNode` | — | **必填。** 在卡片内显示的内容 |
| `side` | `"top" \| "bottom" \| "left" \| "right"` | `"top"` | 卡片出现的侧面 |
| `sideOffset` | `number` | `8` | 距离触发器的像素距离 |
| `children` | `ReactNode` | — | **必填。** 触发卡片的元素 |
| `className` | `string` | — | 卡片的额外 CSS 类名 |

## 可访问性 (Accessibility)

- 基于 Radix Tooltip 构建，具有 200ms 的悬停延迟。
- 键盘可达：聚焦触发器即可显示卡片。
- 屏幕阅读器通过 `role="tooltip"` 播报内容。
- 键盘 `Escape` 键可关闭卡片。
