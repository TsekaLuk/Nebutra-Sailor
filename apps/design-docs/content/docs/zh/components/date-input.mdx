---

## 概览 (Overview)

`DateInput` 重新导出了 `@heroui/date-input` 作为直接替代方案。每个日期部分（年、月、日）都是一个可独立编辑的段。由 `@internationalized/date` 提供驱动。

功能特性：
- 可通过键盘编辑的段（使用 Tab 键在年/月/日/时/分/秒之间切换）
- 多种变体：`flat`、`bordered`、`faded`、`underlined`
- 标签位置：`inside`、`outside`、`outside-left`
- 支持 `ZonedDateTime` 的时区功能
- 支持国际历法（希伯来历、印度历、伊斯兰历、佛历等）
- 最小/最大日期校验
- 粒度控制（`day`、`hour`、`minute`、`second`）

## 使用方法 (Usage)

```tsx
import { DateInput } from "@nebutra/ui/primitives";
import { CalendarDate } from "@internationalized/date";

// 基础用法
<DateInput label="出生日期" />

// 受控模式
const [date, setDate] = useState(new CalendarDate(2024, 4, 4));

<DateInput
  label="活动日期"
  value={date}
  onChange={setDate}
/>

// 带有时区
import { parseZonedDateTime } from "@internationalized/date";

<DateInput
  label="会议时间"
  defaultValue={parseZonedDateTime("2025-03-01T09:00[America/New_York]")}
/>

// 校验
<DateInput
  label="结束日期"
  isRequired
  isInvalid={!date}
  errorMessage="日期是必填项"
/>
```

## 关键属性 (Key Props)

| 属性 | 类型 | 描述 |
|------|------|-------------|
| `label` | `string` | 输入框标签 |
| `value` | `CalendarDate \| ZonedDateTime` | 受控日期值 |
| `defaultValue` | `CalendarDate \| ZonedDateTime` | 非受控初始值 |
| `onChange` | `(date) => void` | 变更回调函数 |
| `variant` | `"flat" \| "bordered" \| "faded" \| "underlined"` | 视觉变体 |
| `granularity` | `"day" \| "hour" \| "minute" \| "second"` | 段的详细级别 |
| `minValue` | `CalendarDate` | 可选的最小日期 |
| `maxValue` | `CalendarDate` | 可选的最大日期 |
| `isRequired` | `boolean` | 标记为必填项 |
| `isInvalid` | `boolean` | 显示错误状态 |
| `isDisabled` | `boolean` | 禁用输入 |
| `errorMessage` | `string` | 校验错误文本 |

## 注意事项 (Notes)

- 需要安装 `@internationalized/date` 以使用 `CalendarDate`/`ZonedDateTime` 类型。
- 支持 HeroUI `DateInput` 的所有属性 —— 这是一个直接的重新导出。
- 有关完整的属性参考，请参阅 [HeroUI DateInput 文档](https://heroui.com/docs/components/date-input)。
