## 概览 (Overview)

`DatePicker` 和 `DateRangePicker` 是基于弹出层 (Popover) 的日期选择组件，由 HeroUI 提供支持。它们将键盘可编辑的日期输入框与日历下拉菜单结合在一起。

## 使用方法 (Usage)

```tsx
import { DatePicker, DateRangePicker } from "@nebutra/ui/primitives";
```

## 属性 (Props)

### DatePicker

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `label` | `string` | — | 输入框标签 |
| `value` | `DateValue` | — | 受控值 |
| `defaultValue` | `DateValue` | — | 非受控默认值 |
| `onChange` | `(value: DateValue) => void` | — | 变更回调函数 |
| `variant` | `"flat" \| "bordered" \| "faded" \| "underlined"` | `"flat"` | 视觉样式 |
| `minValue` | `DateValue` | — | 可选的最早日期 |
| `maxValue` | `DateValue` | — | 可选的最晚日期 |
| `granularity` | `"day" \| "hour" \| "minute" \| "second"` | `"day"` | 段的详细级别 |
| `selectorButtonPlacement` | `"start" \| "end"` | `"end"` | 日历图标位置 |
| `isDisabled` | `boolean` | `false` | 禁用交互 |
| `isRequired` | `boolean` | `false` | 标记为必填 |
| `description` | `string` | — | 输入框下方的辅助文本 |
| `errorMessage` | `string` | — | 输入框下方的错误文本 |

### DateRangePicker

除了 DatePicker 的属性外，还包括：

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `RangeValue<DateValue>` | — | 受控范围值 |
| `onChange` | `(value: RangeValue<DateValue>) => void` | — | 范围变更回调函数 |
| `visibleMonths` | `number` | `1` | 显示的月份数量 |

## 可访问性 (Accessibility)

- 可通过键盘编辑的日期段（使用 Tab 键在年/月/日之间切换）。
- 支持完整方向键导航的日历弹窗。
- 日期更改时的屏幕阅读器播报。
- 输入框与弹出层之间的焦点管理。
