## 使用方法 (Usage)

```tsx
import {
  Dialog,
  DialogTrigger,
  DialogContent,
  DialogHeader,
  DialogTitle,
  DialogDescription,
  DialogFooter,
  DialogClose,
  DialogOverlay,
  DialogPortal,
} from "@nebutra/ui/primitives"
```

## 属性 (Props)

### Dialog (根组件)

扩展自 `@radix-ui/react-dialog` 中的 `Root` 组件属性。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `open` | `boolean` | — | 受控的打开状态 |
| `defaultOpen` | `boolean` | `false` | 非受控的默认打开状态 |
| `onOpenChange` | `(open: boolean) => void` | — | 打开状态改变时的回调函数 |
| `modal` | `boolean` | `true` | 是否使用模态行为（非模态时可与外部交互） |

### DialogTrigger

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | 将属性合并到直接子元素上 |
| `children` | `ReactNode` | — | 触发器元素 |

### DialogContent

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 对话框内容 |

渲染时会在右上角自带一个内置的关闭按钮 (`X`)。

### DialogHeader / DialogFooter

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 头部或底部内容 |

### DialogTitle

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 对话框标题文本 |

### DialogDescription

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 补充说明文本 |

### DialogClose

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | 将属性合并到直接子元素上 |
| `children` | `ReactNode` | — | 关闭触发器元素 |

## 可访问性 (Accessibility)

- 基于 Radix UI Dialog 构建 — 遵循 WAI-ARIA 的 [Dialog pattern](https://www.w3.org/WAI/ARIA/apg/patterns/dialog-modal/)。
- `DialogContent` 具有 `role="dialog"` 和 `aria-modal="true"` 属性。
- `DialogTitle` 通过 `aria-labelledby` 链接，以便屏幕阅读器朗读。
- `DialogDescription` 通过 `aria-describedby` 链接。
- 对话框打开时，焦点会被锁定在对话框内。
- 按下 `Escape` 键可关闭对话框。
- 关闭后，焦点将返回到触发对话框的元素上。
