## 概览 (Overview)

`DisplayCards` 使用 CSS Grid 的 `[grid-area:stack]` 技巧将多个 `DisplayCard` 元素堆叠在一起。卡片具有 `-skew-y-[8deg]` 的倾斜角度，并在悬停时向外动画展开，展现底层内容。非常适合用于装饰性的 Hero 区域和功能展示。

## 安装 (Installation)

```tsx
import { DisplayCards } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 默认堆叠卡片 (Default stacked cards)

```tsx
<DisplayCards />
```

默认渲染三张具有不同偏移量的卡片 —— 最前面的卡片完全可见，中间的卡片向右下偏移并带有灰色叠加层，最后的卡片偏移量最大。

### 自定义卡片 (Custom cards)

```tsx
<DisplayCards
  cards={[
    {
      title: "分析统计 (Analytics)",
      description: "实时跟踪一切",
      date: "今天",
    },
    {
      title: "自动化 (Automation)",
      description: "让您的工作流自动运行",
      date: "2 小时前",
    },
    {
      title: "洞察 (Insights)",
      description: "发现最重要的信息",
      date: "刚刚",
    },
  ]}
/>
```

### 带有自定义图标的单张卡片 (Single card with custom icon)

```tsx
import { Zap } from "lucide-react";

<DisplayCards
  cards={[
    {
      icon: <Zap className="size-4 text-yellow-300" />,
      title: "极速 (Lightning Fast)",
      description: "全球响应时间低于 50ms",
      date: "实时",
      iconClassName: "bg-yellow-800",
      titleClassName: "text-yellow-400",
    },
  ]}
/>
```

## 属性 (Props)

### DisplayCards

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `cards` | `DisplayCardProps[]` | 默认堆叠 | 卡片配置数组 |
| `className` | `string` | — | 网格容器的额外 CSS 类名 |

### DisplayCardProps

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `title` | `string` | `"Featured"` | 卡片标题 |
| `description` | `string` | `"Discover amazing content"` | 卡片描述文本 |
| `date` | `string` | `"Just now"` | 底部显示的时间戳 |
| `icon` | `ReactNode` | `<Sparkles />` | 顶部的图标元素 |
| `iconClassName` | `string` | — | 图标容器的背景类名 |
| `titleClassName` | `string` | `"text-blue-500"` | 标题的颜色类名 |
| `className` | `string` | — | 该卡片的额外 CSS 类名 |

## 注意事项 (Notes)

- 卡片使用了 `select-none` 以防止在交互过程中选中文字。
- 堆叠效果依赖于所有卡片都处于同一个 `[grid-area:stack]` 网格单元格中。
- 默认卡片包含一个 `before:` 伪元素叠加层，在悬停时会淡出，从而实现从去色到彩色的展现效果。
- 建议与背景渐变或图像结合使用，以创建装饰性的 Hero 部分。
