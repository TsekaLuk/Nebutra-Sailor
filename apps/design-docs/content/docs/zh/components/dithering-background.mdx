## 使用方法 (Usage)

```tsx
import { DitheringBackground } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<div className="relative min-h-screen">
  <DitheringBackground />
  <main className="relative z-10">内容 (Content)</main>
</div>
```

### 强制暗黑模式 (Dark Mode Override)

```tsx
<div className="relative h-[400px]">
  <DitheringBackground mode="dark" />
</div>
```

### 强制亮色模式 (Light Mode Override)

```tsx
<div className="relative h-[200px]">
  <DitheringBackground mode="light" />
</div>
```

### 系统自动切换 (System)

```tsx
<DitheringBackground mode="system" />
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `mode` | `"light" \| "dark" \| "system"` | `"system"` | 主题模式 —— `system` 将遵循 `prefers-color-scheme` |
| `className` | `string` | — | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 由 `@paper-design/shaders-react` 的 `Dithering` 组件提供驱动。
- 需要 WebGL 支持。
- 该组件设置为 `absolute inset-0 -z-10` —— 请将其放置在 `relative` 容器中。
- 主题颜色预定义在亮色/暗色配置中。可以通过底层的 `DitheringProps` 使用自定义颜色。
