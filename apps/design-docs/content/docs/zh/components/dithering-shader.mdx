## 概览 (Overview)

`DitheringShader` 直接在 `<canvas>` 元素上使用 WebGL2 渲染动画抖动效果。它不需要外部库 —— 所有 GLSL 着色器都已捆绑在组件中。

功能特性：
- 7 种形状图案：单纯形噪声 (simplex noise)、扭曲 (warp)、点 (dots)、波浪 (wave)、涟漪 (ripple)、旋涡 (swirl)、球体 (sphere)
- 4 种抖动类型：随机噪声、2×2、4×4、8×8 Bayer 有序矩阵
- 可配置的前景/背景颜色（仅限十六进制）
- 可变的像素大小和动画速度
- 设置 `speed=0` 可渲染单个静态帧

如果您需要一个具有颗粒感/暗角/发光叠加层的全屏主题感知背景，请改用 `DitheringBackground`。

## 使用方法 (Usage)

```tsx
import { DitheringShader } from "@nebutra/ui/primitives";

// 默认用法 (单纯形噪声, 8×8 Bayer, 黑底白字)
<DitheringShader width={800} height={800} />

// 卡片中的旋涡效果
<DitheringShader
  shape="swirl"
  type="4x4"
  colorBack="#220011"
  colorFront="#00ffff"
  pxSize={4}
  speed={0.9}
/>

// 静态深色方案
<DitheringShader
  shape="sphere"
  type="8x8"
  colorBack="#0f0f0f"
  colorFront="#e5e7eb"
  speed={0.3}
/>

// 暂停 (单帧)
<DitheringShader shape="wave" speed={0} />
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `width` | `number` | `800` | 画布宽度（像素） |
| `height` | `number` | `800` | 画布高度（像素） |
| `colorBack` | `string` | `"#000000"` | 背景颜色（仅限十六进制格式） |
| `colorFront` | `string` | `"#ffffff"` | 前景/抖动颜色（仅限十六进制格式） |
| `shape` | `DitheringShaderShape` | `"simplex"` | 形状图案（见下表） |
| `type` | `DitheringShaderType` | `"8x8"` | 抖动矩阵类型（见下表） |
| `pxSize` | `number` | `4` | 抖动单元的像素大小。数值越高，像素感越强 |
| `speed` | `number` | `1` | 动画速度倍数。`0` = 暂停（渲染一帧静态预览） |
| `className` | `string` | — | 包装层 `<div>` 的额外 CSS 类名 |
| `style` | `CSSProperties` | — | 包装层 `<div>` 的额外行内样式 |

## 形状图案 (Shape Patterns)

| 取值 | 描述 |
|-------|-------------|
| `"simplex"` | 平滑的单纯形噪声云 |
| `"warp"` | 域扭曲的流动线条 |
| `"dots"` | 动态正弦波点柱 |
| `"wave"` | 水平余弦波 |
| `"ripple"` | 从中心向外扩散的同心圆涟漪 |
| `"swirl"` | 对数螺旋旋涡 |
| `"sphere"` | 带有移动点光源的 3D 球体 |

## 抖动类型 (Dithering Types)

| 取值 | 描述 |
|-------|-------------|
| `"random"` | 伪随机噪声抖动 |
| `"2x2"` | 2×2 有序 Bayer 矩阵 |
| `"4x4"` | 4×4 有序 Bayer 矩阵 |
| `"8x8"` | 8×8 有序 Bayer 矩阵（细节最丰富） |

## 注意事项 (Notes)

- 颜色必须是十六进制字符串 (`#rrggbb`)。不支持 CSS 变量和 `hsl()`。
- 该组件将画布包裹在一个尺寸固定为 `width × height` 像素的 `<div>` 中 —— 它**不会**根据容器自动缩放。
- 需要浏览器支持 WebGL2。在不支持的环境中会优雅降级（显示为空白画布）。
- 更改所有效果参数都会触发 WebGL 的完整重新初始化（通过 `useEffect` 依赖数组实现）。
- 无需外部库 —— GLSL 着色器在浏览器中运行时编译。
