---
title: "点状图案 (Dot Pattern)"
description: "填充其容器的 SVG 点阵网格背景。支持静态和动态（发光）模式、渐变遮罩以及自定义间距。"
---

## 使用方法 (Usage)

```tsx
import { DotPattern } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 静态背景 (Static Background)

```tsx
<div className="relative h-[400px] w-full overflow-hidden">
  <DotPattern className="text-muted-foreground/30" />
  <div className="relative z-10 p-8">内容 (Content)</div>
</div>
```

### 发光点 (Glowing Dots)

```tsx
<div className="relative h-[400px] w-full overflow-hidden">
  <DotPattern glow className="text-primary/50" />
</div>
```

### 渐变遮罩（向角落淡出） (Gradient Mask)

```tsx
<div className="relative h-[400px] w-full overflow-hidden">
  <DotPattern
    className="[mask-image:linear-gradient(to_bottom_right,white,transparent,transparent)]"
  />
</div>
```

### 自定义间距和尺寸 (Custom Spacing and Size)

```tsx
<DotPattern
  width={24}
  height={24}
  cr={1.5}
  className="text-blue-400/40"
/>
```

## 属性 (Props)

`DotPattern` 继承自 `React.SVGProps<SVGSVGElement>`。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `width` | `number` | `16` | 点之间的水平间距 |
| `height` | `number` | `16` | 点之间的垂直间距 |
| `x` | `number` | `0` | 图案的 X 轴偏移 |
| `y` | `number` | `0` | 图案的 Y 轴偏移 |
| `cx` | `number` | `1` | 单个点的 X 轴偏移 |
| `cy` | `number` | `1` | 单个点的 Y 轴偏移 |
| `cr` | `number` | `1` | 每个点的半径 |
| `glow` | `boolean` | `false` | 启用动画发光脉冲效果 |
| `className` | `string` | — | CSS 类名（使用 `text-` 设置点颜色） |

## 注意事项 (Notes)

- 该组件使用绝对定位，填充其容器 (`absolute inset-0 h-full w-full`)。
- 点的颜色通过 Tailwind 的 `text-*` 工具类控制（使用 `currentColor`）。
- 开启 `glow` 后，每个点都会以随机的时间独立进行动画。
- 使用 `mask-image` 工具类实现方向性淡出。
