## 概览 (Overview)

`DottedMap` 将世界地图渲染为填充了圆点的 SVG。可用其展示全球业务范围、办公室地点或用户分布。标记通过经纬度进行定位。

## 使用方法 (Usage)

```tsx
import { DottedMap } from "@nebutra/ui/primitives";

// 基础用法 (空世界地图)
<div className="h-[400px] w-full">
  <DottedMap />
</div>

// 带有位置标记
<div className="h-[400px] w-full">
  <DottedMap
    markers={[
      { lat: 40.7128, lng: -74.006 },   // 纽约
      { lat: 51.5074, lng: -0.1278 },   // 伦敦
      { lat: 35.6762, lng: 139.6503 },  // 东京
    ]}
    markerColor="#FF6900"
  />
</div>

// 在深色背景上使用
<div className="h-[400px] w-full bg-zinc-950 rounded-xl p-4">
  <DottedMap
    markers={[{ lat: 37.7749, lng: -122.4194 }]}
    markerColor="#00e5ff"
    className="text-zinc-700"
  />
</div>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `width` | `number` | `150` | 内部 SVG 宽度单位 |
| `height` | `number` | `75` | 内部 SVG 高度单位 |
| `mapSamples` | `number` | `5000` | 圆点采样点的数量（数值越高，细节越丰富） |
| `markers` | `DottedMapMarker[]` | `[]` | 要绘制的位置标记 |
| `markerColor` | `string` | `"#FF6900"` | 标记点的颜色 |
| `dotRadius` | `number` | `0.2` | 地图圆点的半径 |
| `stagger` | `boolean` | `true` | 启用交错行偏移以获得更好的密度效果 |
| `className` | `string` | — | SVG 类名（通过 `currentColor` 使用 `text-*` 设置圆点颜色） |

### `DottedMapMarker`

| 属性 | 类型 | 描述 |
|------|------|-------------|
| `lat` | `number` | 纬度（–90 到 90） |
| `lng` | `number` | 经度（–180 到 180） |
| `size` | `number` | 标记点大小（可选，默认为 `dotRadius`） |

## 注意事项 (Notes)

- 圆点颜色使用 `currentColor` —— 通过 SVG 上的 Tailwind `text-*` 类进行设置。
- 增加 `mapSamples` 可以提高陆地覆盖的细节，但可能会影响渲染性能。
- 标记使用与圆点相同的 SVG 坐标系进行定位。
- 需要安装 `svg-dotted-map`：`pnpm add svg-dotted-map`
