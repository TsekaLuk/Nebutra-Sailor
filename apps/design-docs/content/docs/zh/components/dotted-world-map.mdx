## 概览 (Overview)

`DottedWorldMap` 使用 `dotted-map` 库生成的预定义斜向点阵网格渲染简化的世界地图。与 `DottedMap` 不同，它使用静态预计算的点集（`height: 55, grid: "diagonal"`）—— 无需运行时生成 —— 因此速度非常快。

最适合用作落地页或仪表盘上的装饰性背景元素。

## 使用方法 (Usage)

```tsx
import { DottedWorldMap } from "@nebutra/ui/primitives";

// 默认
<DottedWorldMap className="w-full" />

// 深色主题
<div className="bg-zinc-950 p-4">
  <DottedWorldMap
    dotColor="#3f3f46"
    backgroundColor="transparent"
  />
</div>

// 主题色圆点
<DottedWorldMap
  dotColor="hsl(var(--primary))"
  backgroundColor="transparent"
  dotRadius={0.2}
/>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `dotRadius` | `number` | `0.15` | 每个圆点的半径 |
| `dotColor` | `string` | `"currentColor"` | 圆点的 CSS 颜色 |
| `backgroundColor` | `string` | `"var(--color-background)"` | SVG 的背景颜色 |
| `className` | `string` | — | SVG 的 CSS 类名 |

## 注意事项 (Notes)

- 地图点在模块初始化时预先计算 —— 无需渲染时计算。
- 使用 `dotted-map` 配置 `{ height: 55, grid: "diagonal" }` —— 运行时不可配置；如需支持标记的运行时生成地图，请使用 `DottedMap`。
- 通过 `w-full` 继承容器宽度；高度遵循 SVG viewBox 的纵横比 (`120 × 60`)。
- 需要安装 `dotted-map`：`pnpm add dotted-map`
