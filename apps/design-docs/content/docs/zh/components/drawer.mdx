## 属性 (Props)

### Drawer (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `open` | `boolean` | — | 受控的打开状态 |
| `onOpenChange` | `(open: boolean) => void` | — | 打开状态改变时的回调函数 |
| `direction` | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` | 滑入方向 |
| `shouldScaleBackground` | `boolean` | `false` | 抽屉打开时是否缩放背景 |

### DrawerContent (抽屉内容)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 抽屉内容 |

## 可访问性 (Accessibility)

- 基于 Vaul 构建 — 提供带有 `aria-modal="true"` 的 `role="dialog"`。
- `DrawerTitle` 通过 `aria-labelledby` 进行关联。
- 抽屉打开时，焦点被锁定在抽屉内。
- 按下 `Escape` 键可关闭抽屉并将焦点返回到触发器。
- 在移动设备上，支持拖拽关闭手势。
