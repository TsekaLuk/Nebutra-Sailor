## 使用方法 (Usage)

```tsx
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuShortcut,
  DropdownMenuCheckboxItem,
  DropdownMenuRadioGroup,
  DropdownMenuRadioItem,
  DropdownMenuSub,
  DropdownMenuSubContent,
  DropdownMenuSubTrigger,
} from "@nebutra/ui/primitives"
```

## 属性 (Props)

### DropdownMenuContent (菜单内容)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"center"` | 相对于触发器的对齐方式 |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | 首选显示侧 |
| `sideOffset` | `number` | `4` | 距离触发器的距离 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 菜单项 |

### DropdownMenuItem (菜单项)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `inset` | `boolean` | `false` | 添加左侧内边距以对齐 |
| `disabled` | `boolean` | `false` | 防止被选择 |
| `onSelect` | `() => void` | — | 选中该项时的回调函数 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 菜单项标签内容 |

## 可访问性 (Accessibility)

- 基于 Radix UI DropdownMenu 构建 — 遵循 WAI-ARIA [菜单按钮模式 (Menu Button pattern)](https://www.w3.org/WAI/ARIA/apg/patterns/menu-button/)。
- `DropdownMenuContent` 具有 `role="menu"` 属性，并带有指向触发器的 `aria-labelledby`。
- 每个条目分别具有 `role="menuitem"`、`role="menuitemcheckbox"` 或 `role="menuitemradio"`。
- 键盘操作：`ArrowUp`/`ArrowDown` 导航，`Enter`/`Space` 选择，`Escape` 关闭。
