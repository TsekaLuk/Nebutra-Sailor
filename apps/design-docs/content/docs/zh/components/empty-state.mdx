## 概览 (Overview)

`EmptyState` 是一个用于处理零内容视图的复合组件。它支持四种设计框架变体 —— 初始空白 (Blank slate)、信息类 (Informational, 带 CTA)、教育类 (Educational) 和引导类 (Guide) —— 所有变体都使用同一套 API：图标、标题、描述以及可选的操作/链接插槽。

## 安装 (Installation)

```tsx
import { EmptyState } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 初始空白 (Blank Slate)

最基础的形式 —— 传达当前的状态，而不引导用户执行特定的操作。

```tsx
import { EmptyState } from "@nebutra/ui/primitives";
import { ChartBarPeak } from "@nebutra/icons";

<EmptyState.Root
  icon={<EmptyState.Icon icon={<ChartBarPeak size={20} />} />}
  title="标题"
  description="传达产品当前状态的消息。"
/>
```

### 信息类 (Informational)

包含一个主要的行动唤起 (CTA) 和一个次要链接，以引导用户进行下一步。

```tsx
<EmptyState.Root
  icon={<EmptyState.Icon icon={<ChartBarPeak size={20} />} />}
  title="尚无部署 (No deployments yet)"
  description="推送到已连接的分支，或手动部署，即可在此处查看动态。"
  action={
    <Button>立即部署 (Deploy now)</Button>
  }
  link={
    <a href="/docs/deployments" className="text-sm text-muted-foreground hover:text-foreground">
      了解更多 (Learn more) ↗
    </a>
  }
/>
```

### 无图标 (No icon)

适用于空间受限场景的极简纯文本变体。

```tsx
<EmptyState.Root
  title="尚无项目 (No projects yet)"
  description="创建您的第一个项目以开始使用。"
  action={<Button>新建项目 (New Project)</Button>}
/>
```

## 属性 (Props)

### `EmptyState.Root`

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `title` | `string` | — | **必填。** 命名该空状态的小标题 |
| `description` | `string` | — | **必填。** 解释状态或下一步操作的辅助文本 |
| `icon` | `ReactNode` | — | 使用 `<EmptyState.Icon icon={...} />` 以保持尺寸一致 |
| `action` | `ReactNode` | — | 主要行动唤起（按钮或锚点） |
| `link` | `ReactNode` | — | 次要行动唤起（如 “了解更多” 链接） |
| `className` | `string` | — | 包裹器的额外 CSS 类名 |

### `EmptyState.Icon`

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `icon` | `ReactNode` | — | **必填。** 图标元素（推荐尺寸：20px） |
| `className` | `string` | — | 图标包裹器的额外 CSS 类名 |

## 可访问性 (Accessibility)

- 图标包裹器在图标元素上使用了 `aria-hidden="true"` —— 所有上下文信息都通过 `title` 和 `description` 文本传达。
- 操作元素应使用带适当标签的标准 `<button>` 或 `<a>` 元素。
