## 演示 (Demo)

<ComponentPreview>
  <ExpandableTabsDemo />

```tsx title="expandable-tabs-demo.tsx"
"use client";

import { ExpandableTabs } from "@nebutra/ui/primitives";
import { Home, Search, Bell, User } from "lucide-react";
import { useState } from "react";

export function ExpandableTabsDemo() {
    const [activeTab, setActiveTab] = useState<number | null>(null);

    const tabs = [
        { title: "Home", icon: Home },
        { title: "Search", icon: Search },
        { title: "Notifications", icon: Bell },
        { title: "Profile", icon: User },
    ];

    return (
        <div className="flex w-full flex-col items-center justify-center p-8 gap-4">
            <ExpandableTabs tabs={tabs} onChange={setActiveTab} />
            <p className="text-sm text-muted-foreground">
                Active tab: <strong>{activeTab !== null ? tabs[activeTab]?.title : "None"}</strong>
            </p>
        </div>
    );
}

```
</ComponentPreview>


## 概览 (Overview)

`ExpandableTabs` 使用仅图标的按钮渲染水平标签栏，在选中时会平滑地展开以显示其标签文本。点击标签栏外部会取消选中当前活动的标签。支持在标签组之间添加可选的分隔线。

## 安装 (Installation)

```tsx
import { ExpandableTabs } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础用法 (Basic)

```tsx
import { Home, Settings, Bell, User } from "lucide-react";

<ExpandableTabs
  tabs={[
    { title: "首页 (Home)", icon: Home },
    { title: "通知 (Notifications)", icon: Bell },
    { title: "个人资料 (Profile)", icon: User },
    { title: "设置 (Settings)", icon: Settings },
  ]}
  onChange={(index) => console.log("选中了标签:", index)}
/>
```

### 带有分隔线 (With separators)

分隔线可以使用垂直线对标签进行视觉上的分组。

```tsx
import { Home, BarChart, Bell, Settings, User } from "lucide-react";

<ExpandableTabs
  tabs={[
    { title: "首页 (Home)", icon: Home },
    { title: "分析 (Analytics)", icon: BarChart },
    { type: "separator" },
    { title: "通知 (Notifications)", icon: Bell },
    { type: "separator" },
    { title: "个人资料 (Profile)", icon: User },
    { title: "设置 (Settings)", icon: Settings },
  ]}
/>
```

### 自定义活动颜色 (Custom active color)

```tsx
<ExpandableTabs
  tabs={[...]}
  activeColor="text-blue-500"
/>
```

### 受控状态 (Controlled state)

```tsx
const [activeTab, setActiveTab] = useState<number | null>(null);

<ExpandableTabs
  tabs={tabs}
  onChange={(index) => {
    setActiveTab(index);
    // 根据 index 导航或显示内容
  }}
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `tabs` | `TabItem[]` | — | **必填。** 标签或分隔线对象的数组 |
| `activeColor` | `string` | `"text-primary"` | 活动标签的 Tailwind 文本颜色类名 |
| `className` | `string` | — | 容器的额外 CSS 类名 |
| `onChange` | `(index: number \| null) => void` | — | 活动标签改变时的回调（取消选中时为 null） |

### TabItem (标签项)

```ts
type TabItem = Tab | Separator;

interface Tab {
  title: string;       // 展开时显示的标签名
  icon: LucideIcon;    // Lucide 图标组件
}

interface Separator {
  type: "separator";   // 渲染一个垂直分隔线
}
```

## 注意事项 (Notes)

- 点击标签栏外部会调用 `onChange(null)` 并折叠所有标签。
- 展开动画使用 Framer Motion 的弹簧效果 (`bounce: 0, duration: 0.6`)。
- 标签文本的动画从 `width: 0, opacity: 0` 渐变为 `width: auto, opacity: 1`。
- 标签按钮使用 `role="tab"` 和 `aria-selected` 以支持无障碍访问。
