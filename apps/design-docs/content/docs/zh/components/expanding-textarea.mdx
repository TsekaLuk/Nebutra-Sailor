## 使用方法 (Usage)

```tsx
import { ExpandingTextarea } from "@nebutra/ui/primitives"
```

## 属性 (Props)

继承自 `React.TextareaHTMLAttributes<HTMLTextAreaElement>`。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `placeholder` | `string` | — | 为空时的占位文本 |
| `value` | `string` | — | 受控值 |
| `defaultValue` | `string` | — | 非受控默认值 |
| `onChange` | `React.ChangeEventHandler<HTMLTextAreaElement>` | — | 变更事件处理函数 |
| `minRows` | `number` | `1` | 最小可见行数 |
| `maxRows` | `number` | — | 开始滚动前的最大行数 |
| `disabled` | `boolean` | `false` | 禁用文本域 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 示例 (Examples)

### 受控可扩展文本域 (Controlled Expanding Textarea)

```tsx
const [value, setValue] = useState("")

<ExpandingTextarea
  value={value}
  onChange={(e) => setValue(e.target.value)}
  placeholder="输入一段描述..."
  className="w-full min-h-[80px]"
/>
```

### 带有最小/最大行数 (With Min/Max Rows)

```tsx
<ExpandingTextarea
  minRows={3}
  maxRows={10}
  placeholder="输入您的消息..."
/>
```

### 在表单中使用 (In a Form)

```tsx
<div className="space-y-2">
  <Label htmlFor="bio">个人简介</Label>
  <ExpandingTextarea
    id="bio"
    name="bio"
    placeholder="向我们介绍一下您自己..."
    className="w-full"
  />
</div>
```

## 可访问性 (Accessibility)

- 渲染原生的 `<textarea>` 元素 —— 继承完整的键盘和屏幕阅读器支持。
- 始终与通过 `htmlFor`/`id` 关联的 `<Label>` 配对使用。
- 高度扩展自动发生 —— 除了打字外不需要用户进行额外操作。
- 通过 `focus-visible:ring-2` 为键盘导航提供可见的焦点环。
