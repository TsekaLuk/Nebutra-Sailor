---
title: 功能箭头卡片 (FeatureArrowCard)
description: 带有图标、标题、副标题、描述、可选内容插槽以及旋转箭头操作按钮的功能展示卡片。
---

## 使用方法 (Usage)

```tsx
import { FeatureArrowCard } from "@nebutra/ui";
import { Zap } from "lucide-react";

<FeatureArrowCard
  icon={Zap}
  title="性能 (Performance)"
  subtitle="亚秒级响应时间 (Sub-second response times)"
  description="覆盖全球 35 个以上的地区"
  onArrowClick={() => router.push("/features/performance")}
/>
```

## 示例 (Examples)

### 基础卡片 (Basic Card)

```tsx
<FeatureArrowCard
  icon={Shield}
  title="安全 (Security)"
  subtitle="企业级保护 (Enterprise-grade protection)"
  description="采用零信任架构并获得 SOC 2 认证"
  onArrowClick={() => console.log("跳转到安全页面")}
/>
```

### 带有卡片内容 (With Card Content)

使用 `cardContent` 在右下角固定显示一个缩略图：

```tsx
<FeatureArrowCard
  icon={BarChart}
  title="分析 (Analytics)"
  subtitle="实时见解 (Real-time insights)"
  description="深入了解您的产品使用情况和增长指标"
  cardContent={
    <img
      src="/analytics-preview.png"
      alt="Analytics preview"
      className="object-cover w-full h-full"
    />
  }
  onArrowClick={() => router.push("/analytics")}
/>
```

### 网格布局 (Grid Layout)

```tsx
<div className="grid grid-cols-1 md:grid-cols-2 gap-4">
  {features.map((feature) => (
    <FeatureArrowCard
      key={feature.id}
      icon={feature.icon}
      title={feature.title}
      subtitle={feature.subtitle}
      description={feature.description}
      onArrowClick={() => router.push(feature.href)}
    />
  ))}
</div>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | — | 功能图标（以 16×16 尺寸渲染） |
| `title` | `string` | — | 显示在副标题上方的小号辅助标签 |
| `subtitle` | `string` | — | 主标题，常规字重 |
| `description` | `string` | — | 副标题后的行内解释文本（淡色显示） |
| `cardContent` | `ReactNode` | — | 固定在右下角的内容插槽 |
| `onArrowClick` | `() => void` | — | 箭头按钮点击时的回调函数 |
| `className` | `string` | — | 额外的 Tailwind 类名 |

## 注意事项 (Notes)

- 箭头按钮在悬停时会通过 `hover:-rotate-45` 旋转 `-45deg` —— 这创造了一种对角线方向的“导航”引导感。
- `cardContent` 被放置在右下角的一个绝对定位卡片中（移动端 `h-20 w-24`，桌面端 `h-32 w-40`），并带有左上圆角。
- 外层容器采用 `relative` 定位，以作为内容插槽和箭头按钮的定位锚点。
