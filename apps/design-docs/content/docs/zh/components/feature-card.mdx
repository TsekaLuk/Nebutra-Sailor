## 使用方法 (Usage)

```tsx
import {
  FeatureCard,
  FeatureCardHeader,
  FeatureCardContent,
} from "@nebutra/ui";
import { MapPin } from "lucide-react";

<FeatureCard>
  <FeatureCardHeader
    icon={MapPin}
    title="实时追踪 (Real-time tracking)"
    description="随时掌握团队动态。"
  />
  <FeatureCardContent>
    <img src="/map-screenshot.png" alt="Map view" />
  </FeatureCardContent>
</FeatureCard>
```

### 配合 DualModeImage 使用 (With DualModeImage)

`DualModeImage` 根据不同主题渲染不同的图像：

```tsx
<FeatureCard>
  <FeatureCardHeader
    icon={Monitor}
    title="控制面板 (Dashboard)"
    description="全面掌控运营状况。"
  />
  <FeatureCardContent>
    <DualModeImage
      lightSrc="/dashboard-light.png"
      darkSrc="/dashboard-dark.png"
      alt="控制面板截图"
      width={800}
      height={500}
    />
  </FeatureCardContent>
</FeatureCard>
```

### CircularUI 装饰 (CircularUI Decoration)

```tsx
<FeatureCard>
  <FeatureCardHeader
    icon={Layers}
    title="灵活权限 (Flexible permissions)"
    description="细粒度的基于角色的访问控制。"
  />
  <FeatureCardContent className="flex gap-4 p-6">
    <CircularUI
      label="管理员 (Admins)"
      circles={[{ pattern: "primary" }, { pattern: "border" }]}
    />
    <CircularUI
      label="成员 (Members)"
      circles={[{ pattern: "border" }, { pattern: "none" }]}
    />
  </FeatureCardContent>
</FeatureCard>
```

## 属性 (Props)

### FeatureCard

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 卡片内容 |
| `className` | `string` | — | 额外的 Tailwind 类名 |

### FeatureCardHeader

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | — | 功能图标 |
| `title` | `string` | — | 功能标签（小号，置灰） |
| `description` | `string` | — | 主标题（大号，半粗体） |
| `className` | `string` | — | 额外的 Tailwind 类名 |

### FeatureCardContent

`CardContent` 的重新导出 —— 接受标准 `div` 属性和 `className`。

### DualModeImage

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `lightSrc` | `string` | — | 浅色模式下的图像 URL |
| `darkSrc` | `string` | — | 深色模式下的图像 URL |
| `alt` | `string` | — | 无障碍提示文本 |
| `width` | `number` | — | 图像宽度 |
| `height` | `number` | — | 图像高度 |
| `className` | `string` | — | 额外的 CSS 类名 |

### CircularUI

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `label` | `string` | — | 圆圈下方的文本 |
| `circles` | `CircleConfig[]` | — | 圆圈配置数组，`pattern` 可选值：`"none" \| "border" \| "primary" \| "blue"` |
| `className` | `string` | — | 额外的 CSS 类名 |

## 注意事项 (Notes)

- `FeatureCard` 默认渲染为 `rounded-none`（无圆角） —— 以便在网格布局中与相邻卡片无缝衔接。建议在父级网格上添加边框，而不是在单个卡片上。
- 角标装饰器 (`FeatureCardDecorator`) 会自动包含，并使用 `text-primary` 颜色。
- `DualModeImage` 使用 CSS `dark:` 变体进行图像切换 —— 需要在应用中配置 Tailwind 的深色模式。
