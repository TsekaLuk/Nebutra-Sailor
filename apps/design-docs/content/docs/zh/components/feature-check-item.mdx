---
title: 功能勾选条目 (FeatureCheckItem)
description: 原语级的功能项，带有勾选图标（或自定义图标）、标题和可选描述。用于功能列表和对比表。
---

## 使用方法 (Usage)

```tsx
import { FeatureCheckItem } from "@nebutra/ui";

<FeatureCheckItem
  title="无限制团队成员 (Unlimited team members)"
  description="免费邀请您的整个组织，无需额外付费。"
/>
```

## 示例 (Examples)

### 功能列表 (Feature List)

```tsx
const features = [
  { title: "无限制团队成员", description: "随心扩展，无按席位计费。" },
  { title: "99.99% 运行时间 SLA", description: "提供财务级别的保障。" },
  { title: "通过 SOC 2 Type II 认证", description: "默认提供企业级安全性。" },
  { title: "24/7 优先支持", description: "专属支持通道。" },
];

<div className="space-y-4">
  {features.map((feature) => (
    <FeatureCheckItem
      key={feature.title}
      title={feature.title}
      description={feature.description}
    />
  ))}
</div>
```

### 仅显示标题（无描述） (Title Only)

```tsx
<div className="grid grid-cols-2 gap-2">
  <FeatureCheckItem title="自定义域名" />
  <FeatureCheckItem title="API 访问" />
  <FeatureCheckItem title="SSO / SAML" />
  <FeatureCheckItem title="审计日志" />
  <FeatureCheckItem title="数据导出" />
  <FeatureCheckItem title="优先支持" />
</div>
```

### 自定义图标 (Custom Icon)

```tsx
import { Sparkles } from "lucide-react";

<FeatureCheckItem
  icon={Sparkles}
  iconClassName="text-yellow-500"
  title="AI 驱动的建议"
  description="根据您团队的工作流程提供上下文感知的推荐。"
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `title` | `string` | — | 功能名称 |
| `description` | `string` | — | 可选的辅助说明文字 |
| `icon` | `LucideIcon` | `Check` | 图标组件 |
| `iconClassName` | `string` | — | 覆盖图标的颜色/尺寸 |
| `className` | `string` | — | 额外的容器类名 |
