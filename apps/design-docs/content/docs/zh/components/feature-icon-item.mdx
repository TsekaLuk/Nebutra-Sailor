---
title: 功能图标条目 (FeatureIconItem)
description: 原语级的功能项，采用紧凑的“图标+文字”行内布局，包含图标、标题和可选描述。
---

## 使用方法 (Usage)

```tsx
import { FeatureIconItem } from "@nebutra/ui";
import { Zap } from "lucide-react";

<FeatureIconItem
  icon={Zap}
  title="极速体验 (Lightning fast)"
  description="全球亚秒级响应。"
/>
```

## 示例 (Examples)

### 功能网格 (Feature Grid)

```tsx
<div className="grid grid-cols-2 gap-6">
  <FeatureIconItem icon={Zap} title="快速" description="针对低延迟进行优化。" />
  <FeatureIconItem icon={Shield} title="安全" description="零信任架构。" />
  <FeatureIconItem icon={Globe} title="全球化" description="全球 35 个以上的边缘节点。" />
  <FeatureIconItem icon={Sparkles} title="AI 就绪" description="内置智能。" />
</div>
```

### 紧凑列表（无描述） (Compact List)

```tsx
<div className="space-y-3">
  <FeatureIconItem icon={Check} title="自定义域名" />
  <FeatureIconItem icon={Check} title="无限制请求" />
  <FeatureIconItem icon={Check} title="优先支持" />
</div>
```

### 自定义图标颜色 (Custom Icon Color)

```tsx
<FeatureIconItem
  icon={Star}
  iconClassName="text-yellow-500"
  title="特色推荐 (Featured)"
  description="深受企业客户好评。"
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | — | 与标题行内显示的图标 |
| `title` | `string` | — | 功能名称 |
| `description` | `string` | — | 标题下方的可选描述文本 |
| `iconClassName` | `string` | — | 覆盖图标的颜色/尺寸 |
| `className` | `string` | — | 额外的容器类名 |

## 注意事项 (Notes)

- **FeatureIconItem vs FeatureCheckItem**: `FeatureIconItem` 要求必须传入 `icon` 属性（无默认值） —— 非常适合使用不同图标的功能网格。`FeatureCheckItem` 默认使用 `Check` 图标 —— 非常适合所有项目共享相同“勾选”视觉效果的任务列表。
