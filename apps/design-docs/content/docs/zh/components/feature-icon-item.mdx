## 使用方法 (Usage)

```tsx
import { FeatureIconItem } from "@nebutra/ui";
import { Zap } from "lucide-react";

<FeatureIconItem
  icon={Zap}
  title="极速体验 (Lightning fast)"
  description="全球亚秒级响应。"
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | — | 与标题行内显示的图标 |
| `title` | `string` | — | 功能名称 |
| `description` | `string` | — | 标题下方的可选描述文本 |
| `iconClassName` | `string` | — | 覆盖图标的颜色/尺寸 |
| `className` | `string` | — | 额外的容器类名 |

## 注意事项 (Notes)

- **FeatureIconItem vs FeatureCheckItem**: `FeatureIconItem` 要求必须传入 `icon` 属性（无默认值） —— 非常适合使用不同图标的功能网格。`FeatureCheckItem` 默认使用 `Check` 图标 —— 非常适合所有项目共享相同“勾选”视觉效果的任务列表。
