## 使用方法 (Usage)

```tsx
import { Field } from "@nebutra/ui/primitives"
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `label` | `string \| ReactNode` | — | 字段标签文本 |
| `description` | `string \| ReactNode` | — | 显示在输入框下方的辅助文本 |
| `error` | `string \| ReactNode` | — | 错误消息；提供时将设置为错误状态 |
| `htmlFor` | `string` | — | 通过 `id` 将标签与输入框关联 |
| `required` | `boolean` | `false` | 在标签上添加必填指示符 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 输入控件（Input, Select 等） |

## 示例 (Examples)

### 必填字段 (Required Field)

```tsx
<Field label="姓名 (Full name)" htmlFor="name" required>
  <Input id="name" placeholder="Jane Doe" />
</Field>
```

### 带有错误信息 (With Error)

```tsx
<Field
  label="密码 (Password)"
  htmlFor="password"
  error="密码长度至少为 8 个字符。"
>
  <Input id="password" type="password" />
</Field>
```

### 与选择器配合使用 (With Select)

```tsx
<Field label="国家 (Country)" htmlFor="country" description="选择您的账单所在国家。">
  <Select>
    <SelectTrigger id="country">
      <SelectValue placeholder="选择国家" />
    </SelectTrigger>
    <SelectContent>
      <SelectItem value="us">美国</SelectItem>
      <SelectItem value="uk">英国</SelectItem>
    </SelectContent>
  </Select>
</Field>
```

## 可访问性 (Accessibility)

- `label` 渲染为 `<label>` 元素，通过 `htmlFor` 与输入框关联。
- `description` 通过 `aria-describedby` 与输入框关联。
- `error` 会在子输入框上触发 `aria-invalid="true"`，并通过 `aria-errormessage` 进行关联。
- 必修字段具有可见的指示符，并在输入框上带有 `aria-required="true"`。
