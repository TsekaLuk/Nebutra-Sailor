title: "闪烁网格 (Flickering Grid)"
description: "基于 Canvas 的动态网格，方块随机在不同透明度之间闪烁。一种精致、具有科技美感的氛围背景。"
---

## 使用方法 (Usage)

```tsx
import { FlickeringGrid } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### Hero 区域背景 (Hero Background)

```tsx
<div className="relative h-[500px] w-full overflow-hidden">
  <FlickeringGrid className="absolute inset-0" />
  <div className="relative z-10 flex h-full items-center justify-center">
    <h1 className="text-4xl font-bold">你好，世界 (Hello World)</h1>
  </div>
</div>
```

### 蓝色网格 (Blue Grid)

```tsx
<FlickeringGrid
  color="rgb(59, 130, 246)"
  maxOpacity={0.4}
  flickerChance={0.5}
/>
```

### 密集小方块 (Dense Small Squares)

```tsx
<FlickeringGrid
  squareSize={2}
  gridGap={3}
  color="rgb(34, 197, 94)"
  maxOpacity={0.3}
/>
```

### 在圆角容器中使用 (In Rounded Container)

```tsx
<div className="relative h-[200px] overflow-hidden rounded-2xl">
  <FlickeringGrid />
</div>
```

## 属性 (Props)

`FlickeringGrid` 继承了 `React.HTMLAttributes<HTMLDivElement>`。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `squareSize` | `number` | `4` | 每个方块的大小（像素） |
| `gridGap` | `number` | `6` | 方块之间的间距（像素） |
| `flickerChance` | `number` | `0.3` | 方块每秒闪烁的概率 (0–1) |
| `color` | `string` | `"rgb(0,0,0)"` | 方块的颜色（任何 CSS 颜色） |
| `maxOpacity` | `number` | `0.3` | 方块的最大透明度 |
| `width` | `number` | — | 固定宽度 (px)，若不指定则填满容器 |
| `height` | `number` | — | 固定高度 (px)，若不指定则填满容器 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 性能 (Performance)

- 使用 `<canvas>` 进行高效渲染。
- 内置 `IntersectionObserver`，在移出屏幕时自动暂停动画。
- 使用 `ResizeObserver` 处理响应式尺寸调整。
- 适配 DPR (Device Pixel Ratio) —— 在 Retina 屏幕上清晰锐利。
