## 概览 (Overview)

`GitHubCalendar` 将一年的活动数据渲染为颜色密度的热力图网格 —— 布局与 GitHub 的贡献图完全一致。适用于：

- 开发者个人资料活动可视化
- 习惯追踪面板
- 内容发布频率展示
- 自定义活动指标

## 使用方法 (Usage)

```tsx
import { GitHubCalendar } from "@nebutra/ui/primitives";

const data = [
  { date: "2025-01-01", count: 3 },
  { date: "2025-01-02", count: 0 },
  { date: "2025-01-05", count: 7 },
];

<GitHubCalendar data={data} />
```

### 自定义颜色 (Custom colors)

```tsx
<GitHubCalendar
  data={data}
  colors={["#1a1a2e", "#16213e", "#0f3460", "#533483", "#e94560"]}
/>
```

### 自定义提示信息 (Custom tooltip)

```tsx
<GitHubCalendar
  data={data}
  tooltipFormatter={(date, count) =>
    count === 0 ? `${date}: 无活动 (No activity)` : `${date}: ${count} 次贡献 (commits)`
  }
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `data` | `ContributionDay[]` | — | **必填。** `{ date, count }` 对象数组 |
| `colors` | `string[]` | GitHub 绿色系 | 从低到高活动的 5 色刻度 |
| `weeks` | `number` | `53` | 显示的周数（53 周等于一年） |
| `thresholds` | `number[]` | `[0,1,2,3,4]` | 每个颜色等级对应的数量阈值 |
| `dayLabels` | `string[]` | `["Sun"..."Sat"]` | 左侧星期指示器的标签 |
| `showDayLabels` | `boolean` | `true` | 显示/隐藏星期标签 |
| `showMonthLabels` | `boolean` | `true` | 显示/隐藏月份标签 |
| `showLegend` | `boolean` | `true` | 显示/隐藏图例 |
| `legendLabels` | `{ less, more }` | — | 自定义图例文本 |
| `tooltipFormatter` | `(date, count) => string` | 默认格式 | 自定义工具提示文本 |
| `className` | `string` | — | 容器 CSS 类名 |
| `cellClassName` | `string` | — | 单个单元格的 CSS 类名 |

### `ContributionDay` (贡献日接口)

```ts
interface ContributionDay {
  date: string;  // ISO 格式: "2025-09-13"
  count: number; // 贡献次数
}
```

## 注意事项 (Notes)

- 日期匹配使用 `date-fns` 的 `isSameDay` —— 请确保日期是有效的 ISO 字符串。
- `data` 中缺失的日期将被视为次数为 `0`（无活动）。
- 日历总是以今天结束，并向前回溯指定的 `weeks` 周数。
- 单元格具有 `hover:scale-125` 效果，以提供微妙的交互反馈。
