## 概览 (Overview)

`Globe` 使用 [COBE](https://github.com/shuding/cobe) 库渲染 WebGL 动力 3D 地球仪。它支持：

- 自动旋转（速度可配置）
- 拖拽旋转（指针事件）
- 使用 经度/纬度 定位的自定义标记
- 可配置颜色（基础颜色、标记颜色、光晕）
- 通过 CSS 实现响应式尺寸

该组件会填满其绝对定位的容器，因此请将其包裹在一个固定尺寸的 `relative` div 中。

## 使用方法 (Usage)

```tsx
import { Globe } from "@nebutra/ui/primitives";

// 基础用法 (使用默认标记和自动旋转)
<div className="relative h-[400px] w-[400px]">
  <Globe />
</div>

// 自定义配置
<div className="relative h-[400px] w-[400px]">
  <Globe
    config={{
      markers: [
        { location: [40.7128, -74.006], size: 0.1 },  // 纽约
        { location: [51.5074, -0.1278], size: 0.08 }, // 伦敦
      ],
      markerColor: [0.1, 0.8, 1],
      dark: 1,
    }}
    rotationSpeed={0.003}
  />
</div>
```

## 属性 (Props)

### `GlobeProps`

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `config` | `GlobeConfig` | 见下方 | 地球仪配置对象 |
| `rotationSpeed` | `number` | `0.005` | 自动旋转速度（设为 `0` 以禁用） |
| `className` | `string` | — | 容器类名 |

### `GlobeConfig`

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `phi` | `number` | `0` | 初始水平旋转弧度 |
| `theta` | `number` | `0.3` | 初始垂直倾斜弧度 |
| `dark` | `number` | `0` | 深色模式（0 = 浅色, 1 = 深色） |
| `diffuse` | `number` | `0.4` | 漫反射光照强度 |
| `mapSamples` | `number` | `16000` | 地图细节级别 |
| `mapBrightness` | `number` | `1.2` | 陆地亮度 |
| `baseColor` | `[r, g, b]` | `[1, 1, 1]` | 基础颜色（每个通道 0–1） |
| `markerColor` | `[r, g, b]` | 橙色 | 标记点颜色 |
| `glowColor` | `[r, g, b]` | `[1, 1, 1]` | 光晕颜色 |
| `markers` | `GlobeMarker[]` | 主要城市 | 位置标记数组 |

### `GlobeMarker`

| 属性 (Prop) | 类型 | 描述 |
|------|------|-------------|
| `location` | `[lat, lng]` | 地理坐标 (纬度, 经度) |
| `size` | `number` | 圆点尺寸 (推荐 0.03–0.1) |

## 注意事项 (Notes)

- 需要安装 `cobe` 包：`pnpm add cobe`
- 容器必须设置为 `position: relative` 并具有固定的宽高比。
- 画布会伴随 500ms 的透明度过渡平滑显示。
- 拖拽交互支持指针和触摸设备。
