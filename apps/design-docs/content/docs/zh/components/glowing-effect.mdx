## 使用方法 (Usage)

```tsx
import { GlowingEffect } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 发光卡片 (Glowing Card)

```tsx
<div className="relative rounded-xl border p-6">
  <GlowingEffect
    spread={40}
    glow
    disabled={false}
    proximity={64}
    inactiveZone={0.7}
  />
  <div className="relative z-10">
    <h2 className="text-xl font-bold">高级卡片 (Premium Card)</h2>
    <p>悬停以查看光晕追踪您的光标位置。</p>
  </div>
</div>
```

### 白色变体（深色背景） (White Variant)

```tsx
<div className="relative rounded-xl border border-white/20 bg-black p-6">
  <GlowingEffect
    variant="white"
    disabled={false}
    glow={false}
    spread={30}
  />
  <div className="relative z-10 text-white">深色模式卡片 (Dark mode card)</div>
</div>
```

### 常驻发光（无需鼠标触发） (Always-on Glow)

```tsx
<div className="relative rounded-xl border p-6">
  <GlowingEffect
    glow={true}
    disabled={false}
    blur={4}
  />
  内容部分
</div>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `disabled` | `boolean` | `true` | 为 `true` 时，渲染静态边框回退方案 |
| `glow` | `boolean` | `false` | 是否始终显示光晕（而不只是在靠近时显示） |
| `variant` | `"default" \| "white"` | `"default"` | 颜色主题 —— default 为五彩，white 为单色 |
| `spread` | `number` | `20` | 光晕展开的角度（度） |
| `blur` | `number` | `0` | 额外的模糊量（像素） |
| `proximity` | `number` | `0` | 距离元素边缘多少像素时激活效果 |
| `inactiveZone` | `number` | `0.7` | 中心非活跃区域的比例 (0–1) |
| `movementDuration` | `number` | `2` | 光晕角度平滑过渡的持续时间 |
| `borderWidth` | `number` | `1` | 边框宽度（像素） |
| `className` | `string` | — | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 必须放置在 `relative` 定位的父元素内。
- `disabled={false}` 可启用鼠标追踪效果 —— 该效果默认是禁用的。
- 光标追踪是通过在整个 `document.body` 上使用被动事件监听器实现的。
- 容器内的内容应具有 `relative z-10` 样式，以确保渲染在发光层之上的位置。
