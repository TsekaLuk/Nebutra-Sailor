---
title: "渐变动画文本 (Gradient Animated Text)"
description: "在渐变和纯色状态之间循环切换的文本，具有错开的时间逻辑。将 3 个变体合用可实现 建立/发布/扩展 (Build/Ship/Scale) 的文字揭示效果。"
---

## 使用方法 (Usage)

```tsx
import { GradientAnimatedText } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 三词错开揭示效果（签名用例） (Three-Word Staggered Reveal)

```tsx
<div className="flex gap-2 text-5xl font-black">
  <GradientAnimatedText variant={1} theme="neon">建立 (Build.)</GradientAnimatedText>
  <GradientAnimatedText variant={2} theme="neon">发布 (Ship.)</GradientAnimatedText>
  <GradientAnimatedText variant={3} theme="neon">扩展 (Scale.)</GradientAnimatedText>
</div>
```

### AI 主题 (AI Theme)

```tsx
<div className="flex gap-2">
  <GradientAnimatedText variant={1} theme="ai">思考 (Think.)</GradientAnimatedText>
  <GradientAnimatedText variant={2} theme="ai">学习 (Learn.)</GradientAnimatedText>
  <GradientAnimatedText variant={3} theme="ai">成长 (Grow.)</GradientAnimatedText>
</div>
```

### 自定义颜色 (Custom Colors)

```tsx
<GradientAnimatedText
  variant={1}
  theme="custom"
  gradientFrom="#f97316"
  gradientTo="#eab308"
>
  自定义 (Custom)
</GradientAnimatedText>
```

### 更慢的动画 (Slower Animation)

```tsx
<GradientAnimatedText variant={1} theme="neon" duration={16}>
  缓慢 (Slow)
</GradientAnimatedText>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `string` | — | 文本内容 |
| `variant` | `1 \| 2 \| 3` | `1` | 动画相位偏移 —— 在不同变体之间实现错开效果 |
| `theme` | `"neon" \| "ai" \| "custom"` | `"neon"` | 预设颜色主题 |
| `gradientFrom` | `string` | — | 自定义渐变起始颜色（覆盖主题配置） |
| `gradientTo` | `string` | — | 自定义渐变结束颜色（覆盖主题配置） |
| `duration` | `number` | `8` | 动画循环周期（秒） |
| `className` | `string` | — | 额外的 CSS 类名 |

## 主题颜色 (Theme Colors)

| 主题 | 变体 1 | 变体 2 | 变体 3 |
|-------|-----------|-----------|-----------|
| `neon` | 绿色 → 青色 | 红粉色 → 洋红色 | 黄色 → 电蓝色 |
| `ai` | 绿色 → 洋红色 | 洋红色 → 青色 | 黄色 → 蓝色 |
| `custom` | 靛蓝色 → 紫色 | 粉色 → 玫瑰色 | 蓝绿色 → 青色 |
