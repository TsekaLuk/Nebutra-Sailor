## 示例 (Examples)


## 属性 (Props)

`GrainGradientBackground` 继承自 `React.HTMLAttributes<HTMLDivElement>` 并接受来自 `@paper-design/shaders-react` 的所有 `GrainGradientProps`（`style` 属性除外）。

底层着色器的关键属性：

| 属性 (Prop) | 类型 | 描述 |
|------|------|-------------|
| `colors` | `string[]` | 用于渐变的 CSS 颜色字符串数组 |
| `softness` | `number` | 渐变柔和度 (0–1) |
| `noise` | `number` | 颗粒噪声强度 (0–1) |
| `speed` | `number` | 动画速度 |
| `className` | `string` | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 该组件默认设为绝对定位，带有 `absolute inset-0 -z-10` 类名。
- 需要 WebGL 支持。在没有 WebGL 的环境中会优雅降级。
- 处于向后兼容性考虑，仍导出了已废弃的 `GrainCloudsBackground` 别名。
