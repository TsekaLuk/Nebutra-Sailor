## 使用方法 (Usage)

```tsx
import { GridFeatureCard } from "@nebutra/ui";
import { Zap } from "lucide-react";

<GridFeatureCard
  icon={Zap}
  title="性能 (Performance)"
  description="全球 35 个边缘节点，提供亚秒级响应时间。"
/>
```

## 示例 (Examples)

### 功能网格 (Feature Grid)

```tsx
<div className="grid grid-cols-1 md:grid-cols-3 divide-x divide-y border">

  />

    icon={Globe}
    title="全球规模 (Global Scale)"
    description="部署到 35 个以上地区，支持自动故障转移。"

```

### 自定义网格大小 (Custom Grid Size)

`gridSize` 属性控制背景 SVG 图案的密度：

```tsx
<GridFeatureCard
  icon={Code}
  title="开发者优先 (Developer First)"
  description="为每个平台提供全面的 API 和 SDK。"
  gridSize={10}
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | — | 功能图标（24×24，细边框样式） |
| `title` | `string` | — | 功能标题 |
| `description` | `string` | — | 功能描述 |
| `gridSize` | `number` | `20` | SVG 图案中网格方块的尺寸 |
| `className` | `string` | — | 额外的 Tailwind 类名 |

## 注意事项 (Notes)

- SVG 网格图案在每次组件挂载时通过 `React.useMemo` **随机生成** —— 方块被随机放置在右上角区域。
- 该图案仅起装饰作用，并通过 `aria-hidden="true"` 对屏幕阅读器隐藏。
- 建议在网格容器上使用 `divide-x divide-y`，以便在卡片之间创建整洁的边框，避免边框重叠。
- 图案使用 `fill-foreground/5 stroke-foreground/25` —— 能够自动适应浅色/深色模式。
