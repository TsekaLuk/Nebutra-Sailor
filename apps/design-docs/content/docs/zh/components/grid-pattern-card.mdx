## 使用方法 (Usage)

```tsx
import { GridPatternCard, GridPatternCardBody } from "@nebutra/ui";

<GridPatternCard>
  <GridPatternCardBody>
    <h3 className="text-xl font-semibold">功能标题 (Feature Title)</h3>
    <p className="text-muted-foreground mt-2">此处填写功能描述。</p>
  </GridPatternCardBody>
</GridPatternCard>
```

## 示例 (Examples)

### 基础卡片 (Basic Card)

```tsx
<GridPatternCard className="max-w-sm">
  <GridPatternCardBody>
    <h3 className="text-xl font-semibold">AI 集成 (AI Integration)</h3>
    <p className="mt-2 text-muted-foreground">
      将您的工作流程与最新的 AI 模型无缝连接。
    </p>
  </GridPatternCardBody>
</GridPatternCard>
```

### 停用入场动画 (Without Entrance Animation)

```tsx
<GridPatternCard disableAnimation className="max-w-sm">
  <GridPatternCardBody>
    <h3>静态展示 (Static display)</h3>
    <p>无淡入效果 —— 立即渲染。</p>
  </GridPatternCardBody>
</GridPatternCard>
```

### 自定义渐变叠加 (Custom Gradient Overlay)

覆盖位于网格图案和内容之间的渐变层：

```tsx
<GridPatternCard
  gradientClassName="from-blue-950/90 via-blue-950/50 to-blue-950/10"
  className="max-w-sm"
>
  <GridPatternCardBody>
    <h3 className="text-white">自定义渐变 (Custom Gradient)</h3>
  </GridPatternCardBody>
</GridPatternCard>
```

### 卡片网格 (Grid of Cards)

```tsx
<div className="grid grid-cols-1 md:grid-cols-3 gap-4">
  {features.map((feature) => (
    <GridPatternCard key={feature.id}>
      <GridPatternCardBody>
        <feature.Icon className="h-8 w-8 mb-4" />
        <h3>{feature.title}</h3>
        <p className="text-muted-foreground">{feature.description}</p>
      </GridPatternCardBody>
    </GridPatternCard>
  ))}
</div>
```

## 属性 (Props)

### GridPatternCard

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 卡片内容 |
| `className` | `string` | — | 卡片容器的额外类名 |
| `patternClassName` | `string` | — | 网格图案层的额外类名 |
| `gradientClassName` | `string` | — | 覆盖渐变叠加层的类名 |
| `disableAnimation` | `boolean` | `false` | 停用淡入式入场动画 |

### GridPatternCardBody

封装了 `text-left p-4 md:p-6` 的 `div` —— 接受标准 `div` 属性和 `className`。

## 注意事项 (Notes)

- 网格图案以内联 SVG 数据 URL 的形式存在 —— 无需外部资源。
- 深色模式：网格颜色会随 `.dark` 类名通过 CSS 变量自动从 `hsla(215, 16%, 47%)`切换为白色。
- 入场动画：在 0.8 秒内实现 `opacity: 0 → 1` 和 `y: -20 → 0` 的平滑过渡（`easeOut`）。在服务端渲染（SSR）或卡片初始化即显示等场景下，建议使用 `disableAnimation`。
