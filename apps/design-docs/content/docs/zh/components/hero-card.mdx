## 概览 (Overview)

`HeroCard` 对 [HeroUI 的 Card](https://www.heroui.com/docs/components/card) 组件进行了封装。它支持背景模糊效果、按压交互（渲染为 `<button>`）、悬停动画，以及由三部分组成的插槽布局（Header / Body / Footer）。

## 安装 (Installation)

```tsx
import {
  HeroCard,
  HeroCardHeader,
  HeroCardBody,
  HeroCardFooter,
} from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础卡片 (Basic card)

```tsx
<HeroCard>
  <HeroCardHeader>
    <h4 className="font-semibold">卡片标题 (Card Title)</h4>
  </HeroCardHeader>
  <HeroCardBody>
    <p>无论您的设计经验如何，都能打造精美的网站。</p>
  </HeroCardBody>
  <HeroCardFooter>
    <button>了解更多 (Learn more)</button>
  </HeroCardFooter>
</HeroCard>
```

### 可悬停 (Hoverable)

```tsx
<HeroCard isHoverable>
  <HeroCardBody>请将鼠标悬停在上方</HeroCardBody>
</HeroCard>
```

### 可按压 (Pressable)

使卡片具备按钮的行为和按压事件。

```tsx
<HeroCard isPressable onPress={() => console.log("被按压了！")}>
  <HeroCardBody>点击我 (Click me)</HeroCardBody>
</HeroCard>
```

### 模糊效果 (Blurred - 玻璃拟态)

需要在具有渐变背景的祖先元素中使用。

```tsx
<div className="bg-gradient-to-tr from-pink-500 to-yellow-500 p-8">
  <HeroCard isBlurred>
    <HeroCardBody>带有模糊效果的玻璃拟态卡片</HeroCardBody>
  </HeroCard>
</div>
```

### 底部模糊 (Blurred footer)

页脚变为覆盖在卡片底部的磨砂玻璃叠加层。

```tsx
<HeroCard isFooterBlurred className="h-[300px] relative">
  <HeroCardBody>背景内容 (Background content)</HeroCardBody>
  <HeroCardFooter className="absolute bottom-0 z-10 backdrop-blur">
    <button>查看详情 (View details)</button>
  </HeroCardFooter>
</HeroCard>
```

## 属性 (Props)

### HeroCard

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `shadow` | `"none" \| "sm" \| "md" \| "lg"` | `"md"` | 阴影深度 |
| `radius` | `"none" \| "sm" \| "md" \| "lg"` | `"lg"` | 圆角半径 |
| `fullWidth` | `boolean` | `false` | 展开以填充容器 |
| `isHoverable` | `boolean` | `false` | 启用悬停缩放效果 |
| `isPressable` | `boolean` | `false` | 渲染为按钮，并启用按压事件 |
| `isBlurred` | `boolean` | `false` | 在卡片背景上应用模糊效果 |
| `isFooterBlurred` | `boolean` | `false` | 在页脚上应用模糊效果 |
| `isDisabled` | `boolean` | `false` | 禁用卡片 |
| `disableAnimation` | `boolean` | `false` | 停用所有动画 |
| `disableRipple` | `boolean` | `false` | 停用按压时的水波纹效果 |
| `allowTextSelectionOnPress` | `boolean` | `false` | 在可按压状态下是否允许选中文本 |
| `classNames` | `Partial<Record<"base" \| "header" \| "body" \| "footer", string>>` | — | 针对各插槽的自定义类名 |
| `onPress` | `(e: PressEvent) => void` | — | 按压事件回调 |
| `onPressStart` | `(e: PressEvent) => void` | — | 按压开始回调 |
| `onPressEnd` | `(e: PressEvent) => void` | — | 按压结束回调 |
| `onPressChange` | `(isPressed: boolean) => void` | — | 按压状态改变回调 |
| `onPressUp` | `(e: PressEvent) => void` | — | 按压释放回调 |

### HeroCardHeader / HeroCardBody / HeroCardFooter

| 属性 (Prop) | 类型 | 描述 |
|------|------|-------------|
| `children` | `ReactNode` | 插槽内容 |
| `className` | `string` | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 按压事件使用 React Aria 的 `PressEvent` 接口，以支持各种指针设备。
- `isBlurred` 在彩色或渐变背景上的卡片效果最佳。
- 需要在项目对应的应用中安装 `@heroui/card`。
