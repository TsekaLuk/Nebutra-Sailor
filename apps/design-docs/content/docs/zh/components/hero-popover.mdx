## 概览 (Overview)

`HeroPopover` 封装了 [HeroUI 的 Popover](https://www.heroui.com/docs/components/popover) 组件。使用 `HeroPopoverTrigger` 来定义打开气泡卡片的触发元素，使用 `HeroPopoverContent` 来定义悬浮的内容区域。

## 安装 (Installation)

```tsx
import {
  HeroPopover,
  HeroPopoverTrigger,
  HeroPopoverContent,
} from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础用法 (Basic)

```tsx
<HeroPopover>
  <HeroPopoverTrigger>
    <button>打开气泡 (Open Popover)</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">
      <p>这是气泡卡片的内容。</p>
    </div>
  </HeroPopoverContent>
</HeroPopover>
```

### 带有箭头和指定定位 (With arrow and placement)

```tsx
<HeroPopover showArrow placement="right">
  <HeroPopoverTrigger>
    <button>右侧显示 (Show right)</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">带有箭头的内容</div>
  </HeroPopoverContent>
</HeroPopover>
```

### 带有背景遮罩 (With backdrop)

```tsx
<HeroPopover backdrop="blur">
  <HeroPopoverTrigger>
    <button>模糊背景 (Blurred backdrop)</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">气泡后的背景已模糊</div>
  </HeroPopoverContent>
</HeroPopover>
```

### 受控模式 (Controlled)

```tsx
const [isOpen, setIsOpen] = useState(false);

<HeroPopover isOpen={isOpen} onOpenChange={setIsOpen}>
  <HeroPopoverTrigger>
    <button onClick={() => setIsOpen(true)}>受控打开 (Open controlled)</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    <div className="p-4">
      <p>受控气泡卡片</p>
      <button onClick={() => setIsOpen(false)}>关闭 (Close)</button>
    </div>
  </HeroPopoverContent>
</HeroPopover>
```

### 带有无障碍标题 (With accessible title)

```tsx
<HeroPopover>
  <HeroPopoverTrigger>
    <button>信息 (Info)</button>
  </HeroPopoverTrigger>
  <HeroPopoverContent>
    {(titleProps) => (
      <div className="p-4">
        <h3 {...titleProps} className="font-semibold">
          无障碍标题
        </h3>
        <p className="mt-1 text-sm">此处是内容。</p>
      </div>
    )}
  </HeroPopoverContent>
</HeroPopover>
```

## 属性 (Props)

### HeroPopover

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `placement` | `HeroPopoverPlacement` | `"bottom"` | 相对于触发器的位置 |
| `color` | `HeroPopoverColor` | `"default"` | 颜色变体 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 气泡尺寸 |
| `radius` | `"none" \| "sm" \| "md" \| "lg" \| "full"` | `"lg"` | 圆角半径 |
| `shadow` | `"none" \| "sm" \| "md" \| "lg"` | `"lg"` | 阴影深度 |
| `backdrop` | `HeroPopoverBackdrop` | `"transparent"` | 背景遮罩样式 |
| `isOpen` | `boolean` | — | 受控打开状态 |
| `defaultOpen` | `boolean` | — | 初始打开状态（非受控） |
| `offset` | `number` | `7` | 距离触发器的像素值 |
| `crossOffset` | `number` | `0` | 在交叉轴上的偏移量 |
| `showArrow` | `boolean` | `false` | 是否显示指向箭头 |
| `shouldFlip` | `boolean` | `true` | 在视口边缘是否自动翻转位置 |
| `triggerScaleOnOpen` | `boolean` | `true` | 当气泡打开时缩放触发元素 |
| `shouldBlockScroll` | `boolean` | `false` | 打开时是否阻止页面滚动 |
| `shouldCloseOnScroll` | `boolean` | `true` | 页面滚动时是否关闭 |
| `isKeyboardDismissDisabled` | `boolean` | `false` | 禁用 Esc 键关闭 |
| `shouldCloseOnBlur` | `boolean` | `false` | 焦点离开时是否关闭 |
| `disableAnimation` | `boolean` | `false` | 停用动画 |
| `motionProps` | `HTMLMotionProps<"div">` | — | 自定义 Framer Motion 属性 |
| `classNames` | `Partial<Record<...>>` | — | 针对各插槽的自定义类名 |
| `onOpenChange` | `(isOpen: boolean) => void` | — | 打开状态改变时的回调 |
| `onClose` | `() => void` | — | 关闭时的回调 |

## 位置选项 (Placement Options)

`top` (上) · `top-start` · `top-end` · `bottom` (下) · `bottom-start` · `bottom-end` · `left` (左) · `left-start` · `left-end` · `right` (右) · `right-start` · `right-end`

## 注意事项 (Notes)

- 需要在项目中安装 `@heroui/popover` 并配置 HeroUI provider。
- `HeroPopoverContent` 可以接受一个渲染函数 `(titleProps: object) => ReactNode` 以实现具有无障碍标识的标题。
