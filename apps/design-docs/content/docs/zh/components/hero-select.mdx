## 概览 (Overview)

`HeroSelect`, `HeroSelectItem` 和 `HeroSelectSection` 是对 [HeroUI 的 Select](https://www.heroui.com/docs/components/select) 组件的重新导出。为了保持设计系统命名的一致性，我们统一添加了 `HeroSelect` 前缀。

## 安装 (Installation)

```tsx
import {
  HeroSelect,
  HeroSelectItem,
  HeroSelectSection,
} from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础用法 (Basic)

```tsx
<HeroSelect label="最喜欢的动物 (Favorite Animal)" placeholder="请选择一种动物">
  <HeroSelectItem key="cat">猫 (Cat)</HeroSelectItem>
  <HeroSelectItem key="dog">狗 (Dog)</HeroSelectItem>
  <HeroSelectItem key="elephant">大象 (Elephant)</HeroSelectItem>
  <HeroSelectItem key="penguin">企鹅 (Penguin)</HeroSelectItem>
</HeroSelect>
```

### 多选模式 (Multiple selection)

```tsx
<HeroSelect
  label="动物 (Animals)"
  selectionMode="multiple"
  placeholder="请选择动物"
>
  <HeroSelectItem key="cat">猫 (Cat)</HeroSelectItem>
  <HeroSelectItem key="dog">狗 (Dog)</HeroSelectItem>
  <HeroSelectItem key="elephant">大象 (Elephant)</HeroSelectItem>
</HeroSelect>
```

### 带有分组 (With sections)

```tsx
<HeroSelect label="地区 (Location)" placeholder="请选择地区">
  <HeroSelectSection title="北美洲 (North America)">
    <HeroSelectItem key="us">美国 (United States)</HeroSelectItem>
    <HeroSelectItem key="ca">加拿大 (Canada)</HeroSelectItem>
    <HeroSelectItem key="mx">墨西哥 (Mexico)</HeroSelectItem>
  </HeroSelectSection>
  <HeroSelectSection title="欧洲 (Europe)">
    <HeroSelectItem key="uk">英国 (United Kingdom)</HeroSelectItem>
    <HeroSelectItem key="fr">法国 (France)</HeroSelectItem>
    <HeroSelectItem key="de">德国 (Germany)</HeroSelectItem>
  </HeroSelectSection>
</HeroSelect>
```

### 受控数值 (Controlled value)

```tsx
const [value, setValue] = useState<string>("");

<HeroSelect
  label="框架 (Framework)"
  selectedKeys={[value]}
  onSelectionChange={(keys) => setValue(Array.from(keys)[0] as string)}
>
  <HeroSelectItem key="react">React</HeroSelectItem>
  <HeroSelectItem key="vue">Vue</HeroSelectItem>
  <HeroSelectItem key="svelte">Svelte</HeroSelectItem>
</HeroSelect>
```

### 虚拟化（用于超长列表） (Virtualized)

```tsx
<HeroSelect
  isVirtualized
  items={largeItemList}
  maxListboxHeight={400}
  label="国家 (Country)"
>
  {(item) => <HeroSelectItem key={item.key}>{item.label}</HeroSelectItem>}
</HeroSelect>
```

## 注意事项 (Notes)

- 这些组件直接从 `@heroui/select` 重新导出 —— 请参考 [HeroUI Select 文档](https://www.heroui.com/docs/components/select) 以查看完整的 API 参考。
- 所有 HeroUI 属性都会被透传，未作修改。
- 需要在项目中安装 `@heroui/select` 并配置 HeroUI provider。
