## 概览 (Overview)

`Highlighter` 将文本包裹在 `<span>` 中，并利用 `rough-notation` 库应用手绘标记笔画过渡动画。支持 7 种标注类型，可自定义颜色、笔画宽度，并支持视口触发动画。

## 安装 (Installation)

```tsx
import { Highlighter } from "@nebutra/ui/primitives";
```

`rough-notation` 是一个同级依赖（peer dependency） —— 如果尚未安装，请通过以下命令添加：

```bash
pnpm add rough-notation
```

## 使用方法 (Usage)

### 基础高亮 (Basic highlight)

```tsx
<p>
  这是一段 <Highlighter>非常重要 (very important)</Highlighter> 的文字。
</p>
```

### 自定义颜色的下划线 (Underline with custom color)

```tsx
<Highlighter action="underline" color="#FF9800">
  带有下划线的文本
</Highlighter>
```

### 圆圈标注 (Circle annotation)

```tsx
<Highlighter action="circle" color="#ef4444" strokeWidth={2}>
  被圈起来了！
</Highlighter>
```

### 视口触发 (Viewport-triggered)

仅当元素滚动到视口中时才播放标注动画：

```tsx
<Highlighter triggerOnView color="#a5f3fc">
  滚动到此处时触发
</Highlighter>
```

### 段落中的混合标注 (Mixed annotations in a paragraph)

```tsx
<p className="text-xl leading-relaxed">
  <Highlighter action="underline" color="#FF9800">
    Magic UI Highlighter
  </Highlighter>{" "}
  能让重要的{" "}
  <Highlighter action="highlight" color="#87CEFA">
    文本脱颖而出
  </Highlighter>{" "}
  ，且毫不费力。
</p>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 需要添加标注的内容 |
| `action` | `HighlighterAction` | `"highlight"` | 标注效果类型 |
| `color` | `string` | `"#ffd1dc"` | 标注笔画/填充颜色（支持任何 CSS 颜色值） |
| `strokeWidth` | `number` | `1.5` | 标注笔画的宽度（像素） |
| `animationDuration` | `number` | `600` | 绘制动画的持续时间（毫秒） |
| `iterations` | `number` | `2` | 绘制迭代次数 —— 值越高，手绘草图感越强 |
| `padding` | `number` | `2` | 元素与标注之间的内边距（像素） |
| `multiline` | `boolean` | `true` | 标注是否跨多行折行 |
| `triggerOnView` | `boolean` | `false` | 仅在元素进入视口时才播放动画 |

## 标注类型 (Annotation Types)

| 取值 (Value) | 描述 |
|-------|-------------|
| `highlight` | 文本背后的填充式高亮（类似记号笔） |
| `underline` | 文本下方的单条手绘线 |
| `box` | 文本四周的矩形框 |
| `circle` | 文本四周的椭圆/圆圈 |
| `strike-through` | 贯穿文本中间的删除线 |
| `crossed-off` | 贯穿文本的两条交叉对角线 |
| `bracket` | 文本旁的方括号 |

## 注意事项 (Notes)

- 当任何属性（颜色、动作等）发生改变时，组件会重新渲染标注。
- 使用 `ResizeObserver` 在元素或文档主体尺寸调整时重绘标注。
- 组件卸载时会正确清理标注。
- 尝试设置 `iterations={3}` 或更高值，可以获得更具“有机感”的手绘外观。
