## 可访问性 (Accessibility)

- 基于 Radix UI HoverCard 构建。
- 悬停卡片的内容对于通过 Tab 键导航的屏幕阅读器用户不可见 — 它仅作为视觉补充。
- 对于键盘用户，请考虑使用 `Tooltip` 或 `Popover` 作为关键信息的替代方案。
- 一旦通过鼠标悬停打开，该卡片是可聚焦且可探索的。
