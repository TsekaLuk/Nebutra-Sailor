## 使用方法 (Usage)

```tsx
import { InteractiveCard } from "@nebutra/ui";

<InteractiveCard
  step="STEP 1"
  title="创建项目 (Create Project)"
  description="首先在您的工作区中创建一个新项目。"
/>
```

## 示例 (Examples)

### 带有动画图标的默认样式 (Default with Animated Icon)

默认图标显示一个带有 PPT 文件的文件夹向上漂浮，以及一个在悬停时产生脉冲效果的下载按钮：

```tsx
<InteractiveCard
  step="STEP 1"
  title="导出数据 (Export your data)"
  description="选择格式，点击一下即可导出您的内容。"
/>
```

### 三步骤流程 (Three-Step Process)

```tsx
<div className="grid grid-cols-3 gap-6">
  <InteractiveCard
    step="STEP 1"
    title="连接 (Connect)"
    description="通过我们的一键集成功能链接您的数据源。"
  />
  <InteractiveCard
    step="STEP 2"
    title="配置 (Configure)"
    description="通过简单的拖拽操作设置您的流水线。"
  />
  <InteractiveCard
    step="STEP 3"
    title="部署 (Deploy)"
    description="通过单条命令即可推送到生产环境。"
  />
</div>
```

### 自定义图标 (Custom Icon)

可以用您自己的图标替换默认的动画图标：

```tsx
import { Sparkles } from "lucide-react";

<InteractiveCard
  step="STEP 1"
  title="AI 分析 (AI Analysis)"
  description="让 AI 自动分析您的数据并提取洞察。"
  icon={
    <div className="mb-8 flex h-40 w-full items-center justify-center">
      <Sparkles className="h-20 w-20 text-primary" />
    </div>
  }
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `step` | `string` | — | 显示在标题上方的步骤标签（如 `"STEP 1"`） |
| `title` | `string` | — | 卡片标题 |
| `description` | `string` | — | 卡片描述文本 |
| `icon` | `ReactNode` | `AnimatedExportIcon` | 用于替换默认动画 SVG 的自定义图标 |
| `className` | `string` | — | 额外的 Tailwind 类名 |

## 注意事项 (Notes)

- 默认的 `AnimatedExportIcon` 也会被导出，可以独立使用。
- 悬停动画是通过 Framer Motion 的 `whileHover="hover"` / `initial="initial"` 组级状态实现的 —— 自定义图标可以通过 `useAnimation` 或使用 `whileHover` 上下文中的 `variants` 来响应此状态。
- 卡片渲染时默认使用 `text-center` 居中对齐 —— 如果需要左对齐，可以通过 `className` 进行覆盖。
- 无障碍友好：`role="group"` 以及基于 `title` 和 `description` 自动设置的 `aria-label`。
