---
title: "Atom 组件 (Atom Components)"
description: "基础 UI 构建块 —— 最小的可复用单元。"
---

Atom 组件是设计系统的基石。它们具有以下特点：

- **自包含** —— 不包含业务逻辑
- **可组合** —— 旨在组合成更大的功能片段
- **易用性 (Accessibility)** —— 内置 ARIA 属性

<Note>
  Atom 组件在 `packages/ui/src/primitives/` 中实现。
  请通过 `@nebutra/ui/primitives` 进行导入。
</Note>
