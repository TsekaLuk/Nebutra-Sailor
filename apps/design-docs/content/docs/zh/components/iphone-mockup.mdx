## 概览 (Overview)

`IphoneMockup` 渲染一个像素级精确的 iPhone 15 Pro SVG 框架。屏幕内容（图片或视频）会被裁剪成与真机一致的圆角。组件具有完全响应式特性 —— 可以通过容器宽度来控制尺寸。

特性：
- 灵动岛 (Dynamic Island) 切口
- 侧边按钮和顶部扬声器
- 通过 Tailwind 类名支持深/浅色框架颜色
- 在屏幕区域内显示图片或视频

## 使用方法 (Usage)

```tsx
import { IphoneMockup } from "@nebutra/ui/primitives";

// 仅框架
<div className="w-48">
  <IphoneMockup />
</div>

// 带有图片
<div className="w-64">
  <IphoneMockup src="/app-screenshot.png" />
</div>

// 带有循环视频
<div className="w-64">
  <IphoneMockup videoSrc="/app-demo.mp4" />
</div>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `src` | `string` | — | 屏幕区域显示的图片 URL |
| `videoSrc` | `string` | — | 视频 URL（自动播放、循环、静音） |
| `className` | `string` | — | 容器 CSS 类名 |
| `style` | `CSSProperties` | — | 容器的内联样式 |

## 注意事项 (Notes)

- 如果同时提供了 `videoSrc` 和 `src`，`videoSrc` 将具有更高优先级。
- 视频播放时带有 `autoPlay loop muted playsInline` 属性 —— 这是符合浏览器安全策略的默认设置。
- 框架颜色通过 `dark:fill-[#404040]` / `dark:fill-[#262626]` 自动适应深色模式。
- 宽高比固定为 `433/882`（iPhone 15 Pro 的实际比例）。
- 屏幕区域坐标：`left 4.9%, top 2.2%, width 89.9%, height 95.6%`，圆角半径 `55.75px`。
