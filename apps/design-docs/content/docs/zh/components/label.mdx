## 使用方法 (Usage)

```tsx
import { Label } from "@nebutra/ui/primitives"
```

## 属性 (Props)

基于原生 HTML `<label>` 元素配合设计系统 Tokens 构建。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `htmlFor` | `string` | — | 通过控件的 `id` 将标签链接到其对应的控件 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 标签文本 |

## 可访问性 (Accessibility)

- 基于 Radix UI Label 构建 —— 渲染为原生 `<label>` 元素。
- 点击标签会通过 `htmlFor` 将焦点转移到关联的控件上。
- 当关联控件被禁用时，标签会自动反映禁用的光标状态。
- 不要使用 `<span>` 或 `<div>` 作为标签 —— 始终使用此 `Label` 组件。
