import { LightRays } from "@nebutra/ui/primitives"

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<div className="relative h-[400px] w-full overflow-hidden rounded-xl border">
  <LightRays />
  <div className="relative z-10 p-8">原始内容</div>
</div>
```

### 金色光束 (Golden Rays)

```tsx
<div className="relative h-[400px] overflow-hidden rounded-xl bg-black">
  <LightRays
    count={10}
    color="rgba(255, 200, 100, 0.3)"
    blur={48}
    speed={18}
    length="80vh"
  />
</div>
```

### 柔和蓝色环境光 (Subtle Blue Ambient)

```tsx
<div className="relative overflow-hidden rounded-xl">
  <LightRays
    count={5}
    color="rgba(99, 102, 241, 0.15)"
    blur={60}
    speed={20}
  />
</div>
```

## 属性 (Props)

`LightRays` 继承自 `React.HTMLAttributes<HTMLDivElement>`。

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `count` | `number` | `7` | 动画光束的数量 |
| `color` | `string` | `"rgba(160,210,255,0.2)"` | 光束渐变的基础颜色 |
| `blur` | `number` | `36` | 模糊滤镜半径（像素） |
| `speed` | `number` | `14` | 每个动画周期的秒数（数值越大速度越慢） |
| `length` | `string` | `"70vh"` | 光束高度的 CSS 长度值 |
| `ref` | `Ref<HTMLDivElement>` | — | 容器的转发 Ref |
| `className` | `string` | — | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 组件设置了 `pointer-events-none` 且会绝对定位充满父容器。
- 每道光束的初始位置、角度、宽度、摇摆幅度和时序均为随机产生，以呈现自然的视觉多样性。
- 请为内容设置 `relative z-10`，以使其渲染在光束之上。
- 在深色背景或渐变背景上效果最佳。
