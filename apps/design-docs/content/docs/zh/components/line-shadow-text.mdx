## 使用方法 (Usage)

```tsx
import { LineShadowText } from "@nebutra/ui/primitives"
```

## CSS 设置 (CSS Setup)

将 `animate-line-shadow` 关键帧添加到您的全局 CSS 或 Tailwind 配置中：

```css
@keyframes line-shadow {
  0%   { background-position: 0 0; }
  100% { background-position: 100% -100%; }
}
.animate-line-shadow::after {
  animation: line-shadow 15s linear infinite;
}
```

该组件还导出了 `LINE_SHADOW_CSS` 字符串，以便通过代码注入：

```tsx
import { LINE_SHADOW_CSS } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<LineShadowText className="text-5xl font-bold">Magic UI</LineShadowText>
```

### 自定义阴影颜色 (Custom Shadow Color)

```tsx
<LineShadowText shadowColor="#3b82f6" className="text-5xl font-bold">
  蓝色阴影 (Blue Shadow)
</LineShadowText>
```

### 作为标题元素渲染 (As Heading Element)

```tsx
<LineShadowText as="h1" className="text-6xl font-black">
  欢迎 (Welcome)
</LineShadowText>
```

### 结合 Framer Motion 进场动画 (With Framer Motion Entry)

```tsx
<LineShadowText
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.6 }}
  className="text-5xl font-bold"
>
  动态进场 (Animated Entry)
</LineShadowText>
```

## 属性 (Props)

`LineShadowText` 同时继承了 `React.HTMLAttributes<HTMLElement>` 和 Framer Motion 的 `MotionProps`。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `string` | — | 文本内容（必须是字符串） |
| `shadowColor` | `string` | `"black"` | 对角线阴影的颜色 |
| `as` | `React.ElementType` | `"span"` | 渲染为指定的 HTML 元素 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `...motionProps` | `MotionProps` | — | 所有 Framer Motion 属性（initial, animate 等） |

## 注意事项 (Notes)

- `children` 必须是纯字符串 —— 如果传入 JSX 节点，组件将报错。
- 阴影是通过 CSS `::after` 伪元素并配合 `data-text` 属性进行渲染的。
