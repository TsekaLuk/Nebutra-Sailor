## 使用方法 (Usage)

```tsx
import { Loader } from "@nebutra/ui";

<Loader />
<Loader variant="dots" size="lg" />
<Loader variant="loading-dots" text="处理中 (Processing)" />
```

## 变体 (Variants)

### 圆形 (Circular) (默认)

简单的旋转环 —— 通用的加载指示器。

```tsx
<Loader variant="circular" />
```

### 经典 (Classic)

iOS 风格的加载旋转器，由 12 条从中心向外辐射的渐隐长条组成。

```tsx
<Loader variant="classic" />
```

### 脉冲 (Pulse)

居中的脉冲式环形动画 —— 简约且含蓄。

```tsx
<Loader variant="pulse" />
```

### 脉冲点 (Pulse Dot)

单个脉冲式的圆点 —— 非常适合内联或紧凑的上下文。

```tsx
<Loader variant="pulse-dot" />
```

### 动画点 (Dots)

三个上下弹跳的圆点 —— 经典的聊天和内容加载指示器。

```tsx
<Loader variant="dots" />
```

### 打字中 (Typing)

聊天打字指示器 —— 三个按顺序循环改变透明度的圆点。

```tsx
<Loader variant="typing" />
```

### 波纹 (Wave)

五个带音频波纹动画的垂直长条 —— 媒体播放器的理想选择。

```tsx
<Loader variant="wave" />
```

### 长条 (Bars)

三个缩放的长条 —— 适用于代码编辑器或仪表盘场景。

```tsx
<Loader variant="bars" />
```

### 终端 (Terminal)

带有 `>` 提示符的终端光标闪烁效果 —— 用于 AI 和 CLI 上下文。

```tsx
<Loader variant="terminal" />
```

### 文字闪烁 (Text Blink)

在前景色和柔和色之间交替闪烁的文字 —— 表示 AI 正在思考。

```tsx
<Loader variant="text-blink" text="思考中 (Thinking)" />
```

### 文字流光 (Text Shimmer)

在文字上扫过的流光渐变效果 —— 高级感的加载状态。

```tsx
<Loader variant="text-shimmer" text="正在生成 (Generating)" />
```

### 加载点 (Loading Dots)

文字后带有动画效果的尾随点 —— 传统的“思考中...”模式。

```tsx
<Loader variant="loading-dots" text="处理中 (Processing)" />
```

## 尺寸 (Sizes)

所有变体均支持 `sm`、`md`（默认）和 `lg`。

```tsx
<Loader size="sm" />
<Loader size="md" />
<Loader size="lg" />
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `variant` | `"circular" \| "classic" \| "pulse" \| "pulse-dot" \| "dots" \| "typing" \| "wave" \| "bars" \| "terminal" \| "text-blink" \| "text-shimmer" \| "loading-dots"` | `"circular"` | 加载器的视觉样式 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 加载器的尺寸 |
| `text` | `string` | `"Thinking"` | 文本标签（仅用于文本类变体：`text-blink`, `text-shimmer`, `loading-dots`） |
| `className` | `string` | — | 额外的 Tailwind 类名 |

## 注意事项 (Notes)

- 所有变体都包含 `<span className="sr-only">正在加载 (Loading)</span>`，以满足屏幕阅读器的无障碍需求。
- `text` 属性仅对 `text-blink`、`text-shimmer` 和 `loading-dots` 变体有效；图形类变体将忽略此属性。
- 各个变体组件也支持独立导出：`CircularLoader`, `ClassicLoader`, `PulseLoader`, `PulseDotLoader`, `DotsLoader`, `TypingLoader`, `WaveLoader`, `BarsLoader`, `TerminalLoader`, `TextBlinkLoader`, `TextShimmerLoader`, `TextDotsLoader`。
