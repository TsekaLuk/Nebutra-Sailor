## 属性 (Props)

`MacbookPro` 组件接受以下属性值：

| 属性 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `width` | `number` | `650` | SVG 的宽度参数。 |
| `height` | `number` | `400` | SVG 的高度参数。 |
| `src` | `string` | `undefined` | 可选的图片 URL，该图片将被渲染在屏幕遮罩内。 |
| `...props` | `SVGProps<SVGSVGElement>` | `{}` | 支持所有原生的 React SVG 元素属性。 |

## 用法

```tsx
import { MacbookPro } from "@nebutra/ui/primitives"

export function MockupSection() {
  return (
    <MacbookPro 
      src="/images/dashboard-screenshot.png"
      className="w-full max-w-4xl" 
    />
  )
}
```
