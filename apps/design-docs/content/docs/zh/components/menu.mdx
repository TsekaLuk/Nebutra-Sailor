## 概述 (Overview)

`Menu` 是一个基于 `DropdownMenu` (Radix) 构建的语义下拉组件。它公开了一个高级 API，在每个条目上都提供了 `prefix`、`suffix`、`locked`、`disabled` 和 `href` 插槽 —— 涵盖了所有标准的 UI 菜单变体，而无需在应用侧暴露 Radix 的内部细节。

当您需要原始构建块（如复选框项、单选组或子菜单）时，请直接使用 `DropdownMenu`。对于标准的导航和操作列表，请使用 `Menu`。

## 权限与状态 (Permissions & States)

### 锁定条目 (Locked Items)

锁定条目受权限管控 —— 可见但不可交互，并自动渲染一个锁定图标。在语义上与 `disabled` 有所区别。

## 链接条目 (Link Items)

当提供 `href` 属性时，条目会自动渲染为 `<a>` 标签。通过 Radix 的 `asChild` 机制保留了键盘和鼠标导航功能。

## 自定义触发器 (Custom Trigger)

您可以通过覆盖类名或使用其他元素（如果使用原始的 `DropdownMenu` 原语并配合 `asChild`）来完全替换 `Menu.Trigger` 的外观。在 `<Menu.Trigger>` 中，您可以传递自定义的子元素。

## 属性 (Props)

该 API 与 Radix DropdownMenu 完全对应，但 `.Item` 接受自定义的渲染属性。

### `Menu.Trigger`

继承 `<button>` 的 HTML 属性。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `chevron` | `boolean` | `false` | 添加一个旋转的 ChevronDown 图标 |
| `className` | `string` | — | 覆盖触发器样式 |

### `Menu.Content`

继承 Radix `DropdownMenuContent` 的属性。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"start"` | 相对于触发器的水平对齐方式 |
| `sideOffset` | `number` | `4` | 触发器与内容之间的间距（像素） |

### `Menu.Item`

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `prefix` | `ReactNode` | — | 左侧的图标/元素 |
| `suffix` | `ReactNode` | — | 右侧的图标/元素（当 `locked` 时被锁定图标覆盖） |
| `disabled` | `boolean` | `false` | 置灰条目，移除指针事件 |
| `locked` | `boolean` | `false` | 权限管控状态 —— 显示锁定图标，防止交互 |
| `href` | `string` | — | 将条目渲染为导航到该 URL 的 `<a>` 标签 |
| `onSelect` | `(e: Event) => void` | — | 条目被选中时的回调 |
| `className` | `string` | — | 额外的 CSS 类名 |
