## 使用方法 (Usage)

```tsx
import {
  Menubar,
  MenubarMenu,
  MenubarTrigger,
  MenubarContent,
  MenubarItem,
  MenubarLabel,
  MenubarSeparator,
  MenubarShortcut,
  MenubarCheckboxItem,
  MenubarRadioGroup,
  MenubarRadioItem,
  MenubarSub,
  MenubarSubContent,
  MenubarSubTrigger,
} from "@nebutra/ui/primitives"
```

## 属性 (Props)

### Menubar (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | `MenubarMenu` 元素 |

### MenubarContent (内容区)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"start"` | 内容对齐方式 |
| `alignOffset` | `number` | `-4` | 相对于触发器的偏移量 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 菜单项 |

### MenubarItem (菜单项)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `inset` | `boolean` | `false` | 添加左侧内边距以对齐 |
| `disabled` | `boolean` | `false` | 防止选中 |
| `onSelect` | `() => void` | — | 条目被选中时的回调函数 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 条目内容 |

## 可访问性 (Accessibility)

- 基于 Base UI 构建 —— 遵循 WAI-ARIA [菜单按钮模式 (Menu Button pattern)](https://www.w3.org/WAI/ARIA/apg/patterns/menu-button/)。
- `Menubar` 具有 `role="menubar"` 属性。
- 每个 `MenubarTrigger` 具有 `role="menuitem"` 属性，并带有 `aria-haspopup="menu"`。
- 键盘操作：`ArrowRight`/`ArrowLeft` 在菜单之间导航，`ArrowDown` 打开，`Escape` 关闭。
