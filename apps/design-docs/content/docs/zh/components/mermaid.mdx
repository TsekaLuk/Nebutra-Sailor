## 使用方法 (Usage)

```tsx
import { Mermaid } from "@nebutra/ui/primitives"
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `chart` | `string` | — | **必需。** Mermaid 图表定义的语法字符串 |
| `className` | `string` | — | 容器上的额外 CSS 类名 |

## 示例 (Examples)

### 时序图 (Sequence Diagram)

```tsx
const sequenceDiagram = `
  sequenceDiagram
    participant Browser as 浏览器
    participant Server as 服务器
    participant Database as 数据库

    Browser->>Server: GET /api/user
    Server->>Database: SELECT * FROM users
    Database-->>Server: 用户数据
    Server-->>Browser: 200 OK + JSON
`

<Mermaid chart={sequenceDiagram} />
```

### 实体关系图 (Entity Relationship)

```tsx
const erDiagram = `
  erDiagram
    USER ||--o{ ORDER : places
    ORDER ||--|{ LINE-ITEM : contains
    PRODUCT }|--|{ LINE-ITEM : includes
`

<Mermaid chart={erDiagram} />
```

### 状态图 (State Diagram)

```tsx
const stateDiagram = `
  stateDiagram-v2
    [*] --> 闲置 (Idle)
    闲置 (Idle) --> 加载中 (Loading): fetch()
    加载中 (Loading) --> 成功 (Success): resolve
    加载中 (Loading) --> 错误 (Error): reject
    错误 (Error) --> 闲置 (Idle): retry
    Success --> [*]
`

<Mermaid chart={stateDiagram} />
```

## 无障碍支持 (Accessibility)

- 组件渲染为 SVG 元素 —— 建议添加 `aria-label` 或可见的标题以提供屏幕阅读器所需的上下文。
- 对于复杂的图表，考虑在旁边提供一段文字描述。
- SVG 图表默认不会被屏幕阅读器播报；可以将其包裹在 `<figure>` 和 `<figcaption>` 中以增强语义背景。

```tsx
<figure>
  <Mermaid chart={chart} />
  <figcaption className="text-sm text-muted-foreground mt-2">
    用户身份验证流程图
  </figcaption>
</figure>
```
