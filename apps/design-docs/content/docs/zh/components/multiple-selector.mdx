## 概览 (Overview)

`MultipleSelector` 是一个功能齐全的多选组件，基于 `cmdk` 构建。选中的选项将以可移除的徽章标签形式显示在输入框内。支持功能：

- 固定（不可移除）选项
- 使用 `groupBy` 进行选项分组
- 通过 `onSearch` 进行异步搜索
- 通过 `onSearchSync` 进行同步搜索
- 最大选择限制
- 键盘导航（Tab、Enter、Backspace 移除）

## 使用方法 (Usage)

```tsx
import { MultipleSelector } from "@nebutra/ui/primitives";

const options = [
  { value: "next", label: "Next.js" },
  { value: "react", label: "React" },
  { value: "vue", label: "Vue.js" },
];

// 基础用法
<MultipleSelector
  defaultOptions={options}
  placeholder="选择框架..."
/>

// 受控模式
const [selected, setSelected] = useState([]);

<MultipleSelector
  defaultOptions={options}
  value={selected}
  onChange={setSelected}
/>

// 异步搜索
<MultipleSelector
  onSearch={async (query) => {
    const results = await searchApi(query);
    return results.map((r) => ({ value: r.id, label: r.name }));
  }}
  placeholder="搜索用户..."
/>

// 带有最大选择限制
<MultipleSelector
  defaultOptions={options}
  maxSelected={3}
  onMaxSelected={(max) => toast(`最多只能选择 ${max} 项`)}
/>

// 分组显示
const grouped = [
  { value: "react", label: "React", group: "前端" },
  { value: "node", label: "Node.js", group: "后端" },
];

<MultipleSelector
  defaultOptions={grouped}
  groupBy="group"
/>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `defaultOptions` | `MultipleSelectorOption[]` | — | 静态选项列表 |
| `options` | `MultipleSelectorOption[]` | — | 手动受控的选项 |
| `value` | `MultipleSelectorOption[]` | — | 受控的已选项 |
| `onChange` | `(options: MultipleSelectorOption[]) => void` | — | 变更回调函数 |
| `placeholder` | `string` | — | 输入框占位符 |
| `onSearch` | `(query: string) => Promise<MultipleSelectorOption[]>` | — | 异步搜索函数 |
| `onSearchSync` | `(query: string) => MultipleSelectorOption[]` | — | 同步搜索函数 |
| `delay` | `number` | — | `onSearch` 的防抖延迟（毫秒） |
| `maxSelected` | `number` | — | 最大选择数量 |
| `onMaxSelected` | `(max: number) => void` | — | 达到最大数量时调用 |
| `groupBy` | `string` | — | 用于分组的选项键名 |
| `disabled` | `boolean` | `false` | 禁用选择器 |
| `hidePlaceholderWhenSelected` | `boolean` | — | 选中选项时隐藏占位符 |
| `emptyIndicator` | `ReactNode` | — | 未找到选项时显示的内容 |
| `loadingIndicator` | `ReactNode` | — | 异步搜索时显示的内容 |
| `className` | `string` | — | 容器 CSS 类名 |

### `MultipleSelectorOption`

```ts
interface MultipleSelectorOption {
  value: string;
  label: string;
  disable?: boolean;   // 不可选择的选项
  fixed?: boolean;     // 选中后不可移除
  [key: string]: string | boolean | undefined; // 用于 groupBy 的键
}
```

## 注意事项 (Notes)

- 选中的选项以带有移除按钮的徽章芯片显示。
- 固定选项 (设置 `fixed: true`) 渲染时不带移除按钮。
- 在空输入框中按 Backspace 会移除最后一个选中的选项。
- `triggerSearchOnFocus` 会在输入框获得焦点时触发 `onSearch` 函数。
