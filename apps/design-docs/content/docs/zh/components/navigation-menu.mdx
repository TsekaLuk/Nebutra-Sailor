## 属性 (Props)

### NavigationMenu (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 菜单方向 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 菜单内容 |

### NavigationMenuTrigger (触发器)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 触发器标签 |

### NavigationMenuContent (内容区)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 浮出面板内容 |

### NavigationMenuLink (链接)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `active` | `boolean` | `false` | 标记为当前页面 |
| `asChild` | `boolean` | `false` | 将属性合并到子元素上 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 链接内容 |

## 可访问性 (Accessibility)

- 基于本地元素和 Base UI 构建 —— 遵循 WAI-ARIA [揭开式导航模式 (Disclosure Navigation pattern)](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/examples/disclosure-navigation/)。
- `NavigationMenuList` 具有 `role="menubar"` 属性。
- 带有浮出内容面板的触发器具有 `aria-expanded` 和 `aria-haspopup="true"` 属性。
- 活动链接具有 `aria-current="page"` 属性。
- 键盘操作：使用 `Tab` 在条目间导航，使用 `Enter`/`Space` 打开浮出面板，使用 `Escape` 关闭。
