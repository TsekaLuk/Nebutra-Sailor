## 概览 (Overview)

`NoisePatternCard` 将内容包裹在一个带有循环颗粒/噪点纹理的深色卡片中。卡片在挂载时会伴随轻微的向上平移动画逐渐显现。使用 `NoisePatternCardBody` 可以获得一致的内部边距。

该噪点模式需要在项目的 CSS 中配置 `bg-noise-pattern` 这一 Tailwind 工具类。

## 安装 (Installation)

```tsx
import {
  NoisePatternCard,
  NoisePatternCardBody,
} from "@nebutra/ui/primitives";
```

## Tailwind 配置 (Tailwind Config)

将噪点纹理背景添加到您的 CSS 或 Tailwind 配置中：

```css
/* globals.css */
@layer utilities {
  .bg-noise-pattern {
    background-image: url("data:image/svg+xml,..."); /* SVG 噪声滤镜 */
  }
}
```

## 使用方法 (Usage)

### 基础卡片 (Basic card)

```tsx
<NoisePatternCard className="max-w-sm">
  <NoisePatternCardBody>
    <h3 className="text-lg font-semibold text-white">标题 (Title)</h3>
    <p className="mt-2 text-sm text-zinc-400">
      一款深邃且具有高级感的暗向卡片，带有有机的颗粒纹理。
    </p>
  </NoisePatternCardBody>
</NoisePatternCard>
```

### 自定义纹理和叠加层 (Custom pattern and overlay)

```tsx
<NoisePatternCard
  patternClassName="opacity-50"
  overlayClassName="bg-zinc-900/50"
  className="rounded-xl"
>
  <NoisePatternCardBody>
    <p className="text-zinc-200">自定义透明度的噪点纹理</p>
  </NoisePatternCardBody>
</NoisePatternCard>
```

## 属性 (Props)

### NoisePatternCard

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 卡片内容 |
| `className` | `string` | — | 外部卡片容器的 CSS 类名 |
| `patternClassName` | `string` | — | 噪点纹理层的 CSS 类名 |
| `overlayClassName` | `string` | — | 包裹子元素的叠加层 div 的 CSS 类名 |

### NoisePatternCardBody

扩展自 `React.HTMLAttributes<HTMLDivElement>`。

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名（默认：`p-4 text-left md:p-6`） |

## 注意事项 (Notes)

- 卡片背景在浅色和深色模式下均为 `bg-zinc-950` —— 专为暗色系 UI 场景设计。
- 入场动画使用 Framer Motion 实现，在 0.8 秒内完成 `opacity: 0 → 1` 和 `y: -20 → 0` 的变化。
- 噪点纹理平铺尺寸通过 `bg-[length:500px_500px]` 设置为 500×500px。
- 叠加层（overlay）在纹理之上添加了 `bg-zinc-950/30` 的透明遮罩。
