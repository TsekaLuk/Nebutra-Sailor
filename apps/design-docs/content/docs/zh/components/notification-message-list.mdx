## 概览 (Overview)

`NotificationMessageList` 渲染一个带有交错缩放入场动画的通知项列表。每个条目包含一个渐变头像、带有时间戳的标题以及内容预览。底部的淡出遮罩暗示了后续仍有溢出内容。

## 安装 (Installation)

```tsx
import { NotificationMessageList } from "@nebutra/ui/primitives";
import type { NotificationMessage } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础用法 (Basic)

```tsx
const messages: NotificationMessage[] = [
  {
    title: "新部署 (New deployment)",
    time: "2分钟前",
    content: "生产环境部署在 42s 内成功完成",
    gradientColor: "from-green-400 to-emerald-600",
  },
  {
    title: "安全警报 (Security alert)",
    time: "15分钟前",
    content: "发现来自德国柏林的新登录",
    gradientColor: "from-red-400 to-pink-600",
  },
  {
    title: "使用额度提醒 (Usage limit)",
    time: "1小时前",
    content: "您已使用了本月 API 配额的 80%",
    gradientColor: "from-amber-400 to-orange-500",
  },
];

<NotificationMessageList messages={messages} />
```

### 自定义高度和动画延迟 (Custom height and animation delay)

```tsx
<NotificationMessageList
  messages={messages}
  maxHeight="400px"
  animationDelay={150}
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `messages` | `NotificationMessage[]` | — | **必需。** 通知项数组 |
| `maxHeight` | `string` | `"280px"` | 容器的 CSS 高度 |
| `animationDelay` | `number` | `300` | 每个条目动画之间的延迟 (单位：毫秒) |
| `className` | `string` | — | 额外的 CSS 类名 |

### NotificationMessage

| 属性 (Prop) | 类型 | 描述 |
|------|------|-------------|
| `title` | `string` | 消息标题 |
| `time` | `string` | 相对时间字符串（如 `"2m ago"`） |
| `content` | `string` | 消息正文（会被截断为 1 行） |
| `gradientColor` | `string` | 用于头像的 Tailwind 渐变类名（例如 `"from-pink-400 to-indigo-500"`） |

## 注意事项 (Notes)

- 动画效果通过 `<style>` 标签注入的 `@keyframes` 代码块实现 —— 不依赖 Framer Motion。
- 每个条目的动画延迟与索引成比例，即 `index` × `animationDelay` 毫秒。
- 底部淡出遮罩使用 `bg-gradient-to-t from-background` 样式，会遮盖最后几个条目。
- 容器使用 `overflow-hidden` 和固定高度来裁剪溢出内容 —— 超过 `maxHeight` 的条目将不可见。
