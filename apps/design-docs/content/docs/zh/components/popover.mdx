## 可访问性 (Accessibility)

- 基于 Radix UI Popover 构建 —— `PopoverContent` 具有 `role="dialog"` 属性。
- 触发器具有 `aria-expanded` 和 `aria-haspopup="dialog"` 属性。
- 打开时焦点移入弹出框；关闭时焦点返回触发器。
- 按下 `Escape` 键关闭弹出框。
- 默认情况下，点击弹出框外部会将其关闭。
