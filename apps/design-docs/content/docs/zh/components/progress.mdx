## 使用方法 (Usage)

```tsx
import { Progress } from "@nebutra/ui/primitives"
```

## 属性 (Props)

继承自 `@radix-ui/react-progress` 的 `Root` 属性。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `number \| null` | — | 进度值，范围在 `0` 到 `max` 之间。`null` 表示状态待定 |
| `max` | `number` | `100` | 最大进度值 |
| `getValueLabel` | `(value: number, max: number) => string` | — | 为屏幕阅读器设置的自定义标签 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 示例 (Examples)

### 动画进度条 (Animated Progress)

<ComponentPreview name="progress-1-demo" className="max-w-[400px]">
  <Progress1Demo />

  ```tsx
import * as React from "react"
import { Progress } from "@nebutra/ui/primitives"

export function ProgressAnimatedDemo() {
  const [progress, setProgress] = React.useState(13)

  React.useEffect(() => {
    const timer = setTimeout(() => setProgress(65), 500)
    return () => clearTimeout(timer)
  }, [])

  return (
    <div className="w-full">
      <Progress value={progress} className="w-full" />
    </div>
  )
}
```
</ComponentPreview>

### 环形进度条 (Circular Progress)

<ComponentPreview name="animated-circular-progress-bar-demo" className="max-w-[400px]">
  <AnimatedCircularProgressBarDemo />

  ```tsx
import { useEffect, useState } from "react"
import { AnimatedCircularProgressBar } from "@nebutra/ui/primitives"

export function AnimatedCircularProgressBarDemo() {
  const [value, setValue] = useState(0)

  useEffect(() => {
    const handleIncrement = (prev: number) => {
      if (prev === 100) return 0
      return prev + 10
    }
    setValue(handleIncrement)
    const interval = setInterval(() => setValue(handleIncrement), 2000)
    return () => clearInterval(interval)
  }, [])

  return (
    <AnimatedCircularProgressBar
      max={100}
      min={0}
      value={value}
      gaugePrimaryColor="hsl(var(--primary))"
    />
  )
}
```
</ComponentPreview>

## 可访问性 (Accessibility)

- 基于 Radix UI Progress 构建 —— 具有 `role="progressbar"` 属性。
- 根据 `value` 设置 `aria-valuemin="0"`、`aria-valuemax` (默认为 100) 和 `aria-valuenow`。
- 应添加 `aria-label` 或 `aria-labelledby` 以提供上下文信息。
- 待定状态 (`value={null}`) 会省略 `aria-valuenow`。
