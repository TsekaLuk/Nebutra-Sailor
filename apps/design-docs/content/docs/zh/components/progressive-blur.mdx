## 使用方法 (Usage)

```tsx
import { ProgressiveBlur } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 底部滚动淡出 (Bottom Scroll Fade)

```tsx
<div className="relative h-[400px] overflow-auto">
  <div className="space-y-4 p-4">
    {items.map(item => <Card key={item.id} />)}
  </div>
  <ProgressiveBlur position="bottom" height="30%" />
</div>
```

### 顶部与底部 (Top and Bottom)

```tsx
<div className="relative h-[300px] overflow-auto">
  <ProgressiveBlur position="both" />
  <Content />
</div>
```

### 柔和淡出（较少的模糊层级） (Subtle Fade)

```tsx
<ProgressiveBlur
  blurLevels={[0.5, 1, 2, 4, 8]}
  height="20%"
  position="bottom"
/>
```

### 带有“查看更多”操作按钮 (With a "See More" CTA)

```tsx
<div className="relative h-[300px] overflow-hidden">
  <ProgressiveBlur position="bottom" height="40%">
    <button className="absolute bottom-4 left-1/2 -translate-x-1/2">
      显示更多 (Show more)
    </button>
  </ProgressiveBlur>
  <Content />
</div>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `position` | `"top" \| "bottom" \| "both"` | `"bottom"` | 应用模糊效果的位置 |
| `height` | `string` | `"30%"` | 模糊区域的高度（CSS 长度单位） |
| `blurLevels` | `number[]` | `[0.5, 1, 2, 4, 8, 16, 32, 64]` | 每层模糊效果的像素值 (px) |
| `children` | `ReactNode` | — | 模糊区域内的可选内容（例如操作按钮） |
| `className` | `string` | — | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 该组件默认设置了 `pointer-events-none`（穿透点击）。其中的子项是可交互的。
- 利用 `backdrop-filter` 配合渐变遮罩（gradient masks）来实现渐进式模糊效果。
- 已包含适配 Safari 浏览器的 `-webkit-backdrop-filter`。
- 当 `position="both"` 时，`height` 属性将被忽略，组件会自动填满整个容器。
