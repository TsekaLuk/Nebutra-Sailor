## 使用方法 (Usage)

```tsx
import { RadioGroupStacked, RadioGroupStackedItem } from "@nebutra/ui/primitives"
```

## 属性 (Props)

### RadioGroupStacked (单选组)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | 受控模式下的选中值 |
| `defaultValue` | `string` | — | 非受控模式下的默认选中值 |
| `onValueChange` | `(value: string) => void` | — | 选中项改变时的回调函数 |
| `disabled` | `boolean` | `false` | 禁用所有选项 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | `RadioGroupStackedItem` 元素集合 |

### RadioGroupStackedItem (单选项目)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | **必需。** 该选项对应的数值 |
| `id` | `string` | — | 用于无障碍标签关联的 ID |
| `disabled` | `boolean` | `false` | 禁用该特定选项 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 选项内容 |

## 示例 (Examples)

### 通知偏好设置 (Notification Preference)

```tsx
<RadioGroupStacked defaultValue="all">
  <RadioGroupStackedItem value="all" id="notif-all">
    <div>
      <div className="font-medium">所有通知 (All notifications)</div>
      <div className="text-sm text-muted-foreground">接收所有活动提醒。</div>
    </div>
  </RadioGroupStackedItem>
  <RadioGroupStackedItem value="mentions" id="notif-mentions">
    <div>
      <div className="font-medium">仅提及 (Mentions only)</div>
      <div className="text-sm text-muted-foreground">仅在被直接提及（@）时提醒。</div>
    </div>
  </RadioGroupStackedItem>
  <RadioGroupStackedItem value="none" id="notif-none">
    <div>
      <div className="font-medium">不接收通知 (No notifications)</div>
      <div className="text-sm text-muted-foreground">保持完全静默。</div>
    </div>
  </RadioGroupStackedItem>
</RadioGroupStacked>
```

## 无障碍支持 (Accessibility)

- 根容器容器带有 `role="radiogroup"` 属性。
- 每个选项项均带有 `role="radio"` 属性以及 `aria-checked` 状态。
- 选中项通过边框高亮进行视觉区分。
- 键盘导航：使用 `ArrowUp`/`ArrowDown` 键移动焦点，使用 `Space` 键进行选中。
