## 使用方法 (Usage)

```tsx
import { RadioGroup, RadioGroupItem } from "@nebutra/ui/primitives"
```

## 属性 (Props)

### RadioGroup (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | 受控的选中值 |
| `defaultValue` | `string` | — | 非受控的默认选中值 |
| `onValueChange` | `(value: string) => void` | — | 选中项改变时的回调函数 |
| `disabled` | `boolean` | `false` | 禁用所有单选条目 |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | 布局方向 |
| `className` | `string` | — | 额外的 CSS 类名 |

### RadioGroupItem (单选条目)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | **必填。** 该选项的值 |
| `disabled` | `boolean` | `false` | 禁用该特定条目 |
| `id` | `string` | — | 用于与 `<Label>` 关联 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 可访问性 (Accessibility)

- 基于 Radix UI RadioGroup 构建 —— 具有 `role="radiogroup"` 属性。
- 每个条目都具有 `role="radio"` 属性，并带有 `aria-checked`。
- 始终通过匹配的 `id`/`htmlFor` 将条目与 `<Label>` 配对。
- 键盘操作：使用 `ArrowUp`/`ArrowDown` (或 `ArrowLeft`/`ArrowRight`) 导航，使用 `Space` 选择。
