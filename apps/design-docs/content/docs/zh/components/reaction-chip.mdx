---
title: 表情回应 (ReactionChip)
description: 悬停显示的表情回应选择器，包含 ReactionBadge 显示组件以及预设的 MessageWithReactions 组件。模拟 Slack/Notion 的表情回应交互体验。
---

## 使用方法 (Usage)

```tsx
import { ReactionChip, ReactionBadge, MessageWithReactions } from "@nebutra/ui";
import { useState } from "react";

// 预设的带回应功能的消息组件
<MessageWithReactions text="这项功能做得太棒了！ 🚀" />

// 独立的表情选择器
const [selected, setSelected] = useState<string>();
<ReactionChip onSelect={setSelected} selected={selected} />
```

## 示例 (Examples)

### MessageWithReactions (预设组件)

最简单的方法 —— 一个在悬停时显示表情选择器的消息卡片：

```tsx
<MessageWithReactions text="这个新设计看起来非常棒！" />
```

自定义表情选项：

```tsx
<MessageWithReactions
  text="来看看这个新组件"
  reactionOptions={["🔥", "💯", "🙌", "✨", "👀"]}
/>
```

自定义子元素：

```tsx
<MessageWithReactions reactionOptions={["👍", "❤️", "😂"]}>
  <div className="flex items-center gap-2">
    <Avatar src="/user.jpg" />
    <p>带有丰富格式的自定义消息内容</p>
  </div>
</MessageWithReactions>
```

### 独立的 ReactionChip (Standalone ReactionChip)

```tsx
const [selected, setSelected] = useState<string | undefined>();

<div className="relative inline-block">
  <div className="rounded-lg border p-3 text-sm">
    悬停以进行回应
  </div>
  <div className="absolute -top-3 right-0">
    <ReactionChip
      emojis={["👍", "❤️", "😂", "🎉", "🚀"]}
      onSelect={setSelected}
      selected={selected}
    />
  </div>
</div>
```

### ReactionBadge (回应徽章)

在消息下方显示各项回应的数量：

```tsx
<div className="flex gap-2 mt-2">
  <ReactionBadge emoji="👍" count={12} />
  <ReactionBadge emoji="❤️" count={5} />
  <ReactionBadge emoji="🎉" count={3} bump={isBumping} />
</div>
```

## 属性 (Props)

### ReactionChip

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `onSelect` | `(emoji: string) => void` | — | 点击表情时的回调函数 |
| `emojis` | `string[]` | `["👍", "❤️", "😂", "🎉"]` | 可选的表情列表 |
| `selected` | `string` | — | 当前选中的表情（会高亮显示） |
| `className` | `string` | — | 额外的类名 |

### ReactionBadge

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `emoji` | `string` | — | 要显示的表情 |
| `count` | `number` | — | 该表情的回应次数 |
| `bump` | `boolean` | `false` | 触发一次短暂的缩放动画效果 |
| `className` | `string` | — | 额外的类名 |

### MessageWithReactions

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `text` | `string` | — | 消息文本内容 |
| `reactionOptions` | `string[]` | `["👍", "❤️", "😂", "🎉"]` | 可选的表情列表 |
| `children` | `ReactNode` | — | 替换 `text` 的自定义内容 |
| `className` | `string` | — | 额外的类名 |

## 注意事项 (Notes)

- `ReactionChip` 在表情点击时使用 `onMouseDown` 配合 `preventDefault` 处理，以防止在选中表情前触发元素的焦点丢失。
- 在 `MessageWithReactions` 中添加回应时，会有持续 180ms 的“脉冲”（bump）动画效果。
- 两次点击同一个表情会取消选中（切换开关逻辑）。
- 无障碍支持：在选择器按钮上设置了 `role="group"`、`aria-label` 以及 `aria-pressed`。
