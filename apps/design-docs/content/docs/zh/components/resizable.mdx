## 使用方法 (Usage)

```tsx
import {
  ResizablePanelGroup,
  ResizablePanel,
  ResizableHandle,
} from "@nebutra/ui/primitives"
```

## 属性 (Props)

### ResizablePanelGroup (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `direction` | `"horizontal" \| "vertical"` | — | **必填。** 分割轴方向 |
| `onLayout` | `(sizes: number[]) => void` | — | 面板尺寸改变时的回调函数 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | `ResizablePanel` 和 `ResizableHandle` 元素 |

### ResizablePanel (面板)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `defaultSize` | `number` | — | 初始尺寸（百分比） |
| `minSize` | `number` | `0` | 最小尺寸（百分比） |
| `maxSize` | `number` | `100` | 最大尺寸（百分比） |
| `collapsible` | `boolean` | `false` | 面板是否可以折叠 |
| `collapsedSize` | `number` | `0` | 折叠时的尺寸（百分比） |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 面板内容 |

### ResizableHandle (调节手柄)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `withHandle` | `boolean` | `false` | 显示可视化的拖拽手柄指示器 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 可访问性 (Accessibility)

- `ResizableHandle` 具有 `role="separator"` 属性，并带有 `aria-orientation`。
- 键盘操作：使用 `ArrowLeft`/`ArrowRight` (或 `ArrowUp`/`ArrowDown`) 进行调节，使用 `Home`/`End` 调至最小/最大。
- 每个手柄都具有 `aria-label`，用于描述它所分割的面板。
