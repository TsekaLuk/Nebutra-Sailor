## 概览 (Overview)

`Safari` 渲染一个 macOS Safari 浏览器窗口的 SVG 框架。与 `BrowserMockup` 不同，它采用细节丰富的 SVG，在 `"default"` 模式下拥有原生风格的工具栏图标（前进/后退、分享、标签页），或在 `"simple"` 模式下提供一个清爽简约的地址栏。

内容（图片或视频）会被裁剪到工具栏下方的浏览器视口区域。

## 使用方法 (Usage)

```tsx
import { Safari } from "@nebutra/ui/primitives";

// 仅显示 URL (无内容)
<div className="w-[800px]">
  <Safari url="https://nebutra.com" />
</div>

// 带有截图
<div className="w-[800px]">
  <Safari
    url="https://app.nebutra.com"
    imageSrc="/screenshots/dashboard.png"
  />
</div>

// 简约模式 (清爽工具栏)
<div className="w-[800px]">
  <Safari
    url="https://nebutra.com"
    imageSrc="/screenshots/homepage.png"
    mode="simple"
  />
</div>

// 带有视频
<div className="w-[800px]">
  <Safari
    url="https://nebutra.com"
    videoSrc="/demos/product.webm"
  />
</div>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `url` | `string` | — | 地址栏中显示的 URL |
| `imageSrc` | `string` | — | 截图或图片的 URL |
| `videoSrc` | `string` | — | 视频 URL（自动播放、循环、静音） |
| `mode` | `"default" \| "simple"` | `"default"` | 工具栏的细节层级 |
| `className` | `string` | — | 容器的 CSS 类名 |

## 注意事项 (Notes)

- `videoSrc` 的优先级高于 `imageSrc`。
- 视频播放时附带 `autoPlay loop muted playsInline` 属性。
- 框架宽高比固定为 `1203/753`；可通过容器宽度控制整体尺寸。
- 框架通过 Tailwind 的 `dark:fill-*` 类名自动适配深色模式。
- 内容视口起始于 `y=52`（即工具栏下方），并具有圆角底边。
