## 使用方法 (Usage)

```tsx
import { ScrollArea, ScrollBar } from "@nebutra/ui/primitives"
```

## 属性 (Props)

### ScrollArea (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `type` | `"auto" \| "always" \| "scroll" \| "hover"` | `"hover"` | 何时显示滚动条 |
| `scrollHideDelay` | `number` | `600` | 隐藏滚动条前的延迟（毫秒） |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 可滚动内容 |

### ScrollBar (滚动条)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | 滚动条方向 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 可访问性 (Accessibility)

- 滚动区域的内容对键盘和屏幕阅读器用户均可访问 —— 保留了原生的滚动语义。
- 自定义滚动条带有 `aria-hidden="true"` 属性 —— 仅作为视觉参考。
- 焦点保留在滚动区域内的内容元素上，而不是滚动条本身。
