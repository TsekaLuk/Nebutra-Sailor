## 概览 (Overview)

`ScrollVelocityContainer` 与 `ScrollVelocityRow` 共同构建出一组走马灯风格的水平内容条，其滚动速度会根据您的页面滚动速度动态增减。

核心特性：
- 移出视野时自动暂停（基于 `IntersectionObserver`）
- 页面标签页隐藏时自动暂停（基于 `visibilitychange` 事件）
- 尊重用户的“减弱动画” (`prefers-reduced-motion`) 偏好设置
- 通过 `ResizeObserver` 测量实现无缝循环滚动，无需手动复制内容

## 使用方法 (Usage)

```tsx
import { ScrollVelocityContainer, ScrollVelocityRow } from "@nebutra/ui/primitives";

// 共享速度的双行布局
<ScrollVelocityContainer className="text-2xl font-bold">
  <ScrollVelocityRow baseVelocity={8} direction={1}>
    <span className="mx-8">Next.js</span>
    <span className="mx-8">TypeScript</span>
    <span className="mx-8">Tailwind</span>
  </ScrollVelocityRow>
  <ScrollVelocityRow baseVelocity={8} direction={-1}>
    <span className="mx-8">React</span>
    <span className="mx-8">Framer Motion</span>
    <span className="mx-8">Storybook</span>
  </ScrollVelocityRow>
</ScrollVelocityContainer>

// 独立行 (使用自身计算的速度)
<ScrollVelocityRow baseVelocity={15}>
  <span className="mx-8">Logo</span>
  <span className="mx-8">Partner</span>
</ScrollVelocityRow>
```

## 属性 (Props)

### `ScrollVelocityContainer` (容器)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | `ScrollVelocityRow` 类型的子组件 |
| `className` | `string` | — | 额外的 CSS 类名 |

为所有子组件行提供一个经过平滑处理 (`useSpring`) 的共享滚动速度。

### `ScrollVelocityRow` (走马灯行)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 走马灯中需要循环展示的内容 |
| `baseVelocity` | `number` | `5` | 基础速度（每秒移动内容宽度的百分比） |
| `direction` | `1 \| -1` | `1` | 滚动方向：`1` 为由左向右，`-1` 为由右向左 |
| `className` | `string` | — | 行容器的 CSS 类名 |

## 注意事项 (Notes)

- 内容会在内部自动复制以填满容器 —— 您无需手动重复输入子元素。
- 建议使用 `ScrollVelocityContainer` 包裹，以获得“滚动加速”效果；虽然独立行也有本地滚动追踪，但通常需要父级滚动环境支持。
- `baseVelocity` 数值越高，基础滚动越快；页面滚动最高可将其速度放大 5 倍。
