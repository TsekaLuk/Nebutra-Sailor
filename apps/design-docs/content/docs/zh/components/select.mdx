## 使用方法 (Usage)

```tsx
import {
  Select,
  SelectTrigger,
  SelectValue,
  SelectContent,
  SelectItem,
  SelectGroup,
  SelectLabel,
  SelectSeparator,
  SelectScrollUpButton,
  SelectScrollDownButton,
} from "@nebutra/ui/primitives"
```

## 属性 (Props)

### Select (根组件)

继承自 `@radix-ui/react-select` 的 `Root` 属性.

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | 受控的选中值 |
| `defaultValue` | `string` | — | 非受控的默认值 |
| `onValueChange` | `(value: string) => void` | — | 选中改变时的回调函数 |
| `open` | `boolean` | — | 受控的打开状态 |
| `onOpenChange` | `(open: boolean) => void` | — | 打开状态改变时的回调函数 |
| `disabled` | `boolean` | `false` | 禁用选择器 |

### SelectTrigger (触发器)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 触发器内容 (通常为 `SelectValue`) |

### SelectValue (显示值)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `placeholder` | `string" | — | 未选中任何值时显示的占位文本 |

### SelectContent (内容区)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string" | — | 额外的 CSS 类名 |
| `position` | `"item-aligned" \| "popper"` | `"popper"` | 定位策略 |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | 首选打开侧 |
| `align` | `"start" \| "center" \| "end"` | `"start"` | 沿触发器的对齐方式 |

### SelectItem (选择条目)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string" | — | **必填。** 该选项的值 |
| `disabled` | `boolean" | `false` | 防止该条目被选中 |
| `className` | `string" | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 条目标签文本 |

### SelectGroup / SelectLabel / SelectSeparator (分组/标签/分隔线)

| 组件 | 描述 |
|-----------|-------------|
| `SelectGroup` | 将相关的条目分在一起 |
| `SelectLabel` | 组的非可选标签 |
| `SelectSeparator` | 组之间的视觉分隔线 |

## 可访问性 (Accessibility)

- 基于 Radix UI Select 构建 —— 遵循 WAI-ARIA [列表框模式 (Listbox pattern)](https://www.w3.org/WAI/ARIA/apg/patterns/listbox/)。
- `SelectTrigger` 具有 `role="combobox"` 属性，并带有 `aria-expanded` 和 `aria-haspopup="listbox"`。
- `SelectContent` 具有 `role="listbox"` 属性。
- 每个 `SelectItem` 具有 `role="option"` 属性和 `aria-selected`。
- 完善的键盘导航：使用 `ArrowUp`/`ArrowDown` 导航，使用 `Enter`/`Space` 选择，使用 `Escape` 关闭。
