## 使用方法 (Usage)

```tsx
import { Separator } from "@nebutra/ui/primitives"
```

## 属性 (Props)

原生的 HTML `<hr>` 分隔符。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 分隔线的方向 |
| `decorative` | `boolean` | `true` | 为 `true` 时，分隔线具有 `aria-hidden` 属性（仅用于装饰） |
| `className` | `string` | — | 额外的 CSS 类名 |

## 可访问性 (Accessibility)

- 默认情况下 `decorative={true}` 会设置 `aria-hidden="true"` 和 `role="none"` —— 屏幕阅读器不可见。
- `decorative={false}` 会设置 `role="separator"` 并带有 `aria-orientation`，用于标识语义上的内容边界。
- 仅当分隔线传达有意义的结构信息时，才使用 `decorative={false}`。
