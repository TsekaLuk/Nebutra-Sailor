## 属性 (Props)

### Sheet (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `open` | `boolean` | — | 受控的打开状态 |
| `defaultOpen` | `boolean` | `false` | 非受控的默认打开状态 |
| `onOpenChange` | `(open: boolean) => void` | — | 打开状态改变时的回调函数 |

### SheetContent (内容区)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"right"` | 抽屉滑出的边缘 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 抽屉内容 |

## 可访问性 (Accessibility)

- 基于 Radix UI Dialog 构建 —— `SheetContent` 具有 `role="dialog"` 和 `aria-modal="true"` 属性。
- `SheetTitle` 通过 `aria-labelledby` 进行关联。
- 打开时焦点会被锁定在抽屉内部。
- 按下 `Escape` 键会关闭抽屉并将焦点返回触发器。
