## 使用方法 (Usage)

```tsx
import { ShineBorder } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 覆盖模式 (Overlay Mode - 无子元素)

为任何现有元素添加发光效果：

```tsx
<div className="relative rounded-xl p-6">
  <ShineBorder shineColor={["#A07CFE", "#FE8FB5", "#FFBE7B"]} />
  <p>此处放置您的内容</p>
</div>
```

### 容器模式 (Container Mode - 带有子元素)

带有发光边框的独立卡片：

```tsx
<ShineBorder
  shineColor="#A07CFE"
  borderRadius={16}
  borderWidth={2}
>
  <h2>发光卡片</h2>
  <p>内部内容</p>
</ShineBorder>
```

### 多色渐变 (Multi-Color Gradient)

```tsx
<ShineBorder shineColor={["#FF6B6B", "#FFE66D", "#4ECDC4"]}>
  <span>彩虹边框</span>
</ShineBorder>
```

### 更快的动画速度 (Faster Animation)

```tsx
<ShineBorder shineColor="#6366f1" duration={6}>
  内容
</ShineBorder>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `shineColor` | `string \| string[]` | — | 发光渐变的颜色（单个或数组） |
| `color` | `string \| string[]` | `"#000000"` | `shineColor` 的已弃用别名 |
| `borderWidth` | `number` | `1` | 边框宽度（像素） |
| `duration` | `number` | `14` | 动画循环周期（秒） |
| `borderRadius` | `number` | `8` | 边框圆角（仅限容器模式） |
| `children` | `ReactNode` | — | 包含子元素时将切换至容器模式 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 模式说明 (Modes)

| 模式 | 触发条件 | 定位方式 |
|------|----------|-------------|
| **覆盖 (Overlay)** | 无 `children` | `absolute inset-0`，继承父级的 `border-radius` |
| **容器 (Container)** | 带有 `children` | 独立的 `div` 容器，包裹内部内容 |

## 注意事项 (Notes)

- 需要在 Tailwind 配置或全局 CSS 中定义 `animate-shine` 关键帧动画。
- 自动应用 `motion-safe:` 前缀 —— 当系统开启“减弱动画”偏好设置时，动画将自动禁用。
