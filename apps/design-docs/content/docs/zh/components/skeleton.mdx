## 使用方法 (Usage)

```tsx
import { Skeleton } from "@nebutra/ui/primitives"
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `string` | — | **必填。** 通过 Tailwind 类设置宽度、高度和圆角。 |

<Info>
  `Skeleton` 没有固定尺寸 —— 请始终通过 `className` 提供 `w-*`、`h-*` 以及可选的 `rounded-*` 工具类，以匹配它所替换的内容形状。
</Info>

### 条件加载 (Conditional Loading)

```tsx
function UserCard({ userId }: { userId: string }) {
  const { data: user, isLoading } = useUser(userId)

  if (isLoading) {
    return (
      <div className="flex items-center space-x-4">
        <Skeleton className="h-12 w-12 rounded-full" />
        <div className="space-y-2">
          <Skeleton className="h-4 w-48" />
          <Skeleton className="h-4 w-36" />
        </div>
      </div>
    )
  }

  return <UserInfo user={user} />
}
```

## 可访问性 (Accessibility)

- 骨架屏元素默认具有 `aria-hidden="true"` 属性 —— 它们仅作为装饰性的加载指示器。
- 在容器上包裹 `aria-busy="true"` 以播报加载状态。
- 对于复杂的骨架屏布局，请同时提供可见或仅供屏幕阅读器识读的 "正在加载... (Loading...)" 标签。

```tsx
<div aria-busy="true" aria-label="正在加载用户资料 (Loading user profile)">
  <Skeleton className="h-12 w-12 rounded-full" />
</div>
```
