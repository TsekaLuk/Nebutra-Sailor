## 概览 (Overview)

`SnippetStatus` 是一款受 Vercel 设计系统启发的精简型代码片段组件。它具有状态配色背景（成功/警告/错误）、`$` 提示符前缀、对多行文本的支持，以及带有图标点击切换动画的剪切板复制功能。

与通用的 `Snippet` 组件不同，`SnippetStatus` 使用了 Vercel 风格的颜色令牌，并内置了用于图标过渡的 CSS 关键帧动画。

## 安装 (Installation)

```tsx
import { SnippetStatus } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础用法 (Basic)

```tsx
<SnippetStatus text="npm install package" />
```

### 状态变体 (Status variants)

```tsx
<SnippetStatus text="部署完成 (Deployment complete)" type="success" />
<SnippetStatus text="构建超时，已持续 30s" type="warning" />
<SnippetStatus text="错误：找不到模块 (Error: module not found)" type="error" />
```

### 深色模式 (Dark mode)

```tsx
<SnippetStatus text="npm run dev" dark />
```

### 多行 (Multiline)

```tsx
<SnippetStatus text={["npm install", "npm run dev"]} />
```

### 无提示符 (Without prompt)

```tsx
<SnippetStatus text="const x = 1" prompt={false} />
```

### 复制回调 (Copy callback)

```tsx
<SnippetStatus
  text="git clone https://github.com/org/repo"
  onCopy={() => console.log("已复制！")}
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `text` | `string \| string[]` | — | **必需。** 代码文本或行数组 |
| `width` | `string` | `"100%"` | 容器的 CSS 宽度 |
| `onCopy` | `() => void` | — | 文本被复制时的回调函数 |
| `prompt` | `boolean` | `true` | 是否在每行前显示 `$` 前缀 |
| `dark` | `boolean` | `false` | 使用深色/反色主题 |
| `type` | `"success" \| "warning" \| "error"` | — | 状态色变体 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 颜色变体对照表 (Color Variants)

| 类型 (Type) | 背景色 | 文本色 |
|------|-----------|------|
| 默认 (default) | `bg-background-100` | `text-gray-1000` |
| 反色 (inverted/dark) | `bg-gray-1000` | `text-gray-100` |
| 成功 (success) | `bg-blue-100` | `text-blue-900` |
| 警告 (warning) | `bg-amber-100` | `text-amber-900` |
| 错误 (error) | `bg-red-100` | `text-red-900` |

## 注意事项 (Notes)

- 复制动画使用了通过 `<style>` 标签动态注入的内联 `@keyframes` —— 无需外部动画库。
- 点击后，复制图标会淡出 (`animate-snippet-fade-out`)，同时勾选图标会淡入 (`animate-snippet-fade-in`)，持续时间为 2 秒。
- 如果动画未生效，请确保在全局 CSS 中添加了 `animate-snippet-fade-out` 和 `animate-snippet-fade-in` 的关键帧定义。
