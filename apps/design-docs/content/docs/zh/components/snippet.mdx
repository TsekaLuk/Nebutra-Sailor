## 概览 (Overview)

`Snippet` 组件用于渲染带有 `$` 前缀和复制按钮的代码。支持单行及多行代码展示，提供 4 种视觉变体、6 种配色方案、3 种尺寸规格，且支持自定义圆角。

## 安装 (Installation)

```tsx
import { Snippet } from "@nebutra/ui/primitives";
```

## 使用方法 (Usage)

### 基础用法 (Basic)

```tsx
<Snippet>npm install @nebutra/ui</Snippet>
```

### 隐藏符号 (Without symbol)

```tsx
<Snippet hideSymbol>const x = 1;</Snippet>
```

### 自定义符号 (Custom symbol)

```tsx
<Snippet symbol=">">{`git clone https://github.com/org/repo`}</Snippet>
```

### 多行 (Multiline)

```tsx
<Snippet>
  {`npm install package
yarn add package
pnpm add package`}
</Snippet>
```

### 状态配色变体 (Colored variants)

```tsx
<Snippet color="success">部署成功！ (Deployment successful!)</Snippet>
<Snippet color="danger">构建失败：缺少依赖 (Build failed: missing dependency)</Snippet>
<Snippet color="warning">使用了已弃用的 API (Deprecated API in use)</Snippet>
```

### 视觉变体 (Visual variants)

```tsx
<Snippet variant="flat">扁平化背景 (Flat)</Snippet>
<Snippet variant="bordered">边框模式 (Bordered)</Snippet>
<Snippet variant="shadow">带有阴影 (Shadow)</Snippet>
<Snippet variant="solid">实心反白 (Inverted solid)</Snippet>
```

### 隐藏复制按钮 (Without copy button)

```tsx
<Snippet hideCopyButton>只读代码</Snippet>
```

### 复制回调 (Copy callback)

```tsx
<Snippet onCopy={(text) => console.log("已复制:", text)}>
  npm run dev
</Snippet>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 代码内容 |
| `codeString` | `string` | — | 覆盖实际复制的文本（若与 children 展示的不同） |
| `variant` | `"flat" \| "bordered" \| "shadow" \| "solid"` | `"flat"` | 视觉风格 |
| `color` | `"default" \| "primary" \| "secondary" \| "success" \| "warning" \| "danger"` | `"default"` | 配色方案 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 尺寸大小 |
| `radius` | `"none" \| "sm" \| "md" \| "lg" \| "full"` | `"lg"` | 边框圆角 |
| `symbol` | `string \| ReactNode` | `"$"` | 前缀符号 |
| `hideSymbol` | `boolean` | `false` | 是否隐藏符号前缀 |
| `hideCopyButton` | `boolean` | `false` | 是否隐藏复制按钮 |
| `disableCopy` | `boolean` | `false` | 禁用复制功能 |
| `copyIcon` | `ReactNode` | `<Copy />` | 自定义复制图标 |
| `checkIcon` | `ReactNode` | `<Check />` | 自定义复制后（勾选）状态图标 |
| `timeout` | `number` | `2000` | 复制状态重置前的持续时间 (ms) |
| `onCopy` | `(value: string) => void` | — | 文本被复制时的回调函数 |
| `className` | `string` | — | 额外的 CSS 类名 |
