## 演示 (Demo)

```tsx
import { toast } from "sonner"
import { Button } from "@nebutra/ui/primitives"
import { Toaster } from "@nebutra/ui/primitives"

export function SonnerDemo() {
  return (
    <>
      <Toaster />
      <div className="flex flex-wrap gap-2">
        <Button onClick={() => toast("事件已创建 (Event has been created)")}>
          默认通知 (Default Toast)
        </Button>
        <Button onClick={() => toast.success("个人资料已更新！ (Profile updated!)")}>
          成功 (Success)
        </Button>
        <Button onClick={() => toast.error("出错了。 (Something went wrong.)")}>
          错误 (Error)
        </Button>
        <Button onClick={() => toast.warning("磁盘空间不足。 (Low disk space.)")}>
          警告 (Warning)
        </Button>
        <Button onClick={() => toast.info("有新版本可用。 (New update available.)")}>
          信息 (Info)
        </Button>
      </div>
    </>
  )
}
```

## 使用方法 (Usage)

```tsx
// 1. 将 Toaster 添加到您的根布局 (仅需添加一次):
import { Toaster } from "@nebutra/ui/primitives"
export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Toaster />
      </body>
    </html>
  )
}

// 2. 在应用中的任何位置触发通知:
import { toast } from "sonner"
toast("你好，世界 (Hello world)")
```

## 属性 (Props)

### Toaster (通知容器)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `position` | `"top-left" \| "top-center" \| "top-right" \| "bottom-left" \| "bottom-center" \| "bottom-right"` | `"bottom-right"` | 在屏幕上的显示位置 |
| `expand` | `boolean` | `false` | 是否始终展开显示所有通知 |
| `richColors` | `boolean` | `false` | 是否使用丰富的语义化颜色 |
| `duration` | `number` | `4000` | 默认持续时间（毫秒） |
| `closeButton` | `boolean` | `false` | 是否在所有通知上显示关闭按钮 |
| `theme` | `"light" \| "dark" \| "system"` | `"system"` | 颜色主题 |
| `toastOptions` | `ToastOptions` | — | 所有通知的默认选项 |

## 示例 (Examples)

### Promise 通知 (Promise Toast)

```tsx
toast.promise(saveUser(userData), {
  loading: "正在保存... (Saving...)",
  success: "个人资料更新成功！ (Profile updated successfully!)",
  error: "保存个人资料失败。 (Failed to save profile.)",
})
```

### 带有操作的通知 (Toast with Action)

```tsx
toast("文件已删除 (File deleted)", {
  action: {
    label: "撤销 (Undo)",
    onClick: () => restoreFile(),
  },
  duration: 5000,
})
```

### 自定义描述通知 (Custom Description Toast)

```tsx
toast("计划维护 (Scheduled maintenance)", {
  description: "系统将在 UTC 时间凌晨 2-4 点下线。 (The system will be down from 2–4 AM UTC.)",
})
```

### 以编程方式关闭 (Dismiss Programmatically)

```tsx
const toastId = toast.loading("处理中... (Processing...)")
await doSomething()
toast.dismiss(toastId)
toast.success("完成！ (Done!)")
```

## 可访问性 (Accessibility)

- 通知渲染在具有 `aria-label="通知 (Notifications)"` 的 `<section>` 元素中。
- 每个通知具有 `role="status"` 属性（错误通知为 `role="alert"`），并带有 `aria-live`。
- 通知出现时会向屏幕阅读器播报。
- `Toaster` 会附加在文档底部 —— 处于模态框焦点锁定范围之外。
