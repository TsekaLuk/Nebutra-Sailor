            <StarsCanvas

                hue={280} // Purple-ish stars
                brightness={0.8}
                speedMultiplier={0.8}
                position="absolute"
                zIndex={1}

            <div className="relative z-10 flex flex-col items-center gap-6 p-6 text-center">
                <div className="space-y-2 max-w-md">
                    <h2 className="text-4xl font-bold tracking-tighter text-white">Atmospheric Effects</h2>
                    <p className="text-zinc-400">
                        Create immersive depth with performant HTML5 canvas particle systems.
                        Perfect for hero sections and dark mode experiences.
                    </p>
                </div>

                <Button
                    variant="outline"
                    size="sm"
                    className="bg-black/50 border-zinc-800 text-white hover:bg-white hover:text-black transition-colors"
                    onClick={() => setPaused(!paused)}
                    prefix={paused ? <Play className="w-4 h-4" /> : <Pause className="w-4 h-4" />}
                >
                    {paused ? "Resume Animation" : "Pause Animation"}
                </Button>

        </div>

## 使用方法 (Usage)

```tsx
import { StarsCanvas } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 全屏背景 (Full Page Background)

```tsx
<>
  <StarsCanvas />
  <main className="relative z-10">内容区域</main>
</>
```

### 局部容器 (Contained Section)

```tsx
<div className="relative overflow-hidden rounded-xl h-[400px]">
  <StarsCanvas position="absolute" zIndex={0} />
  <div className="relative z-10 p-8">内部内容</div>
</div>
```

### 紫色繁星 (Purple Stars)

```tsx
<StarsCanvas hue={280} brightness={0.8} />
```

### 缓慢且密集的星场 (Slow Dense Starfield)

```tsx
<StarsCanvas
  maxStars={2000}
  speedMultiplier={0.3}
  twinkleIntensity={10}
/>
```

### 透明模式（用于覆盖） (Transparent - overlay)

```tsx
<StarsCanvas transparent position="absolute" />
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `maxStars` | `number` | `1200` | 星星的总数量 |
| `hue` | `number` | `217` | 星星颜色的色相 (0–360) |
| `brightness` | `number` | `1` | 亮度倍率 (0–1) |
| `speedMultiplier` | `number` | `1` | 动画速度倍率 |
| `twinkleIntensity` | `number` | `20` | 闪烁频率（数值越低，闪烁越频繁） |
| `transparent` | `boolean` | `false` | 背景透明（用于在其他元素上层覆盖） |
| `paused` | `boolean` | `false` | 暂停动画 |
| `position` | `"fixed" \| "absolute" \| "relative"` | `"fixed"` | 画布的定位方式 |
| `zIndex` | `number` | `-1` | Z 轴层级 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 默认情况下使用 `position="fixed"`，画布会填充整个视口作为页面背景。
- 在 `relative overflow-hidden` 容器内使用 `position="absolute"` 可实现局部背景效果。
- 星星会绕着画布中心公转，营造出类似视差的深度感。
- 窗口尺寸调整时会自动调整画布大小。
