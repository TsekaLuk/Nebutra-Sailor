## 使用方法 (Usage)

```tsx
import { Switch } from "@nebutra/ui/primitives"
```

## 属性 (Props)

继承自 `@radix-ui/react-switch` 的 `Root` 属性。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `checked` | `boolean` | — | 受控的选中状态 |
| `defaultChecked` | `boolean` | `false` | 非受控的默认选中状态 |
| `onCheckedChange` | `(checked: boolean) => void` | — | 状态改变时的回调函数 |
| `disabled` | `boolean` | `false` | 禁用开关 |
| `required` | `boolean` | `false` | 在表单上下文中标记为必填 |
| `name` | `string` | — | 表单字段名称 |
| `value` | `string` | `"on"` | 随表单提交的值 |
| `className` | `string` | — | 额外的 CSS 类名 |

### 在表单中使用 (In a Form)

```tsx
<FormField
  control={form.control}
  name="marketingEmails"
  render={({ field }) => (
    <FormItem className="flex items-center justify-between rounded-lg border p-4">
      <div>
        <FormLabel>营销邮件 (Marketing emails)</FormLabel>
        <FormLabel>获取我们的最新消息和优惠。 (Receive our latest news and offers.)</FormLabel>
      </div>
      <FormControl>
        <Switch checked={field.value} onCheckedChange={field.onChange} />
      </FormControl>
    </FormItem>
  )}
/>
```

## 可访问性 (Accessibility)

- 基于 Radix UI Switch 构建 —— 具有 `role="switch"` 属性并带有 `aria-checked`。
- 始终通过 `htmlFor`/`id` 将开关与 `<Label>` 配对。
- 键盘操作：使用 `Space` 键切换开关。
- 通过 `aria-checked="true"` / `"false"` 向屏幕阅读器传达状态变化。
