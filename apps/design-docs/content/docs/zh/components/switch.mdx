
## 属性 (Props)

### Switch (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `name` | `string` | `"default"` | 内部单选输入组合的表单名称 |
| `size` | `"small" \| "medium" \| "large"` | `"medium"` | 定义控件的大小 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `style` | `React.CSSProperties` | — | 额外的内联样式 |

### Switch.Control (子控件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `label` | `string` | — | 控件选项的文本标签 |
| `value` | `string` | — | **必填。** 该特定选项的值 |
| `defaultChecked` | `boolean` | `false` | 此选项是否应默认选中 |
| `disabled` | `boolean` | `false` | 禁用此特定控件 |
| `icon` | `ReactNode` | — | 可选图标，显示以代替文本标签 |
