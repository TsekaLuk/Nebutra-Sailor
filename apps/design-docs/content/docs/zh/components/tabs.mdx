## 使用方法 (Usage)

```tsx
import {
  Tabs,
  TabsList,
  TabsTrigger,
  TabsContent,
} from "@nebutra/ui/primitives"
```

## 属性 (Props)

### Tabs (根组件)

基于 Base UI 标签页组件构建。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `defaultValue` | `string` | — | 非受控的默认激活选项卡 |
| `value` | `string` | — | 受控的激活选项卡值 |
| `onValueChange` | `(value: string) => void` | — | 激活选项卡改变时的回调函数 |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 选项卡列表的方向 |
| `className` | `string` | — | 额外的 CSS 类名 |

### TabsList (选项卡列表)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `variant` | `"default" \| "button" \| "line"` | `"default"` | 选项卡列表的视觉样式 |
| `size` | `"xs" \| "sm" \| "md" \| "lg"` | `"md"` | 选项卡的大小 |
| `shape` | `"default" \| "pill"` | `"default"` | 选项卡触发器的形状 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | `TabsTrigger` 元素 |

### TabsTrigger (选项卡触发器)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | **必填。** 匹配 `TabsContent` 的唯一标识符 |
| `disabled` | `boolean` | `false` | 防止选中此选项卡 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 选项卡标签（文本 + 可选图标） |

### TabsContent (选项卡内容)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `value` | `string` | — | **必填。** 匹配 `TabsTrigger` 的值 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 该选项卡激活时显示的内容 |

### 受控选项卡 (Controlled Tabs)

```tsx
const [activeTab, setActiveTab] = useState("overview")

<Tabs value={activeTab} onValueChange={setActiveTab}>
  <TabsList>
    <TabsTrigger value="overview">概览 (Overview)</TabsTrigger>
    <TabsTrigger value="details">详情 (Details)</TabsTrigger>
  </TabsList>
</Tabs>
```

## 可访问性 (Accessibility)

- 基于 Radix UI Tabs 构建 —— 遵循 WAI-ARIA [选项卡模式](https://www.w3.org/WAI/ARIA/apg/patterns/tabs/)。
- `TabsList` 具有 `role="tablist"` 属性。
- 每个 `TabsTrigger` 具有 `role="tab"`、`aria-selected` 和 `aria-controls` 属性。
- 每个 `TabsContent` 具有 `role="tabpanel"` 和 `aria-labelledby` 属性。
- 键盘导航：使用 `ArrowLeft`/`ArrowRight` 在选项卡间切换，使用 `Home`/`End` 跳转至首个/最后一个。
