## 使用方法 (Usage)

```tsx
import { TextLoop } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 基础旋转词汇 (Basic Rotating Words)

```tsx
<TextLoop>
  {["设计师", "开发者", "创始人"].map(t => (
    <span key={t}>{t}</span>
  ))}
</TextLoop>
```

### 与静态文本混排 (Inline With Static Text)

```tsx
<p className="text-2xl">
  专为{" "}
  <TextLoop interval={1.5}>
    {["速度", "扩展性", "易用性"].map(t => (
      <span key={t} className="font-bold text-primary">{t}</span>
    ))}
  </TextLoop>
  打造
</p>
```

### 自定义过渡效果 (Custom Transition)

```tsx
<TextLoop
  interval={3}
  transition={{ duration: 0.5, ease: "easeInOut" }}
>
  {["你好", "Bonjour", "Hola", "Ciao"].map(t => (
    <span key={t}>{t}</span>
  ))}
</TextLoop>
```

### 带有索引回调 (With Index Callback)

```tsx
<TextLoop onIndexChange={(i) => console.warn("当前索引:", i)}>
  {items.map(item => <span key={item.id}>{item.label}</span>)}
</TextLoop>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode[]` | — | 要循环播放的项目数组 |
| `interval` | `number` | `2` | 每次切换之间的间隔秒数 |
| `transition` | `Transition` | `{ duration: 0.3 }` | Framer Motion 过渡配置 |
| `variants` | `Variants` | 向上/向下滑动 | 自定义 Framer Motion 动画变体 |
| `onIndexChange` | `(index: number) => void` | — | 当前激活索引变更时的回调函数 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 默认动画 (Default Animation)

项目在进入时向上滑动，在退出时也向上滑动（类似于垂直滚轮的感觉）：

```
initial: { y: 20, opacity: 0 }
animate: { y: 0, opacity: 1 }
exit:    { y: -20, opacity: 0 }
```
