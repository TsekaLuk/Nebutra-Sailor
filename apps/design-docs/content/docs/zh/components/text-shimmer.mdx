## 使用方法 (Usage)

```tsx
import { TextShimmer } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 加载状态 (Loading State)

```tsx
<TextShimmer duration={1}>正在生成代码... (Generating code...)</TextShimmer>
```

### AI 输入指示器 (AI Typing Indicator)

```tsx
<TextShimmer duration={1.5} className="text-sm text-muted-foreground">
  Claude 正在思考中... (Claude is thinking...)
</TextShimmer>
```

### 通过 CSS 变量自定义颜色 (Custom Colors via CSS Variables)

```tsx
<TextShimmer
  duration={1.2}
  className="text-xl [--base-color:theme(colors.blue.600)] [--base-gradient-color:theme(colors.blue.200)]"
>
  高级功能 (Premium Feature)
</TextShimmer>
```

### 作为标题渲染 (As Heading)

```tsx
<TextShimmer as="h1" className="text-4xl font-bold" duration={3}>
  欢迎 (Welcome)
</TextShimmer>
```

### 暂停状态（静态显示） (Paused)

```tsx
<TextShimmer paused>静态文本 (Static Text)</TextShimmer>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `string` | — | 文本内容（必须是字符串） |
| `as` | `React.ElementType` | `"p"` | 渲染为指定的 HTML 元素 |
| `duration` | `number` | `2` | 微光扫过一周的持续时间（秒） |
| `spread` | `number` | `2` | 微光束的扩散范围（与文本长度成比例） |
| `paused` | `boolean` | `false` | 是否暂停动画 |
| `className` | `string` | — | 额外的 CSS 类名 |

## CSS 变量 (CSS Variables)

可以通过覆盖 CSS 变量来自定义微光颜色：

| 变量 | 默认值 (浅色) | 默认值 (深色) | 用途 |
|----------|-----------------|----------------|---------|
| `--base-color` | `#a1a1aa` | `#71717a` | 文本基础颜色 |
| `--base-gradient-color` | `#000` | `#ffffff` | 微光亮点颜色 |
