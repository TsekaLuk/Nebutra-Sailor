## 使用方法 (Usage)

```tsx
import { Text } from "@nebutra/ui";

<Text variant="body">正文文本内容 (Body text content)</Text>
```

## 示例 (Examples)

### 变体 (Variants)

```tsx
<Text variant="body">正文 — 默认段落样式 (Body text — default paragraph style)</Text>
<Text variant="body-sm">小号正文 — 用于说明文字和次要内容 (Small body — captions and secondary content)</Text>
<Text variant="label">标签 — 表单标签和元数据 (Label — form labels and metadata)</Text>
<Text variant="label-sm">小号标签 — 微型标签和 tag (Small label — micro labels and tags)</Text>
<Text variant="code">等宽字体 — 行内代码片段 (Monospace — code snippets inline)</Text>
```

### 颜色 Token (Color Tokens)

```tsx
<Text color="default">默认前景色 (Default foreground color)</Text>
<Text color="muted">柔和色 — 次要文本、占位符 (Muted — secondary text, placeholders)</Text>
<Text color="accent">强调色 — 突出显示的文本 (Accent — highlighted text)</Text>
```

### 文本截断 (Truncation)

```tsx
{/* 单行截断 */}
<Text truncate className="max-w-xs">
  如果文本超过容器宽度，将以省略号截断 (This text will be truncated with an ellipsis if it exceeds the container width)
</Text>

{/* 多行截断 (Clamp) */}
<Text lineClamp={2} className="max-w-sm">
  此文本将被限制在最多两行，并在末尾显示省略号。第二行之后的内容将被隐藏。 (This text will be clamped to a maximum of two lines with an ellipsis at the end.)
</Text>
```

### 对齐方式 (Alignment)

```tsx
<Text align="center" variant="body-sm" color="muted">
  居中的辅助文本 (Centered supporting text)
</Text>
```

### 多态元素 (Polymorphic Element)

```tsx
{/* 渲染为 span 以用于行内场景 */}
<Text as="span" variant="label" color="muted">
  发布于 2025年1月15日
</Text>

{/* 渲染为 blockquote */}
<Text as="blockquote" variant="body" className="border-l-4 pl-4 italic">
  "设计不仅仅是外在的表现和感觉，设计是它的运作方式。" (Design is not just what it looks like and feels like. Design is how it works.)
</Text>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `as` | `React.ElementType` | `"p"` | 渲染为不同的 HTML 元素 |
| `variant` | `TextStyle` | `"body"` | 来自排版 Token 系统的视觉变体 |
| `color` | `TextColor` | `"default"` | 文本颜色 Token |
| `truncate` | `boolean` | `false` | 以省略号截断单行文本 |
| `lineClamp` | `1 \| 2 \| 3 \| 4 \| 5 \| 6` | — | 将文本限制在 N 行并显示省略号 |
| `align` | `"left" \| "center" \| "right"` | — | 文本对齐方式 |
| `className` | `string` | — | 额外的 Tailwind 类名 |

## 注意事项 (Notes)

- `truncate` 和 `lineClamp` 是互斥的 —— `truncate` 具有更高优先级。请二选一使用。
- `lineClamp` 需要元素具有 `overflow: hidden`，该样式通过 Tailwind 的 `line-clamp-*` 工具类自动应用。
- 对于语义化的标题元素 (h1–h6)，请改用 `Heading` 组件。
