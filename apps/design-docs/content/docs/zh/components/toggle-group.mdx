## 使用方法 (Usage)

```tsx
import { ToggleGroup, ToggleGroupItem } from "@nebutra/ui";
```

## 属性 (Props)

### ToggleGroup (根组件)

| 属性 | 类型 | 默认值 | 描述 |
| -------------- | ------------------------------- | ----------- | ------------------------------------------------ |
| `type` | `"single" \| "multiple"` | **必填** | 允许按下一个还是多个条目。 |
| `value` | `string \| string[]` | — | 受控的选中值。 |
| `defaultValue` | `string \| string[]` | — | 非受控的默认值。 |
| `onValueChange` | `(value: string \| string[]) => void` | — | 选中值改变时的回调函数。 |
| `variant` | `"default" \| "outline"` | `"default"` | 应用于所有条目的视觉变体。 |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | 应用于所有条目的尺寸变体。 |
| `disabled` | `boolean` | `false` | 禁用所有条目。 |

### ToggleGroupItem (开关条目)

| 属性 | 类型 | 默认值 | 描述 |
| ---------- | --------- | ------- | ----------------------------------- |
| `value` | `string` | **必填** | 条目的值。 |
| `disabled` | `boolean` | `false` | 禁用此特定条目。 |
| `asChild` | `boolean` | `false` | 作为子元素渲染。 |

## 示例 (Examples)

### 多选模式 (Multiple Selection)

```tsx
<ComponentPreview className="min-h-[150px] flex justify-center items-center">
  <ToggleGroup type="multiple" defaultValue={["bold", "italic"]}>
    <ToggleGroupItem value="bold" aria-label="切换加粗 (Toggle bold)">
      <Bold className="size-4" />
    </ToggleGroupItem>
    <ToggleGroupItem value="italic" aria-label="切换斜体 (Toggle italic)">
      <Italic className="size-4" />
    </ToggleGroupItem>
    <ToggleGroupItem value="underline" aria-label="切换下划线 (Toggle underline)">
      <Underline className="size-4" />
    </ToggleGroupItem>
  </ToggleGroup>
</ComponentPreview>
```

### 受控单选 (Controlled Single Selection)

```tsx
<ComponentPreview className="min-h-[150px] flex justify-center items-center">
  <ToggleGroup type="single" defaultValue="center">
    <ToggleGroupItem value="left">左 (Left)</ToggleGroupItem>
    <ToggleGroupItem value="center">中 (Center)</ToggleGroupItem>
    <ToggleGroupItem value="right">右 (Right)</ToggleGroupItem>
  </ToggleGroup>
</ComponentPreview>
```

## 可访问性 (Accessibility)

实现了 WAI-ARIA 工具栏 (toolbar) 模式。使用方向键在条目间导航。每个条目都具有 `aria-pressed` 属性。对于仅含图标的条目，请提供 `aria-label`。
