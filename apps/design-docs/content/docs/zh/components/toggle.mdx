
## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
| ---------------- | ----------------------------- | ----------- | ------------------------------------------------ |
| `pressed`        | `boolean`                     | —           | 受控的按钮按下状态。 |
| `defaultPressed` | `boolean`                     | `false`     | 初始按钮按下状态（非受控）。 |
| `onPressedChange`| `(pressed: boolean) => void`  | —           | 状态改变时的回调函数。 |
| `variant`        | `"default" \| "outline"`      | `"default"` | 视觉变体。 |
| `size`           | `"default" \| "sm" \| "lg"`   | `"default"` | 尺寸变体。 |
| `disabled`       | `boolean`                     | `false`     | 禁用开关按钮。 |
| `asChild`        | `boolean`                     | `false`     | 作为子元素渲染。 |

## 可访问性 (Accessibility)

Toggle 渲染一个带有 `aria-pressed` 属性的 `<button>`。当开关按钮仅包含图标时，请提供 `aria-label`。键盘操作：使用 `Space` 或 `Enter` 键激活。
