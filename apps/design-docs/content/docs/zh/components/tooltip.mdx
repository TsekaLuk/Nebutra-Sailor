## 可访问性 (Accessibility)

- 基于 Radix UI Tooltip 构建 —— 遵循 WAI-ARIA [文字提示模式](https://www.w3.org/WAI/ARIA/apg/patterns/tooltip/)。
- 文字提示内容通过 `aria-describedby` 与触发器关联。
- 在获得焦点和悬停时均会打开 —— 方便键盘用户使用。
- 按下 `Escape` 键会关闭文字提示。
- 不要仅在文字提示中放置核心、非冗余的信息 —— 它们在触摸设备上无法通过悬停访问。
