## 属性 (Props)

### TooltipProvider (提示提供者)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `delayDuration` | `number` | `700` | 文字提示出现前的延迟（毫秒） |
| `skipDelayDuration` | `number` | `300` | 当另一个文字提示已打开时，新提示打开的速度 |
| `disableHoverableContent` | `boolean` | `false` | 防止文字提示内容可被悬停选中 |

### Tooltip (根组件)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `open` | `boolean` | — | 受控的打开状态 |
| `defaultOpen` | `boolean` | `false` | 非受控的默认打开状态 |
| `onOpenChange` | `(open: boolean) => void` | — | 打开状态改变时的回调函数 |
| `delayDuration` | `number` | — | 为此特定提示覆盖提供者的延迟时间 |

### TooltipTrigger (触发器)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `asChild` | `boolean" | `false` | 将属性合并到直接子元素上 |
| `children` | `ReactNode` | — | 触发文字提示的元素 |

### TooltipContent (内容区)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"top"` | 首选显示侧 |
| `sideOffset` | `number` | `4` | 距离触发器的距离（像素） |
| `align` | `"start" \| "center" \| "end"` | `"center"` | 沿触发器的对齐方式 |
| `className` | `string` | — | 额外的 CSS 类名 |
| `children` | `ReactNode` | — | 文字提示内容 |

## 可访问性 (Accessibility)

- 基于 Radix UI Tooltip 构建 —— 遵循 WAI-ARIA [文字提示模式](https://www.w3.org/WAI/ARIA/apg/patterns/tooltip/)。
- 文字提示内容通过 `aria-describedby` 与触发器关联。
- 在获得焦点和悬停时均会打开 —— 方便键盘用户使用。
- 按下 `Escape` 键会关闭文字提示。
- 不要仅在文字提示中放置核心、非冗余的信息 —— 它们在触摸设备上无法通过悬停访问。
