## 演示 (Demo)

```tsx
import { TreeView, TreeViewItem } from "@nebutra/ui";
import { Folder, File } from "lucide-react";

export default function Example() {
  return (
    <TreeView>
      <TreeViewItem label="src" icon={<Folder className="size-4" />} defaultExpanded>
        <TreeViewItem label="components" icon={<Folder className="size-4" />}>
          <TreeViewItem label="Button.tsx" icon={<File className="size-4" />} />
          <TreeViewItem label="Input.tsx" icon={<File className="size-4" />} />
        </TreeViewItem>
        <TreeViewItem label="index.ts" icon={<File className="size-4" />} />
      </TreeViewItem>
    </TreeView>
  );
}
```

## 使用方法 (Usage)

```tsx
import { TreeView, TreeViewItem } from "@nebutra/ui";
```

## 属性 (Props)

### TreeView (树形视图容器)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
| ----------- | ----------- | ------- | ------------------------------------------------ |
| `className` | `string`    | —       | 额外的 CSS 类名。 |
| `children`  | `ReactNode` | —       | `TreeViewItem` 子元素集合。 |

### TreeViewItem (树形项目)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
| ------------------ | ----------- | ------- | ------------------------------------------------ |
| `label`            | `string`    | **必填** | 展示的标签文本。 |
| `icon`             | `ReactNode` | —       | 标签前显示的图标。 |
| `defaultExpanded`  | `boolean`   | `false` | 该项目初始是否处于展开状态。 |
| `expanded`         | `boolean`   | —       | 受控的展开状态。 |
| `onExpandedChange` | `(expanded: boolean) => void` | — | 展开状态改变时的回调。 |
| `selected`         | `boolean`   | `false` | 该项目是否被选中。 |
| `onSelect`         | `() => void`| —       | 项目被选中时的回调。 |
| `disabled`         | `boolean`   | `false` | 禁用选择和展开操作。 |
| `children`         | `ReactNode` | —       | 嵌套的子级 `TreeViewItem` 元素。 |

## 示例 (Examples)

### 受控展开 (Controlled Expansion)

```tsx
const [expanded, setExpanded] = useState<Record<string, boolean>>({});

function ControlledTree() {
  return (
    <TreeView>
      <TreeViewItem
        label="root"
        expanded={expanded["root"] ?? false}
        onExpandedChange={(open) =>
          setExpanded((prev) => ({ ...prev, root: open }))
        }
      >
        <TreeViewItem label="child-1" />
        <TreeViewItem label="child-2" />
      </TreeViewItem>
    </TreeView>
  );
}
```

### 可选择项目 (Selectable Items)

```tsx
const [selected, setSelected] = useState<string | null>(null);

<TreeView>
  {items.map((item) => (
    <TreeViewItem
      key={item.id}
      label={item.name}
      selected={selected === item.id}
      onSelect={() => setSelected(item.id)}
    />
  ))}
</TreeView>
```

## 无障碍支持 (Accessibility)

实现了 WAI-ARIA 的 `tree` 和 `treeitem` 角色。支持键盘导航：`ArrowDown`/`ArrowUp` 在项目间移动，`ArrowRight` 展开，`ArrowLeft` 折叠或移动到父级，`Enter`/`Space` 进行选中。
