## 概览 (Overview)

`VideoPlayer` 是一个基于 HTML5 `<video>` 和 Framer Motion 动画构建的功能完备视频播放器：

- **播放/暂停** —— 点击视频区域或按钮即可切换。
- **进度条** —— 带有弹性效果的滑动条；点击可直接跳转进度。
- **音量调节** —— 支持静音切换的音量滑动条。
- **播放倍速** —— 可配置的倍速选项（默认：0.5×, 1×, 1.5×, 2×）。
- **自动隐藏控制栏** —— 控制层在悬停时淡入，鼠标离开时淡出。

## 使用方法 (Usage)

```tsx
import { VideoPlayer } from "@nebutra/ui/primitives";

// 基础用法
<VideoPlayer src="/demo.mp4" />

// 带有封面图和自定义倍速
<VideoPlayer
  src="/tutorial.mp4"
  poster="/thumbnail.jpg"
  speeds={[0.5, 1, 1.25, 1.5, 2]}
  initialVolume={0.8}
/>

// 自动播放循环（浏览器要求必须静音）
<VideoPlayer src="/background.mp4" autoPlay loop />
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `src` | `string` | — | **必需。** 视频源文件 URL |
| `poster` | `string` | — | 播放前显示的封面图 URL |
| `autoPlay` | `boolean` | `false` | 挂载时自动播放（会自动开启静音） |
| `loop` | `boolean` | `false` | 循环播放视频 |
| `speeds` | `number[]` | `[0.5, 1, 1.5, 2]` | 可选的播放倍速列表 |
| `initialVolume` | `number` | `1` | 初始音量 (0–1) |
| `showControlsOnMount` | `boolean` | `false` | 挂载后立即显示控制栏 |
| `className` | `string` | — | 容器的 CSS 类名 |

## 注意事项 (Notes)

- 控制层浮窗定位在视频底部的中心位置。
- `autoPlay` 会自动将视频设为静音（这是现代浏览器的硬性要求）。
- 音量图标会自动适配：`VolumeX` (静音), `Volume1` (< 50%), `Volume2` (≥ 50%)。
- 播放器默认包装在 `max-w-4xl` 容器中；可使用 `className` 限制宽度。
