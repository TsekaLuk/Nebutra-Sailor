## 使用方法 (Usage)

```tsx
import { VideoText } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<div className="relative h-[300px] w-full overflow-hidden bg-black">
  <VideoText src="/videos/ocean.webm">大海 (OCEAN)</VideoText>
</div>
```

### 自定义字体设置 (Custom Font Settings)

```tsx
<div className="relative h-[400px] w-full overflow-hidden bg-black">
  <VideoText
    src="/videos/fire.webm"
    fontSize={15}
    fontWeight={900}
    fontFamily="Impact"
  >
    火焰 (FIRE)
  </VideoText>
</div>
```

### 关闭自动播放（手动控制） (No Autoplay)

```tsx
<div className="relative h-[300px] w-full overflow-hidden">
  <VideoText src="/video.webm" autoPlay={false} loop={false} muted={false}>
    播放 (PLAY)
  </VideoText>
</div>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `src` | `string` | — | 视频资源 URL |
| `children` | `ReactNode` | — | 作为蒙版的文本内容 |
| `autoPlay` | `boolean` | `true` | 是否自动播放视频 |
| `muted` | `boolean` | `true` | 是否静音视频 |
| `loop` | `boolean` | `true` | 是否循环播放 |
| `preload` | `"auto" \| "metadata" \| "none"` | `"auto"` | 视频预加载策略 |
| `fontSize` | `string \| number` | `20` | 以 `vw` 为单位的字号数字，或 CSS 字符串 |
| `fontWeight` | `string \| number` | `"bold"` | CSS 字体权重 (font-weight) |
| `fontFamily` | `string` | `"sans-serif"` | CSS 字体族 (font-family) |
| `textAnchor` | `string` | `"middle"` | SVG 文本锚点 (text-anchor) |
| `dominantBaseline` | `string` | `"middle"` | SVG 显性基准线 (dominant-baseline) |
| `as` | `ElementType` | `"div"` | 容器元素类型 |
| `className` | `string` | — | 容器上的额外 CSS 类名 |

## 可访问性 (Accessibility)

- 使用了一个带有 `.sr-only` 类的 `<span>` 以便屏幕阅读器读取文本。
- 视频配置了 `playsInline` 属性，以防止在 iOS 上强制进入全屏。
- 请在 `children` 中提供有意义的文案 —— 它将作为无障碍文本标签。

## 注意事项 (Notes)

- 必须包裹在设定了明确尺寸且 `overflow-hidden` 的容器中。
- 蒙版使用了内联的 `data:image/svg+xml` URL，会在窗口尺寸调整时更新。
- 为了达到最佳可读性，建议使用高对比度视频（如黑底明亮视频或白底深色视频）。
