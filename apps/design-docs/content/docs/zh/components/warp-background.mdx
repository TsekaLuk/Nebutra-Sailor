## 示例 (Examples)

### 基础用法 (Basic)

```tsx
<WarpBackground>
  <h2 className="text-2xl font-bold">欢迎开启未来 (Welcome to the Future)</h2>
  <p>体验下一代技术革新。</p>
</WarpBackground>
```

### 引导行动区域 (CTA Section)

```tsx
<WarpBackground
  perspective={150}
  beamsPerSide={5}
  beamSize={3}
  beamDuration={4}
  gridColor="rgba(59, 130, 246, 0.3)"
  className="p-12"
>
  <h2 className="text-4xl font-black text-center">即刻开始</h2>
  <Button size="lg" className="mt-6">免费注册</Button>
</WarpBackground>
```

### 戏剧化的特写效果 (Dramatic Close-up)

```tsx
<WarpBackground perspective={50} beamSize={8}>
  <PricingCard />
</WarpBackground>
```

## 属性 (Props)

`WarpBackground` 继承自 `React.HTMLAttributes<HTMLDivElement>`。

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | 渲染在容器内部的内容 |
| `perspective` | `number` | `100` | CSS 透视深度（单位 px，越低效果越强烈） |
| `beamsPerSide` | `number` | `3` | 网格每侧的光束数量 |
| `beamSize` | `number` | `5` | 光束尺寸（占容器的百分比） |
| `beamDelayMax` | `number` | `3` | 最大启动延迟（秒） |
| `beamDelayMin` | `number` | `0` | 最小启动延迟（秒） |
| `beamDuration` | `number` | `3` | 每道光束动画的持续时间（秒） |
| `gridColor` | `string` | `"var(--border)"` | 网格线的 CSS 颜色 |
| `className` | `string` | — | 额外的 CSS 类名（默认包含 `p-20`） |

## 注意事项 (Notes)

- 网格背景使用了 CSS 容器查询 (`cqmax`, `cqh`, `cqi`) —— 需要现代浏览器支持。
- 每道光束都具有随机生成的颜色 (HSL) 和长宽比，以增加视觉的多样性。
- 3D 效果是通过在网格的四个面应用 CSS `perspective` 和 `rotateX` 变换来实现的。
