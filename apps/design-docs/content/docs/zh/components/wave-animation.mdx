---
title: "波浪动画 (Wave Animation)"
description: "基于 Canvas 实现的动态波浪条纹视觉效果。营造类似音频可视化器的波浪模式，并带有拖尾效果 —— 非常适合音乐、创意及节日活动页面。"
---

## 使用方法 (Usage)

```tsx
import { WaveAnimation } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 全屏背景 (Full Page Background)

```tsx
<>
  <WaveAnimation />
  <main className="relative z-10">内容区域</main>
</>
```

### 局部背景 (Section Background)

```tsx
<div className="relative">
  <WaveAnimation position="absolute" zIndex={-1} />
  <section className="relative z-10 py-24">
    <h2>创意版块</h2>
  </section>
</div>
```

### 紫色调色板 (Purple Palette)

```tsx
<WaveAnimation
  palette={["#6366f1", "#8b5cf6", "#a855f7", "#d946ef"]}
  speed={1.5}
  barCount={30}
/>
```

### 缓慢且宽阔 (Slow and Wide)

```tsx
<WaveAnimation
  barCount={10}
  barWidth={200}
  speed={0.5}
  amplitude={1.5}
  trailOpacity={0.05}
/>
```

## 属性 (Props)

| 属性 (Prop) | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `palette` | `string[]` | 海洋/日落渐变 | 波浪条纹的配色方案 |
| `barCount` | `number` | `20` | 波浪条纹的数量 |
| `barWidth` | `number` | `100` | 每条条纹的像素宽度 |
| `speed` | `number` | `1` | 动画速度倍率 |
| `trailOpacity` | `number` | `0.03` | 拖尾消失的透明度（数值越低，拖尾越长） |
| `amplitude` | `number` | `1` | 波浪高度倍率 |
| `paused` | `boolean` | `false` | 暂停动画 |
| `position` | `"fixed" \| "absolute" \| "relative"` | `"fixed"` | 画布的定位方式 |
| `zIndex` | `number` | `0` | Z 轴层级 |
| `className` | `string` | — | 额外的 CSS 类名 |
