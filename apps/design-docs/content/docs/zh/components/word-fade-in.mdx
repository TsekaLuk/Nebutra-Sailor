---
title: "单词淡入 (Word Fade In)"
description: "由 Framer Motion 驱动的逐词交错淡入动画。为大标题、名言和欢迎消息提供优雅的揭示效果。"
---

## 使用方法 (Usage)

```tsx
import { WordFadeIn } from "@nebutra/ui/primitives"
```

## 示例 (Examples)

### 基础 Hero 区域标题 (Basic Hero Headline)

```tsx
<WordFadeIn words="欢迎来到我们的平台 (Welcome to our platform)" />
```

### 渲染为段落 (As Paragraph)

```tsx
<WordFadeIn
  as="p"
  words="每个单词都会逐一淡入 (Each word fades in one by one)"
  className="text-lg text-muted-foreground"
/>
```

### 更慢的交错延迟 (Slower Stagger)

```tsx
<WordFadeIn words="慢慢来 (Take your time)" delay={0.3} />
```

### 滚动触发 (Scroll-Triggered)

```tsx
<WordFadeIn
  words="滚动进入视口时才会出现 (This appears when scrolled into view)"
  trigger="inView"
  className="text-3xl font-bold"
/>
```

### 受控模式（初始暂停） (Controlled)

```tsx
<WordFadeIn
  words="点击时开始播放 (Starts when you click)"
  animate={isStarted}
/>
```

## 属性 (Props)

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `words` | `string` | — | 要执行动画的文本（以空格分隔） |
| `delay` | `number` | `0.15` | 单词之间的交错延迟时间（秒） |
| `as` | `"h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6" \| "p" \| "span" \| "div"` | `"h1"` | 要渲染的 HTML 元素 |
| `animate` | `boolean` | `true` | 开关动画 |
| `trigger` | `"mount" \| "inView"` | `"mount"` | 何时开始动画 |
| `variants` | `Variants` | 交错淡入 | 自定义 Framer Motion 变体 |
| `className` | `string` | — | 额外的 CSS 类名 |

## 注意事项 (Notes)

- 设置 `trigger="inView"` 时，动画会在元素进入视口时播放一次 (`viewport: { once: true }`)。
- 默认样式：`text-center text-4xl font-bold tracking-tight` —— 您可以随时通过 `className` 自行覆盖。
