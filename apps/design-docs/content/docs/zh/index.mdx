## 为什么选择 Nebutra？

Nebutra 建立在现代前端原生能力之上，以确保最佳的性能和 DX（开发者体验）。

<CardGroup>
  <Card title="开箱即用的美观" icon={<Sparkles className="text-blue-500" />}>
    精心打磨的美学设计，包含高级排版、优化的间距，以及开箱即用的无缝暗色模式支持。
  </Card>
  <Card title="极致性能" icon={<Zap className="text-yellow-500" />}>
    针对 React Server Components 进行优化。极小的客户端 JavaScript 开销，Tailwind v4 带来零运行时 CSS 体积。
  </Card>
  <Card title="无障碍访问 (a11y)" icon={<UserCircle className="text-green-500" />}>
    完全符合 WAI-ARIA 标准。严格保障键盘导航和屏幕阅读器支持。
  </Card>
  <Card title="复制可用" icon={<FileCode className="text-purple-500" />}>
    拒绝臃肿的 npm 包，将代码的控制权交还给你。你可以完全控制、调整和微调每一个组件的代码。
  </Card>
</CardGroup>

## 快速上手 

准备好开始了吗？直接跳转到演练场，或者通过 CLI 安装。

```bash title="Terminal"
npm install @nebutra/ui my-project
```
