## Demo

```tsx
import { Badge } from "@nebutra/custom-ui/primitives"

export function BadgeDemo() {
  return (
    <div className="flex flex-wrap gap-2">
      <Badge>Default</Badge>
      <Badge variant="secondary">Secondary</Badge>
      <Badge variant="destructive">Destructive</Badge>
      <Badge variant="outline">Outline</Badge>
    </div>
  )
}
```

## Usage

```tsx
import { Badge } from "@nebutra/custom-ui/primitives"
```

## Props

`Badge` extends `React.HTMLAttributes<HTMLDivElement>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline"` | `"default"` | Visual style and semantic meaning |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Badge label content |

## Examples

### Semantic Status Badges

```tsx
<Badge variant="default">Active</Badge>
<Badge variant="secondary">Draft</Badge>
<Badge variant="destructive">Error</Badge>
<Badge variant="outline">Beta</Badge>
```

### In Context — Table Row Status

```tsx
<TableRow>
  <TableCell>Invoice #1024</TableCell>
  <TableCell>
    <Badge variant="default">Paid</Badge>
  </TableCell>
</TableRow>
```

### With Icon

```tsx
import { CircleCheck } from "lucide-react"

<Badge>
  <CircleCheck className="mr-1 h-3 w-3" />
  Verified
</Badge>
```

### Count Badge

```tsx
<div className="relative inline-flex">
  <Bell className="h-5 w-5" />
  <Badge className="absolute -top-2 -right-2 h-4 w-4 p-0 text-[10px] flex items-center justify-center">
    3
  </Badge>
</div>
```

## Accessibility

- Renders as a `<div>` — not interactive by default.
- For status information, consider adding `aria-label` to clarify context.
- When used as a notification count, wrap in a container with `aria-live="polite"` for dynamic updates.
