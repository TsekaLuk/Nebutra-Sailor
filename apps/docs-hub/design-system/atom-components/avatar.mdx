---
title: "Avatar"
description: "User profile image component with automatic fallback to initials, built on Radix UI."
---

## Demo

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "@nebutra/custom-ui/primitives"

export function AvatarDemo() {
  return (
    <div className="flex gap-4 items-center">
      <Avatar>
        <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
        <AvatarFallback>SC</AvatarFallback>
      </Avatar>

      <Avatar>
        <AvatarImage src="/broken-image.jpg" alt="Broken image" />
        <AvatarFallback>JD</AvatarFallback>
      </Avatar>

      <Avatar className="h-14 w-14">
        <AvatarImage src="https://github.com/shadcn.png" alt="Large avatar" />
        <AvatarFallback>LG</AvatarFallback>
      </Avatar>
    </div>
  )
}
```

## Usage

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "@nebutra/custom-ui/primitives"
```

## Props

### Avatar

Extends `@radix-ui/react-avatar` `Root` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes. Use to override default `h-10 w-10` size |
| `children` | `ReactNode` | — | `AvatarImage` and/or `AvatarFallback` |

Default size is `40×40px` (`h-10 w-10`) with a circular shape (`rounded-full`).

### AvatarImage

Extends `@radix-ui/react-avatar` `Image` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | Image URL |
| `alt` | `string` | — | **Required.** Alt text for screen readers |
| `className` | `string` | — | Additional CSS classes |

The image is hidden automatically if it fails to load, revealing `AvatarFallback`.

### AvatarFallback

Extends `@radix-ui/react-avatar` `Fallback` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `delayMs` | `number` | — | Delay before showing fallback (prevents flash on fast image loads) |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Fallback content (typically 1–2 initials) |

```tsx
<Avatar className="h-6 w-6">
  <AvatarFallback className="text-xs">XS</AvatarFallback>

<Avatar className="h-8 w-8">

</Avatar>

<Avatar className="h-14 w-14">
  <AvatarFallback className="text-lg">LG</AvatarFallback>
</Avatar>
```

### Avatar with User Data

```tsx
function UserAvatar({ user }: { user: { name: string; image?: string } }) {
  const initials = user.name
    .split(" ")
    .map((n) => n[0])
    .join("")
    .toUpperCase()
    .slice(0, 2)

  return (
    <Avatar>
      {user.image && <AvatarImage src={user.image} alt={user.name} />}
      <AvatarFallback>{initials}</AvatarFallback>
    </Avatar>
  )
}
```

### Avatar Group

```tsx
<div className="flex -space-x-2">
  {users.map((user) => (
    <Avatar key={user.id} className="h-8 w-8 border-2 border-background">
      <AvatarImage src={user.image} alt={user.name} />
      <AvatarFallback>{user.initials}</AvatarFallback>
    </Avatar>
  ))}
</div>
```

### With Delay Fallback

```tsx
<Avatar>
  <AvatarImage src={profileImageUrl} alt="Profile" />
  <AvatarFallback delayMs={600}>AB</AvatarFallback>
</Avatar>
```

## Accessibility

- `AvatarImage` renders an `<img>` element — always provide a descriptive `alt` attribute.
- When the image fails to load, `AvatarFallback` is shown — ensure the fallback initials are meaningful.
- The root element has `role="img"` semantics via Radix UI.
- Avoid using avatars as the sole means of identifying a user — combine with visible name text.
