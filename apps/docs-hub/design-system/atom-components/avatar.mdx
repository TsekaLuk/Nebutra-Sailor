## Usage

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "@nebutra/custom-ui/primitives"
```

## Examples

### Basic

```tsx
<Avatar>
  <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
  <AvatarFallback>SC</AvatarFallback>
</Avatar>
```

### Size Variants

Default is `40×40px`. Override with className:

```tsx
<Avatar className="h-5 w-5">   {/* xs — 20px */}
  <AvatarFallback className="text-[10px]">XS</AvatarFallback>
</Avatar>

<Avatar className="h-8 w-8">   {/* sm — 32px */}
  <AvatarFallback className="text-xs">SM</AvatarFallback>
</Avatar>

<Avatar>                          {/* md — 40px (default) */}
  <AvatarFallback>MD</AvatarFallback>
</Avatar>

<Avatar className="h-14 w-14"> {/* lg — 56px */}
  <AvatarFallback>LG</AvatarFallback>
</Avatar>

<Avatar className="h-20 w-20"> {/* xl — 80px */}
  <AvatarFallback className="text-xl">XL</AvatarFallback>
</Avatar>
```

### Avatar Group

Stack avatars with `border-background` to separate them visually:

```tsx
<div className="flex -space-x-2">
  {users.map((user) => (
    <Avatar key={user.id} className="h-8 w-8 border-2 border-background">
      <AvatarImage src={user.avatar} alt={user.name} />
      <AvatarFallback>{user.initials}</AvatarFallback>
    </Avatar>
  ))}
  <Avatar className="h-8 w-8 border-2 border-background">
    <AvatarFallback className="bg-muted text-muted-foreground text-xs">
      +{remaining}
    </AvatarFallback>
  </Avatar>
</div>
```

### From User Data

```tsx
function UserAvatar({ user }: { user: { name: string; image?: string } }) {
  const initials = user.name
    .split(" ")
    .map((n) => n[0])
    .join("")
    .toUpperCase()
    .slice(0, 2)

  return (
    <Avatar>
      {user.image && <AvatarImage src={user.image} alt={user.name} />}
      <AvatarFallback>{initials}</AvatarFallback>
    </Avatar>
  )
}
```

## Props

### Avatar

Extends `@radix-ui/react-avatar` Root props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Use to set size — e.g. `h-8 w-8` for 32px |
| `children` | `ReactNode` | — | `AvatarImage` and/or `AvatarFallback` |

Default size: `40×40px` (`h-10 w-10`), circular (`rounded-full`).

### AvatarImage

Extends `@radix-ui/react-avatar` Image props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | Image URL |
| `alt` | `string` | — | **Required.** Alt text for screen readers |
| `className` | `string` | — | Additional CSS classes |

### AvatarFallback

Renders when the image fails to load or while loading.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `delayMs` | `number` | `600` | Delay before showing fallback — prevents flash on fast loads |
| `className` | `string` | — | Override `bg-muted` for custom colors |
| `children` | `ReactNode` | — | 1–2 letter initials |

## Accessibility

- Always provide a meaningful `alt` on `AvatarImage`.
- When the avatar triggers an action (e.g., opens a user menu), wrap in `<button>` with `aria-label`.
- Ensure fallback background has sufficient contrast against initials text.
