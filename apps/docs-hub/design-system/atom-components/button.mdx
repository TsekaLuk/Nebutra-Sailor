## Usage

```tsx
import { Button } from "@nebutra/custom-ui/primitives"
```

## Examples

### Variants

```tsx
<Button variant="default">Default</Button>
<Button variant="outline">Outline</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="destructive">Delete</Button>
<Button variant="link">Learn more</Button>
```

### Sizes

```tsx
<Button size="sm">Small</Button>       {/* h-9 — 36px */}
<Button size="default">Default</Button> {/* h-10 — 40px */}
<Button size="lg">Large</Button>       {/* h-11 — 44px */}
<Button size="icon"><PlusIcon /></Button> {/* 40×40px */}
```

### As Child (polymorphic)

```tsx
import Link from "next/link"
import { Button } from "@nebutra/custom-ui/primitives"

<Button asChild>
  <Link href="/dashboard">Go to Dashboard</Link>
</Button>
```

### Loading State

```tsx
import { Loader2 } from "lucide-react"

<Button disabled aria-busy="true">
  <Loader2 className="mr-2 h-4 w-4 animate-spin" />
  Saving…
</Button>
```

### With Icon

```tsx
import { Mail } from "lucide-react"

<Button>
  <Mail className="mr-2 h-4 w-4" />
  Login with Email
</Button>
```

## Props

`Button` extends all native `React.ButtonHTMLAttributes<HTMLButtonElement>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"` | Visual style variant |
| `size` | `"default" \| "sm" \| "lg" \| "icon"` | `"default"` | Size preset |
| `asChild` | `boolean` | `false` | Merge props onto the immediate child via Radix `Slot` |
| `disabled` | `boolean` | `false` | Disables the button |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Button label content |

### Size Reference

| Size | Height | Horizontal Padding |
|------|--------|--------------------|
| `sm` | 36px | 12px |
| `default` | 40px | 16px |
| `lg` | 44px | 32px |
| `icon` | 40×40px | — |

## Design Tokens

Primary variant consumes `--color-primary` (VI: `#0033FE` 云毓蓝).
Focus ring uses `--focus-ring` — brand-blue, 2px width, 2px offset.

## Accessibility

- Renders as native `<button>` — keyboard accessible by default.
- Set `aria-busy="true"` and `disabled` together during loading states.
- For icon-only buttons (`size="icon"`), always provide `aria-label`.
- Focus ring is always visible — never suppressed for keyboard users.
