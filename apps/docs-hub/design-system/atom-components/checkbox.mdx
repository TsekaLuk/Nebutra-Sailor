## Demo

```tsx
import { Checkbox } from "@nebutra/custom-ui/primitives"
import { Label } from "@nebutra/custom-ui/primitives"

export function CheckboxDemo() {
  return (
    <div className="flex items-center space-x-2">
      <Checkbox id="terms" />
      <Label htmlFor="terms">Accept terms and conditions</Label>
    </div>
  )
}
```

## Usage

```tsx
import { Checkbox } from "@nebutra/custom-ui/primitives"
```

## Props

Extends `@radix-ui/react-checkbox` `Root` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean \| "indeterminate"` | — | Controlled checked state |
| `defaultChecked` | `boolean` | `false` | Uncontrolled default checked state |
| `onCheckedChange` | `(checked: boolean \| "indeterminate") => void` | — | Callback when state changes |
| `disabled` | `boolean` | `false` | Disables the checkbox |
| `required` | `boolean` | `false` | Marks as required in form context |
| `name` | `string` | — | Form field name |
| `value` | `string` | `"on"` | Value submitted with form |
| `className` | `string` | — | Additional CSS classes |

## Examples

### Controlled Checkbox

```tsx
const [checked, setChecked] = useState(false)

<div className="flex items-center space-x-2">
  <Checkbox
    id="newsletter"
    checked={checked}
    onCheckedChange={setChecked}
  />
  <Label htmlFor="newsletter">Subscribe to newsletter</Label>
</div>
```

### Indeterminate State

```tsx
<Checkbox checked="indeterminate" />
```

### Checkbox Group

```tsx
{options.map((option) => (
  <div key={option.id} className="flex items-center space-x-2">
    <Checkbox
      id={option.id}
      checked={selected.includes(option.id)}
      onCheckedChange={(checked) => toggleOption(option.id, checked)}
    />
    <Label htmlFor={option.id}>{option.label}</Label>
  </div>
))}
```

## Accessibility

- Built on Radix UI Checkbox — has `role="checkbox"` with `aria-checked`.
- Supports `true`, `false`, and `"indeterminate"` (maps to `aria-checked="mixed"`).
- Always pair with a `<Label>` element linked via `htmlFor`/`id`.
- Keyboard: `Space` toggles the checkbox state.
