## Overview

`CodeBlock` renders a tabbed code viewer with full syntax highlighting powered by `react-syntax-highlighter` (Prism). It auto-detects the language from the file extension, adapts to light/dark mode, and provides copy-to-clipboard on a per-tab basis.

Geist-aligned features include highlighted lines, diff-style added/removed lines, clickable line references, and a language switcher dropdown.

## Installation

```tsx
import { CodeBlock } from "@nebutra/custom-ui/primitives";
```

`react-syntax-highlighter` is a peer dependency:

```bash
pnpm add react-syntax-highlighter @types/react-syntax-highlighter
```

## Usage

### Single file

```tsx
<CodeBlock
  files={[{
    title: "hello.ts",
    code: `const greeting = "Hello World";\nconsole.log(greeting);`
  }]}
/>
```

### Multiple files

```tsx
<CodeBlock
  files={[
    {
      title: "theme.ts",
      code: `export const theme = { primary: "#7C3AED" };`,
      language: "typescript",
    },
    {
      title: "styles.css",
      code: `:root { --primary: #7C3AED; }`,
      language: "css",
    },
  ]}
  defaultTitle="theme.ts"
/>
```

### With line numbers and custom height

```tsx
<CodeBlock
  files={[{ title: "config.json", code: JSON.stringify(config, null, 2) }]}
  maxHeight={300}
  showLineNumbers
/>
```

### Highlighted lines

Emphasize specific lines with an accent background and left border indicator.

```tsx
<CodeBlock
  files={[{
    title: "counter.tsx",
    code: counterCode,
    highlightedLines: [3, 4, 5],
  }]}
  showLineNumbers
/>
```

### Diff lines (added / removed)

Show diff-style green/red decoration with `+`/`-` gutter markers.

```tsx
<CodeBlock
  files={[{
    title: "counter.tsx",
    code: diffCode,
    addedLines: [5, 9, 10],
    removedLines: [4],
  }]}
  showLineNumbers
/>
```

### Referenced lines

Click a line number to copy a `#L{n}` anchor to the clipboard.

```tsx
<CodeBlock
  files={[{ title: "app.tsx", code: appCode }]}
  showLineNumbers
  enableLineReferences
  onLineReference={(line) => console.log(`Referenced line ${line}`)}
/>
```

### Language switcher

Override the auto-detected language with a dropdown in the header.

```tsx
<CodeBlock
  files={[{ title: "example.txt", code: someCode }]}
  showLanguageSwitcher
  languages={["typescript", "javascript", "python", "go"]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `files` | `CodeBlockFile[]` | — | **Required.** Array of code files to display |
| `defaultTitle` | `string` | `files[0].title` | Initially active file tab |
| `className` | `string` | — | Additional CSS classes for the outer container |
| `maxHeight` | `string \| number` | `400` | Max height of code area (number = px) |
| `showLineNumbers` | `boolean` | `false` | Show line numbers |
| `enableLineReferences` | `boolean` | `false` | Click line numbers to copy `#L{n}` anchor |
| `showLanguageSwitcher` | `boolean` | `false` | Show a language dropdown in the header |
| `languages` | `string[]` | built-in list | Languages available in the switcher |
| `onLineReference` | `(lineNumber: number) => void` | — | Callback when a line reference is clicked |

### CodeBlockFile

| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | File name shown in the tab |
| `code` | `string` | Code content to display and copy |
| `language` | `string` | Syntax highlighting language (auto-detected from extension if omitted) |
| `highlightedLines` | `number[]` | Line numbers to highlight with accent background (1-indexed) |
| `addedLines` | `number[]` | Line numbers marked as added with green background (1-indexed) |
| `removedLines` | `number[]` | Line numbers marked as removed with red background (1-indexed) |

## Auto-Detected Languages

Language is inferred from file extension when `language` is not provided:

| Extension | Language |
|-----------|---------|
| `.js` `.jsx` | javascript / jsx |
| `.ts` `.tsx` | typescript / tsx |
| `.html` | html |
| `.css` `.scss` | css / scss |
| `.json` | json |
| `.md` | markdown |
| `.py` | python |
| `.go` | go |
| `.rs` | rust |
| `.sh` `.bash` `.zsh` | bash |
| `.yaml` `.yml` | yaml |
| `.sql` | sql |
| `.graphql` `.gql` | graphql |

## Notes

- Light/dark theme is auto-detected via a `MutationObserver` watching for the `dark` class on `<html>`.
- The light theme adapts Prism's nightOwl with design-system CSS variables.
- File icons are shown in tabs: `FileCode` for JS/TS, `FileText` for CSS, `File` for others.
- Tab labels are hidden on mobile (`hidden sm:inline`) to save space.
- Highlighted and diff line styles use inline `rgba()` colors that adapt to light/dark mode.
- Diff markers (`+`/`-`) are rendered via scoped CSS `::before` pseudo-elements.
- The language switcher resets when switching between file tabs.
