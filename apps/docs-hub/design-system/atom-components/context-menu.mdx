## Accessibility

- Built on Radix UI Context Menu ظ¤ follows the WAI-ARIA [Menu pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menu/).
- `ContextMenuContent` has `role="menu"`.
- Each `ContextMenuItem` has `role="menuitem"`.
- Keyboard: `ArrowUp`/`ArrowDown` to navigate, `Enter`/`Space` to select, `Escape` to close.
