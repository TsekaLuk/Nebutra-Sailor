---
title: "Gradient Animated Text"
description: "Text that cycles through gradient and solid states with staggered timing. Use 3 variants together for a build/ship/scale word reveal effect."
---

<Note>
  **Storybook** — Interactive demo: `apps/storybook` → **Primitives / GradientAnimatedText**
</Note>

## Usage

```tsx
import { GradientAnimatedText } from "@nebutra/custom-ui/primitives"
```

## Examples

### Three-Word Staggered Reveal (signature use case)

```tsx
<div className="flex gap-2 text-5xl font-black">
  <GradientAnimatedText variant={1} theme="neon">Build.</GradientAnimatedText>
  <GradientAnimatedText variant={2} theme="neon">Ship.</GradientAnimatedText>
  <GradientAnimatedText variant={3} theme="neon">Scale.</GradientAnimatedText>
</div>
```

### AI Theme

```tsx
<div className="flex gap-2">
  <GradientAnimatedText variant={1} theme="ai">Think.</GradientAnimatedText>
  <GradientAnimatedText variant={2} theme="ai">Learn.</GradientAnimatedText>
  <GradientAnimatedText variant={3} theme="ai">Grow.</GradientAnimatedText>
</div>
```

### Custom Colors

```tsx
<GradientAnimatedText
  variant={1}
  theme="custom"
  gradientFrom="#f97316"
  gradientTo="#eab308"
>
  Custom
</GradientAnimatedText>
```

### Slower Animation

```tsx
<GradientAnimatedText variant={1} theme="neon" duration={16}>
  Slow
</GradientAnimatedText>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text content |
| `variant` | `1 \| 2 \| 3` | `1` | Animation phase offset — stagger across variants |
| `theme` | `"neon" \| "ai" \| "custom"` | `"neon"` | Color theme preset |
| `gradientFrom` | `string` | — | Custom gradient start color (overrides theme) |
| `gradientTo` | `string` | — | Custom gradient end color (overrides theme) |
| `duration` | `number` | `8` | Animation cycle duration in seconds |
| `className` | `string` | — | Additional CSS classes |

## Theme Colors

| Theme | Variant 1 | Variant 2 | Variant 3 |
|-------|-----------|-----------|-----------|
| `neon` | green → cyan | red-pink → magenta | yellow → electric blue |
| `ai` | green → magenta | magenta → cyan | yellow → blue |
| `custom` | indigo → violet | pink → rose | teal → cyan |
