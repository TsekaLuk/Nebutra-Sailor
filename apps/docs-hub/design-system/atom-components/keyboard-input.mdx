## Overview

`Kbd` renders a styled `<kbd>` element. It supports:
- **Modifier shortcuts** — `meta` / `shift` / `alt` / `ctrl` render platform symbols (⌘ ⇧ ⌥ ⌃)
- **Combinations** — multiple modifier props on one element combine their symbols
- **Small variant** — compact size for inline prose
- **Custom children** — plain text for arbitrary key names (e.g. `Enter`, `Esc`)

## Installation

```tsx
import { Kbd } from "@nebutra/ui/primitives";
```

## Usage

### Modifiers

```tsx
<Kbd meta />    {/* ⌘ */}
<Kbd shift />   {/* ⇧ */}
<Kbd alt />     {/* ⌥ */}
<Kbd ctrl />    {/* ⌃ */}
```

### Combination

```tsx
<Kbd meta shift />   {/* ⌘⇧ */}
```

### Small

```tsx
<Kbd small>/</Kbd>
<Kbd small meta />
```

### With text content

```tsx
<Kbd>Enter</Kbd>
<Kbd>Esc</Kbd>
<Kbd meta>K</Kbd>
```

### In context

```tsx
<p>
  Press <Kbd small meta /> <Kbd small>K</Kbd> to open the command menu.
</p>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `meta` | `boolean` | — | Renders ⌘ symbol |
| `shift` | `boolean` | — | Renders ⇧ symbol |
| `alt` | `boolean` | — | Renders ⌥ symbol |
| `ctrl` | `boolean` | — | Renders ⌃ symbol |
| `small` | `boolean` | — | Smaller size variant |
| `children` | `ReactNode` | — | Custom key label text |
| `className` | `string` | — | Additional CSS classes |

All other `<kbd>` HTML attributes are forwarded to the underlying element.

## Modifier order

When multiple modifiers are combined, they render in the order: **⌘ ⌃ ⌥ ⇧** (meta → ctrl → alt → shift), matching standard macOS conventions.
