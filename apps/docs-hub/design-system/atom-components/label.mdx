## Demo

```tsx
import { Label } from "@nebutra/custom-ui/primitives"
import { Input } from "@nebutra/custom-ui/primitives"
import { Checkbox } from "@nebutra/custom-ui/primitives"

export function LabelDemo() {
  return (
    <div className="space-y-4">
      <div className="space-y-2">
        <Label htmlFor="email">Email address</Label>
        <Input id="email" type="email" placeholder="you@example.com" />
      </div>

      <div className="flex items-center space-x-2">
        <Checkbox id="terms" />
        <Label htmlFor="terms">Accept terms and conditions</Label>
      </div>
    </div>
  )
}
```

## Usage

```tsx
import { Label } from "@nebutra/custom-ui/primitives"
```

## Props

Extends `@radix-ui/react-label` `Root` props (extends `React.LabelHTMLAttributes<HTMLLabelElement>`).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `htmlFor` | `string` | — | Links label to its control via the control's `id` |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Label text |

## Examples

### Required Field Label

```tsx
<Label htmlFor="name">
  Full name
  <span className="ml-1 text-destructive" aria-hidden="true">*</span>
</Label>
<Input id="name" required aria-required="true" />
```

### Label with Description

```tsx
<div className="space-y-1">
  <Label htmlFor="bio">Bio</Label>
  <p id="bio-hint" className="text-xs text-muted-foreground">
    Tell us a bit about yourself. Max 200 characters.
  </p>
  <Textarea id="bio" aria-describedby="bio-hint" maxLength={200} />
</div>
```

### Disabled Label

```tsx
<Label htmlFor="disabled-input" className="opacity-50 cursor-not-allowed">
  Disabled field
</Label>
<Input id="disabled-input" disabled />
```

## Accessibility

- Built on Radix UI Label — renders as a native `<label>` element.
- Clicking the label transfers focus to the associated control via `htmlFor`.
- When the associated control is disabled, the label automatically reflects the disabled cursor.
- Do not use `<span>` or `<div>` as labels — always use this `Label` component.
