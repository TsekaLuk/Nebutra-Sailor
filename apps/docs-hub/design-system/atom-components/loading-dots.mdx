## Overview

`LoadingDots` renders three animated circles with a staggered fade-pulse effect to indicate background activity.

- **Standalone** — three dots on their own
- **With text** — label + dots inline via `children`
- **Size control** — `size` prop sets dot diameter in pixels
- **Color inheritance** — dots use `currentColor`, so they inherit the surrounding text color

## Installation

```tsx
import { LoadingDots } from "@nebutra/custom-ui/primitives";
```

## Usage

### Default

```tsx
<LoadingDots />
```

### With text

```tsx
<LoadingDots size={4}>
  <p>Loading</p>
</LoadingDots>
```

### Custom size

```tsx
<LoadingDots size={4} />   {/* small — for inline / text use */}
<LoadingDots size={6} />   {/* default */}
<LoadingDots size={10} />  {/* large */}
```

### Color inheritance

```tsx
<span className="text-blue-500">
  <LoadingDots />
</span>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `6` | Diameter of each dot in pixels |
| `children` | `ReactNode` | — | Content rendered before the dots (e.g. label text) |
| `className` | `string` | — | Additional CSS classes on the outer `<span>` |

## Animation

Each dot pulses with a 1.2-second cycle. The three dots are staggered by 0.2 s each, producing a wave effect. The animation is defined by a `@keyframes loading-dot` injected inline — no additional CSS configuration required.
