## Usage

```tsx
import { Spinner } from "@nebutra/ui";

<Spinner />
<Spinner variant="ring" size={32} />
<Spinner variant="infinite" className="text-primary" />
```

## Variants

### Default

Lucide `Loader` icon with spin animation ‚Äî the classic dashed circle.

```tsx
<Spinner variant="default" />
```

### Circle

Lucide `LoaderCircle` icon ‚Äî solid arc spinner.

```tsx
<Spinner variant="circle" />
```

### Pinwheel

Lucide `LoaderPinwheel` icon ‚Äî pinwheel rotation.

```tsx
<Spinner variant="pinwheel" />
```

### Circle Filled

Two overlapping `LoaderCircle` icons with offset opacity ‚Äî creates a depth effect.

```tsx
<Spinner variant="circle-filled" />
```

### Ellipsis

Three dots that bounce up sequentially using native SVG `<animate>`.

```tsx
<Spinner variant="ellipsis" />
```

### Ring

Expanding ring pulses radiating outward ‚Äî like a sonar ping.

```tsx
<Spinner variant="ring" />
```

### Bars

Three vertical bars that scale in sequence ‚Äî audio visualizer style.

```tsx
<Spinner variant="bars" />
```

### Infinite

Animated infinity symbol traced with a dashed stroke.

```tsx
<Spinner variant="infinite" />
```

## Size and Color

`Spinner` extends `LucideProps`, so `size` and `className` control dimensions and color:

```tsx
<Spinner size={16} className="mr-2 text-white" />
<Spinner variant="ring" size={48} className="text-blue-500" />
```

### In a Button

```tsx
<button disabled className="flex items-center gap-2 px-4 py-2">
  <Spinner size={16} />
  Saving...
</button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "circle" \| "pinwheel" \| "circle-filled" \| "ellipsis" \| "ring" \| "bars" \| "infinite"` | `"default"` | Spinner animation style |
| `size` | `number \| string` | `24` | Width and height in pixels (from `LucideProps`) |
| `className` | `string` | ‚Äî | Tailwind classes ‚Äî use `text-*` to control color |

## Notes

- `Spinner` uses `currentColor` for all custom SVG variants ‚Äî color is controlled via `text-*` Tailwind classes.
- The `ellipsis`, `ring`, `bars`, and `infinite` variants use native SVG `<animate>` elements, which respect `prefers-reduced-motion` in browsers that support it.
- For text-based loading indicators (e.g., "Thinking..."), use the `Loader` component's `text-shimmer` or `loading-dots` variants instead.
