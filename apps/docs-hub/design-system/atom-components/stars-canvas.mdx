---
title: "Stars Canvas"
description: "Canvas-based animated starfield with orbiting, twinkling stars. A space-themed ambient background for hero sections, dashboards, and loading screens."
---

<Note>
  **Storybook** — Interactive demo: `apps/storybook` → **Primitives / StarsCanvas**
</Note>

## Usage

```tsx
import { StarsCanvas } from "@nebutra/custom-ui/primitives"
```

## Examples

### Full Page Background

```tsx
<>
  <StarsCanvas />
  <main className="relative z-10">Content</main>
</>
```

### Contained Section

```tsx
<div className="relative overflow-hidden rounded-xl h-[400px]">
  <StarsCanvas position="absolute" zIndex={0} />
  <div className="relative z-10 p-8">Content</div>
</div>
```

### Purple Stars

```tsx
<StarsCanvas hue={280} brightness={0.8} />
```

### Slow Dense Starfield

```tsx
<StarsCanvas
  maxStars={2000}
  speedMultiplier={0.3}
  twinkleIntensity={10}
/>
```

### Transparent (overlay)

```tsx
<StarsCanvas transparent position="absolute" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxStars` | `number` | `1200` | Total number of stars |
| `hue` | `number` | `217` | Star color hue (0–360) |
| `brightness` | `number` | `1` | Brightness multiplier (0–1) |
| `speedMultiplier` | `number` | `1` | Animation speed multiplier |
| `twinkleIntensity` | `number` | `20` | Twinkling frequency (lower = more twinkling) |
| `transparent` | `boolean` | `false` | Transparent background (for overlaying) |
| `paused` | `boolean` | `false` | Pause the animation |
| `position` | `"fixed" \| "absolute" \| "relative"` | `"fixed"` | Canvas position style |
| `zIndex` | `number` | `-1` | Z-index layer |
| `className` | `string` | — | Additional CSS classes |

## Notes

- With `position="fixed"` (default), the canvas covers the full viewport as a page background.
- Use `position="absolute"` inside a `relative overflow-hidden` container for section-level backgrounds.
- Stars orbit around the canvas center, creating a parallax-like depth effect.
- Auto-resizes with the window.
