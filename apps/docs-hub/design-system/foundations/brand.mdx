## Logo

The Nebutra logo uses the letter **N** as its foundational form — not displayed literally, but constructed through positive and negative geometric space to create an implied hexagonal structure. The Blue→Cyan gradient fill gives it a forward-looking, technology-driven character.

### Logo Components

Two implementations are available. Choose based on your context:

| Component | Package | How it loads | When to use |
|-----------|---------|-------------|-------------|
| `Logo` / `Logomark` / `Wordmark` | `@nebutra/brand` | `<img src="/brand/logo-*.svg">` | When the public folder is available (web app, Next.js app dir) |
| `LogomarkSVG` / `WordmarkEnSVG` / `LogoEnSVG` | `@nebutra/brand` | Inline SVG paths (`fill="currentColor"`) | No public folder needed; color via Tailwind `text-*`; email, PDF, Storybook |

#### File-based (img)

```tsx
import { Logo, Logomark, Wordmark } from "@nebutra/brand"

// Hexagonal N icon, 32px, white on dark background
<Logomark variant="inverse" size={32} />

// Full logo with English wordmark, 150px wide, inverted to white
<Logo variant="en" size={150} inverted />
```

#### Inline SVG (`fill="currentColor"`)

```tsx
import { LogomarkSVG, WordmarkEnSVG, LogoEnSVG } from "@nebutra/brand"

// Icon only — color via CSS
<LogomarkSVG className="text-white w-8 h-8" />

// Wordmark only — "Nebutra" text
<WordmarkEnSVG className="text-white" width={150} />

// Full logo (icon + wordmark side-by-side)
<LogoEnSVG className="text-white" width={160} />

// On a dark background (Navbar example)
<LogoEnSVG className="text-white" aria-label="Nebutra" width={140} />

// Framer Motion color animation
<motion.div>
  <LogomarkSVG className="text-indigo-400" width={32} />
</motion.div>
```

<Warning>
  **Landing page is forced dark** — always use `text-white` for inline SVG logos.
  Never conditionally swap between `text-black` and `text-white` based on theme on the landing page.
</Warning>

### Clearance & Minimum Size

- **Minimum clearance**: All sides ≥ ¼ of the logo height
- **Print minimum height**: ≥ 6 mm
- **Digital minimum height**: ≥ 35 px

### Usage Rules

<Warning>
  **Do not** distort, rotate, recolor, add drop shadows, or place on low-contrast backgrounds.
  **Do not** use the reverse (white) logo on light or complex backgrounds.
  Always use the provided brand gradient — never substitute with a solid color.
</Warning>

## Brand Gradients

The **Blue→Cyan linear gradient** is Nebutra's most distinctive visual signature. It is a **first-class brand asset** — used on hero sections, primary CTAs, and logo fills.

<CardGroup cols={2}>
  <Card title="Brand (Primary)" icon="arrow-right">
    `linear-gradient(135deg, #0033FE 0%, #0BF1C3 100%)`
    **Usage** Hero sections, primary CTA buttons, logo gradient fill
  </Card>
  <Card title="Brand Hover (Reverse)" icon="arrow-left">
    `linear-gradient(135deg, #0BF1C3 0%, #0033FE 100%)`
    **Usage** Hover state on gradient interactive elements
  </Card>
  <Card title="Section (Vertical)" icon="arrow-down">
    `linear-gradient(180deg, #0033FE 0%, #0BF1C3 100%)`
    **Usage** Page section dividers, feature highlight strips
  </Card>
  <Card title="Glow (Radial)" icon="circle">
    `radial-gradient(circle, #0BF1C3 0%, #0033FE 100%)`
    **Usage** Background halo effects, focus glow, radial emphasis
  </Card>
</CardGroup>

### Using Gradients in Code

```tsx
// CSS custom properties (defined in globals.css)
<div style={{ background: 'var(--gradient-brand)' }} />

// Tailwind (via @theme inline in globals.css)
<div className="bg-[var(--gradient-brand)]" />

// Button — apply gradient as primary
<Button className="bg-[linear-gradient(135deg,#0033FE_0%,#0BF1C3_100%)] text-white border-0">
  Get Started
</Button>
```

## Typography

### English — Poppins (Primary)

Poppins is the primary brand typeface for all English UI text, headlines, and marketing copy.

| Weight | Name | Use case |
|--------|------|----------|
| 400 | Poppins Regular | Body copy, descriptions |
| 500 | Poppins Medium | Labels, navigation, captions |
| 600 | Poppins SemiBold | Subheadings, button labels |
| 700 | Poppins Bold | Main headings, hero copy |

### Chinese — vivo Sans (Primary)

vivo Sans is the primary brand typeface for all Chinese UI text. Falls back to system CJK fonts.

| Weight | Name |
|--------|------|
| 300 | vivo Sans Light |
| 400 | vivo Sans Regular |
| 500 | vivo Sans Medium |
| 600 | vivo Sans DemiBold |
| 700 | vivo Sans Bold |

### Font Stack (Code)

```css
/* English / Default */
font-family: "Poppins", "vivo Sans", "PingFang SC", "Microsoft YaHei",
             "Noto Sans SC", -apple-system, BlinkMacSystemFont, sans-serif;

/* Chinese UI */
font-family: "vivo Sans", "PingFang SC", "Microsoft YaHei", "Noto Sans SC", sans-serif;

/* Display / Hero headlines */
font-family: "Poppins", sans-serif;

/* Code / Monospace */
font-family: "JetBrains Mono", "Fira Code", ui-monospace, Consolas, monospace;
```

```tsx
// Tailwind usage
<h1 className="font-display">Hero Headline</h1>
<p className="font-sans">Body copy</p>
<p className="font-cn">中文正文</p>
<code className="font-mono">const x = 1</code>
```

## Semantic Colors

These are derived from the VI manual and mapped to CSS variables for theming.

| Role | Light | Dark | Source |
|------|-------|------|--------|
| Primary | `#0033FE` | `#5c7cfa` | VI: 云毓蓝 |
| Secondary | `#0BF1C3` | `#1af7c8` | VI: 云毓青 |
| Success | `#22c55e` | `#16a34a` | VI: specified exact value |
| Warning | `#f59e0b` | `#d97706` | VI: §Color Specifications |
| Info | `#0033FE` | `#5c7cfa` | VI: Info = Brand Blue |
| Destructive | `#ef4444` | `#dc2626` | Standard |
| Background | `#ffffff` | `#020617` | VI: white light / neutral-950 dark |

<Warning>
  **Info = Brand Blue** — The VI manual explicitly specifies that the Info semantic color uses `#0033FE` (cloud blue), not a sky/teal color.
  This is a common mistake — do not use `sky-500` or `#06b6d4` for info states.
</Warning>
