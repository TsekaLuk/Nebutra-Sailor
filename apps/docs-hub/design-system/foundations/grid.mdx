## Layout Grid

All Nebutra product UIs use a **12-column fluid grid** with fixed gutters. The column count collapses at breakpoints for responsive layouts.

| Breakpoint | Name | Min width | Columns | Gutter | Margin |
|-----------|------|-----------|---------|--------|--------|
| `sm` | Mobile | 640px | 4 | 16px | 16px |
| `md` | Tablet | 768px | 8 | 24px | 24px |
| `lg` | Desktop | 1024px | 12 | 24px | 32px |
| `xl` | Wide | 1280px | 12 | 32px | 40px |
| `2xl` | Ultra | 1536px | 12 | 32px | 48px |

## Container Widths

Standard page containers constrain content to readable line lengths.

```tsx
// Full-width page wrapper
<div className="mx-auto w-full max-w-screen-xl px-4 md:px-6 lg:px-8">

// Prose / documentation
<div className="mx-auto max-w-3xl px-4">

// Dashboard content
<div className="mx-auto max-w-7xl px-4 sm:px-6 lg:px-8">

// Narrow form / modal content
<div className="mx-auto max-w-md">
```

| Name | Max width | Usage |
|------|-----------|-------|
| `max-w-sm` | 384px | Narrow cards, modals |
| `max-w-md` | 448px | Forms, dialogs |
| `max-w-lg` | 512px | Wide modals |
| `max-w-2xl` | 672px | Prose, documentation |
| `max-w-4xl` | 896px | Feature sections |
| `max-w-6xl` | 1152px | Dashboard layouts |
| `max-w-7xl` | 1280px | Full product layouts |

## Spacing Scale

The spacing scale follows Tailwind's default 4px base unit. Use these for margins, paddings, and gaps consistently.

| Class | Value | Usage |
|-------|-------|-------|
| `space-1` | 4px | Tight icon gap, inline elements |
| `space-2` | 8px | Inner padding, badge gap |
| `space-3` | 12px | Form element internal padding |
| `space-4` | 16px | Card padding (sm), section gap |
| `space-6` | 24px | Card padding (md), grid gap |
| `space-8` | 32px | Card padding (lg), section spacing |
| `space-12` | 48px | Page section dividers |
| `space-16` | 64px | Hero section padding |
| `space-24` | 96px | Large section breaks |

## Common Layout Patterns

### Page Shell

```tsx
<div className="min-h-screen bg-background">
  <header className="sticky top-0 z-50 border-b border-border bg-background/80 backdrop-blur">
    <div className="mx-auto max-w-7xl px-4 sm:px-6 lg:px-8">
      <nav className="flex h-16 items-center justify-between">
        {/* Nav content */}
      </nav>
    </div>
  </header>

  <main className="mx-auto max-w-7xl px-4 py-8 sm:px-6 lg:px-8">
    {/* Page content */}
  </main>
</div>
```

### Dashboard Grid

```tsx
{/* Stats row — 4 equal cards */}
<div className="grid gap-4 sm:grid-cols-2 lg:grid-cols-4">
  <Card variant="elevated" padding="md">...</Card>
  <Card variant="elevated" padding="md">...</Card>
  <Card variant="elevated" padding="md">...</Card>
  <Card variant="elevated" padding="md">...</Card>
</div>

{/* Main + Sidebar — 2/3 + 1/3 */}
<div className="mt-6 grid gap-6 lg:grid-cols-3">
  <div className="lg:col-span-2">
    {/* Main content */}
  </div>
  <div>
    {/* Sidebar */}
  </div>
</div>
```

### Feature Section (3-column)

```tsx
<section className="py-16">
  <div className="mx-auto max-w-6xl px-4">
    <h2 className="text-3xl font-semibold text-center mb-12">Features</h2>
    <div className="grid gap-6 sm:grid-cols-2 lg:grid-cols-3">
      {features.map((f) => (
        <Card key={f.title} variant="bordered" padding="lg">
          <Card.Header>
            <Card.Icon size="md">{f.icon}</Card.Icon>
            <Card.Title>{f.title}</Card.Title>
          </Card.Header>
          <Card.Body>
            <Card.Description>{f.description}</Card.Description>
          </Card.Body>
        </Card>
      ))}
    </div>
  </div>
</section>
```

### Centered Auth / Onboarding

```tsx
<div className="flex min-h-screen items-center justify-center bg-background px-4">
  <div className="w-full max-w-md">
    <Card variant="elevated" padding="lg">
      {/* Form content */}
    </Card>
  </div>
</div>
```

### Split Layout (Marketing / Auth)

```tsx
<div className="grid min-h-screen lg:grid-cols-2">
  {/* Left — brand panel */}
  <div className="hidden lg:flex flex-col items-center justify-center
                  bg-[linear-gradient(135deg,#0033FE_0%,#0BF1C3_100%)] p-12">
    {/* Logo, hero copy */}
  </div>

  {/* Right — form */}
  <div className="flex items-center justify-center p-8">
    <div className="w-full max-w-md">
      {/* Auth form */}
    </div>
  </div>
</div>
```

## Stack & Flex Utilities

### Vertical Stack

```tsx
{/* Stack with consistent gap */}
<div className="flex flex-col gap-4">
  <Component />
  <Component />
  <Component />
</div>
```

### Horizontal Cluster

```tsx
{/* Inline elements that wrap */}
<div className="flex flex-wrap items-center gap-2">
  <Badge variant="success" dot>Live</Badge>
  <Badge variant="info">v2.1</Badge>
</div>
```

### Space Between

```tsx
{/* Header with left/right content */}
<div className="flex items-center justify-between">
  <h2 className="text-lg font-semibold">Title</h2>
  <Button size="sm" variant="outline">View all</Button>
</div>
```

## Responsive Patterns

Use Tailwind breakpoint prefixes consistently:

```tsx
{/* Mobile-first: 1 col → 2 → 4 */}
<div className="grid grid-cols-1 gap-4 sm:grid-cols-2 lg:grid-cols-4">

{/* Hidden on mobile, visible on desktop */}
<aside className="hidden lg:block">

{/* Full width mobile, auto width desktop */}
<Button className="w-full sm:w-auto">Submit</Button>

{/* Smaller text on mobile */}
<h1 className="text-2xl font-bold sm:text-3xl lg:text-4xl">
```
