---
title: "EmptyState"
description: "Animated empty state component with an icon trio, title, description, and optional action button."
---

## Demo

```tsx
import { EmptyState } from "@nebutra/custom-ui";
import { FolderOpen, Plus, Rocket } from "lucide-react";

<EmptyState
  title="No projects yet"
  description="Create your first project to get started with Nebutra."
  icons={[<FolderOpen />, <Plus />, <Rocket />]}
  action={{
    label: "Create project",
    icon: <Plus className="w-4 h-4" />,
    onClick: () => router.push("/projects/new"),
  }}
  theme="light"
  size="default"
/>
```

## Usage

```tsx
import { EmptyState } from "@nebutra/custom-ui";
import { Database, Plus, Zap } from "lucide-react";

export function PipelinesPage() {
  const { pipelines } = usePipelines();

  if (pipelines.length === 0) {
    return (
      <EmptyState
        title="No pipelines"
        description="Build your first pipeline to start processing events."
        icons={[<Database />, <Zap />, <Plus />]}
        action={{ label: "New pipeline", onClick: openCreateDialog }}
        theme="light"
      />
    );
  }

  return <PipelineList pipelines={pipelines} />;
}
```

## Props

### `EmptyStateProps`

| Prop | Type | Default | Description |
|---|---|---|---|
| `title` | `string` | — | **Required.** Heading text rendered as `<h2>` |
| `description` | `string` | — | Supporting body copy beneath the title |
| `icons` | `React.ReactNode[]` | — | Array of exactly 3 icons displayed in an animated trio |
| `action` | `EmptyStateAction` | — | Configuration for the action button |
| `variant` | `"default" \| "subtle" \| "error"` | `"default"` | Visual style of the container |
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | Controls padding and text scale |
| `theme` | `"light" \| "dark" \| "neutral"` | `"light"` | Color palette; match your page background |
| `isIconAnimated` | `boolean` | `true` | When `true`, icons fan out and lift on hover |
| `className` | `string` | `""` | Additional Tailwind classes on the root element |

### `EmptyStateAction`

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | — | **Required.** Button label text |
| `icon` | `React.ReactNode` | — | Optional icon rendered before the label; rotates 90° on hover |
| `onClick` | `() => void` | — | Click handler |
| `disabled` | `boolean` | — | Disables the action button |

## Examples

### Dark theme

```tsx
<EmptyState
  title="No logs available"
  description="Start emitting events to see logs appear here."
  icons={[<Terminal />, <Activity />, <Zap />]}
  theme="dark"
  variant="default"
/>
```

### Error variant

```tsx
<EmptyState
  title="Failed to load data"
  description="An error occurred while fetching your resources. Please try again."
  icons={[<AlertCircle />, <RefreshCcw />, <HelpCircle />]}
  action={{ label: "Retry", onClick: refetch }}
  variant="error"
  theme="light"
/>
```

### Subtle — no border

```tsx
<EmptyState
  title="Nothing here yet"
  description="Items will appear once data is available."
  variant="subtle"
  size="sm"
/>
```

### Static icons (no hover animation)

```tsx
<EmptyState
  title="All caught up"
  description="No new notifications."
  icons={[<Bell />, <Check />, <Star />]}
  isIconAnimated={false}
/>
```

## Accessibility

- The root element renders as `<section>` with `role="region"` and `aria-labelledby` pointing to the title.
- When `description` is present, `aria-describedby` references the description paragraph.
- The action button renders as a native `<button>` with `type="button"`.
- `disabled` action buttons have `aria-disabled="true"` and `disabled` attributes.
- Icon containers are `aria-hidden="true"` — decorative only.
- Framer Motion animations respect the user's `prefers-reduced-motion` setting via `LazyMotion` with `domAnimation`.
