## Demo

```tsx
<LogsBarChart
  data={[
    { timestamp: "2024-01-15T10:00:00Z", info: 320, warn: 45, error: 12 },
    { timestamp: "2024-01-15T11:00:00Z", info: 280, warn: 38, error: 3 },
    { timestamp: "2024-01-15T12:00:00Z", info: 410, warn: 72, error: 28 },
    { timestamp: "2024-01-15T13:00:00Z", info: 190, warn: 21, error: 5 },
  ]}
  interval="1h"
  height={200}
/>
```

## Usage

```tsx
import { LogsBarChart } from "@nebutra/custom-ui";

export function LogsOverview() {
  const { data, isLoading } = useLogVolume({ range: "24h", interval: "1h" });

  return (
    <div className="space-y-2">
      <h3 className="text-sm font-medium text-gray-700">Log volume (24h)</h3>
      <LogsBarChart
        data={data}
        interval="1h"
        isLoading={isLoading}
        height={160}
        onBarClick={(bucket) => setTimeRange(bucket.timestamp)}
      />
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `LogBucket[]` | — | **Required.** Array of time-bucketed log counts |
| `interval` | `"1m" \| "5m" \| "15m" \| "1h" \| "1d"` | `"1h"` | Time bucket granularity; controls x-axis label formatting |
| `height` | `number` | `160` | Chart height in pixels |
| `isLoading` | `boolean` | `false` | Renders a skeleton placeholder |
| `showLegend` | `boolean` | `true` | Toggles the severity legend below the chart |
| `onBarClick` | `(bucket: LogBucket) => void` | — | Called when a bar column is clicked |
| `className` | `string` | — | Additional classes on the root container |

### `LogBucket`

```ts
interface LogBucket {
  timestamp: string;  // ISO 8601
  info?: number;
  warn?: number;
  error?: number;
  debug?: number;
}
```

## Examples

### Loading state

```tsx
<LogsBarChart
  data={[]}
  interval="5m"
  isLoading={true}
  height={160}
/>
```

### Click-to-filter time range

```tsx
<LogsBarChart
  data={volumeData}
  interval="1h"
  onBarClick={(bucket) => {
    const start = new Date(bucket.timestamp);
    const end = addHours(start, 1);
    setTimeRange({ start, end });
  }}
/>
```

### Without legend

```tsx
<LogsBarChart
  data={volumeData}
  interval="1h"
  showLegend={false}
  height={80}
/>
```

## Accessibility

- The chart container has `role="img"` with `aria-label` describing the data range.
- A visually hidden data table is rendered below the chart for screen reader users.
- Interactive bars have `role="button"` with `aria-label="{count} events at {time}"`.
- Color is not the only differentiator between severity levels — pattern fills are applied for high-contrast mode.
- `isLoading` skeleton has `aria-busy="true"` and `aria-label="Loading chart"`.
