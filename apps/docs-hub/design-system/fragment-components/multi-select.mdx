## Demo

```tsx
<MultiSelect
  options={[
    { value: "node", label: "Node.js" },
    { value: "python", label: "Python" },
    { value: "go", label: "Go" },

    { value: "java", label: "Java" },
  ]}
  value={["node", "python"]}
  onChange={setSelected}
  placeholder="Select runtimes..."
/>
```

## Usage

```tsx
import { MultiSelect } from "@nebutra/custom-ui";

export function RuntimeFilter() {
  const [runtimes, setRuntimes] = useState<string[]>([]);

  return (
    <FormItemLayout label="Runtimes" htmlFor="runtimes">
      <MultiSelect
        id="runtimes"
        options={RUNTIME_OPTIONS}
        value={runtimes}
        onChange={setRuntimes}
        placeholder="Select runtimes..."
        maxSelected={5}
      />
    </FormItemLayout>
  );
}
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `options` | `SelectOption[]` | — | **Required.** Full list of selectable options |
| `value` | `string[]` | — | Controlled array of selected values |
| `onChange` | `(values: string[]) => void` | — | Called whenever the selection changes |
| `placeholder` | `string` | `"Select..."` | Placeholder shown when no values are selected |
| `searchPlaceholder` | `string` | `"Search..."` | Placeholder inside the search input in the dropdown |
| `maxSelected` | `number` | — | Cap on number of selections; further selections are disabled |
| `disabled` | `boolean` | `false` | Disables the entire input |
| `clearable` | `boolean` | `true` | Renders a clear-all button when values are selected |
| `className` | `string` | — | Additional classes on the root trigger element |

### `SelectOption`

```ts
interface SelectOption {
  value: string;
  label: string;
  icon?: React.ReactNode;
  disabled?: boolean;
}
```

## Accessibility

- The trigger renders as a `<button>` with `aria-haspopup="listbox"` and `aria-expanded`.
- The dropdown list has `role="listbox"` and `aria-multiselectable="true"`.
- Each option has `role="option"` and `aria-selected`.
- Selected value chips have a remove button with `aria-label="Remove {label}"`.
- Keyboard: `Enter` / `Space` opens the dropdown; arrow keys navigate options; `Enter` toggles selection; `Escape` closes.
- The search input inside the dropdown has `aria-label="Search options"`.
- `maxSelected` limit is announced via `aria-live="polite"` when reached.
