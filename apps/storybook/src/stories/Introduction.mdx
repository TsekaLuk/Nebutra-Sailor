# Nebutra UI Component Library

Welcome to the Nebutra Storybook. This is the interactive component explorer for `@nebutra/ui`.

## What's here

- **Design Tokens** – Visual documentation for all brand tokens (replaces Figma for solo devs)
  - **Colors** – Brand scales (云毓蓝 / 云毓青), 12-step semantic scales, gradients, dark mode
  - **Typography** – Type scale, font families, brand gradient text
  - **Motion** – Animation presets (`AnimateIn`), stagger groups, duration scale
  - **Shadows** – Elevation scale, brand glow shadows
- **Primitives** – Low-level UI building blocks (buttons, inputs, modals, etc.)
- **Components** – Composed UI patterns built from primitives
- **Layouts** – Page-level layout shells

## Getting started

Browse components using the sidebar on the left. Each story demonstrates a component in isolation with controls to adjust its props.

## Running locally

```bash
pnpm --filter @nebutra/storybook dev
```
