# Post-Launch Checklist

> **Timeline:** T+1 to T+30 (Day after launch to one month)  
> **Goal:** Capitalize on momentum, convert interest to users

---

## T+1 (Day After Launch)

### Immediate (Morning)

- [ ] Final results screenshot saved
- [ ] Record all metrics in METRICS.md
- [ ] Thank-you post on Twitter
- [ ] Thank-you email to supporters
- [ ] Rest and recover üò¥

### Afternoon

- [ ] Review all PH comments
- [ ] Respond to any unanswered comments
- [ ] Check GitHub for new issues/stars
- [ ] Review signup/waitlist numbers

---

## T+2 to T+3 (Days 2-3)

### Follow-Up Engagement

- [ ] Personal thank-you DMs to top engagers
- [ ] Reply to any Twitter mentions
- [ ] Engage with posts that shared your launch
- [ ] Follow up on any press inquiries

### Content

- [ ] Draft launch retrospective blog post
- [ ] Prepare "lessons learned" thread
- [ ] Collect and save testimonials from comments

### Lead Nurturing

- [ ] Segment new signups
- [ ] Send welcome email sequence
- [ ] Identify high-intent leads for personal outreach

---

## T+4 to T+7 (Week 1)

### Analytics Review

- [ ] Complete metrics analysis
- [ ] Compare to targets
- [ ] Identify top traffic sources
- [ ] Note what worked/didn't work

### Content Publishing

- [ ] Publish launch retrospective
- [ ] Share "lessons learned" thread
- [ ] Post behind-the-scenes content

### Community Building

- [ ] Onboard new community members
- [ ] Answer questions in Discord/GitHub
- [ ] Start conversations with new followers

### Business Development

- [ ] Follow up with interested parties
- [ ] Schedule demos if applicable
- [ ] Note partnership opportunities

---

## T+8 to T+14 (Week 2)

### Momentum Maintenance

- [ ] Continue posting valuable content
- [ ] Engage with new users
- [ ] Share user testimonials/wins

### Product Improvements

- [ ] Address feature requests from launch
- [ ] Fix any bugs discovered
- [ ] Plan roadmap based on feedback

### Secondary Distribution

- [ ] Submit to other directories (AlternativeTo, etc.)
- [ ] Guest posts / podcast appearances
- [ ] Newsletter features

---

## T+15 to T+30 (Weeks 3-4)

### Long-Term Follow-Up

- [ ] Check for PH "Product of the Week/Month"
- [ ] Apply for PH Golden Kitty (if applicable)
- [ ] Plan next launch milestone

### Conversion Focus

- [ ] Optimize onboarding based on data
- [ ] A/B test landing page
- [ ] Nurture leads that didn't convert

### Documentation

- [ ] Complete post-mortem document
- [ ] Archive launch materials
- [ ] Update this playbook for next time
