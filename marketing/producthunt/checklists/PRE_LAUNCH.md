# Pre-Launch Checklist

> **Timeline:** T-30 to T-1 (30 days before launch to day before)  
> **Goal:** Build momentum and prepare all assets

---

## T-30 to T-21 (Week 4)

### Product Readiness

- [ ] Landing page is live and polished
- [ ] Demo/trial is accessible
- [ ] Documentation is up to date
- [ ] GitHub README is compelling
- [ ] All critical bugs are fixed
- [ ] Performance is acceptable

### PH Setup

- [ ] Hunter identified (self or external)
- [ ] PH Maker profile complete and professional
- [ ] PH Ship page created
- [ ] Initial subscribers on Ship page (team + early supporters)

### Copy Preparation

- [ ] Tagline finalized
- [ ] Description written
- [ ] First comment drafted
- [ ] FAQ responses prepared

---

## T-21 to T-14 (Week 3)

### Asset Creation

- [ ] Thumbnail created (240x240)
- [ ] Gallery images created (3-8 images)
- [ ] All images tested on PH preview
- [ ] Demo video/GIF recorded
- [ ] OG image for social sharing

### Community Building

- [ ] Email list: 100+ subscribers
- [ ] Twitter following engaged
- [ ] Ship page: 50+ subscribers
- [ ] Discord/community notified

### Content Drafts

- [ ] Twitter launch thread drafted
- [ ] LinkedIn post drafted
- [ ] Email to supporters drafted
- [ ] HackerNews post drafted

---

## T-14 to T-7 (Week 2)

### Pre-Launch Buzz

- [ ] Start teaser campaign on Twitter/X
- [ ] "Coming soon" posts in relevant communities
- [ ] Engage daily with PH community
- [ ] Comment on 3-5 other launches daily
- [ ] DM 10+ potential supporters

### Outreach

- [ ] Contact 5+ tech newsletters
- [ ] Reach out to 10+ influential accounts
- [ ] Notify beta users / early adopters
- [ ] Brief team on launch roles

### Technical Prep

- [ ] Set up UTM tracking links
- [ ] Configure analytics events
- [ ] Test all CTAs and links
- [ ] Prepare monitoring dashboard

---

## T-7 to T-2 (Week 1)

### Content Scheduling

- [ ] Schedule Twitter thread (12:01 AM PT)
- [ ] Schedule email blast (12:05 AM PT)
- [ ] Schedule LinkedIn post (6:00 AM PT)
- [ ] Prepare HN "Show HN" post
- [ ] Set calendar reminders for all posts

### Community Final Push

- [ ] Send "we're launching soon" email
- [ ] Post teaser in communities
- [ ] Personal DMs to top 20 supporters
- [ ] Brief all team members

### Final Review

- [ ] Proofread all copy
- [ ] Test all links
- [ ] Review gallery images
- [ ] Check mobile responsiveness
- [ ] Verify Ship page subscriber count

---

## T-1 (Day Before)

### Morning (by 12:00 PM PT)

- [ ] Submit product to PH (schedule for 12:01 AM PT)
- [ ] Verify submission details are correct
- [ ] Confirm all gallery images uploaded
- [ ] Confirm video/GIF is working

### Afternoon

- [ ] Prepare first comment (copy to clipboard)
- [ ] Set up comment notification alerts
- [ ] Test PH mobile app notifications
- [ ] Final check of all scheduled posts

### Evening

- [ ] Team briefing call
- [ ] Confirm everyone's timezone and availability
- [ ] Set alarms for launch time
