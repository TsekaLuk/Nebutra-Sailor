## Overview

`AvatarCircles` renders a row of overlapping circular avatar images with `-space-x-4` negative spacing. An optional overflow counter (e.g., `+99`) appears after the last avatar, linking to `overflowUrl`.

Common uses:
- Team member previews
- GitHub contributor lists
- Social proof ("Join 1,200+ users")
- Comment thread participants

## Usage

```tsx
import { AvatarCircles } from "@nebutra/custom-ui/primitives";

const avatars = [
  { imageUrl: "https://github.com/octocat.png", profileUrl: "https://github.com/octocat" },
  { imageUrl: "https://github.com/user2.png", profileUrl: "https://github.com/user2" },
  { imageUrl: "https://github.com/user3.png", profileUrl: "https://github.com/user3" },
];

// Basic
<AvatarCircles avatarUrls={avatars} />

// With overflow count
<AvatarCircles avatarUrls={avatars} numPeople={99} />

// With custom size
<AvatarCircles avatarUrls={avatars} numPeople={50} size={48} />

// Social proof pattern
<div className="flex items-center gap-4">
  <AvatarCircles avatarUrls={avatars} numPeople={1200} />
  <span className="text-sm text-muted-foreground">Join 1,200+ developers</span>
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `avatarUrls` | `AvatarCircleItem[]` | — | **Required.** Array of avatar configurations |
| `numPeople` | `number` | — | Number displayed in the overflow badge |
| `size` | `number` | `40` | Avatar diameter in pixels |
| `overflowUrl` | `string` | `""` | Link for the overflow count badge |
| `className` | `string` | — | Container CSS class |

### `AvatarCircleItem`

| Prop | Type | Description |
|------|------|-------------|
| `imageUrl` | `string` | Avatar image URL |
| `profileUrl` | `string` | Link URL when avatar is clicked |
| `alt` | `string` | Alt text (defaults to `"Avatar {index}"`) |

## Notes

- Avatars open in a new tab (`target="_blank" rel="noopener noreferrer"`)
- Hover effect: `scale-110` with `z-10` to bring avatar to front
- The overflow badge is dark-mode aware (black on light, white on dark)
- RTL support: uses `rtl:space-x-reverse` for right-to-left layouts
