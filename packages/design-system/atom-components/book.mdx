---
title: "Book"
description: "3-D book cover component for showcasing e-books, reports, and documentation. Supports custom color, illustration, grain texture, and responsive width."
---

<Note>
  **Storybook** — Interactive demo: `apps/storybook` → **Primitives / Book**
</Note>

## Import

```tsx
import { Book } from "@nebutra/custom-ui"
```

---

## Default

A book with spine perspective effect, amber cover, and title.

```tsx
<Book title="The user experience of the Frontend Cloud" />
```

---

## Variants

`"default"` renders a 3-D perspective book with a visible spine shadow.
`"simple"` renders a flat, shadowless cover — use in grids or lists.

```tsx
<Book
  title="The user experience of the Frontend Cloud"
  variant="simple"
  width={196}
/>

<Book
  title="The user experience of the Frontend Cloud"
  variant="default"
  width={196}
/>
```

---

## Custom Color

Pass any CSS color string to `color`. Set `textColor` for contrast.

```tsx
<Book
  color="#9D2127"
  title="How Nebutra improves your deployment pipeline"
/>

<Book
  color="#7DC1C1"
  textColor="white"
  title="Design Engineering at Nebutra"
  variant="simple"
/>

<Book
  color="#0033FE"
  textColor="white"
  title="AI-Powered Infrastructure Guide"
/>
```

---

## Custom Illustration

The `illustration` prop accepts any ReactNode and fills the upper ~55% of the cover.

```tsx
// Colored circles (social proof / team illustration)
<Book
  title="The user experience of the Frontend Cloud"
  illustration={
    <div className="flex items-end justify-center gap-3 h-full pb-4">
      <div className="h-14 w-14 rounded-full bg-blue-500" />
      <div className="h-10 w-10 rounded-full bg-red-500" />
      <div className="h-12 w-12 rounded-full bg-green-500" />
    </div>
  }
/>

// Full-bleed image
<Book
  title="Nebutra Architecture Deep Dive"
  color="#0f172a"
  textColor="white"
  illustration={
    <img
      src="/covers/architecture.png"
      className="h-full w-full object-cover"
      alt=""
    />
  }
/>
```

## Textured

Add grain/paper texture to any color. Works best with saturated colors.

```tsx
<Book color="#7DC1C1" textured title="Cloud Computing Patterns" />
<Book color="#9D2127" textured title="Security Engineering" />
<Book color="#FED954" textured title="Frontend Performance" />

{/* Textured + simple variant */}
<Book
  color="#7DC1C1"
  textColor="white"
  textured
  variant="simple"
  title="Design Engineering"
/>
```

---

## Responsive Width

Pass `{ sm, md }` to `width` for responsive sizing using CSS `clamp()`.

```tsx
<Book
  title="AI-Powered Search"
  width={{ sm: 130, md: 196 }}
/>
```

---

## Width Options

```tsx
<Book title="Wide book" width={300} />
<Book title="Default book" width={196} />
<Book title="Compact book" width={140} />
```

---

## With Logo Mark

The `logo` prop renders a brand mark at the bottom-right of the cover.

```tsx
import { LogomarkSVG } from "@nebutra/brand"

<Book
  title="Nebutra Platform Guide"
  color="#0033FE"
  textColor="white"
  logo={<LogomarkSVG width={16} height={16} />}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | **Required.** Cover title text |
| `variant` | `"default" \| "simple"` | `"default"` | 3-D spine vs flat |
| `color` | `string` | `"#f59e0b"` | Cover background color (CSS color) |
| `textColor` | `string` | `"#000000"` | Title and logo text color |
| `illustration` | `ReactNode` | — | JSX rendered in upper 55% of cover |
| `width` | `number \| { sm?, md? }` | `160` | Width in px, or responsive breakpoints |
| `textured` | `boolean` | `false` | Grain/paper texture overlay |
| `logo` | `ReactNode` | — | Brand mark at bottom-right |
| `className` | `string` | — | Additional CSS classes |

---

## Design Notes

- **Aspect ratio** is fixed at `1 : 1.45` (portrait). Height is computed automatically from `width`.
- **Title font size** scales proportionally with `width` — no manual size adjustment needed.
- **Spine effect** (default variant only) uses CSS `perspective(700px) rotateY(-10deg)` with a left-to-right gradient overlay to simulate a printed book spine.
- **Texture** uses an SVG `feTurbulence` fractalNoise filter rendered at `200 × 200px` tiles with `mix-blend-mode: overlay`.

---

## Accessibility

- The component renders as a non-interactive `<div>`.
- All decorative overlays (spine gradient, gloss, texture) carry `aria-hidden`.
- Always provide meaningful `title` text — it is the only accessible label.
- If the book is clickable (e.g., links to a PDF), wrap with `<a>` and add `aria-label`.
