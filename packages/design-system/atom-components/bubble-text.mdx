---
title: "Bubble Text"
description: "Interactive text where hovering a character creates a bold bubble ripple effect across neighboring characters."
---

<Note>
  **Storybook** — Interactive demo: `apps/storybook` → **Primitives / BubbleText**
</Note>

## Usage

```tsx
import { BubbleText } from "@nebutra/ui/primitives"
```

## Examples

### Basic

```tsx
<BubbleText className="text-5xl text-indigo-300">
  Hover over me!
</BubbleText>
```

### As a Hero Heading

```tsx
<BubbleText as="h1" className="text-6xl font-bold text-blue-400">
  Interactive Hero
</BubbleText>
```

### Wider Ripple Distance

```tsx
<BubbleText rippleDistance={4} className="text-4xl text-purple-400">
  Wide Ripple Effect
</BubbleText>
```

### Narrow Ripple

```tsx
<BubbleText rippleDistance={1} className="text-5xl text-emerald-400">
  Tight Bubble
</BubbleText>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text content (must be a string) |
| `as` | `"h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6" \| "p" \| "span"` | `"h2"` | HTML element to render |
| `rippleDistance` | `number` | `2` | Number of characters affected by hover ripple in each direction |
| `className` | `string` | — | Additional CSS classes |

## Design Notes

The ripple effect applies weight transitions:
- **Hovered char**: `font-black` + `brightness-150`
- **±1 distance**: `font-medium` + `brightness-125`
- **±2 distance**: `font-light`
- **Beyond**: No change

## Accessibility

- Renders each character as a `<span>` — semantically neutral.
- Mouse interaction only; no keyboard event handling. Ensure the text remains readable without hover interaction.
