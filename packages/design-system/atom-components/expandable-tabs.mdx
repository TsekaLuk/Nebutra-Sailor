## Overview

`ExpandableTabs` renders a horizontal tab bar using icon-only buttons that smoothly expand to show their label text when selected. Clicking outside the tab bar deselects the active tab. Supports optional separators between groups of tabs.

## Installation

```tsx
import { ExpandableTabs } from "@nebutra/ui/primitives";
```

## Usage

### Basic

```tsx
import { Home, Settings, Bell, User } from "lucide-react";

<ExpandableTabs
  tabs={[
    { title: "Home", icon: Home },
    { title: "Notifications", icon: Bell },
    { title: "Profile", icon: User },
    { title: "Settings", icon: Settings },
  ]}
  onChange={(index) => console.log("Selected tab:", index)}
/>
```

### With separators

Separators visually group tabs with a vertical divider line.

```tsx
import { Home, BarChart, Bell, Settings, User } from "lucide-react";

<ExpandableTabs
  tabs={[
    { title: "Home", icon: Home },
    { title: "Analytics", icon: BarChart },
    { type: "separator" },
    { title: "Notifications", icon: Bell },
    { type: "separator" },
    { title: "Profile", icon: User },
    { title: "Settings", icon: Settings },
  ]}
/>
```

### Custom active color

```tsx
<ExpandableTabs
  tabs={[...]}
  activeColor="text-blue-500"
/>
```

### Controlled state

```tsx
const [activeTab, setActiveTab] = useState<number | null>(null);

<ExpandableTabs
  tabs={tabs}
  onChange={(index) => {
    setActiveTab(index);
    // navigate or show content based on index
  }}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `tabs` | `TabItem[]` | — | **Required.** Array of tab or separator objects |
| `activeColor` | `string` | `"text-primary"` | Tailwind text color class for the active tab |
| `className` | `string` | — | Additional CSS classes for the container |
| `onChange` | `(index: number \| null) => void` | — | Callback when active tab changes (null when deselected) |

### TabItem

```ts
type TabItem = Tab | Separator;

interface Tab {
  title: string;       // Tab label shown when expanded
  icon: LucideIcon;    // Lucide icon component
}

interface Separator {
  type: "separator";   // Renders a vertical divider
}
```

## Notes

- Clicking outside the tab bar calls `onChange(null)` and collapses all tabs.
- The expansion animation uses Framer Motion with a spring transition (`bounce: 0, duration: 0.6`).
- The label text animates from `width: 0, opacity: 0` to `width: auto, opacity: 1`.
- Tab buttons use `role="tab"` and `aria-selected` for accessibility.
