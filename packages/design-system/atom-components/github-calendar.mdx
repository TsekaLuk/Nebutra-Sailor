## Overview

`GitHubCalendar` renders a year of activity data as a color-intensity heatmap grid — identical in layout to GitHub's contribution graph. Useful for:

- Developer profile activity visualization
- Habit tracking dashboards
- Content creation frequency
- Custom activity metrics

## Usage

```tsx
import { GitHubCalendar } from "@nebutra/ui/primitives";

const data = [
  { date: "2025-01-01", count: 3 },
  { date: "2025-01-02", count: 0 },
  { date: "2025-01-05", count: 7 },
];

<GitHubCalendar data={data} />
```

### Custom colors

```tsx
<GitHubCalendar
  data={data}
  colors={["#1a1a2e", "#16213e", "#0f3460", "#533483", "#e94560"]}
/>
```

### Custom tooltip

```tsx
<GitHubCalendar
  data={data}
  tooltipFormatter={(date, count) =>
    count === 0 ? `${date}: No activity` : `${date}: ${count} commits`
  }
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `ContributionDay[]` | — | **Required.** Array of `{ date, count }` objects |
| `colors` | `string[]` | GitHub greens | 5-color scale from least to most activity |
| `weeks` | `number` | `53` | Number of weeks to display (53 = one year) |
| `thresholds` | `number[]` | `[0,1,2,3,4]` | Count thresholds for each color level |
| `dayLabels` | `string[]` | `["Sun"..."Sat"]` | Labels for the left-side day indicators |
| `showDayLabels` | `boolean` | `true` | Show/hide day labels |
| `showMonthLabels` | `boolean` | `true` | Show/hide month labels |
| `showLegend` | `boolean` | `true` | Show/hide the color legend |
| `legendLabels` | `{ less, more }` | — | Custom legend text |
| `tooltipFormatter` | `(date, count) => string` | Default format | Custom tooltip text |
| `className` | `string` | — | Container CSS class |
| `cellClassName` | `string` | — | Individual cell CSS class |

### `ContributionDay`

```ts
interface ContributionDay {
  date: string;  // ISO format: "2025-09-13"
  count: number; // Number of contributions
}
```

## Notes

- Date matching uses `isSameDay` from `date-fns` — ensure dates are valid ISO strings
- Missing dates in `data` are treated as count `0` (no activity)
- The calendar always ends today and extends `weeks` weeks back
- Cells have `hover:scale-125` for subtle interactive feedback
