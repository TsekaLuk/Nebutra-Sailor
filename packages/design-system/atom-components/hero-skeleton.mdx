## Overview

`HeroSkeleton` wraps [HeroUI's Skeleton](https://www.heroui.com/docs/components/skeleton) component. It takes the shape of its children and shows a shimmer animation while content loads. Set `isLoaded={true}` to transition from skeleton to content.

## Installation

```tsx
import { HeroSkeleton } from "@nebutra/ui/primitives";
```

## Usage

### Takes child's shape

```tsx
<HeroSkeleton>
  <div className="h-24 w-24 rounded-lg" />
</HeroSkeleton>
```

### Standalone text lines

```tsx
<div className="flex flex-col gap-3">
  <HeroSkeleton className="h-3 w-3/5 rounded-lg" />
  <HeroSkeleton className="h-3 w-4/5 rounded-lg" />
  <HeroSkeleton className="h-3 w-2/5 rounded-lg" />
</div>
```

### With loaded state

```tsx
const [isLoaded, setIsLoaded] = useState(false);

<HeroSkeleton isLoaded={isLoaded} className="rounded-lg">
  <div className="h-24 w-24 bg-primary/20" />
</HeroSkeleton>
```

### Card skeleton

Combine with `HeroCard` for a full card loading state:

```tsx
<HeroCard className="w-[200px] space-y-5 p-4">
  <HeroSkeleton className="rounded-lg">
    <div className="h-24 rounded-lg bg-default-300" />
  </HeroSkeleton>
  <div className="space-y-3">
    <HeroSkeleton className="w-3/5 rounded-lg">
      <div className="h-3 w-3/5 rounded-lg bg-default-200" />
    </HeroSkeleton>
    <HeroSkeleton className="w-4/5 rounded-lg">
      <div className="h-3 w-4/5 rounded-lg bg-default-200" />
    </HeroSkeleton>
    <HeroSkeleton className="w-2/5 rounded-lg">
      <div className="h-3 w-2/5 rounded-lg bg-default-300" />
    </HeroSkeleton>
  </div>
</HeroCard>
```

## Notes

- When `isLoaded` transitions from `false` to `true`, the content fades in smoothly.
- Apply `rounded-*` classes matching your content's border radius for a polished look.
- Requires `@heroui/skeleton` to be installed in the consuming app.
