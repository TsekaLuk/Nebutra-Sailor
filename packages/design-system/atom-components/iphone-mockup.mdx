## Overview

`IphoneMockup` renders a pixel-accurate iPhone 15 Pro SVG frame. The screen content (image or video) is clipped with rounded corners matching the real device. Fully responsive — control size via the container width.

Features:
- Dynamic Island cutout
- Side buttons and top speaker
- Light/dark frame colors via Tailwind classes
- Image or video inside the screen area

## Usage

```tsx
import { IphoneMockup } from "@nebutra/custom-ui/primitives";

// Frame only
<div className="w-48">
  <IphoneMockup />
</div>

// With image
<div className="w-64">
  <IphoneMockup src="/app-screenshot.png" />
</div>

// With looping video
<div className="w-64">
  <IphoneMockup videoSrc="/app-demo.mp4" />
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | Image URL to display in the screen area |
| `videoSrc` | `string` | — | Video URL (autoplay, loop, muted) |
| `className` | `string` | — | Container CSS class |
| `style` | `CSSProperties` | — | Inline styles for the container |

## Notes

- `videoSrc` takes priority over `src` if both are provided
- Video plays with `autoPlay loop muted playsInline` — browser-safe defaults
- Frame colors adapt to dark mode via `dark:fill-[#404040]` / `dark:fill-[#262626]`
- Aspect ratio is fixed at `433/882` (iPhone 15 Pro dimensions)
- Screen region coordinates: `left 4.9%, top 2.2%, width 89.9%, height 95.6%`, radius `55.75px`
