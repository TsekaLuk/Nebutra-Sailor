## Usage

```tsx
import { Loader } from "@nebutra/ui";

<Loader />
<Loader variant="dots" size="lg" />
<Loader variant="loading-dots" text="Processing" />
```

## Variants

### Circular (default)

Simple spinning ring — universal loading indicator.

```tsx
<Loader variant="circular" />
```

### Classic

iOS-style spinner with 12 fading bars radiating from center.

```tsx
<Loader variant="classic" />
```

### Pulse

Pulsing ring animation — subtle and minimal.

```tsx
<Loader variant="pulse" />
```

### Pulse Dot

Single pulsing dot — ideal for inline or compact contexts.

```tsx
<Loader variant="pulse-dot" />
```

### Dots

Three bouncing dots — classic chat and content loading indicator.

```tsx
<Loader variant="dots" />
```

### Typing

Chat typing indicator — three dots with sequential opacity animation.

```tsx
<Loader variant="typing" />
```

### Wave

Five vertical bars with audio-wave animation — ideal for media players.

```tsx
<Loader variant="wave" />
```

### Bars

Three scaling bars — code editor or dashboard context.

```tsx
<Loader variant="bars" />
```

### Terminal

Terminal cursor blink with `>` prompt — for AI and CLI contexts.

```tsx
<Loader variant="terminal" />
```

### Text Blink

Blinking text that alternates between foreground and muted — AI thinking.

```tsx
<Loader variant="text-blink" text="Thinking" />
```

### Text Shimmer

Shimmer gradient sweep on text — premium loading state.

```tsx
<Loader variant="text-shimmer" text="Generating" />
```

### Loading Dots

Animated trailing dots after text — "Thinking..." pattern.

```tsx
<Loader variant="loading-dots" text="Processing" />
```

## Sizes

All variants support `sm`, `md` (default), and `lg`.

```tsx
<Loader size="sm" />
<Loader size="md" />
<Loader size="lg" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"circular" \| "classic" \| "pulse" \| "pulse-dot" \| "dots" \| "typing" \| "wave" \| "bars" \| "terminal" \| "text-blink" \| "text-shimmer" \| "loading-dots"` | `"circular"` | Visual style of the loader |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Size of the loader |
| `text` | `string` | `"Thinking"` | Text label (for text-based variants: `text-blink`, `text-shimmer`, `loading-dots`) |
| `className` | `string` | — | Additional Tailwind classes |

## Notes

- All variants include `<span className="sr-only">Loading</span>` for screen reader accessibility.
- The `text` prop is only relevant for `text-blink`, `text-shimmer`, and `loading-dots` variants; it is ignored by graphic variants.
- Individual variant components are also exported: `CircularLoader`, `ClassicLoader`, `PulseLoader`, `PulseDotLoader`, `DotsLoader`, `TypingLoader`, `WaveLoader`, `BarsLoader`, `TerminalLoader`, `TextBlinkLoader`, `TextShimmerLoader`, `TextDotsLoader`.
