---
title: MagicCard
description: Card with a gradient spotlight and glowing border effect that follows the mouse cursor. Perfect for pricing cards and feature highlights.
---

<Note>
  View interactive examples in [Storybook](http://localhost:6006/?path=/docs/primitives-magiccard--docs).
</Note>

## Usage

```tsx
import { MagicCard } from "@nebutra/ui";

<MagicCard className="rounded-xl p-6">
  <h3 className="text-xl font-bold">Pro Plan</h3>
  <p className="mt-2 text-muted-foreground">$49/month</p>
</MagicCard>
```

## Examples

### Basic Card

```tsx
<MagicCard className="rounded-xl p-8 cursor-pointer">
  <h3 className="text-xl font-semibold">Move your mouse</h3>
  <p className="mt-2 text-sm text-muted-foreground">
    The gradient follows your cursor.
  </p>
</MagicCard>
```

### Custom Gradient Colors

```tsx
<MagicCard
  gradientFrom="#00C6FF"
  gradientTo="#0072FF"
  gradientColor="#0a1a3e"
  className="rounded-xl p-8"
>
  <h3 className="text-xl font-semibold">Blue Theme</h3>
</MagicCard>
```

### Pricing Card

```tsx
<MagicCard className="rounded-2xl p-8 cursor-pointer w-72">
  <div className="mb-4 text-xs font-semibold uppercase tracking-wider text-primary">
    Most Popular
  </div>
  <div className="text-3xl font-bold">$49<span className="text-lg text-muted-foreground">/mo</span></div>
  <div className="mt-6 space-y-3">
    {["Unlimited projects", "API access", "Priority support"].map((f) => (
      <div key={f} className="flex items-center gap-2 text-sm">
        <Check className="h-4 w-4 text-primary" />
        {f}
      </div>
    ))}
  </div>
  <button className="mt-6 w-full rounded-lg bg-primary py-2 text-sm font-medium text-primary-foreground">
    Get started
  </button>
</MagicCard>
```

### Grid of Magic Cards

```tsx
<div className="grid grid-cols-3 gap-4">
  <MagicCard className="rounded-xl p-6 cursor-pointer">...</MagicCard>
  <MagicCard className="rounded-xl p-6 cursor-pointer">...</MagicCard>

```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Card content |
| `gradientSize` | `number` | `200` | Radius of the spotlight gradient in pixels |
| `gradientColor` | `string` | `"#262626"` | Color of the inner spotlight fill |
| `gradientOpacity` | `number` | `0.8` | Opacity of the inner spotlight |
| `gradientFrom` | `string` | `"#9E7AFF"` | Start color of the border gradient |
| `gradientTo` | `string` | `"#FE8BBB"` | End color of the border gradient |
| `className` | `string` | — | Applied to the outer wrapper |

## Notes

- `MagicCard` wraps children in a `relative rounded-[inherit]` container — the border radius should be set via `className`.
- The gradient border is only visible when the cursor is over the card (tracked via `onPointerMove`).
- The spotlight resets to off-screen when the cursor leaves the card, the window loses focus, or the tab becomes hidden.
- For the border effect to look right, ensure the parent element doesn't clip overflow.
