## Usage

```tsx
import { NavMenu, NavMenuItem, NavMenuLink } from "@nebutra/custom-ui/primitives"
```

## Props

### NavMenu

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | `NavMenuItem` elements |

### NavMenuLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | — | Navigation target URL |
| `active` | `boolean` | `false` | Highlights the link as the current page |
| `disabled` | `boolean` | `false` | Prevents navigation |
| `asChild` | `boolean` | `false` | Merge props onto the immediate child (for router links) |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Link content (icon + label) |

## Examples

### With Next.js Link

```tsx
import Link from "next/link"
import { usePathname } from "next/navigation"

function AppNav() {
  const pathname = usePathname()

  return (
    <NavMenu>
      {navItems.map((item) => (
        <NavMenuItem key={item.href}>
          <NavMenuLink asChild active={pathname === item.href}>
            <Link href={item.href}>
              <item.icon className="h-4 w-4" />
              {item.label}
            </Link>
          </NavMenuLink>
        </NavMenuItem>
      ))}
    </NavMenu>
  )
}
```

## Accessibility

- Renders as a `<nav>` element with `role="navigation"`.
- Active link has `aria-current="page"` for screen reader announcement.
- Disabled links have `aria-disabled="true"` and `tabIndex={-1}`.
- Each link is keyboard focusable in natural tab order.
