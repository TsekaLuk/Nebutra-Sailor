## Overview

`NotificationMessageList` renders a list of notification items with staggered scale-up animations. Each item shows a gradient avatar, title with timestamp, and content preview. A fade overlay at the bottom suggests overflow content.

## Installation

```tsx
import { NotificationMessageList } from "@nebutra/custom-ui/primitives";
import type { NotificationMessage } from "@nebutra/custom-ui/primitives";
```

## Usage

### Basic

```tsx
const messages: NotificationMessage[] = [
  {
    title: "New deployment",
    time: "2m ago",
    content: "Production deploy succeeded in 42s",
    gradientColor: "from-green-400 to-emerald-600",
  },
  {
    title: "Security alert",
    time: "15m ago",
    content: "New login from Berlin, Germany",
    gradientColor: "from-red-400 to-pink-600",
  },
  {
    title: "Usage limit",
    time: "1h ago",
    content: "You've used 80% of your monthly API quota",
    gradientColor: "from-amber-400 to-orange-500",
  },
];

<NotificationMessageList messages={messages} />
```

### Custom height and animation delay

```tsx
<NotificationMessageList
  messages={messages}
  maxHeight="400px"
  animationDelay={150}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `messages` | `NotificationMessage[]` | — | **Required.** Array of notification items |
| `maxHeight` | `string` | `"280px"` | CSS height of the container |
| `animationDelay` | `number` | `300` | Delay between each item's animation (ms) |
| `className` | `string` | — | Additional CSS classes |

### NotificationMessage

| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | Message title |
| `time` | `string` | Relative time string (e.g., `"2m ago"`) |
| `content` | `string` | Message body (clamped to 1 line) |
| `gradientColor` | `string` | Tailwind gradient classes for the avatar (e.g., `"from-pink-400 to-indigo-500"`) |

## Notes

- Animation uses a `@keyframes` block injected via a `<style>` tag — no Framer Motion dependency.
- Each item is animated with `animation-delay` proportional to its index × `animationDelay` ms.
- The bottom fade overlay uses `bg-gradient-to-t from-background` and overlaps the last few items.
- The container clips overflow with `overflow-hidden` and fixed height — items beyond `maxHeight` are not visible.
