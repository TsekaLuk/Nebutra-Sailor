title: "Word Fade In"
description: "Staggered word-by-word fade-in animation powered by Framer Motion. Elegant reveal for headlines, quotes, and welcome messages."
---

<Note>
  **Storybook** — Interactive demo: `apps/storybook` → **Primitives / WordFadeIn**
</Note>

## Usage

```tsx
import { WordFadeIn } from "@nebutra/custom-ui/primitives"
```

## Examples

### Basic Hero Headline

```tsx
<WordFadeIn words="Welcome to our platform" />
```

### As Paragraph

```tsx
<WordFadeIn
  as="p"
  words="Each word fades in one by one"
  className="text-lg text-muted-foreground"
/>
```

### Slower Stagger

```tsx
<WordFadeIn words="Take your time" delay={0.3} />
```

### Scroll-Triggered

```tsx
<WordFadeIn
  words="This appears when scrolled into view"
  trigger="inView"
  className="text-3xl font-bold"
/>
```

### Controlled (start paused)

```tsx
<WordFadeIn
  words="Starts when you click"
  animate={isStarted}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `words` | `string` | — | Space-separated text to animate |
| `delay` | `number` | `0.15` | Stagger delay between words in seconds |
| `as` | `"h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6" \| "p" \| "span" \| "div"` | `"h1"` | HTML element to render |
| `animate` | `boolean` | `true` | Toggle animation on/off |
| `trigger` | `"mount" \| "inView"` | `"mount"` | When to start the animation |
| `variants` | `Variants` | staggered fade | Custom Framer Motion variants |
| `className` | `string` | — | Additional CSS classes |

## Notes

- With `trigger="inView"`, animation plays once when the element enters the viewport (`viewport: { once: true }`).
- Default styles: `text-center text-4xl font-bold tracking-tight` — override freely with `className`.
