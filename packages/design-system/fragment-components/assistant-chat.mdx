## Demo

```tsx
<AssistantChat
  assistantName="Nebula"
  assistantAvatar="/avatars/nebula.png"
  messages={[
    {
      id: "1",
      role: "user",
      content: "Summarize the last 7 days of errors.",
      timestamp: "10:01 AM",
    },
    {
      id: "2",
      role: "assistant",
      content: "Over the past 7 days, 94% of errors originated from the payments service...",
      timestamp: "10:01 AM",
    },
  ]}
  onSend={(message) => console.log(message)}
/>
```

## Usage

```tsx
import { AssistantChat } from "@nebutra/custom-ui";

export function SupportPanel() {
  const [messages, setMessages] = useState<ChatMessage[]>([]);

  async function handleSend(content: string) {
    const userMsg = { id: crypto.randomUUID(), role: "user", content, timestamp: new Date().toLocaleTimeString() };
    setMessages((prev) => [...prev, userMsg]);

    const reply = await fetchAIReply(content);
    setMessages((prev) => [...prev, { id: crypto.randomUUID(), role: "assistant", content: reply, timestamp: new Date().toLocaleTimeString() }]);
  }

  return (
    <AssistantChat
      assistantName="Nebula"
      messages={messages}
      onSend={handleSend}
      isLoading={false}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `assistantName` | `string` | `"Assistant"` | Display name shown in the assistant message header |
| `assistantAvatar` | `string` | — | URL for the assistant avatar image |
| `messages` | `AssistantChatMessage[]` | `[]` | Array of messages to display in order |
| `onSend` | `(content: string) => void` | — | Called when the user submits a message |
| `isLoading` | `boolean` | `false` | Shows a typing indicator beneath the last assistant message |
| `inputPlaceholder` | `string` | `"Ask anything..."` | Placeholder text for the message input |
| `className` | `string` | — | Additional classes on the root container |

### `AssistantChatMessage`

```ts
interface AssistantChatMessage {
  id: string;
  role: "user" | "assistant";
  content: string;
  timestamp: string;
}
```

## Examples

### Empty state

When `messages` is empty, the component renders a welcome screen with suggested prompts.

```tsx
<AssistantChat
  assistantName="Nebula"
  messages={[]}
  onSend={handleSend}
/>
```

### With loading indicator

```tsx
<AssistantChat
  assistantName="Nebula"
  messages={messages}
  onSend={handleSend}
  isLoading={true}
/>
```

### Full-page layout

```tsx
<div className="h-screen flex flex-col">
  <AssistantChat
    assistantName="Nebula"
    messages={messages}
    onSend={handleSend}
    className="flex-1"
  />
</div>
```

## Accessibility

- Message list is rendered inside a `role="log"` region with `aria-live="polite"` so screen readers announce new messages.
- The send button has `aria-label="Send message"`.
- The `isLoading` typing indicator has `aria-label="Assistant is typing"` and `aria-busy="true"`.
- User can submit with `Enter`; `Shift+Enter` inserts a line break.
- The input has an associated visible label via `aria-label`.
